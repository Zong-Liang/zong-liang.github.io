## 代码

### 行内代码

你可以使用 `<code>` 标签或者反引号 `` ` `` 来标记行内代码，例如 `variable_name = "value"`。

### 代码块

以下是一个 Python 代码块的示例：

```python
import time

def say_hello(name):
  """这是一个简单的问候函数"""
  print(f"你好, {name}!")
  print(f"当前时间是: {time.strftime('%Y-%m-%d %H:%M:%S')}")

if __name__ == "__main__":
  say_hello("世界")
