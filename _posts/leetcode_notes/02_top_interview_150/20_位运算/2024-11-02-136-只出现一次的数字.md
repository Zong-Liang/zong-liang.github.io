## 题目

**描述：**

给你一个 **非空** 整数数组 `nums`，除了某个元素只出现一次以外，其余每个元素均出现两次。找出那个只出现了一次的元素。

你必须设计并实现线性时间复杂度的算法来解决此问题，且该算法只使用常数量额外空间。

**示例 1:**

- **输入:** `nums = [2,2,1]`
- **输出:** 1

**示例 2:**

- **输入:** `nums = [4,1,2,1,2]`
- **输出:** 4

**示例 3:**

- **输入:** `nums = [1]`
- **输出:** 1

**提示:**

- `1 <= nums.length <= 3 * 10^4`
- `-3 * 10^4 <= nums[i] <= 3 * 10^4`
- 除了某个元素只出现一次以外，其余每个元素均出现两次。

## 题解


题目给了一个很强的结构：除了一个数外，其余都出现两次。最省事的解法就是全数组异或。

原因是异或满足两条关键性质：`a ^ a = 0`，`a ^ 0 = a`。再加上交换律和结合律，所有成对数字都会互相抵消，最后只剩那个只出现一次的数。

实现上从 `result = 0` 开始，逐个做 `result ^= num` 即可，不需要额外容器。

时间复杂度是 $O(n)$，空间复杂度是 $O(1)$。

这题可以当作位运算入门的标志题：当数据里存在“成对抵消”结构时，优先尝试异或。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static int singleNumber(int[] nums) {
        int result = 0;
        for (int num : nums) {
            result ^= num;
        }
        return result;
    }

    public static void main(String[] args) {
        int[] nums1 = {2, 2, 1};
        System.out.println("示例 1 输入: " + Arrays.toString(nums1));
        System.out.println("示例 1 输出: " + singleNumber(nums1));
        System.out.println("-----------------------------");

        int[] nums2 = {4, 1, 2, 1, 2};
        System.out.println("示例 2 输入: " + Arrays.toString(nums2));
        System.out.println("示例 2 输出: " + singleNumber(nums2));
        System.out.println("-----------------------------");

        int[] nums3 = {1};
        System.out.println("示例 3 输入: " + Arrays.toString(nums3));
        System.out.println("示例 3 输出: " + singleNumber(nums3));
        System.out.println("-----------------------------");
    }
}
```
