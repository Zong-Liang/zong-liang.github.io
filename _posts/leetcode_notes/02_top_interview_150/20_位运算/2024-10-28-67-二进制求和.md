## 题目

**描述：**

给你两个二进制字符串 `a` 和 `b`，以二进制字符串的形式返回它们的和。

**示例 1:**

- **输入:** `a = "11"`, `b = "1"`
- **输出:** `"100"`

**示例 2:**

- **输入:** `a = "1010"`, `b = "1011"`
- **输出:** `"10101"`

**提示:**

- `1 <= a.length, b.length <= 10^4`
- `a` 和 `b` 仅由字符 `'0'` 或 `'1'` 组成
- 字符串如果不是 `"0"`，就不含前导零

## 题解


二进制求和和十进制竖式加法是同一件事：从低位往高位走，维护进位。

代码用两个指针从 `a`、`b` 末尾向前扫，把当前位和 `carry` 加到 `sum`，再把 `sum % 2` 作为当前结果位，`sum / 2` 作为新的进位。任一字符串先耗尽都没关系，缺位按 0 处理。

循环条件写成 `i >= 0 || j >= 0 || carry != 0`，可以自然覆盖“最后还剩一位进位”的情况。由于我们是从低位往高位追加，最后反转一次得到答案。

时间复杂度是 $O(\max(m,n))$，空间复杂度也是 $O(\max(m,n))$（结果字符串）。

这题可以当作“多源双指针 + 进位状态”的模板，十六进制、任意进制都能按同样思路扩展。

完整代码如下：

```java
public class Main {

    public static String addBinary(String a, String b) {
        StringBuilder sb = new StringBuilder();
        int i = a.length() - 1;
        int j = b.length() - 1;
        int carry = 0;

        while (i >= 0 || j >= 0 || carry != 0) {
            int sum = carry;

            if (i >= 0) {
                sum += a.charAt(i) - '0';
                i--;
            }

            if (j >= 0) {
                sum += b.charAt(j) - '0';
                j--;
            }

            sb.append(sum % 2);
            carry = sum / 2;
        }

        return sb.reverse().toString();
    }

    public static void main(String[] args) {
        String a1 = "11";
        String b1 = "1";
        System.out.println("示例 1 输入: a = \"" + a1 + "\", b = \"" + b1 + "\"");
        System.out.println("示例 1 输出: \"" + addBinary(a1, b1) + "\"");
        System.out.println("-----------------------------");

        String a2 = "1010";
        String b2 = "1011";
        System.out.println("示例 2 输入: a = \"" + a2 + "\", b = \"" + b2 + "\"");
        System.out.println("示例 2 输出: \"" + addBinary(a2, b2) + "\"");
        System.out.println("-----------------------------");

        String a3 = "111";
        String b3 = "1";
        System.out.println("边界测试 输入: a = \"" + a3 + "\", b = \"" + b3 + "\"");
        System.out.println("边界测试 输出: \"" + addBinary(a3, b3) + "\"");
    }
}
```
