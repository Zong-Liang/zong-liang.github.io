## 题目

**描述：**

将两个升序链表合并为一个新的 **升序** 链表并返回。新链表是通过拼接给定的两个链表的所有节点组成的。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2020/10/03/merge_ex1.jpg)

- **输入:** `l1 = [1,2,4]`, `l2 = [1,3,4]`
- **输出:** `[1,1,2,3,4,4]`

**示例 2:**

- **输入:** `l1 = []`, `l2 = []`
- **输出:** `[]`

**示例 3:**

- **输入:** `l1 = []`, `l2 = [0]`
- **输出:** `[0]`

**提示:**

- 两个链表的节点数目范围是 `[0, 50]`
- `-100 <= Node.val <= 100`
- `l1` 和 `l2` 均按 **非递减顺序** 排列

## 题解

### 先想清楚这题在做什么

这题本质上和“合并两个有序数组”是一个思路，只不过这里的载体从数组换成了链表。

两个链表本身已经是升序的，所以合并时并不需要重新排序。我们真正要做的事只有一件：

**每次从两个链表当前的头节点里，选出较小的那个，接到结果链表后面。**

谁小就先接谁，然后把对应链表的指针往后挪一位，继续比较。  
一直重复这个过程，直到其中一个链表走完为止。

### 为什么这题适合用双指针

因为两个链表都是有序的，所以每一轮比较时：

- `l1` 当前节点是 `l1` 这条链里最小的未处理节点
- `l2` 当前节点是 `l2` 这条链里最小的未处理节点

那整个结果链表里，下一个应该放谁，其实只取决于 `l1.val` 和 `l2.val` 谁更小。

也就是说，我们不需要回头看，也不需要额外存一堆节点，只要让两个指针一起往前走就够了。

这就是这题最核心的判断依据。

### 为什么通常会先准备一个哑节点

如果直接拼接结果链表，很快就会遇到一个麻烦：

**结果链表的头节点到底是谁？**

第一次比较时：

- 可能是 `l1` 的头节点更小
- 也可能是 `l2` 的头节点更小

如果不加哑节点，代码里就得额外处理“头节点还没确定”的情况。  
这种写法不是不能做，只是会更绕。

所以更顺手的方式，是先准备一个哑节点 `dummy`，再用一个指针 `curr` 始终指向结果链表的最后一个节点。这样无论当前接的是谁，都统一写成：

```java
curr.next = ...
curr = curr.next
```

最后返回 `dummy.next` 就可以了。

对于这种“不断往结果链表尾部接节点”的题，哑节点基本就是最省事的写法。

### 整个合并过程怎么走

假设：

```text
l1 = [1,2,4]
l2 = [1,3,4]
```

一开始比较两个头节点：

- `1` 和 `1`

谁先接都可以。这里假设先接 `l1` 的 `1`，结果链表变成：

```text
1
```

然后继续比较：

- `2` 和 `1`

这次 `l2` 的 `1` 更小，把它接上：

```text
1 -> 1
```

再往后比较：

- `2` 和 `3`

接 `2`：

```text
1 -> 1 -> 2
```

继续比较：

- `4` 和 `3`

接 `3`：

```text
1 -> 1 -> 2 -> 3
```

继续比较：

- `4` 和 `4`

任选其一都可以。假设先接 `l1` 的 `4`：

```text
1 -> 1 -> 2 -> 3 -> 4
```

这时 `l1` 已经走完了，`l2` 还剩一个 `4`。  
因为 `l2` 剩下的部分本来就是有序的，而且它们都比前面已经接好的节点大，所以直接把剩余部分整体接到结果链表后面就行。

最后结果就是：

```text
[1,1,2,3,4,4]
```

### 真正需要注意的只有两个点

第一个点是：  
**比较完后，移动的是被选中的那个链表指针。**

如果这轮接上的是 `l1` 当前节点，那就让 `l1 = l1.next`；  
如果接上的是 `l2` 当前节点，那就让 `l2 = l2.next`。

第二个点是：  
**循环结束后，不要忘了把剩余链表直接接上。**

因为退出循环时，一定是 `l1` 或 `l2` 其中一个已经为空了。  
而另一个链表剩下的部分本来就已经有序，不需要再一个个比较，直接整体接过去就行。

这一步很短，但很关键。

### 迭代写法

这题最推荐的还是迭代法，思路直，代码也稳。

```java
public ListNode mergeTwoLists(ListNode l1, ListNode l2) {
    ListNode dummy = new ListNode(-1);
    ListNode curr = dummy;

    while (l1 != null && l2 != null) {
        if (l1.val <= l2.val) {
            curr.next = l1;
            l1 = l1.next;
        } else {
            curr.next = l2;
            l2 = l2.next;
        }
        curr = curr.next;
    }

    curr.next = (l1 != null) ? l1 : l2;

    return dummy.next;
}
```

### 这段代码怎么理解

可以把 `curr` 理解成“结果链表当前的尾巴”。

每轮循环做的事情都一样：

1. 比较 `l1.val` 和 `l2.val`
2. 把较小的那个节点接到 `curr.next`
3. 把被选中的链表指针后移一位
4. 让 `curr` 也往后走一位

循环结束后，再把没走完的那条链直接接到后面。

整个过程没有新建真正的数据节点，只是在重新连接原链表里的节点，所以空间开销很小。

### 递归写法也可以做

这题还有一种很常见的写法是递归。

它的思路其实和迭代完全一样，只不过是把“谁更小，谁当头节点”这个过程交给递归去展开。

如果：

- `l1` 为空，直接返回 `l2`
- `l2` 为空，直接返回 `l1`

否则就比较 `l1.val` 和 `l2.val`：

- 如果 `l1.val` 更小，那么合并后链表的头节点就是 `l1`
- 它后面的部分，等于 `l1.next` 和 `l2` 继续合并的结果

反过来也一样。

代码如下：

```java
public ListNode mergeTwoListsRecursive(ListNode l1, ListNode l2) {
    if (l1 == null) {
        return l2;
    }
    if (l2 == null) {
        return l1;
    }

    if (l1.val <= l2.val) {
        l1.next = mergeTwoListsRecursive(l1.next, l2);
        return l1;
    } else {
        l2.next = mergeTwoListsRecursive(l1, l2.next);
        return l2;
    }
}
```

递归写法更短，也很优雅，但它会额外消耗递归栈空间。  
如果是面试或者平时刷题，通常还是优先把迭代写法掌握扎实。

### 复杂度分析

迭代法的时间复杂度是 $O(m+n)$，其中 `m` 和 `n` 分别是两个链表的长度。因为两个链表里的每个节点最多只会被访问一次。

空间复杂度是 $O(1)$。  
除了几个指针变量，没有额外申请和链表长度相关的空间。

如果用递归写法，时间复杂度同样是 $O(m+n)$，但空间复杂度会变成 $O(m+n)$，因为递归调用会占用栈空间。

### 这题真正值得记住的地方

这题最值得记住的，不只是“两个有序链表怎么合并”，而是这种思路：

**当两个结构本身已经有序时，合并的关键不是重排，而是始终拿当前最小的那个。**

数组里可以这么做，链表里也可以这么做。  
区别只在于：

- 数组可以通过下标访问
- 链表只能靠指针往后走

所以这题其实是在练一种很基础但很重要的能力：  
**把“有序合并”这个思路，从数组迁移到链表。**

### 完整代码

```java
public class Main {

    public static class ListNode {
        int val;
        ListNode next;

        ListNode() {}

        ListNode(int val) {
            this.val = val;
        }

        ListNode(int val, ListNode next) {
            this.val = val;
            this.next = next;
        }
    }

    public static ListNode mergeTwoLists(ListNode l1, ListNode l2) {
        ListNode dummy = new ListNode(-1);
        ListNode curr = dummy;

        while (l1 != null && l2 != null) {
            if (l1.val <= l2.val) {
                curr.next = l1;
                l1 = l1.next;
            } else {
                curr.next = l2;
                l2 = l2.next;
            }
            curr = curr.next;
        }

        curr.next = (l1 != null) ? l1 : l2;

        return dummy.next;
    }

    public static ListNode createList(int[] arr) {
        ListNode dummy = new ListNode(-1);
        ListNode curr = dummy;
        for (int val : arr) {
            curr.next = new ListNode(val);
            curr = curr.next;
        }
        return dummy.next;
    }

    public static void printList(ListNode head) {
        StringBuilder sb = new StringBuilder();
        sb.append("[");
        while (head != null) {
            sb.append(head.val);
            if (head.next != null) {
                sb.append(", ");
            }
            head = head.next;
        }
        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        int[] arr1 = {1, 2, 4};
        int[] arr2 = {1, 3, 4};
        ListNode l1 = createList(arr1);
        ListNode l2 = createList(arr2);

        System.out.println("示例 1 输入:");
        System.out.print("l1 = ");
        printList(l1);
        System.out.print("l2 = ");
        printList(l2);

        ListNode res1 = mergeTwoLists(l1, l2);
        System.out.print("示例 1 输出: ");
        printList(res1);
        System.out.println("-----------------------------");

        int[] arr3 = {};
        int[] arr4 = {};
        ListNode l3 = createList(arr3);
        ListNode l4 = createList(arr4);

        System.out.println("示例 2 输入:");
        System.out.print("l1 = ");
        printList(l3);
        System.out.print("l2 = ");
        printList(l4);

        ListNode res2 = mergeTwoLists(l3, l4);
        System.out.print("示例 2 输出: ");
        printList(res2);
        System.out.println("-----------------------------");

        int[] arr5 = {};
        int[] arr6 = {0};
        ListNode l5 = createList(arr5);
        ListNode l6 = createList(arr6);

        System.out.println("示例 3 输入:");
        System.out.print("l1 = ");
        printList(l5);
        System.out.print("l2 = ");
        printList(l6);

        ListNode res3 = mergeTwoLists(l5, l6);
        System.out.print("示例 3 输出: ");
        printList(res3);
    }
}
```
