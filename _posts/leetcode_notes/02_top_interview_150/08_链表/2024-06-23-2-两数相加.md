### 题目

**描述：**

给你两个 **非空** 的链表，表示两个非负的整数。它们每位数字都是按照 **逆序** 的方式存储的，并且每个节点只能存储 **一位** 数字。

请你将两个数相加，并以相同形式返回一个表示和的链表。

你可以假设除了数字 0 之外，这两个数都不会以 0 开头。

**示例 1:**

![示例1图片](https://assets.leetcode-cn.com/aliyun-lc-upload/uploads/2021/01/02/addtwonumber1.jpg)

- **输入:** `l1 = [2,4,3]`, `l2 = [5,6,4]`
- **输出:** `[7,0,8]`
- **解释:** `342 + 465 = 807`.

**示例 2:**

- **输入:** `l1 = [0]`, `l2 = [0]`
- **输出:** `[0]`

**示例 3:**

- **输入:** `l1 = [9,9,9,9,9,9,9]`, `l2 = [9,9,9,9]`
- **输出:** `[8,9,9,9,0,0,0,1]`

**提示:**

- 每个链表中的节点数在范围 `[1, 100]` 内
- `0 <= Node.val <= 9`
- 题目数据保证列表表示的数字不含前导零

## 题解

### 先看清这题为什么适合直接模拟

这题乍一看是在做链表题，但真正的核心其实不是链表操作本身，而是**按位相加**。

题目已经把两个数字按逆序存进了链表里，也就是说：

- 头节点是个位
- 后面的节点依次是十位、百位、千位……

这和我们平时手算加法的顺序是完全一致的。手算加法本来就是从个位开始，一位一位往后算，同时把进位带到下一位。

所以这题最自然的做法，不是什么复杂技巧，而是老老实实把笔算加法在链表上模拟出来。

### 这题本质上就是“逐位相加 + 处理进位”

假设当前来到两个链表的某一位：

- 当前位的值分别是 `x` 和 `y`
- 上一位计算后可能还带着一个进位 `carry`

那么当前这一位的和就是：

```text
sum = x + y + carry
```

接下来要做两件事：

- 当前位应该放进结果链表的值是 `sum % 10`
- 新的进位是 `sum / 10`

这就是整道题最核心的逻辑。  
后面的代码，无非是在不断重复这个过程。

### 为什么要用哑节点

这题最后要返回一个新的链表，所以我们需要一边计算，一边把结果节点接起来。

这里比较常见的写法，是先创建一个哑节点 `dummy`，然后再用一个指针 `curr` 指向结果链表的末尾。

这样做的好处很直接：不需要单独讨论“结果链表的第一个节点怎么处理”。不管当前是不是第一次添加节点，统一写成：

```java
curr.next = new ListNode(...);
curr = curr.next;
```

最后返回 `dummy.next` 就可以了。

对于这种“动态构造新链表”的题，哑节点基本是最顺手的写法。

### 遍历时要注意什么

这题遍历时，不能只看 `l1` 和 `l2` 有没有走完，还得把“最后一次进位”考虑进去。

比如：

- `l1 = [5]`
- `l2 = [5]`

第一轮算出来是 `0`，但进位是 `1`。  
如果这时候因为两个链表都走完了就直接结束，结果就会少掉最后这一位。

所以循环条件应该写成：

```java
while (l1 != null || l2 != null || carry != 0)
```

只要还有节点没处理，或者还有进位没放进去，循环就不能停。

这也是这题最容易漏掉的地方。

### 链表长度不同也不用额外处理

这题还有一个细节很适合提前想清楚：两个链表不一定一样长。

例如：

- `l1 = [9,9,9,9,9,9,9]`
- `l2 = [9,9,9,9]`

走到后面时，`l2` 会先为空，但加法并不会因此结束，因为 `l1` 可能还有剩余位，甚至还可能继续产生进位。

处理这个问题的方法也很简单：  
如果某个链表已经走到头了，就把它当前位的值当成 `0`。

也就是说：

```java
int x = (l1 != null) ? l1.val : 0;
int y = (l2 != null) ? l2.val : 0;
```

这样整个过程就统一了，不需要专门分情况讨论“谁更长”。

### 用示例走一遍

以示例 1 为例：

```text
l1 = [2,4,3]
l2 = [5,6,4]
```

因为链表是逆序存储，所以它们表示的数字分别是：

- `342`
- `465`

现在开始逐位相加。

第一轮：

- `2 + 5 + 0 = 7`
- 当前位写入 `7`
- 进位是 `0`

结果链表现在是：

```text
[7]
```

第二轮：

- `4 + 6 + 0 = 10`
- 当前位写入 `0`
- 进位变成 `1`

结果链表变成：

```text
[7,0]
```

第三轮：

- `3 + 4 + 1 = 8`
- 当前位写入 `8`
- 进位变成 `0`

结果链表就是：

```text
[7,0,8]
```

这正好对应 `807`。

整个过程和我们平时手算加法其实没有区别，只不过这里是把每一位的结果挂到链表后面。

### 代码

```java
public ListNode addTwoNumbers(ListNode l1, ListNode l2) {
    ListNode dummyHead = new ListNode(0);
    ListNode curr = dummyHead;
    int carry = 0;

    while (l1 != null || l2 != null || carry != 0) {
        int x = (l1 != null) ? l1.val : 0;
        int y = (l2 != null) ? l2.val : 0;

        int sum = x + y + carry;
        carry = sum / 10;

        curr.next = new ListNode(sum % 10);
        curr = curr.next;

        if (l1 != null) {
            l1 = l1.next;
        }
        if (l2 != null) {
            l2 = l2.next;
        }
    }

    return dummyHead.next;
}
```

### 这段代码怎么理解

代码里真正重要的只有三件事：

第一，取当前位的值。  
如果某个链表已经走完了，就按 `0` 处理。

第二，算当前位和新的进位。

```java
int sum = x + y + carry;
carry = sum / 10;
```

第三，把当前位结果接到答案链表后面。

```java
curr.next = new ListNode(sum % 10);
curr = curr.next;
```

然后把 `l1`、`l2` 往后移动，继续处理下一位。

整道题说到底，就是把这个过程重复到不能再重复为止。

### 复杂度分析

时间复杂度是 $O(\max(m, n))$，其中 `m` 和 `n` 分别是两个链表的长度。因为我们只需要从头到尾遍历一次，循环次数最多和较长的链表一样多，再加上最后可能多处理一次进位。

空间复杂度是 $O(1)$。  
这里不把返回结果链表本身算进额外空间，所以真正额外用到的只有几个指针和一个进位变量。

### 这题真正值得记住的地方

这题最值得记住的，不是“链表相加”这个表面形式，而是这个判断：

**只要题目给出的存储顺序，刚好和运算顺序一致，就可以直接模拟。**

这道题里，数字是逆序存储的，刚好对应“从个位往高位算”；  
所以最自然的办法就是一边遍历链表，一边模拟笔算加法。

以后如果再遇到类似“按位处理”“带进位”“结果逐步构造”的题，也可以优先往这个方向想。

### 完整代码

```java
public class Main {

    public static class ListNode {
        int val;
        ListNode next;

        ListNode() {}

        ListNode(int val) {
            this.val = val;
        }

        ListNode(int val, ListNode next) {
            this.val = val;
            this.next = next;
        }
    }

    public static ListNode addTwoNumbers(ListNode l1, ListNode l2) {
        ListNode dummyHead = new ListNode(0);
        ListNode curr = dummyHead;
        int carry = 0;

        while (l1 != null || l2 != null || carry != 0) {
            int val1 = (l1 != null) ? l1.val : 0;
            int val2 = (l2 != null) ? l2.val : 0;

            int sum = val1 + val2 + carry;
            carry = sum / 10;

            curr.next = new ListNode(sum % 10);
            curr = curr.next;

            if (l1 != null) {
                l1 = l1.next;
            }
            if (l2 != null) {
                l2 = l2.next;
            }
        }

        return dummyHead.next;
    }

    public static ListNode createList(int[] arr) {
        ListNode dummy = new ListNode(0);
        ListNode curr = dummy;
        for (int val : arr) {
            curr.next = new ListNode(val);
            curr = curr.next;
        }
        return dummy.next;
    }

    public static void printList(ListNode head) {
        StringBuilder sb = new StringBuilder();
        sb.append("[");
        while (head != null) {
            sb.append(head.val);
            if (head.next != null) {
                sb.append(", ");
            }
            head = head.next;
        }
        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        int[] arr1 = {2, 4, 3};
        int[] arr2 = {5, 6, 4};
        ListNode l1 = createList(arr1);
        ListNode l2 = createList(arr2);

        System.out.println("示例 1 输入:");
        System.out.print("l1 = ");
        printList(l1);
        System.out.print("l2 = ");
        printList(l2);

        ListNode res1 = addTwoNumbers(l1, l2);
        System.out.print("示例 1 输出: ");
        printList(res1);
        System.out.println("-----------------------------");

        int[] arr3 = {0};
        int[] arr4 = {0};
        ListNode l3 = createList(arr3);
        ListNode l4 = createList(arr4);

        System.out.println("示例 2 输入:");
        System.out.print("l1 = ");
        printList(l3);
        System.out.print("l2 = ");
        printList(l4);

        ListNode res2 = addTwoNumbers(l3, l4);
        System.out.print("示例 2 输出: ");
        printList(res2);
        System.out.println("-----------------------------");

        int[] arr5 = {9, 9, 9, 9, 9, 9, 9};
        int[] arr6 = {9, 9, 9, 9};
        ListNode l5 = createList(arr5);
        ListNode l6 = createList(arr6);

        System.out.println("示例 3 输入:");
        System.out.print("l1 = ");
        printList(l5);
        System.out.print("l2 = ");
        printList(l6);

        ListNode res3 = addTwoNumbers(l5, l6);
        System.out.print("示例 3 输出: ");
        printList(res3);
    }
}
```
