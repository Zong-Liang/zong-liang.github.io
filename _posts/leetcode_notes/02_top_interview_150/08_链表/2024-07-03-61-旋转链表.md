## 题目

**描述：**

给你一个链表的头节点 `head`，旋转链表，将链表每个节点向右移动 `k` 个位置。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2020/11/13/rotate1.jpg)

- **输入:** `head = [1,2,3,4,5]`, `k = 2`
- **输出:** `[4,5,1,2,3]`

**示例 2:**

![示例2图片](https://assets.leetcode.com/uploads/2020/11/13/roate2.jpg)

- **输入:** `head = [0,1,2]`, `k = 4`
- **输出:** `[2,0,1]`

**提示:**

- 链表中节点的数目在范围 `[0, 500]` 内
- `-100 <= Node.val <= 100`
- `0 <= k <= 2 * 10^9`

## 题解

### 先想清楚“向右旋转”到底发生了什么

这题表面上是在做链表旋转，但如果只盯着“每个节点向右移动 `k` 个位置”这句话，写起来会比较别扭。

更容易理解的角度是：

**向右旋转 `k` 次，本质上是把链表最后面的 `k` 个节点整体搬到最前面。**

比如：

```text
1 -> 2 -> 3 -> 4 -> 5
k = 2
```

向右旋转之后会变成：

```text
4 -> 5 -> 1 -> 2 -> 3
```

也就是说，新的头节点其实是原链表的倒数第 `k` 个节点，  
新的尾节点则是它前面的那个节点。

这么看之后，这题就不再是“所有节点都怎么移动”，而变成了：

**找到新的头和新的尾，然后把链表重新接一下。**

### 先注意一个很容易忽略的问题：`k` 可能特别大

题目里 `k` 的范围很大，甚至可能远大于链表长度。

比如链表长度是 5：

- 向右旋转 1 次
- 向右旋转 6 次
- 向右旋转 11 次

结果其实都一样。

因为每转满一圈，链表都会回到原状。  
所以真正有意义的，不是 `k` 本身，而是：

```text
k % n
```

其中 `n` 是链表长度。

这一步一定要先想到，不然会把问题想复杂。

### 这题最顺手的做法：先连成环，再断开

这题有一个很经典、也很自然的思路，就是先把链表首尾相连，变成一个环。

为什么这么做会简单？

因为原来你想把“后面一段接到前面”，需要考虑很多断开和重连；  
但如果先把它变成一个环，那么所谓“旋转”其实就只剩下一件事：

**在正确的位置把这个环断开。**

也就是说：

- 原链表首尾相连之后，所有节点已经天然首尾相接了
- 我们只需要找到新的尾节点
- 然后把它的 `next` 断掉
- 它后面的那个节点自然就成了新的头节点

这就是这题最核心的技巧。

### 为什么这个思路成立

还是看这个例子：

```text
1 -> 2 -> 3 -> 4 -> 5
k = 2
```

如果先把尾节点 `5` 连回头节点 `1`，链表就变成了一个环：

```text
1 -> 2 -> 3 -> 4 -> 5
^                   |
|___________________|
```

现在题目要求向右旋转 2 次，结果应该是：

```text
4 -> 5 -> 1 -> 2 -> 3
```

那么新的头节点就是 `4`，  
新的尾节点就是 `3`。

所以在这个环里，我们只要找到 `3`，把它和 `4` 之间断开，就得到答案了。

这样一来，整题的关键就变成了：

**如何找到新的尾节点。**

### 新尾节点应该在什么位置

假设链表长度是 `n`，有效旋转次数是：

```text
k % n
```

那么向右旋转 `k` 次之后：

- 新头节点是原链表的第 `n - k % n + 1` 个节点
- 新尾节点就是原链表的第 `n - k % n` 个节点

换句话说，新尾节点正好在“从头开始走 `n - k % n - 1` 步”的位置。

不过实际写代码时，有一种更顺手的写法：  
我们先走到原来的尾节点，把链表连成环，然后从这个旧尾节点继续向前走 `n - k % n` 步，就会停在新的尾节点上。

这个写法比重新从头再走一遍更自然一些。

### 用示例走一遍

以示例 1 为例：

```text
head = [1,2,3,4,5], k = 2
```

先遍历链表，得到：

- 长度 `n = 5`
- 尾节点是 `5`

然后计算有效旋转次数：

```text
k % n = 2
```

所以新的尾节点应该在：

```text
n - k % n = 3
```

也就是说，从旧尾节点 `5` 出发，再往前走 3 步：

- 第 1 步到 `1`
- 第 2 步到 `2`
- 第 3 步到 `3`

停下时正好来到新的尾节点 `3`。

这时候：

- `3.next` 就是新的头节点 `4`
- 只要把 `3.next` 断开

链表就变成：

```text
4 -> 5 -> 1 -> 2 -> 3
```

正好是答案。

### 为什么有时候根本不用旋转

还有一种情况要提前处理：

如果：

```text
k % n == 0
```

说明转完以后链表和原来一模一样。  
比如长度是 3，右移 3 次、6 次、9 次，本质上都等于没动。

这种情况下直接返回原链表就行，不需要真的连成环再断开。

这一点虽然简单，但最好尽早判断掉，代码会更干净。

### 代码

```java
public ListNode rotateRight(ListNode head, int k) {
    if (head == null || head.next == null || k == 0) {
        return head;
    }

    ListNode iter = head;
    int n = 1;
    while (iter.next != null) {
        iter = iter.next;
        n++;
    }

    int steps = n - k % n;
    if (steps == n) {
        return head;
    }

    iter.next = head;

    while (steps-- > 0) {
        iter = iter.next;
    }

    ListNode newHead = iter.next;
    iter.next = null;

    return newHead;
}
```

### 这段代码怎么理解

这段代码其实就是按前面的思路一步一步翻译过来的。

第一步，先处理边界情况：

```java
if (head == null || head.next == null || k == 0) {
    return head;
}
```

如果链表为空、只有一个节点，或者 `k = 0`，都不用转。

第二步，统计链表长度，同时找到尾节点：

```java
ListNode iter = head;
int n = 1;
while (iter.next != null) {
    iter = iter.next;
    n++;
}
```

循环结束后：

- `n` 是链表长度
- `iter` 停在旧尾节点

第三步，计算真正需要走的步数：

```java
int steps = n - k % n;
```

这一步的意思其实就是：  
为了找到新的尾节点，我们要从旧尾节点开始，再往前走 `n - k % n` 步。

第四步，把链表连成环：

```java
iter.next = head;
```

第五步，走到新的尾节点：

```java
while (steps-- > 0) {
    iter = iter.next;
}
```

第六步，断开环：

```java
ListNode newHead = iter.next;
iter.next = null;
```

这里：

- `iter` 停在新的尾节点
- `iter.next` 就是新的头节点

把它断开之后，结果就出来了。

### 复杂度分析

时间复杂度是 $O(n)$。  
我们最多遍历链表两次左右：一次求长度，一次定位新的断点，总体仍然是线性时间。

空间复杂度是 $O(1)$。  
只用了几个指针变量，没有额外开辟和链表长度相关的空间。

### 这题真正值得记住的地方

这题真正值得记住的，不只是“旋转链表”这个结论，而是这种转换思路：

**有些链表重排问题，直接改来改去会很绕，但如果先把链表变成环，再在合适的位置断开，问题会一下子简单很多。**

这题就是很典型的例子。  
表面上是在做“向右移动”，实际上是在做：

- 找尾节点
- 连成环
- 找新尾节点
- 断开

一旦把问题转成这个角度，代码就会非常顺。

### 完整代码

```java
public class Main {

    public static class ListNode {
        int val;
        ListNode next;

        ListNode() {}

        ListNode(int val) {
            this.val = val;
        }

        ListNode(int val, ListNode next) {
            this.val = val;
            this.next = next;
        }
    }

    public static ListNode rotateRight(ListNode head, int k) {
        if (head == null || head.next == null || k == 0) {
            return head;
        }

        ListNode iter = head;
        int n = 1;
        while (iter.next != null) {
            iter = iter.next;
            n++;
        }

        int steps = n - k % n;
        if (steps == n) {
            return head;
        }

        iter.next = head;

        while (steps-- > 0) {
            iter = iter.next;
        }

        ListNode newHead = iter.next;
        iter.next = null;

        return newHead;
    }

    public static ListNode createList(int[] arr) {
        if (arr.length == 0) {
            return null;
        }

        ListNode head = new ListNode(arr[0]);
        ListNode curr = head;
        for (int i = 1; i < arr.length; i++) {
            curr.next = new ListNode(arr[i]);
            curr = curr.next;
        }
        return head;
    }

    public static void printList(ListNode head) {
        StringBuilder sb = new StringBuilder();
        sb.append("[");
        ListNode cur = head;
        while (cur != null) {
            sb.append(cur.val);
            if (cur.next != null) {
                sb.append(", ");
            }
            cur = cur.next;
        }
        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        int[] arr1 = {1, 2, 3, 4, 5};
        int k1 = 2;
        ListNode head1 = createList(arr1);

        System.out.println("示例 1 输入:");
        System.out.print("head = ");
        printList(head1);
        System.out.println("k = " + k1);

        ListNode res1 = rotateRight(head1, k1);

        System.out.print("示例 1 输出: ");
        printList(res1);
        System.out.println("-----------------------------");

        int[] arr2 = {0, 1, 2};
        int k2 = 4;
        ListNode head2 = createList(arr2);

        System.out.println("示例 2 输入:");
        System.out.print("head = ");
        printList(head2);
        System.out.println("k = " + k2);

        ListNode res2 = rotateRight(head2, k2);

        System.out.print("示例 2 输出: ");
        printList(res2);
        System.out.println("-----------------------------");

        int[] arr3 = {1, 2};
        int k3 = 2;
        ListNode head3 = createList(arr3);

        System.out.println("示例 3 输入:");
        System.out.print("head = ");
        printList(head3);
        System.out.println("k = " + k3);

        ListNode res3 = rotateRight(head3, k3);

        System.out.print("示例 3 输出: ");
        printList(res3);
    }
}
```
