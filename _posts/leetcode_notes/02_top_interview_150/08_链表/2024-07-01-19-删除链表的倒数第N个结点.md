## 题目

**描述：**

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

- **输入:** `head = [1,2,3,4,5]`, `n = 2`
- **输出:** `[1,2,3,5]`

**示例 2:**

- **输入:** `head = [1]`, `n = 1`
- **输出:** `[]`

**示例 3:**

- **输入:** `head = [1,2]`, `n = 1`
- **输出:** `[1]`

**提示:**

- 链表中结点的数目为 `sz`
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

**进阶：** 你能尝试使用一趟扫描实现吗？

## 题解

### 先想清楚这题到底在删谁

题目说的是“删除链表的倒数第 `n` 个结点”。

这个说法看起来绕，其实可以先换个角度理解：

如果链表总长度是 `L`，那么倒数第 `n` 个节点，其实就是正数第 `L - n + 1` 个节点。

所以最直接的办法当然是：

- 先遍历一遍，算出链表长度
- 再找到正数第 `L - n + 1` 个节点
- 把它删掉

这个思路没有问题，也很好想，但题目进阶要求是一趟扫描，所以真正更值得掌握的，是双指针写法。

### 先说最直观的做法：先算长度

如果不考虑“一趟扫描”，这题其实很简单。

先把链表长度算出来，假设长度是 `L`，那么要删除的节点前一个位置，就是正数第 `L - n` 个节点。只要找到这个前驱节点，再执行：

```java
pre.next = pre.next.next;
```

删除就完成了。

代码如下：

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0, head);
    int length = 0;
    ListNode cur = head;

    while (cur != null) {
        length++;
        cur = cur.next;
    }

    ListNode pre = dummy;
    for (int i = 0; i < length - n; i++) {
        pre = pre.next;
    }

    pre.next = pre.next.next;

    return dummy.next;
}
```

这个方法的好处是稳定、直接，不容易绕。  
时间复杂度是 $O(L)$，空间复杂度是 $O(1)$。

不过它需要走两遍链表，所以还不是这题最想考的地方。

### 这题真正关键的地方，是怎么一趟找到“前驱节点”

删除链表节点，真正要改的其实不是这个节点本身，而是它前一个节点的 `next`。

所以这题的核心不是“怎么找到倒数第 `n` 个节点”，而是：

**怎么在一趟扫描里，找到倒数第 `n` 个节点的前一个节点。**

这时候双指针就很自然了。

### 双指针为什么能做到一趟扫描

我们让两个指针都从 `dummy` 出发：

- `fast` 先往前走 `n` 步
- `slow` 先不动

这样一来，`fast` 和 `slow` 之间就始终隔着 `n` 个节点。

接下来让它们一起往前走：

- `fast` 走一步
- `slow` 也走一步

直到 `fast` 来到链表最后一个节点。

这时候，由于两者之间一直保持 `n` 个节点的间距，所以 `slow` 正好会停在“倒数第 `n` 个节点的前一个位置”。

这就是这题最核心的技巧。

### 为什么一定要找前一个节点

因为单链表删除一个节点，通常并不是直接把它“删掉”，而是让它前一个节点绕过它：

```java
slow.next = slow.next.next;
```

所以如果你最后停在的是待删除节点本身，反而不太好操作。  
而双指针这种写法，天然就能让 `slow` 停在它前面，正好适合做删除。

这也是这道题里最值得记住的点。

### 用示例走一遍就很清楚了

以示例 1 为例：

```text
head = [1,2,3,4,5], n = 2
```

我们想删除的是倒数第 2 个节点，也就是值为 `4` 的那个节点。

先加一个 `dummy`，链表就可以理解成：

```text
dummy -> 1 -> 2 -> 3 -> 4 -> 5
```

现在让 `fast` 先走 2 步。

此时：

- `slow` 还在 `dummy`
- `fast` 来到节点 `2`

接着两个指针一起走。

第一轮后：

- `slow` 在 `1`
- `fast` 在 `3`

第二轮后：

- `slow` 在 `2`
- `fast` 在 `4`

第三轮后：

- `slow` 在 `3`
- `fast` 在 `5`

这时 `fast.next == null`，说明 `fast` 已经到末尾了，停止。

现在 `slow` 正好停在 `3`，也就是待删除节点 `4` 的前一个位置。  
接下来执行：

```java
slow.next = slow.next.next;
```

链表就变成：

```text
1 -> 2 -> 3 -> 5
```

正好是答案。

### 为什么要用 dummy 节点

这题里 `dummy` 很重要，不是可有可无。

因为有一种情况特别麻烦：要删除的正好是头节点。

比如：

```text
head = [1], n = 1
```

如果没有 `dummy`，那头节点前面就没有前驱节点，代码会变得别扭。  
加上 `dummy` 后，哪怕删除的是原来的头节点，`slow` 也总能停在一个合法的前驱位置上，逻辑就统一了。

所以这题里 `dummy` 的作用，其实就是：

**把“删除头节点”也变成普通删除。**

### 双指针代码

这题更推荐掌握下面这个版本：

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0, head);
    ListNode slow = dummy;
    ListNode fast = dummy;

    for (int i = 0; i < n; i++) {
        fast = fast.next;
    }

    while (fast.next != null) {
        fast = fast.next;
        slow = slow.next;
    }

    slow.next = slow.next.next;

    return dummy.next;
}
```

### 这段代码怎么理解

这段代码真正重要的，其实就三步。

第一步，让 `fast` 先走 `n` 步。  
这样就把两者之间的距离固定住了。

第二步，让 `fast` 和 `slow` 同时往前走。  
因为间距始终不变，所以等 `fast` 到链表末尾时，`slow` 就刚好到待删除节点的前一个位置。

第三步，执行删除：

```java
slow.next = slow.next.next;
```

这一步本身没什么难点，真正难的是前面怎么把 `slow` 定位到正确位置。

所以整道题看起来在考删除，实际上更像是在考：

**如何用双指针精确定位一个节点的前驱。**

### 复杂度分析

时间复杂度是 $O(L)$，其中 `L` 是链表长度。  
虽然写法上分成“快指针先走”和“双指针一起走”两段，但整体仍然只是在链表上做一次线性扫描。

空间复杂度是 $O(1)$，只用了几个指针变量。

### 这题真正值得记住的地方

这题真正值得记住的，不只是“删除倒数第 `n` 个节点”这个题型，而是这个双指针思路：

**当题目要求在单链表里定位“倒数第几”时，通常可以考虑让两个指针先拉开固定距离，再一起移动。**

这样做的好处是，不需要真的把“倒数”转换成“正数第几个”再回头找，而是直接在一次遍历里把目标位置逼出来。

这类思路在链表题里非常常见，后面遇到类似题时也能直接迁移。

### 完整代码

```java
import java.util.*;

// 链表节点定义
class ListNode {
    int val;
    ListNode next;

    ListNode() {}

    ListNode(int val) {
        this.val = val;
    }

    ListNode(int val, ListNode next) {
        this.val = val;
        this.next = next;
    }
}

public class Main {

    public static ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode dummy = new ListNode(0, head);
        ListNode slow = dummy;
        ListNode fast = dummy;

        for (int i = 0; i < n; i++) {
            fast = fast.next;
        }

        while (fast.next != null) {
            fast = fast.next;
            slow = slow.next;
        }

        slow.next = slow.next.next;

        return dummy.next;
    }

    public static void printList(ListNode head) {
        System.out.print("[");
        ListNode curr = head;
        while (curr != null) {
            System.out.print(curr.val);
            if (curr.next != null) {
                System.out.print(", ");
            }
            curr = curr.next;
        }
        System.out.println("]");
    }

    public static ListNode buildList(int[] vals) {
        if (vals.length == 0) {
            return null;
        }

        ListNode dummy = new ListNode(0);
        ListNode curr = dummy;
        for (int val : vals) {
            curr.next = new ListNode(val);
            curr = curr.next;
        }
        return dummy.next;
    }

    public static void main(String[] args) {
        int[] vals1 = {1, 2, 3, 4, 5};
        int n1 = 2;
        ListNode head1 = buildList(vals1);

        System.out.println("示例 1 输入: head = " + Arrays.toString(vals1) + ", n = " + n1);
        ListNode res1 = removeNthFromEnd(head1, n1);
        System.out.print("示例 1 输出: ");
        printList(res1);
        System.out.println("-----------------------------");

        int[] vals2 = {1};
        int n2 = 1;
        ListNode head2 = buildList(vals2);

        System.out.println("示例 2 输入: head = " + Arrays.toString(vals2) + ", n = " + n2);
        ListNode res2 = removeNthFromEnd(head2, n2);
        System.out.print("示例 2 输出: ");
        printList(res2);
        System.out.println("-----------------------------");

        int[] vals3 = {1, 2};
        int n3 = 1;
        ListNode head3 = buildList(vals3);

        System.out.println("示例 3 输入: head = " + Arrays.toString(vals3) + ", n = " + n3);
        ListNode res3 = removeNthFromEnd(head3, n3);
        System.out.print("示例 3 输出: ");
        printList(res3);
    }
}
```
