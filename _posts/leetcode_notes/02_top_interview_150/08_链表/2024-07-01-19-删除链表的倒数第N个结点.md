## 题目

**描述：**

给你一个链表，删除链表的倒数第 `n` 个结点，并且返回链表的头结点。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2020/10/03/remove_ex1.jpg)

- **输入:** `head = [1,2,3,4,5]`, `n = 2`
- **输出:** `[1,2,3,5]`

**示例 2:**

- **输入:** `head = [1]`, `n = 1`
- **输出:** `[]`

**示例 3:**

- **输入:** `head = [1,2]`, `n = 1`
- **输出:** `[1]`

**提示:**

- 链表中结点的数目为 `sz`
- `1 <= sz <= 30`
- `0 <= Node.val <= 100`
- `1 <= n <= sz`

**进阶：** 你能尝试使用一趟扫描实现吗？

## 题解

倒数第 `n` 个节点在单链表里不好直接定位，因为我们只能从前往后走。把它转成“固定间距”的问题就简单了：让快指针先走 `n` 步，再让快慢指针一起走，直到快指针到尾部，此时慢指针正好停在待删除节点的前一个位置。

这题强烈建议加哑节点。因为当删除目标是原头节点时，如果没有哑节点就要专门分支；有了哑节点，删除动作始终统一为 `slow.next = slow.next.next`，不用区分是否删头。

实现细节上，先让 `fast` 从 `dummy` 出发走 `n` 步，再同步推进两指针直到 `fast.next == null`。这个停止条件对应的正是“`slow` 指向待删节点前驱”的时刻。

代码如下：

```java
public ListNode removeNthFromEnd(ListNode head, int n) {
    ListNode dummy = new ListNode(0, head);
    ListNode fast = dummy;
    ListNode slow = dummy;

    for (int i = 0; i < n; i++) {
        fast = fast.next;
    }

    while (fast.next != null) {
        fast = fast.next;
        slow = slow.next;
    }

    slow.next = slow.next.next;
    return dummy.next;
}
```

时间复杂度是 $O(L)$，`L` 为链表长度；空间复杂度是 $O(1)$。

“倒数第几”这类描述，常见改写就是双指针固定间距。把相对位置关系提前建立起来，后面只要同步移动就能到目标。

完整代码如下：

```java
import java.util.*;
// 链表节点定义
class ListNode {
    int val;
    ListNode next;
    ListNode() {}
    ListNode(int val) {
        this.val = val;
    }
    ListNode(int val, ListNode next) {
        this.val = val;
        this.next = next;
    }
}
public class Main {
    public static ListNode removeNthFromEnd(ListNode head, int n) {
        ListNode dummy = new ListNode(0, head);
        ListNode slow = dummy;
        ListNode fast = dummy;
        for (int i = 0; i < n; i++) {
            fast = fast.next;
        }
        while (fast.next != null) {
            fast = fast.next;
            slow = slow.next;
        }
        slow.next = slow.next.next;
        return dummy.next;
    }
    public static void printList(ListNode head) {
        System.out.print("[");
        ListNode curr = head;
        while (curr != null) {
            System.out.print(curr.val);
            if (curr.next != null) {
                System.out.print(", ");
            }
            curr = curr.next;
        }
        System.out.println("]");
    }
    public static ListNode buildList(int[] vals) {
        if (vals.length == 0) {
            return null;
        }
        ListNode dummy = new ListNode(0);
        ListNode curr = dummy;
        for (int val : vals) {
            curr.next = new ListNode(val);
            curr = curr.next;
        }
        return dummy.next;
    }
    public static void main(String[] args) {
        int[] vals1 = {1, 2, 3, 4, 5};
        int n1 = 2;
        ListNode head1 = buildList(vals1);
        System.out.println("示例 1 输入: head = " + Arrays.toString(vals1) + ", n = " + n1);
        ListNode res1 = removeNthFromEnd(head1, n1);
        System.out.print("示例 1 输出: ");
        printList(res1);
        System.out.println("-----------------------------");
        int[] vals2 = {1};
        int n2 = 1;
        ListNode head2 = buildList(vals2);
        System.out.println("示例 2 输入: head = " + Arrays.toString(vals2) + ", n = " + n2);
        ListNode res2 = removeNthFromEnd(head2, n2);
        System.out.print("示例 2 输出: ");
        printList(res2);
        System.out.println("-----------------------------");
        int[] vals3 = {1, 2};
        int n3 = 1;
        ListNode head3 = buildList(vals3);
        System.out.println("示例 3 输入: head = " + Arrays.toString(vals3) + ", n = " + n3);
        ListNode res3 = removeNthFromEnd(head3, n3);
        System.out.print("示例 3 输出: ");
        printList(res3);
    }
}
```
