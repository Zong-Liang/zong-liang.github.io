## 题目

**描述：**

给定一个已排序的链表的头 `head`，删除原始链表中所有重复数字的节点，只留下不同的数字。返回 **已排序的链表**。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/01/04/linkedlist1.jpg)

- **输入:** `head = [1,2,3,3,4,4,5]`
- **输出:** `[1,2,5]`

**示例 2:**

![示例2图片](https://assets.leetcode.com/uploads/2021/01/04/linkedlist2.jpg)

- **输入:** `head = [1,1,1,2,3]`
- **输出:** `[2,3]`

**提示:**

- 链表中节点数目在范围 `[0, 300]` 内
- `-100 <= Node.val <= 100`
- 题目数据保证链表已经按 **升序** 排列

## 题解

### 先看清这题删的不是“多余的副本”

这题最容易想错的地方是：很多人会把它和“删除排序链表中的重复元素”那道题混在一起。

那道题的意思是：

- 重复的值可以保留一个
- 比如 `1 -> 1 -> 2`，最后变成 `1 -> 2`

但这道题不是这样。  
这道题要求的是：

**只要某个值出现了不止一次，那么这个值对应的所有节点都要删掉，一个都不能留。**

比如：

```text
1 -> 2 -> 3 -> 3 -> 4 -> 4 -> 5
```

最后应该变成：

```text
1 -> 2 -> 5
```

因为 `3` 和 `4` 都重复了，所以它们整段都不能保留。

所以这题真正要做的不是“去重保留一个”，而是：

**把所有重复值整组删除，只留下只出现一次的节点。**

### 有序这个条件为什么特别重要

题目已经说了链表是升序排列的，这一点非常关键。

因为链表有序，所以相同的值一定会挨在一起。  
也就是说，如果某个值重复了，它一定会形成一整段连续区间。

例如：

```text
1 -> 2 -> 2 -> 2 -> 3 -> 4 -> 4 -> 5
```

这里重复值不会散着出现，而是会集中成块。

这就意味着，遍历时只要发现当前节点和下一个节点值相同，就可以立刻判断：

**当前这一整段值相同的节点，最后都不能要。**

这也是这题能在线性时间里解决的根本原因。

### 为什么一定要用 dummy 节点

这题里 `dummy` 非常重要。

因为头节点本身也可能属于重复值的一部分。  
比如：

```text
1 -> 1 -> 1 -> 2 -> 3
```

最后结果应该是：

```text
2 -> 3
```

也就是说，原来的头节点要被删掉，而且不只是删一个，是整段都删掉。

如果没有 `dummy`，处理这种“头节点整段被删除”的情况就会很别扭。  
加上 `dummy` 之后，逻辑会统一很多：

- `cur` 始终指向“当前确认保留下来的最后一个节点”
- 更准确一点说，是指向“当前正在检查位置的前一个节点”

这样无论删除的是中间一段，还是开头一段，操作方式都一样。

### 这题真正的核心思路

我们用一个指针 `cur` 从 `dummy` 开始往后走。

每次重点看的是 `cur` 后面的两个节点：

- `cur.next`
- `cur.next.next`

为什么要看这两个？

因为只靠一个节点，无法判断它是不是重复；  
但如果相邻两个节点值相同，就说明重复序列出现了。

接下来分两种情况：

#### 情况一：`cur.next.val != cur.next.next.val`

说明当前这个节点没有和后一个节点重复。  
在有序链表里，这就意味着 `cur.next` 目前是安全的，可以保留。

这时候直接让：

```java
cur = cur.next;
```

继续往后看就行。

#### 情况二：`cur.next.val == cur.next.next.val`

说明发现了一段重复值。

这时候不能只删一个，因为题目要求这个值一个都不能留。  
所以要先把这个重复值记下来，然后不断跳过所有值等于它的节点。

比如重复值是 `3`，那就一直做：

```java
cur.next = cur.next.next;
```

直到 `cur.next` 不再是 `3` 为止。

这就是整道题最核心的逻辑。

### 用示例走一遍

以示例 1 为例：

```text
1 -> 2 -> 3 -> 3 -> 4 -> 4 -> 5
```

先加一个 `dummy`，可以理解成：

```text
dummy -> 1 -> 2 -> 3 -> 3 -> 4 -> 4 -> 5
```

开始时，`cur` 指向 `dummy`。

先看 `1` 和 `2`：

- 不相等
- 说明 `1` 可以保留

于是 `cur` 往后走到 `1`

再看 `2` 和 `3`：

- 不相等
- 说明 `2` 可以保留

于是 `cur` 往后走到 `2`

再看 `3` 和 `3`：

- 相等
- 说明发现了重复值 `3`

这时候不能让 `cur` 往后走，而是要一直删掉所有值为 `3` 的节点。删完之后链表变成：

```text
dummy -> 1 -> 2 -> 4 -> 4 -> 5
```

此时 `cur` 仍然停在 `2`，因为新的 `cur.next` 还要继续检查。

接着看 `4` 和 `4`：

- 相等
- 再删掉所有值为 `4` 的节点

删完之后链表变成：

```text
dummy -> 1 -> 2 -> 5
```

最后看不到重复了，结束。

答案就是：

```text
[1,2,5]
```

### 为什么发现重复后 `cur` 不能立刻后移

这是这题里一个非常关键的细节。

当发现重复值并开始删除时，`cur` 不要动。  
原因是删完这一段之后，新的 `cur.next` 可能还是另一组重复值的开头。

比如刚才的例子里：

```text
1 -> 2 -> 3 -> 3 -> 4 -> 4 -> 5
```

删掉 `3` 那一段后，`cur` 还是停在 `2`，而新的 `cur.next` 已经变成了 `4`。  
这时候如果你提前把 `cur` 往后挪了，就会错过后面对 `4` 这一段重复值的判断。

所以正确做法是：

- 只有在确认 `cur.next` 这个节点可以保留时，才移动 `cur`
- 一旦进入“删除重复段”的流程，`cur` 就保持不动

这个点一旦想清楚，代码就不会写乱。

### 代码

```java
public ListNode deleteDuplicates(ListNode head) {
    if (head == null) {
        return null;
    }

    ListNode dummy = new ListNode(0, head);
    ListNode cur = dummy;

    while (cur.next != null && cur.next.next != null) {
        if (cur.next.val == cur.next.next.val) {
            int x = cur.next.val;
            while (cur.next != null && cur.next.val == x) {
                cur.next = cur.next.next;
            }
        } else {
            cur = cur.next;
        }
    }

    return dummy.next;
}
```

### 这段代码怎么理解

代码里真正关键的只有两层判断。

外层循环：

```java
while (cur.next != null && cur.next.next != null)
```

意思是：只要 `cur` 后面至少还有两个节点，就还有判断“是否重复”的必要。

内层判断：

```java
if (cur.next.val == cur.next.next.val)
```

意思是：如果后面两个节点值相等，说明遇到重复段了。

这时候先记住重复值：

```java
int x = cur.next.val;
```

然后一直删：

```java
while (cur.next != null && cur.next.val == x) {
    cur.next = cur.next.next;
}
```

如果后面两个值不相等，那就说明 `cur.next` 可以保留，于是 `cur` 正常后移一位。

整道题说到底，就是在不断做这两种分支判断。

### 复杂度分析

时间复杂度是 $O(n)$。  
虽然看起来有一层 `while` 套一层 `while`，但每个节点最多被访问和跳过常数次，所以总体仍然是线性的。

空间复杂度是 $O(1)$。  
除了几个指针变量，没有额外使用和链表长度相关的空间。

### 这题真正值得记住的地方

这题真正值得记住的，不只是“删除排序链表中的重复元素 II”这个题型，而是这种判断方式：

**当链表有序时，重复值一定成段出现；一旦题目要求“重复的一个都不留”，那就不是保留一个节点，而是整段跳过。**

很多链表题的难点不在操作本身，而在“遇到一段相同元素时，到底该留一个，还是整段删除”。  
这道题练的就是这个判断。

### 完整代码

```java
public class Main {

    public static class ListNode {
        int val;
        ListNode next;

        ListNode() {}

        ListNode(int val) {
            this.val = val;
        }

        ListNode(int val, ListNode next) {
            this.val = val;
            this.next = next;
        }
    }

    public static ListNode deleteDuplicates(ListNode head) {
        if (head == null) {
            return null;
        }

        ListNode dummy = new ListNode(0, head);
        ListNode cur = dummy;

        while (cur.next != null && cur.next.next != null) {
            if (cur.next.val == cur.next.next.val) {
                int x = cur.next.val;
                while (cur.next != null && cur.next.val == x) {
                    cur.next = cur.next.next;
                }
            } else {
                cur = cur.next;
            }
        }

        return dummy.next;
    }

    public static ListNode createList(int[] arr) {
        ListNode dummy = new ListNode(-1);
        ListNode cur = dummy;
        for (int val : arr) {
            cur.next = new ListNode(val);
            cur = cur.next;
        }
        return dummy.next;
    }

    public static void printList(ListNode head) {
        StringBuilder sb = new StringBuilder();
        sb.append("[");
        ListNode cur = head;
        while (cur != null) {
            sb.append(cur.val);
            if (cur.next != null) {
                sb.append(", ");
            }
            cur = cur.next;
        }
        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        int[] arr1 = {1, 2, 3, 3, 4, 4, 5};
        ListNode head1 = createList(arr1);
        System.out.println("示例 1 输入:");
        printList(head1);

        ListNode res1 = deleteDuplicates(head1);

        System.out.println("示例 1 输出:");
        printList(res1);
        System.out.println("-----------------------------");

        int[] arr2 = {1, 1, 1, 2, 3};
        ListNode head2 = createList(arr2);
        System.out.println("示例 2 输入:");
        printList(head2);

        ListNode res2 = deleteDuplicates(head2);

        System.out.println("示例 2 输出:");
        printList(res2);
        System.out.println("-----------------------------");

        int[] arr3 = {1, 1};
        ListNode head3 = createList(arr3);
        System.out.println("示例 3 输入:");
        printList(head3);

        ListNode res3 = deleteDuplicates(head3);

        System.out.println("示例 3 输出:");
        printList(res3);
    }
}
```
