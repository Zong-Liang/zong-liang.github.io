## 题目

**描述：**

给你一个链表的头节点 `head` 和一个特定值 `x`，请你对链表进行分隔，使得所有 **小于** `x` 的节点都出现在 **大于或等于** `x` 的节点之前。

你应当 **保留** 两个分区中每个节点的初始相对位置。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/01/04/partition.jpg)

- **输入:** `head = [1,4,3,2,5,2]`, `x = 3`
- **输出:** `[1,2,2,4,3,5]`

**示例 2:**

- **输入:** `head = [2,1]`, `x = 2`
- **输出:** `[1,2]`

**提示:**

- 链表中节点的数目在范围 `[0, 200]` 内
- `-100 <= Node.val <= 100`
- `-200 <= x <= 200`

## 题解

### 先看这题真正要求的是什么

这题不是简单地把“小于 `x` 的节点放前面，大于等于 `x` 的放后面”就结束了。

它还有一个很关键的要求：

**每个分区内部，节点原本的相对顺序不能变。**

这句话非常重要。

比如：

```text
1 -> 4 -> 3 -> 2 -> 5 -> 2
x = 3
```

最后结果应该是：

```text
1 -> 2 -> 2 -> 4 -> 3 -> 5
```

注意：

- 小于 `3` 的节点是 `1、2、2`，它们在结果里仍然保持原来的先后顺序
- 大于等于 `3` 的节点是 `4、3、5`，它们在结果里也保持原来的先后顺序

所以这题真正要做的，不是排序，也不是交换位置，而是：

**把原链表按条件拆成两段，再按顺序接起来。**

### 为什么不能想着原地随便交换

如果只看到“把小的放前面，大的放后面”，很多人第一反应可能会往“原地交换节点”上想。

但这题的难点恰恰在于：  
它不是只关心分区结果，还关心稳定性，也就是相对顺序不能乱。

而一旦开始原地交换，就很容易把顺序打乱。  
所以这题更自然的思路，不是边遍历边交换，而是：

- 准备一条“小于 `x`”的链表
- 再准备一条“大于等于 `x`”的链表
- 遍历原链表时，把节点按顺序尾插到对应链表里
- 最后把两条链表拼起来

这个思路既直观，又天然能保证顺序不乱。

### 这题本质上是在做“稳定分组”

你可以把这题理解成链表版的稳定分组。

所谓稳定，意思就是：

- 属于同一组的元素，前后顺序不能改变

所以当我们遍历原链表时，每遇到一个节点，其实只需要做一个判断：

- 如果 `head.val < x`，就把它接到“小链表”的尾部
- 否则，就把它接到“大链表”的尾部

注意这里必须是**尾插**，不能头插。

因为如果头插，顺序就会反过来，题目要求就不满足了。  
而尾插正好能保证：节点出现的顺序，就是它们在各自分区里的顺序。

### 为什么要用两个哑节点

这题里最顺手的写法，就是给两条链表都准备一个哑节点：

- `smallHead`：小于 `x` 的那条链表的伪头
- `largeHead`：大于等于 `x` 的那条链表的伪头

然后分别再用两个尾指针：

- `small`
- `large`

这样做的好处很直接：  
不用每次判断“这条链表是不是还没有头节点”，直接统一做尾插就行。

如果没有哑节点，代码里会多出很多“第一次插入怎么办”的分支判断。  
而加了哑节点之后，逻辑就很干净：

- 谁属于哪一组，就接到哪一组的尾巴后面
- 最后再把两组拼起来

这类“动态构造链表”的题，哑节点基本都是最稳的写法。

### 用示例走一遍

以示例 1 为例：

```text
head = [1,4,3,2,5,2], x = 3
```

我们一边遍历，一边把节点分到两条链表里。

先看 `1`：

- `1 < 3`
- 接到 small 链表后面

此时：

```text
small: 1
large: 空
```

再看 `4`：

- `4 >= 3`
- 接到 large 链表后面

此时：

```text
small: 1
large: 4
```

再看 `3`：

- `3 >= 3`
- 接到 large 后面

此时：

```text
small: 1
large: 4 -> 3
```

再看 `2`：

- `2 < 3`
- 接到 small 后面

此时：

```text
small: 1 -> 2
large: 4 -> 3
```

再看 `5`：

```text
small: 1 -> 2
large: 4 -> 3 -> 5
```

最后看 `2`：

```text
small: 1 -> 2 -> 2
large: 4 -> 3 -> 5
```

遍历结束后，只要把 small 链表的尾巴接到 large 链表的头上，结果就是：

```text
1 -> 2 -> 2 -> 4 -> 3 -> 5
```

这正好符合题目要求。

### 这题最容易忽略的一步：记得断尾

这题里有一个非常容易漏掉的细节：

**最后一定要让 `large.next = null`。**

为什么？

因为我们复用的是原链表里的节点，而这些节点原本都带着旧的 `next` 指针。  
如果不把 `large` 这条链表的尾节点断开，它可能还会保留对原链表后续节点的连接，最终把新链表搞成环，或者把已经分好的结构又串乱。

所以正确顺序是：

1. 遍历时分别尾插到两条链表
2. 遍历结束后先把 `large` 尾巴断开
3. 再把 `small` 和 `large` 接起来

这一步虽然只是一行代码，但非常关键。

### 代码

```java
public ListNode partition(ListNode head, int x) {
    ListNode smallHead = new ListNode(0);
    ListNode small = smallHead;
    ListNode largeHead = new ListNode(0);
    ListNode large = largeHead;

    while (head != null) {
        if (head.val < x) {
            small.next = head;
            small = small.next;
        } else {
            large.next = head;
            large = large.next;
        }
        head = head.next;
    }

    large.next = null;
    small.next = largeHead.next;

    return smallHead.next;
}
```

### 这段代码怎么理解

代码本身不复杂，核心就是两条链表同时维护。

每轮循环里只做一件事：  
判断当前节点属于哪一边，然后尾插到对应链表。

如果当前节点值小于 `x`：

```java
small.next = head;
small = small.next;
```

如果当前节点值大于等于 `x`：

```java
large.next = head;
large = large.next;
```

然后继续遍历下一个节点：

```java
head = head.next;
```

遍历结束后：

```java
large.next = null;
small.next = largeHead.next;
```

第一句是断尾，第二句是拼接。

整个过程其实就是在做一次分类收集，最后再合并。

### 为什么这种写法能保证相对顺序不变

因为我们对每个节点的处理方式始终是**按原链表顺序遍历，按遇到的顺序尾插到目标链表**。

假设原链表里，小于 `x` 的节点依次出现为：

```text
a -> b -> c
```

那么它们会按这个顺序依次被接到 `small` 链表尾部，最后得到的仍然是：

```text
a -> b -> c
```

大于等于 `x` 的部分也是一样。

也就是说，这种做法天然就是稳定的，不需要再额外维护什么顺序信息。

### 复杂度分析

时间复杂度是 $O(n)$，其中 `n` 是链表长度。  
因为我们只遍历了一遍链表。

空间复杂度是 $O(1)$。  
这里只用了几个额外指针，没有新建与输入规模相关的额外节点空间。

### 这题真正值得记住的地方

这题真正值得记住的，不只是“分隔链表”这个题型，而是这种处理方式：

**当题目要求按条件把链表拆成两部分，并且还要保留相对顺序时，最稳的办法通常不是原地交换，而是分别收集，再统一拼接。**

这类题的关键不在于指针有多花，而在于你能不能先把问题转换成“两条稳定链表的构造”。  
一旦想到这一层，代码就会非常顺。

### 完整代码

```java
public class Main {

    public static class ListNode {
        int val;
        ListNode next;

        ListNode() {}

        ListNode(int val) {
            this.val = val;
        }

        ListNode(int val, ListNode next) {
            this.val = val;
            this.next = next;
        }
    }

    public static ListNode partition(ListNode head, int x) {
        ListNode smallHead = new ListNode(0);
        ListNode small = smallHead;

        ListNode largeHead = new ListNode(0);
        ListNode large = largeHead;

        while (head != null) {
            if (head.val < x) {
                small.next = head;
                small = small.next;
            } else {
                large.next = head;
                large = large.next;
            }
            head = head.next;
        }

        large.next = null;
        small.next = largeHead.next;

        return smallHead.next;
    }

    public static ListNode createList(int[] arr) {
        if (arr.length == 0) {
            return null;
        }

        ListNode head = new ListNode(arr[0]);
        ListNode curr = head;
        for (int i = 1; i < arr.length; i++) {
            curr.next = new ListNode(arr[i]);
            curr = curr.next;
        }
        return head;
    }

    public static void printList(ListNode head) {
        StringBuilder sb = new StringBuilder();
        sb.append("[");
        ListNode curr = head;
        while (curr != null) {
            sb.append(curr.val);
            if (curr.next != null) {
                sb.append(", ");
            }
            curr = curr.next;
        }
        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        int[] arr1 = {1, 4, 3, 2, 5, 2};
        int x1 = 3;
        ListNode head1 = createList(arr1);

        System.out.println("示例 1 输入:");
        System.out.print("head = ");
        printList(head1);
        System.out.println("x = " + x1);

        ListNode res1 = partition(head1, x1);

        System.out.print("示例 1 输出: ");
        printList(res1);
        System.out.println("-----------------------------");

        int[] arr2 = {2, 1};
        int x2 = 2;
        ListNode head2 = createList(arr2);

        System.out.println("示例 2 输入:");
        System.out.print("head = ");
        printList(head2);
        System.out.println("x = " + x2);

        ListNode res2 = partition(head2, x2);

        System.out.print("示例 2 输出: ");
        printList(res2);
    }
}
```
