## 题目

**描述：**

给定一个单词数组 `words` 和一个长度 `maxWidth` ，重新排版单词，使其成为每行恰好有 `maxWidth` 个字符，且左右两端对齐的文本。

你应该使用“贪心算法”来放置给定的单词；也就是说，尽可能多地往每行中放置单词。必要时可用空格 `' '` 填充，使得每行恰好有 `maxWidth` 个字符。

要求尽可能均匀分配单词间的空格数量。如果某一行单词间的空格不能均匀分配，则左侧放置的空格数要多于右侧的空格数。

文本的最后一行应为左对齐，且单词之间不插入额外的空格。

**注意：**

- 单词是指由非空格字符组成的字符序列。
- 每个单词的长度大于 0，小于等于 `maxWidth`。
- 输入单词数组 `words` 至少包含一个单词。

**示例 1:**

- **输入:** `words = ["This", "is", "an", "example", "of", "text", "justification."], maxWidth = 16`
- **输出:**
  ```
  [
     "This    is    an",
     "example  of text",
     "justification.  "
  ]
  ```

**示例 2:**

- **输入:** `words = ["What","must","be","acknowledgment","shall","be"], maxWidth = 16`
- **输出:**
  ```
  [
    "What   must   be",
    "acknowledgment  ",
    "shall be        "
  ]
  ```
- **解释:** 注意最后一行和只包含一个单词的行的格式。

**示例 3:**

- **输入:** `words = ["Science","is","what","we","understand","well","enough","to","explain","to","a","computer.","Art","is","everything","else","we","do"], maxWidth = 20`
- **输出:**
  ```
  [
    "Science  is  what we",
    "understand      well",
    "enough to explain to",
    "a  computer.  Art is",
    "everything  else  we",
    "do                  "
  ]
  ```

**提示：**

- `1 <= words.length <= 300`
- `1 <= words[i].length <= 20`
- `words[i]` 由小写英文字母和符号组成
- `1 <= maxWidth <= 100`
- `words[i].length <= maxWidth`

## 题解

### 这题难的不是拼字符串，而是规则很多，而且要同时满足

这题如果只要求“每行放单词”，并不难。  
真正麻烦的是它有好几条规则一起生效：

- 每一行都要尽可能多放单词
- 普通行要左右对齐
- 空格要尽量均匀分
- 如果分不匀，左边的空隙要比右边多
- 最后一行不能左右对齐，而是左对齐
- 如果一行只有一个单词，也按左对齐处理

所以这题不能只盯着“怎么补空格”，而是要先把整个过程拆开。

更自然的做法其实只有两步：

**先贪心确定这一行放哪些单词，再根据这一行的类型决定空格怎么补。**

### 第一步其实很明确：一行里能塞多少就塞多少

题目已经明说了要用贪心。  
所以每一行都不用纠结，直接从当前位置开始往后试：

- 当前行先放第一个单词
- 然后继续尝试下一个单词
- 只要“已有内容长度 + 至少一个分隔空格 + 新单词长度”不超过 `maxWidth`，就继续放
- 一旦超了，就停下来

这样就能得到当前行的单词范围。

这里有个容易忽略的小点：  
在判断能不能放下一个单词时，单词之间至少要留一个空格。也就是说，哪怕最后这一行将来可能不是这么排，判断“能不能塞进这一行”时，也要先按最基本的一空格分隔去算长度。

### 真正的分叉发生在第二步：这行到底属于哪一种

当一行里的单词已经确定后，接下来只需要分两种情况。

第一种是：

- 这是最后一行
- 或者这一行只有一个单词

这两种都按左对齐处理。  
也就是：

- 单词之间只放一个空格
- 剩下的位置全部补到行尾

第二种是普通行。  
普通行才需要做“左右对齐”和“空格均匀分配”。

### 普通行的关键，其实就是先算出一共有多少空格可分

假设当前这一行已经选中了若干个单词。

先把这行所有单词本身的长度加起来，记作：

```text
wordsTotalLen
```

那么这一行总共需要补的空格数就是：

```text
totalSpaces = maxWidth - wordsTotalLen
```

再看这行有多少个空隙。  
如果这行有 `numWords` 个单词，那么空隙数就是：

```text
gaps = numWords - 1
```

接下来问题就很简单了：  
把 `totalSpaces` 分给 `gaps` 个空隙。

- 每个空隙至少分到 `avgSpaces = totalSpaces / gaps`
- 多出来的那部分是 `extraSpaces = totalSpaces % gaps`

题目要求左边空隙更宽，所以前 `extraSpaces` 个空隙，每个再多放一个空格。

这就是整题最核心的计算。

### 用示例 1 走一遍，会更容易看清楚

看这一行：

```text
"This", "is", "an"
```

`maxWidth = 16`

这三个单词长度分别是：

- `This` -> 4
- `is` -> 2
- `an` -> 2

单词总长度是：

```text
4 + 2 + 2 = 8
```

所以这一行总共需要补：

```text
16 - 8 = 8
```

空隙一共有两个，所以：

```text
avgSpaces = 8 / 2 = 4
extraSpaces = 8 % 2 = 0
```

说明两个空隙都放 4 个空格，结果就是：

```text
"This    is    an"
```

再看下一行：

```text
"example", "of", "text"
```

单词总长度：

```text
7 + 2 + 4 = 13
```

需要补空格：

```text
16 - 13 = 3
```

空隙还是两个，所以：

```text
avgSpaces = 3 / 2 = 1
extraSpaces = 3 % 2 = 1
```

也就是说：

- 左边第一个空隙放 `1 + 1 = 2` 个空格
- 右边第二个空隙放 `1` 个空格

结果就是：

```text
"example  of text"
```

这正好对应题目给出的答案。

### 为什么最后一行不能套这套公式

因为最后一行的规则不一样。

题目明确要求最后一行左对齐，所以最后一行根本不用做“平均分空格”这件事。  
它只需要：

- 单词之间放一个空格
- 后面剩多少补多少

比如：

```text
"justification."
```

长度是 14，`maxWidth = 16`。  
那就直接在后面补两个空格：

```text
"justification.  "
```

如果最后一行有多个单词，也是一样的逻辑。  
先正常用一个空格连接，再把剩余空格补到行尾。

### 单个单词单独成行时，也按左对齐处理

这个分支也很重要。

如果某一行只有一个单词，比如：

```text
"acknowledgment"
```

那它根本没有“单词之间的空隙”可以分配。  
这时候自然也不能做均匀分配，只能：

- 先把这个单词放进去
- 然后把剩余空间全部补到右边

所以“最后一行”和“单个单词成行”其实都归到同一类处理里。

### 代码

```java
public List<String> fullJustify(String[] words, int maxWidth) {
    List<String> res = new ArrayList<>();
    int n = words.length;
    int i = 0;

    while (i < n) {
        int left = i;
        int lineLen = 0;

        while (i < n) {
            int nextLen = lineLen + words[i].length() + (lineLen > 0 ? 1 : 0);
            if (nextLen > maxWidth) {
                break;
            }
            lineLen = nextLen;
            i++;
        }

        int right = i;
        int numWords = right - left;
        int gaps = numWords - 1;

        StringBuilder sb = new StringBuilder();

        if (right == n || numWords == 1) {
            for (int j = left; j < right; j++) {
                sb.append(words[j]);
                if (j < right - 1) {
                    sb.append(" ");
                }
            }
            while (sb.length() < maxWidth) {
                sb.append(" ");
            }
        } else {
            int wordsTotalLen = 0;
            for (int j = left; j < right; j++) {
                wordsTotalLen += words[j].length();
            }

            int totalSpaces = maxWidth - wordsTotalLen;
            int avgSpaces = totalSpaces / gaps;
            int extraSpaces = totalSpaces % gaps;

            for (int j = left; j < right - 1; j++) {
                sb.append(words[j]);

                for (int k = 0; k < avgSpaces; k++) {
                    sb.append(" ");
                }
                if (j - left < extraSpaces) {
                    sb.append(" ");
                }
            }

            sb.append(words[right - 1]);
        }

        res.add(sb.toString());
    }

    return res;
}
```

时间复杂度是 $O(N)$，这里的 `N` 可以理解为所有单词字符总数加上结果构造的总量。  
空间复杂度主要来自返回结果本身。

### 这题真正值得记住的地方

这题最值得记住的，不是某个空格公式，而是这种拆题方式：

**先用贪心确定一行装什么，再在这一行内部做格式化。**

也就是说，整题其实分成两个层次：

- 行与行之间，解决“哪些单词属于同一行”
- 行内部，解决“这一行的空格怎么分”

一旦这两层分开想，题目就会清楚很多。  
很多看起来麻烦的字符串模拟题，真正难的都不是实现，而是有没有把规则拆到合适的层次。

### 完整代码

```java
import java.util.ArrayList;
import java.util.List;

public class Main {

    public static List<String> fullJustify(String[] words, int maxWidth) {
        List<String> res = new ArrayList<>();
        int n = words.length;
        int i = 0;

        while (i < n) {
            int left = i;
            int lineLen = 0;

            while (i < n) {
                int nextLen = lineLen + words[i].length() + (lineLen > 0 ? 1 : 0);
                if (nextLen > maxWidth) {
                    break;
                }
                lineLen = nextLen;
                i++;
            }

            int right = i;
            int numWords = right - left;
            int gaps = numWords - 1;

            StringBuilder sb = new StringBuilder();

            if (right == n || numWords == 1) {
                for (int j = left; j < right; j++) {
                    sb.append(words[j]);
                    if (j < right - 1) {
                        sb.append(" ");
                    }
                }
                while (sb.length() < maxWidth) {
                    sb.append(" ");
                }
            } else {
                int wordsTotalLen = 0;
                for (int j = left; j < right; j++) {
                    wordsTotalLen += words[j].length();
                }

                int totalSpaces = maxWidth - wordsTotalLen;
                int avgSpaces = totalSpaces / gaps;
                int extraSpaces = totalSpaces % gaps;

                for (int j = left; j < right - 1; j++) {
                    sb.append(words[j]);

                    for (int k = 0; k < avgSpaces; k++) {
                        sb.append(" ");
                    }
                    if (j - left < extraSpaces) {
                        sb.append(" ");
                    }
                }

                sb.append(words[right - 1]);
            }

            res.add(sb.toString());
        }

        return res;
    }

    public static void main(String[] args) {
        String[] words1 = {"This", "is", "an", "example", "of", "text", "justification."};
        int maxWidth1 = 16;

        System.out.println("示例 1 输出:");
        printResult(fullJustify(words1, maxWidth1));
        System.out.println("-----------------------------");

        String[] words2 = {"What", "must", "be", "acknowledgment", "shall", "be"};
        int maxWidth2 = 16;

        System.out.println("示例 2 输出:");
        printResult(fullJustify(words2, maxWidth2));
        System.out.println("-----------------------------");

        String[] words3 = {"Science", "is", "what", "we", "understand", "well", "enough", "to", "explain", "to", "a", "computer.", "Art", "is", "everything", "else", "we", "do"};
        int maxWidth3 = 20;

        System.out.println("示例 3 输出:");
        printResult(fullJustify(words3, maxWidth3));
    }

    private static void printResult(List<String> result) {
        for (String line : result) {
            System.out.println("\"" + line + "\"");
        }
    }
}
```
