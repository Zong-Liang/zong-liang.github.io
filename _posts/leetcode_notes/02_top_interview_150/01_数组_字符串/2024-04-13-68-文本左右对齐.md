## 题目

**描述：**

给定一个单词数组 `words` 和一个长度 `maxWidth` ，重新排版单词，使其成为每行恰好有 `maxWidth` 个字符，且左右两端对齐的文本。

你应该使用“贪心算法”来放置给定的单词；也就是说，尽可能多地往每行中放置单词。必要时可用空格 `' '` 填充，使得每行恰好有 `maxWidth` 个字符。

要求尽可能均匀分配单词间的空格数量。如果某一行单词间的空格不能均匀分配，则左侧放置的空格数要多于右侧的空格数。

文本的最后一行应为左对齐，且单词之间不插入额外的空格。

**注意：**

- 单词是指由非空格字符组成的字符序列。
- 每个单词的长度大于 0，小于等于 `maxWidth`。
- 输入单词数组 `words` 至少包含一个单词。

**示例 1:**

- **输入:** `words = ["This", "is", "an", "example", "of", "text", "justification."], maxWidth = 16`
- **输出:**
  ```
  [
     "This    is    an",
     "example  of text",
     "justification.  "
  ]
  ```

**示例 2:**

- **输入:** `words = ["What","must","be","acknowledgment","shall","be"], maxWidth = 16`
- **输出:**
  ```
  [
    "What   must   be",
    "acknowledgment  ",
    "shall be        "
  ]
  ```
- **解释:** 注意最后一行和只包含一个单词的行的格式。

**示例 3:**

- **输入:** `words = ["Science","is","what","we","understand","well","enough","to","explain","to","a","computer.","Art","is","everything","else","we","do"], maxWidth = 20`
- **输出:**
  ```
  [
    "Science  is  what we",
    "understand      well",
    "enough to explain to",
    "a  computer.  Art is",
    "everything  else  we",
    "do                  "
  ]
  ```

**提示：**

- `1 <= words.length <= 300`
- `1 <= words[i].length <= 20`
- `words[i]` 由小写英文字母和符号组成
- `1 <= maxWidth <= 100`
- `words[i].length <= maxWidth`

## 题解

这题看着像是在补空格，最容易绕乱的地方其实是规则一下子全挤在一起：一行里要尽量多放单词，普通行要左右对齐，空格要尽量平均，不够平均时左边多一点，最后一行和单词数为 `1` 的行又都要左对齐。要是上来就盯着“这里该补几个空格”，很容易把所有分支写成一团。

更顺的拆法其实只有两层。第一层先解决“这一行装哪些单词”，第二层再解决“这一行怎么排版”。前一层完全可以贪心：从当前单词开始，尽量往这一行里塞，只要“已有内容长度 + 至少一个分隔空格 + 下一个单词长度”不超过 `maxWidth`，就继续放。这样一趟下来，这一行包含哪些单词就定了。

单词范围定下来以后，事情就清楚很多了。如果这是最后一行，或者这一行只有一个单词，那就按左对齐处理：单词之间只放一个空格，剩下的空位全补到行尾。普通行则单独算空格。先把这一行所有单词本身的长度加起来，得到 `wordsTotalLen`，那么这一行总共需要补的空格数就是 `totalSpaces = maxWidth - wordsTotalLen`；如果这一行有 `gaps` 个空隙，就先让每个空隙分到 `totalSpaces / gaps` 个空格，多出来的 `totalSpaces % gaps` 个空格再从左到右一个个补给前面的空隙。

拿题目里的 `"example", "of", "text"` 这一行看会很直观。三个单词长度一共是 `7 + 2 + 4 = 13`，`maxWidth = 16`，所以总共还要补 `3` 个空格。这一行有两个空隙，平均下来每个至少 `1` 个，还多出 `1` 个，所以左边空隙放 `2` 个空格，右边放 `1` 个，最后得到的就是 `"example  of text"`。题目要求“左边的空隙更多”，其实正是通过这一步实现的。

这样做的时候，最后一行和单个单词成行都不用套普通行那套公式，因为它们根本不存在“把空格平均分到多个空隙里”这件事。把这两类单独归成左对齐，代码会干净很多，也不容易在 `gaps = 0` 这种地方出问题。

所以整题最重要的不是某个空格公式本身，而是先把这道模拟题拆成“选词”和“排版”两个层次。前一层只管一行里能放多少，后一层才去关心这一行该怎么补空格。层次分开以后，规则虽然多，但每一段都还是很直的。

代码如下：

```java
public List<String> fullJustify(String[] words, int maxWidth) {
    List<String> res = new ArrayList<>();
    int n = words.length;
    int i = 0;

    while (i < n) {
        int left = i;
        int lineLen = words[i].length();
        i++;

        while (i < n && lineLen + 1 + words[i].length() <= maxWidth) {
            lineLen += 1 + words[i].length();
            i++;
        }

        int right = i;
        int numWords = right - left;
        int gaps = numWords - 1;
        StringBuilder sb = new StringBuilder();

        if (right == n || numWords == 1) {
            for (int j = left; j < right; j++) {
                if (j > left) {
                    sb.append(" ");
                }
                sb.append(words[j]);
            }
            while (sb.length() < maxWidth) {
                sb.append(" ");
            }
        } else {
            int wordsTotalLen = 0;
            for (int j = left; j < right; j++) {
                wordsTotalLen += words[j].length();
            }

            int totalSpaces = maxWidth - wordsTotalLen;
            int avgSpaces = totalSpaces / gaps;
            int extraSpaces = totalSpaces % gaps;

            for (int j = left; j < right - 1; j++) {
                sb.append(words[j]);
                for (int k = 0; k < avgSpaces; k++) {
                    sb.append(" ");
                }
                if (j - left < extraSpaces) {
                    sb.append(" ");
                }
            }

            sb.append(words[right - 1]);
        }

        res.add(sb.toString());
    }

    return res;
}
```

时间复杂度可以记作 `O(N)`，其中 `N` 是所有单词字符总数加上结果构造的总量；空间复杂度主要来自返回结果本身。

这题挺适合顺手记住一种拆规则的方法：当一堆格式要求同时出现时，先把“选什么内容”和“内容怎么摆”拆开，很多原本看着很乱的字符串模拟题就会清楚不少。

完整代码如下：

```java
import java.util.ArrayList;
import java.util.List;

public class Main {

    public static List<String> fullJustify(String[] words, int maxWidth) {
        List<String> res = new ArrayList<>();
        int n = words.length;
        int i = 0;

        while (i < n) {
            int left = i;
            int lineLen = words[i].length();
            i++;

            while (i < n && lineLen + 1 + words[i].length() <= maxWidth) {
                lineLen += 1 + words[i].length();
                i++;
            }

            int right = i;
            int numWords = right - left;
            int gaps = numWords - 1;
            StringBuilder sb = new StringBuilder();

            if (right == n || numWords == 1) {
                for (int j = left; j < right; j++) {
                    if (j > left) {
                        sb.append(" ");
                    }
                    sb.append(words[j]);
                }
                while (sb.length() < maxWidth) {
                    sb.append(" ");
                }
            } else {
                int wordsTotalLen = 0;
                for (int j = left; j < right; j++) {
                    wordsTotalLen += words[j].length();
                }

                int totalSpaces = maxWidth - wordsTotalLen;
                int avgSpaces = totalSpaces / gaps;
                int extraSpaces = totalSpaces % gaps;

                for (int j = left; j < right - 1; j++) {
                    sb.append(words[j]);
                    for (int k = 0; k < avgSpaces; k++) {

                    if (j - left < extraSpaces) {
                        sb.append(" ");
                    }

    public static void main(String[] args) {
        String[] words1 = {"This", "is", "an", "example", "of", "text", "justification."};
        int maxWidth1 = 16;

        System.out.println("示例 1 输出:");
        printResult(fullJustify(words1, maxWidth1));
        System.out.println("-----------------------------");

        String[] words2 = {"What", "must", "be", "acknowledgment", "shall", "be"};
        int maxWidth2 = 16;

        System.out.println("示例 2 输出:");
        printResult(fullJustify(words2, maxWidth2));
        System.out.println("-----------------------------");

        String[] words3 = {"Science", "is", "what", "we", "understand", "well", "enough", "to", "explain", "to", "a", "computer.", "Art", "is", "everything", "else", "we", "do"};
        int maxWidth3 = 20;

        System.out.println("示例 3 输出:");
        printResult(fullJustify(words3, maxWidth3));
    }

    private static void printResult(List<String> result) {
        for (String line : result) {
            System.out.println("\"" + line + "\"");
        }
    }
}
```
