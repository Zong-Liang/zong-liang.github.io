## 题目

**描述：**

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**示例 1:**

- **输入:** `strs = ["flower","flow","flight"]`
- **输出:** `"fl"`

**示例 2:**

- **输入:** `strs = ["dog","racecar","car"]`
- **输出:** `""`
- **解释:** 输入不存在公共前缀。

**提示:**

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` 如果非空，则仅由小写英文字母组成

## 题解

这题别急着想“怎么把公共前缀找出来”，先换个角度去看：随着参与比较的字符串越来越多，公共前缀只会变短，不会变长。这个性质一旦想清楚，写法其实很自然——先拿一个最长的候选前缀出来，再不断把它缩短到所有字符串都能接受为止。

最顺手的起点，就是把第一个字符串先当成当前公共前缀。接着从第二个字符串开始逐个比较：如果当前字符串本来就以 `prefix` 开头，那说明这个候选还站得住；如果不是，就说明 `prefix` 太长了，需要把最后一个字符砍掉，再试一次。因为答案只可能变短，所以这件事做得越往后越明确，不会出现“比较了更多字符串，前缀反而变长”的情况。

拿 `strs = ["flower", "flow", "flight"]` 走一遍会很直观。开始时 `prefix = "flower"`。和 `"flow"` 比较时，它并不是以 `"flower"` 开头的，于是把 `prefix` 缩成 `"flowe"`、`"flow"`，这时终于匹配上了。然后再拿 `"flow"` 去和 `"flight"` 比，不匹配，就继续缩成 `"flo"`、`"fl"`，最后得到的就是答案。整个过程其实没有“重新搜索”，只是一直在收缩候选前缀。

像 `strs = ["dog", "racecar", "car"]` 这种例子也很好理解。`prefix` 从 `"dog"` 一路缩到空串都还是对不上，说明根本不存在公共前缀，直接返回 `""` 就可以了。这里空串其实就是最自然的失败信号。

所以这题的核心并不在某个技巧性的 API，而在这个单调收缩的过程：当前候选如果太长，就缩一点，再拿它去试；直到它能匹配，或者已经缩没了为止。写成代码时，用 `startsWith` 会比 `indexOf(prefix) == 0` 更直接一些，也更符合这里的语义。

代码如下：

```java
public String longestCommonPrefix(String[] strs) {
    if (strs == null || strs.length == 0) {
        return "";
    }

    String prefix = strs[0];

    for (int i = 1; i < strs.length; i++) {
        while (!strs[i].startsWith(prefix)) {
            prefix = prefix.substring(0, prefix.length() - 1);
            if (prefix.isEmpty()) {
                return "";
            }
        }
    }

    return prefix;
}
```

时间复杂度可以记作 `O(S)`，其中 `S` 是所有字符串字符总数的量级；空间复杂度是 `O(1)`，因为没有使用额外的复杂数据结构。

这题比较值得带走的，不一定是“横向扫描”这个名字，而是这种思路：当候选答案只会单调收缩时，先假设一个最大的候选，再一点点把它削到合法，往往会比从零开始找更顺。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static String longestCommonPrefix(String[] strs) {
        if (strs == null || strs.length == 0) {
            return "";
        }

        String prefix = strs[0];

        for (int i = 1; i < strs.length; i++) {
            while (!strs[i].startsWith(prefix)) {
                prefix = prefix.substring(0, prefix.length() - 1);
                if (prefix.isEmpty()) {
                    return "";
                }
            }
        }

        return prefix;
    }

    public static void main(String[] args) {
        String[] strs1 = new String[]{"flower", "flow", "flight"};
        System.out.println("示例 1 输入: " + Arrays.toString(strs1));
        System.out.println("示例 1 输出: \"" + longestCommonPrefix(strs1) + "\"");

        String[] strs2 = new String[]{"dog", "racecar", "car"};
        System.out.println("示例 2 输入: " + Arrays.toString(strs2));
        System.out.println("示例 2 输出: \"" + longestCommonPrefix(strs2) + "\"");

        String[] strs3 = new String[]{"ab", "a"};
        System.out.println("额外测试 输入: " + Arrays.toString(strs3));
        System.out.println("额外测试 输出: \"" + longestCommonPrefix(strs3) + "\"");
    }
}
```
