## 题目

**描述：**

给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

**示例 1:**

- **输入:** `[7,1,5,3,6,4]`
- **输出:** 5
- **解释:** 在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

**示例 2:**

- **输入:** `prices = [7,6,4,3,1]`
- **输出:** 0
- **解释:** 在这种情况下, 没有交易完成, 所以最大利润为 0。

**提示:**

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

## 题解

很多人第一次看到这题，会先盯着“卖出”去想：到底在哪一天卖最赚？但这样想很容易把问题想复杂，因为利润不只取决于卖价高不高，还取决于你之前有没有在更低的价格买到。真正需要一直盯住的，其实不是“今天卖不卖”，而是：如果今天卖出，我之前能买到的最低价格是多少。

最直接的做法当然是把所有买卖组合都枚举一遍，枚举买入日，再枚举它后面的卖出日，逐个计算利润，最后取最大值。这个方法没问题，但时间复杂度是 $O(n^2)$，题目给到 `10^5` 的规模时肯定撑不住。既然如此，问题就变成了：能不能在遍历到当天时，顺手就知道“以今天卖出最多能赚多少”。

这样一来，遍历过程中真正有价值的信息其实只有两个：到今天为止出现过的最低价格，以及到今天为止能拿到的最大利润。于是我们只需要维护 `minPrice` 和 `maxProfit` 两个变量。每看到一个新价格 `price`，如果它比当前最低价还低，就更新 `minPrice`；否则就计算 `price - minPrice`，看看能不能刷新 `maxProfit`。

这个思路之所以成立，关键就在“先买后卖”这条限制上。假设今天是卖出日，那合法的买入位置只能出现在今天之前；而想让今天这笔交易利润最大，买入价当然应该尽可能低。所以对每一天来说，最优选择其实都很明确：用今天的价格减去之前出现过的最低价格。把每一天都这样算一遍，最大值自然就是答案。

拿示例 `prices = [7,1,5,3,6,4]` 走一遍。开始时 `minPrice` 还很大，`maxProfit = 0`。看到 `7` 时，最低价更新成 `7`；看到 `1` 时，最低价更新成 `1`；看到 `5` 时，可以赚 `5 - 1 = 4`，于是 `maxProfit = 4`；看到 `3` 时，利润只有 `2`，不用更新；看到 `6` 时，利润变成 `5`，于是 `maxProfit` 刷新成 `5`；最后看到 `4` 时，利润是 `3`，仍然不如 `5`。因此最终答案就是 `5`。

像示例 2 这种一路下降的情况也很好解释。只要后面的价格始终不高于前面出现过的最低价，算出来的利润就不可能是正数。题目要求不能获利时返回 `0`，所以 `maxProfit` 从一开始就设成 `0`，正好把这种情况兜住了。

代码如下：

```java
public int maxProfit(int[] prices) {
    int minPrice = Integer.MAX_VALUE;
    int maxProfit = 0;

    for (int price : prices) {
        if (price < minPrice) {
            minPrice = price;
        } else if (price - minPrice > maxProfit) {
            maxProfit = price - minPrice;
        }
    }

    return maxProfit;
}
```

时间复杂度是 $O(n)$，因为整个数组只遍历了一遍；空间复杂度是 $O(1)$，因为只用了两个额外变量。

这题更适合记住的不是“股票题要用贪心”这种标签，而是这种压缩信息的方式：如果某个位置的最优答案只依赖于它前面的一部分最值信息，那就没必要把所有历史都留着，只维护那个最关键的最值就够了。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static int maxProfit(int[] prices) {
        int minPrice = Integer.MAX_VALUE;
        int maxProfit = 0;

        for (int price : prices) {
            if (price < minPrice) {
                minPrice = price;
            } else if (price - minPrice > maxProfit) {
                maxProfit = price - minPrice;
            }
        }

        return maxProfit;
    }

    public static void main(String[] args) {
        int[] prices1 = new int[]{7, 1, 5, 3, 6, 4};

        System.out.println("示例 1 输入:");
        System.out.println("prices = " + Arrays.toString(prices1));

        int result1 = maxProfit(prices1);

        System.out.println("示例 1 输出: " + result1);

        int[] prices2 = new int[]{7, 6, 4, 3, 1};

        System.out.println("示例 2 输入:");
        System.out.println("prices = " + Arrays.toString(prices2));

        int result2 = maxProfit(prices2);

        System.out.println("示例 2 输出: " + result2);
    }
}
```
