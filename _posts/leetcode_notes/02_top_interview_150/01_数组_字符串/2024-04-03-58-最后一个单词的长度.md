## 题目

**描述：**

给你一个字符串 `s`，由若干单词组成，单词前后用一些空格字符隔开。返回字符串中 **最后一个** 单词的长度。

**单词** 是指仅由字母组成、不包含任何空格字符的最大子字符串。

**示例 1:**

- **输入:** `s = "Hello World"`
- **输出:** 5
- **解释:** 最后一个单词是“World”，长度为 5。

**示例 2:**

- **输入:** `s = "   fly me   to   the moon  "`
- **输出:** 4
- **解释:** 最后一个单词是“moon”，长度为 4。

**示例 3:**

- **输入:** `s = "luffy is still joyboy"`
- **输出:** 6
- **解释:** 最后一个单词是长度为 6 的“joyboy”。

**提示:**

- `1 <= s.length <= 10^4`
- `s` 仅有英文字母和空格 `' '` 组成
- `s` 中至少存在一个单词

## 题解

这题不难，容易绕的地方反而是把它做重了。像 `trim()`、`split()` 当然都能用，但它们会把前面那些根本不关心的内容也一起处理掉。题目只问最后一个单词有多长，所以更顺手的写法其实是直接从字符串尾部往前看。

这么做的原因也很自然，因为字符串末尾可能先出现的是空格，而不是最后一个单词本身。像 `"   fly me   to   the moon  "`，字符串的结尾是两个空格，不先跳过它们，就根本碰不到 `moon`。所以整个过程其实只有两步：先跳过末尾所有空格，再从最后一个非空字符开始往前数，直到再次遇到空格为止。

拿 `"Hello World"` 这个例子看，末尾没有空格，指针直接停在 `d` 上，然后一路往前数到 `W`，长度就是 `5`。再看 `"   fly me   to   the moon  "`，先跳过尾部空格，指针会落在 `n` 上，再继续往前数 `moon` 这四个字符，遇到空格后停止，所以答案是 `4`。整个过程都只围着字符串最后那一小段打转，不需要把整串切开。

题目还保证了字符串里至少有一个单词，所以这里也不用担心“全是空格”这种额外情况。边界一旦明确，这题就很适合写成一个很短的双 `while`：第一段负责跳过空格，第二段负责计数。

代码如下：

```java
public int lengthOfLastWord(String s) {
    int index = s.length() - 1;

    while (index >= 0 && s.charAt(index) == ' ') {
        index--;
    }

    int length = 0;
    while (index >= 0 && s.charAt(index) != ' ') {
        length++;
        index--;
    }

    return length;
}
```

时间复杂度是 `O(n)`，最坏情况下要从后往前扫完整个字符串；空间复杂度是 `O(1)`，因为只用了几个变量。

这题可以顺手留下一个挺实用的观察：如果题目只关心字符串尾部附近的信息，优先考虑从后往前扫描，往往会比先切分整个字符串更省事。

完整代码如下：

```java
public class Main {

    public static int lengthOfLastWord(String s) {
        int index = s.length() - 1;

        while (index >= 0 && s.charAt(index) == ' ') {
            index--;
        }

        int length = 0;
        while (index >= 0 && s.charAt(index) != ' ') {
            length++;
            index--;
        }

        return length;
    }

    public static void main(String[] args) {
        String s1 = "Hello World";
        System.out.println("示例 1 输入: s = \"" + s1 + "\"");
        System.out.println("示例 1 输出: " + lengthOfLastWord(s1));

        String s2 = "   fly me   to   the moon  ";
        System.out.println("示例 2 输入: s = \"" + s2 + "\"");
        System.out.println("示例 2 输出: " + lengthOfLastWord(s2));

        String s3 = "luffy is still joyboy";
        System.out.println("示例 3 输入: s = \"" + s3 + "\"");
        System.out.println("示例 3 输出: " + lengthOfLastWord(s3));
    }
}
```
