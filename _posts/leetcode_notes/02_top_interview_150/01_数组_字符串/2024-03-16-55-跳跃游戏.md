## 题目

**描述：**

给你一个非负整数数组 `nums` ，你最初位于数组的 **第一个下标** 。数组中的每个元素代表你在该位置可以跳跃的最大长度。

判断你是否能够到达最后一个下标，如果可以，返回 `true` ；否则，返回 `false` 。

**示例 1:**

- **输入:** `nums = [2,3,1,1,4]`
- **输出:** `true`
- **解释:** 可以先跳 1 步，从下标 0 到达下标 1, 然后再从下标 1 跳 3 步到达最后一个下标。

**示例 2:**

- **输入:** `nums = [3,2,1,0,4]`
- **输出:** `false`
- **解释:** 无论怎样，总会到达下标为 3 的位置。但该下标的最大跳跃长度是 0 ，所以永远不可能到达最后一个下标。

**提示:**

- `1 <= nums.length <= 10^4`
- `0 <= nums[i] <= 10^5`

## 题解

很多人看到“每个位置可以跳多远”，第一反应会去想：这一步到底该跳几格，要不要把所有跳法都试一遍，或者干脆用搜索把路径全找出来。其实这题根本不要求你给出具体跳法，它只问一件事：能不能到最后一个位置。

一旦只关心“能不能到”，真正有价值的信息就不是“这一步跳到哪里”，而是“到当前为止，最远能覆盖到哪个位置”。所以这题最合适维护的变量其实只有一个：`rightmost`，表示当前已知最远能到达的下标。

从左到右扫描数组时，每来到一个位置 `i`，先看它是不是可达。如果 `i > rightmost`，说明这个位置已经超出了当前覆盖范围，那后面自然也不用再看了，直接返回 `false`；如果 `i <= rightmost`，说明它在当前覆盖范围内，那就可以继续用 `i + nums[i]` 去扩展最远可达位置。整个过程本质上就是在不断维护一段“当前还能覆盖到的区间”。

这也是为什么这题不需要真的决定“下一步跳哪”。对于一个可达位置 `i` 来说，它带来的价值只有一个：它能不能把最远可达边界继续往右推。至于你具体是怎么走到 `i` 的，题目并不关心。所以这题其实不是路径选择题，而更像是在做区间覆盖。

拿 `nums = [2,3,1,1,4]` 看一下。开始时 `rightmost = 0`。来到下标 `0` 时，因为它可达，所以最远更新成 `0 + 2 = 2`；来到下标 `1` 时，因为 `1 <= 2`，说明这个位置也可达，于是最远更新成 `1 + 3 = 4`。而最后一个下标本来就是 `4`，这时说明终点已经被覆盖到，可以直接返回 `true`。

再看 `nums = [3,2,1,0,4]`。开始时最远能到 `3`，中间的 `1`、`2`、`3` 虽然都还能到，但它们并不能把覆盖范围继续往后推进，因为走到下标 `3` 时 `nums[3] = 0`，最远位置仍然停在 `3`。等准备检查下标 `4` 时，会发现 `4 > rightmost`，也就是说终点已经落在了覆盖范围之外，自然就到不了。

代码如下：

```java
public boolean canJump(int[] nums) {
    int rightmost = 0;

    for (int i = 0; i < nums.length; i++) {
        if (i > rightmost) {
            return false;
        }

        rightmost = Math.max(rightmost, i + nums[i]);

        if (rightmost >= nums.length - 1) {
            return true;
        }
    }

    return true;
}
```

时间复杂度是 $O(n)$，因为整个数组只遍历了一遍；空间复杂度是 $O(1)$，因为只用了一个额外变量。

这题更适合记住的是这个判断：当题目只问“能不能到达”，而不要求具体路径时，很多时候根本不需要真的去模拟每一步，只要维护当前能扩展到的最远边界就够了。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static boolean canJump(int[] nums) {
        int rightmost = 0;

        for (int i = 0; i < nums.length; i++) {
            if (i > rightmost) {
                return false;
            }

            rightmost = Math.max(rightmost, i + nums[i]);

            if (rightmost >= nums.length - 1) {
                return true;
            }
        }

        return true;
    }

    public static void main(String[] args) {
        int[] nums1 = new int[]{2, 3, 1, 1, 4};

        System.out.println("示例 1 输入:");
        System.out.println("nums = " + Arrays.toString(nums1));

        boolean result1 = canJump(nums1);

        System.out.println("示例 1 输出: " + result1);

        int[] nums2 = new int[]{3, 2, 1, 0, 4};

        System.out.println("示例 2 输入:");
        System.out.println("nums = " + Arrays.toString(nums2));

        boolean result2 = canJump(nums2);

        System.out.println("示例 2 输出: " + result2);
    }
}
```
