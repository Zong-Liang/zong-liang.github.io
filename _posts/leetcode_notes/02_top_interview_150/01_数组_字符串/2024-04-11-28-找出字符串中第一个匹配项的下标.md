## 题目

**描述：**

给你两个字符串 `haystack` 和 `needle` ，请你在 `haystack` 字符串中找出 `needle` 字符串的第一个匹配项的下标（下标从 0 开始）。如果 `needle` 不是 `haystack` 的一部分，则返回 `-1` 。

**示例 1:**

- **输入:** `haystack = "sadbutsad"`, `needle = "sad"`
- **输出:** 0
- **解释:** `"sad"` 在下标 0 和 6 处匹配。第一个匹配项的下标是 0 ，所以返回 0 。

**示例 2:**

- **输入:** `haystack = "leetcode"`, `needle = "leeto"`
- **输出:** -1
- **解释:** `"leeto"` 没有在 `"leetcode"` 中出现，所以返回 -1 。

**提示:**

- `1 <= haystack.length, needle.length <= 10^4`
- `haystack` 和 `needle` 仅由小写英文字符组成

## 题解

这题如果只是想把答案写出来，最直接的办法其实就是枚举每一个可能的起点，看看 `needle` 能不能从那里完整匹配上。这个思路本身没有任何问题，而且它正好也是理解 KMP 的最好起点。因为 KMP 并不是把匹配过程推翻重来，它只是专门解决一件事：失配以后，别把已经比较过的内容白白再比一遍。

先看暴力匹配为什么会浪费。比如 `haystack = "aaab"`，`needle = "aab"`。从下标 `0` 开始时，前两个字符都已经匹配上了，第三个字符才失配。暴力法接下来只能把起点挪到 `1`，然后让模式串从头再比一遍，刚才那两个已经比过的 `a` 又会重新走一次。浪费就浪费在这里：一旦失配，前面匹配成功的信息完全没有被利用。

KMP 的思路，就是把这部分信息先存下来。它用一个 `next` 数组记录：对于模式串的每个前缀，内部有没有一段“既是前缀又是后缀”的结构可以复用。比如 `"abab"` 这段里，最长的相同前后缀是 `"ab"`，长度是 `2`。这意味着如果我们已经匹配到了 `abab`，接下来又失配了，其实没必要让模式串退回到开头，因为开头那段 `ab` 和结尾那段 `ab` 是一样的，完全可以直接从还能接上的位置继续试。

所以 KMP 在正式匹配时做的事很简单：主串指针 `i` 一直往前，不回退；模式串指针 `j` 一旦失配，就跳到 `next[j - 1]` 指向的位置。换句话说，它不是把当前这次尝试整个推倒重来，而是尽量保住已经匹配成功的那一段里还能复用的部分。

拿 `needle = "aabaaf"` 这样的模式串来说，假设已经匹配到了前面的某一段，现在在位置 `j` 失配。此时 `j = next[j - 1]` 这一步，含义并不神秘，它只是说：前面那段已经对上的内容里，还有多长的一段前后缀是相同的，我就把这段保留下来，模式串从那里继续对齐。主串不用回头，所以整个匹配过程才能压到线性时间。

写代码时通常分成两步：先为 `needle` 构造 `next` 数组，再拿它去扫 `haystack`。构造 `next` 的过程和正式匹配很像，本质上也是在做“失配时利用已有结构跳转”。如果想兼容老版本 `strStr` 的定义，`needle` 为空时直接返回 `0` 就可以。

代码如下：

```java
public int strStr(String haystack, String needle) {
    if (needle.length() == 0) {
        return 0;
    }

    int[] next = new int[needle.length()];
    buildNext(next, needle);

    int j = 0;
    for (int i = 0; i < haystack.length(); i++) {
        while (j > 0 && haystack.charAt(i) != needle.charAt(j)) {
            j = next[j - 1];
        }

        if (haystack.charAt(i) == needle.charAt(j)) {
            j++;
        }

        if (j == needle.length()) {
            return i - needle.length() + 1;
        }
    }

    return -1;
}

private void buildNext(int[] next, String s) {
    int j = 0;

    for (int i = 1; i < s.length(); i++) {
        while (j > 0 && s.charAt(i) != s.charAt(j)) {
            j = next[j - 1];
        }

        if (s.charAt(i) == s.charAt(j)) {
            j++;
        }

        next[i] = j;
    }
}
```

时间复杂度是 `O(n + m)`，其中 `n` 是 `haystack` 的长度，`m` 是 `needle` 的长度；空间复杂度是 `O(m)`，因为需要一个长度为 `m` 的 `next` 数组。

这题更适合记住的，不是某个公式本身，而是 KMP 背后的出发点：如果一个匹配过程里总在重复比较已经看过的内容，那就该想办法把“还能复用的那部分”提前记录下来，失配时直接接着用。

完整代码如下：

```java
public class Main {

    public static int strStr(String haystack, String needle) {
        if (needle.length() == 0) {
            return 0;
        }

        int[] next = new int[needle.length()];
        buildNext(next, needle);

        int j = 0;
        for (int i = 0; i < haystack.length(); i++) {
            while (j > 0 && haystack.charAt(i) != needle.charAt(j)) {
                j = next[j - 1];
            }

            if (haystack.charAt(i) == needle.charAt(j)) {
                j++;
            }

            if (j == needle.length()) {
                return i - needle.length() + 1;
            }
        }

        return -1;
    }

    private static void buildNext(int[] next, String s) {
        int j = 0;

        for (int i = 1; i < s.length(); i++) {
            while (j > 0 && s.charAt(i) != s.charAt(j)) {
                j = next[j - 1];
            }

            if (s.charAt(i) == s.charAt(j)) {
                j++;
            }

            next[i] = j;
        }
    }

    public static void main(String[] args) {
        String haystack1 = "sadbutsad";
        String needle1 = "sad";

        System.out.println("示例 1 输入: haystack = \"" + haystack1 + "\", needle = \"" + needle1 + "\"");
        System.out.println("示例 1 输出: " + strStr(haystack1, needle1));

        String haystack2 = "leetcode";
        String needle2 = "leeto";

        System.out.println("示例 2 输入: haystack = \"" + haystack2 + "\", needle = \"" + needle2 + "\"");
        System.out.println("示例 2 输出: " + strStr(haystack2, needle2));

        String haystack3 = "hello";
        String needle3 = "ll";

        System.out.println("额外测试 输入: haystack = \"" + haystack3 + "\", needle = \"" + needle3 + "\"");
        System.out.println("额外测试 输出: " + strStr(haystack3, needle3));
    }
}
```
