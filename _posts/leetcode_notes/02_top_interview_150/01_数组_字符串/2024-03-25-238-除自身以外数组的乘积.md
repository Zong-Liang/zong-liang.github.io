## 题目

**描述：**

给你一个整数数组 `nums`，返回 数组 `answer` ，其中 `answer[i]` 等于 `nums` 中除 `nums[i]` 之外其余各元素的乘积。

题目数据 **保证** 数组 `nums`之中任意元素的全部前缀元素和后缀的乘积都在 **32 位** 整数范围内。

请 **不要使用除法**，且在 `O(n)` 时间复杂度内完成此题。

**示例 1:**

- **输入:** `nums = [1,2,3,4]`
- **输出:** `[24,12,8,6]`

**示例 2:**

- **输入:** `nums = [-1,1,0,-3,3]`
- **输出:** `[0,0,9,0,0]`

**提示:**

- `2 <= nums.length <= 10^5`
- `-30 <= nums[i] <= 30`
- 输入 **保证** 数组 `answer[i]` 在 **32 位** 整数范围内

**进阶：** 你可以在 `O(1)` 的额外空间复杂度内完成这个题目吗？（出于对空间复杂度分析的目的，输出数组 **不被视为** 额外空间。）

## 题解

这题如果不看限制，最直接的想法其实就是先把整个数组乘起来，再让每个位置去除以自己。但题目偏偏把这条路堵死了：不能用除法，而且还要求在线性时间里完成。所以做这题时，视角最好尽快从“总乘积”切过去，改成去想：对位置 `i` 来说，除了 `nums[i]` 以外的乘积，到底由哪两部分组成。

答案其实很朴素，它就是“左边所有元素的乘积”乘上“右边所有元素的乘积”。比如 `nums = [1, 2, 3, 4]`，位置 `2` 的答案不是靠什么特殊技巧算出来的，它只是左边 `1 * 2` 和右边 `4` 的乘积。这样一拆，题目立刻就清楚了：我们要做的，不是想办法把自己从整体乘积里剔掉，而是高效地拿到每个位置左边和右边的贡献。

最直观的写法，是先准备两个辅助数组：`left[i]` 表示位置 `i` 左边所有元素的乘积，`right[i]` 表示位置 `i` 右边所有元素的乘积，最后 `answer[i] = left[i] * right[i]`。这个版本不难想到，也已经能在线性时间里解决问题。不过题目后面还追问能不能做到 `O(1)` 额外空间，这时就得再往前走一步：既然返回数组本来就一定要开出来，不如先拿它存左边乘积，再在第二遍从右往左时，把右边乘积顺手乘进去。

还是看 `[1, 2, 3, 4]`。第一遍从左往右时，让 `answer[i]` 先表示当前位置左边所有元素的乘积，处理完后会得到 `[1, 1, 2, 6]`。接下来再用一个变量 `rightProduct` 维护当前位置右侧的乘积，从右往左扫描：到下标 `3` 时，右边没有元素，所以 `answer[3] = 6 * 1`；然后把 `rightProduct` 更新成 `4`；到下标 `2` 时，`answer[2] = 2 * 4 = 8`；继续往前，最后整个数组就变成 `[24, 12, 8, 6]`。这样除了输出数组本身，我们只多用了一个变量。

这种写法还有一个很省心的地方，就是根本不用单独讨论 `0`。因为从头到尾我们只是在累乘左边和右边的贡献，没有做除法，所以数组里有 `0`、有负数都能被自然处理掉。像示例 `[-1, 1, 0, -3, 3]`，照样会老老实实得到 `[0, 0, 9, 0, 0]`，不需要额外打补丁。

代码如下：

```java
public int[] productExceptSelf(int[] nums) {
    int n = nums.length;
    int[] answer = new int[n];

    answer[0] = 1;
    for (int i = 1; i < n; i++) {
        answer[i] = answer[i - 1] * nums[i - 1];
    }

    int rightProduct = 1;
    for (int i = n - 1; i >= 0; i--) {
        answer[i] *= rightProduct;
        rightProduct *= nums[i];
    }

    return answer;
}
```

时间复杂度是 `O(n)`，因为数组总共只遍历了两遍；额外空间复杂度是 `O(1)`，因为除了返回数组外，只用了一个 `rightProduct` 变量。

这题可以顺手记成一种拆问题的方式：如果当前位置的答案同时依赖左边整体信息和右边整体信息，往往可以先存一边，再在第二遍把另一边补上；而输出数组本来就存在时，很多时候还能顺便拿它做状态复用。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static int[] productExceptSelf(int[] nums) {
        int n = nums.length;
        int[] answer = new int[n];

        answer[0] = 1;
        for (int i = 1; i < n; i++) {
            answer[i] = answer[i - 1] * nums[i - 1];
        }

        int rightProduct = 1;
        for (int i = n - 1; i >= 0; i--) {
            answer[i] *= rightProduct;
            rightProduct *= nums[i];
        }

        return answer;
    }

    public static void main(String[] args) {
        int[] nums1 = new int[]{1, 2, 3, 4};

        System.out.println("示例 1 输入:");
        System.out.println("nums = " + Arrays.toString(nums1));

        int[] result1 = productExceptSelf(nums1);

        System.out.println("示例 1 输出: " + Arrays.toString(result1));

        int[] nums2 = new int[]{-1, 1, 0, -3, 3};

        System.out.println("示例 2 输入:");
        System.out.println("nums = " + Arrays.toString(nums2));

        int[] result2 = productExceptSelf(nums2);

        System.out.println("示例 2 输出: " + Arrays.toString(result2));
    }
}
```
