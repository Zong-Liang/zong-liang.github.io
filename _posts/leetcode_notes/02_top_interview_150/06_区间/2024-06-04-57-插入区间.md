## 题目

**描述：**

给你一个 **无重叠** 的 ，按照区间起始端点排序的区间列表 `intervals` ，其中 `intervals[i] = [start_i, end_i]` 表示第 `i` 个区间的开始和结束，并且 `intervals` 按照 `start_i` **升序排列** 。

同样给定一个区间 `newInterval = [start, end]` 表示另一个区间的开始和结束。

在 `intervals` 中插入区间 `newInterval`，使得 `intervals` 依然按照 `start_i` 升序排列，且区间之间不重叠（如果有必要的话，可以合并区间）。

返回插入之后的 `intervals`。

**注意** 你不需要原地修改 `intervals`。你可以创建一个新数组然后返回它。

**示例 1:**

- **输入:** `intervals = [[1,3],[6,9]]`, `newInterval = [2,5]`
- **输出:** `[[1,5],[6,9]]`

**示例 2:**

- **输入:** `intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]]`, `newInterval = [4,8]`
- **输出:** `[[1,2],[3,10],[12,16]]`
- **解释:** 这是因为新的区间 `[4,8]` 与 `[3,5],[6,7],[8,10]` 重叠。

**提示:**

- `0 <= intervals.length <= 10^4`
- `intervals[i].length == 2`
- `0 <= start_i <= end_i <= 10^5`
- `intervals` 根据 `start_i` 按 **升序** 排列
- `newInterval.length == 2`
- `0 <= start <= end <= 10^5`

## 题解

### 这题和“合并区间”长得像，但重点不一样

上一题“合并区间”里，区间整体是乱的，所以第一步一定要先排序。  
但这题不一样，题目已经明确告诉你：

- 原区间列表本来就是按起点升序排列的
- 原区间之间本来就互不重叠

这两个条件非常关键。它意味着我们根本不用重新排序，也不用把所有区间再完整合并一遍。

这题真正要做的，其实只是：

**把一个新区间插进去，然后看看它会和中间哪一段发生重叠。**

所以这题的重点不是“排序后合并”，而是“利用原数组已经有序且无重叠这个条件，直接一次扫描完成插入”。

### 既然原区间已经排好序，整个过程天然会分成三段

把 `newInterval` 放进原区间序列之后，整体结构其实很清楚，只会出现三部分：

1. 在 `newInterval` 左边、且和它完全没有交集的区间
2. 和 `newInterval` 有重叠、需要被合并进去的区间
3. 在 `newInterval` 右边、且和它完全没有交集的区间

也就是说，这题不是在一堆混乱区间里找关系，而是在一条已经排好序的线上，顺着扫过去，依次处理这三段。

这也是为什么它的标准写法经常被称为“三阶段遍历”。

### 第一段最简单：左边完全不重叠的，直接收下

如果当前区间的结束位置还在 `newInterval` 开始位置的左边，也就是：

```text
intervals[i][1] < newInterval[0]
```

那说明这个区间不可能和 `newInterval` 重叠。  
而且因为原数组有序，后面这些区间的起点只会越来越大，所以当前这个区间肯定应该原样放进结果里。

这种区间完全不用改，直接加入结果即可。

### 第二段才是核心：一旦重叠，就不断扩展 `newInterval`

什么时候算和 `newInterval` 重叠？

只要当前区间的起点还没有跑到 `newInterval` 右边去，也就是：

```text
intervals[i][0] <= newInterval[1]
```

就说明它和 `newInterval` 还有交集，或者刚好贴边相接，需要合并。

这时候不用把当前区间单独加入结果，而是直接拿它去更新 `newInterval` 的范围：

- 新起点取两者较小值
- 新终点取两者较大值

也就是：

```java
newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
```

这一步其实很关键。  
你可以把它理解成：`newInterval` 不断“吃掉”所有和它重叠的区间，直到再也吃不到为止。

等这段处理完之后，`newInterval` 就已经变成了合并后的最终区间。

### 第三段也很简单：右边剩下的，原样接上

当前面所有重叠区间都处理完之后，后面的区间一定都在 `newInterval` 的右边，而且不会再和它重叠。

原因很简单：

- 原数组本来有序
- 前面重叠的已经连续处理完了
- 一旦遇到第一个不重叠且在右边的区间，后面只会更靠右

所以剩下的部分也不需要额外操作，直接依次加入结果即可。

### 用示例 1 走一遍

看示例 1：

```text
intervals = [[1,3],[6,9]]
newInterval = [2,5]
```

先看 `[1,3]`：

- 它不在 `newInterval` 左边，因为 `3 < 2` 不成立
- 它和 `newInterval` 有重叠，因为 `1 <= 5`

所以进入第二段，把 `newInterval` 更新为：

```text
[min(2,1), max(5,3)] = [1,5]
```

再看 `[6,9]`：

- 它已经不和 `[1,5]` 重叠了，因为 `6 <= 5` 不成立

说明第二段结束。  
这时先把合并后的 `[1,5]` 放进结果，再把剩下的 `[6,9]` 接上。

最终答案就是：

```text
[[1,5],[6,9]]
```

### 再看示例 2，会更能看出“三段”的感觉

```text
intervals = [[1,2],[3,5],[6,7],[8,10],[12,16]]
newInterval = [4,8]
```

先处理左边：

- `[1,2]` 完全在 `newInterval` 左边，因为 `2 < 4`，直接放进结果

接着进入重叠段：

- `[3,5]` 和 `[4,8]` 重叠，合并成 `[3,8]`
- `[6,7]` 也重叠，还是 `[3,8]`
- `[8,10]` 也重叠，更新成 `[3,10]`

再往后看 `[12,16]`：

- `12 <= 10` 不成立，说明重叠段结束

于是把合并后的 `[3,10]` 放进结果，最后再接上右边剩下的 `[12,16]`。

最终结果就是：

```text
[[1,2],[3,10],[12,16]]
```

### 为什么这题不需要排序，也不需要回头

因为原始区间已经满足两个特别好的条件：

- 有序
- 无重叠

这两个条件保证了：

1. 所有和 `newInterval` 重叠的区间，一定会连续出现
2. 一旦某个区间已经落在 `newInterval` 右边，后面所有区间也都在右边

所以整个过程只需要从左到右扫一遍，不需要来回找，也不需要额外整理顺序。

这正是这题和“合并区间”最大的差别：  
上一题要先把顺序理出来，这一题顺序本来就是理好的。

### 代码

```java
public int[][] insert(int[][] intervals, int[] newInterval) {
    List<int[]> result = new ArrayList<>();
    int i = 0;
    int n = intervals.length;

    while (i < n && intervals[i][1] < newInterval[0]) {
        result.add(intervals[i]);
        i++;
    }

    while (i < n && intervals[i][0] <= newInterval[1]) {
        newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
        newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
        i++;
    }

    result.add(newInterval);

    while (i < n) {
        result.add(intervals[i]);
        i++;
    }

    return result.toArray(new int[result.size()][]);
}
```

时间复杂度是 $O(n)$，因为整个数组只会被顺着扫描一遍。  
空间复杂度如果不算返回结果本身，就是 $O(1)$；额外只用了少量变量。

### 这题真正值得记住的地方

这题最值得记住的，不是“三阶段遍历”这个名字，而是背后的判断：

**如果输入本身已经有序，而且结构足够干净，就不要再套一遍通用模板，应该直接利用它的顺序性质做线性处理。**

这题就是一个典型例子：

- 如果你把它当“合并区间”去做，也能做
- 但那样会把已经给出的有序、无重叠条件浪费掉

更自然的想法应该是：

**既然原序列本来就是干净的，那新区间插进去之后，受影响的只会是中间连续的一段。**

把这种“利用输入已有结构”的意识练出来，后面很多题都会更顺。

### 完整代码

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Main {

    public static int[][] insert(int[][] intervals, int[] newInterval) {
        List<int[]> result = new ArrayList<>();
        int i = 0;
        int n = intervals.length;

        while (i < n && intervals[i][1] < newInterval[0]) {
            result.add(intervals[i]);
            i++;
        }

        while (i < n && intervals[i][0] <= newInterval[1]) {
            newInterval[0] = Math.min(newInterval[0], intervals[i][0]);
            newInterval[1] = Math.max(newInterval[1], intervals[i][1]);
            i++;
        }

        result.add(newInterval);

        while (i < n) {
            result.add(intervals[i]);
            i++;
        }

        return result.toArray(new int[result.size()][]);
    }

    public static void main(String[] args) {
        int[][] intervals1 = {{1, 3}, {6, 9}};
        int[] newInterval1 = {2, 5};

        System.out.println("示例 1 输入:");
        System.out.println("intervals = " + Arrays.deepToString(intervals1));
        System.out.println("newInterval = " + Arrays.toString(newInterval1));

        int[][] res1 = insert(intervals1, newInterval1);
        System.out.println("示例 1 输出: " + Arrays.deepToString(res1));
        System.out.println("-----------------------------");

        int[][] intervals2 = {{1, 2}, {3, 5}, {6, 7}, {8, 10}, {12, 16}};
        int[] newInterval2 = {4, 8};

        System.out.println("示例 2 输入:");
        System.out.println("intervals = " + Arrays.deepToString(intervals2));
        System.out.println("newInterval = " + Arrays.toString(newInterval2));

        int[][] res2 = insert(intervals2, newInterval2);
        System.out.println("示例 2 输出: " + Arrays.deepToString(res2));
        System.out.println("-----------------------------");

        int[][] intervals3 = {{2, 3}, {5, 7}};
        int[] newInterval3 = {0, 1};

        System.out.println("示例 3 输入:");
        System.out.println("intervals = " + Arrays.deepToString(intervals3));
        System.out.println("newInterval = " + Arrays.toString(newInterval3));

        int[][] res3 = insert(intervals3, newInterval3);
        System.out.println("示例 3 输出: " + Arrays.deepToString(res3));
    }
}
```
