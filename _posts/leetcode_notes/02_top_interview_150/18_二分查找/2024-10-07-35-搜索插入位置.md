## 题目

**描述：**

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 `O(log n)` 的算法。

**示例 1:**

- **输入:** `nums = [1,3,5,6]`, `target = 5`
- **输出:** 2**示例 2:**

- **输入:** `nums = [1,3,5,6]`, `target = 2`
- **输出:** 1**示例 3:**

- **输入:** `nums = [1,3,5,6]`, `target = 7`
- **输出:** 4**提示:**

- `1 <= nums.length <= 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 为 **无重复元素** 的**升序** 排列数组
- `-10^4 <= target <= 10^4`

## 题解
这题看起来是“查找目标值”，但更顺手的理解是：找到第一个大于等于 `target` 的位置。找到就返回它本身，找不到时这个位置就是插入点，所以两件事其实是同一个问题。

写法上就是标准二分。维护闭区间 `[left, right]`，每次看 `mid`：如果 `nums[mid] < target`，说明插入点一定在右侧；否则插入点可能就是 `mid`，要把右边界收回来。循环结束时区间被压空，`left` 正好停在第一个大于等于 `target` 的位置。

这也是为什么最后直接返回 `left`，不需要额外分“找到”和“没找到”两套分支。比如 `[1,3,5,6]` 查 `2`，边界收缩到最后时 `left` 会落在下标 `1`；查 `7` 时 `left` 会走到 `4`，刚好是数组尾后插入点。

代码如下：

```java
import java.util.Arrays;

public class Main {

    public static int searchInsert(int[] nums, int target) {
        int left = 0;
        int right = nums.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return left;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 3, 5, 6};
        int target1 = 5;
        System.out.println("示例 1 输入: nums = " + Arrays.toString(nums1) + ", target = " + target1);
        System.out.println("示例 1 输出: " + searchInsert(nums1, target1));
        System.out.println("-----------------------------");

        int[] nums2 = {1, 3, 5, 6};
        int target2 = 2;
        System.out.println("示例 2 输入: nums = " + Arrays.toString(nums2) + ", target = " + target2);
        System.out.println("示例 2 输出: " + searchInsert(nums2, target2));
        System.out.println("-----------------------------");

        int[] nums3 = {1, 3, 5, 6};
        int target3 = 7;
        System.out.println("示例 3 输入: nums = " + Arrays.toString(nums3) + ", target = " + target3);
        System.out.println("示例 3 输出: " + searchInsert(nums3, target3));
        System.out.println("-----------------------------");

        int[] nums4 = {1, 3, 5, 6};
        int target4 = 0;
        System.out.println("示例 4 输入: nums = " + Arrays.toString(nums4) + ", target = " + target4);
        System.out.println("示例 4 输出: " + searchInsert(nums4, target4));
    }
}
```

复杂度分析：时间复杂度 `O(log n)`，空间复杂度 `O(1)`。

这个题型可以直接记成一句话：二分不一定是“找等于”，也可以是“找第一个满足条件的位置”。
