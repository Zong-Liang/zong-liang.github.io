## 题目

**描述：**

给定一个排序数组和一个目标值，在数组中找到目标值，并返回其索引。如果目标值不存在于数组中，返回它将会被按顺序插入的位置。

请必须使用时间复杂度为 `O(log n)` 的算法。

**示例 1:**

- **输入:** `nums = [1,3,5,6]`, `target = 5`
- **输出:** 2

**示例 2:**

- **输入:** `nums = [1,3,5,6]`, `target = 2`
- **输出:** 1

**示例 3:**

- **输入:** `nums = [1,3,5,6]`, `target = 7`
- **输出:** 4

**提示:**

- `1 <= nums.length <= 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 为 **无重复元素** 的 **升序** 排列数组
- `-10^4 <= target <= 10^4`

## 题解

### 这题其实是在找什么

题目表面上是在问：目标值在不在数组里。  
但如果把题意再往前推一步，会发现它真正想让我们找的是：

**第一个大于等于 `target` 的位置。**

为什么这么说？

- 如果数组里刚好有 `target`，那么这个位置就是它本来的下标
- 如果数组里没有 `target`，那么这个位置就是它应该插入的位置

所以这题并不是“找到就返回，找不到再另外处理”的两件事，而是从头到尾都在找同一个东西。

既然数组本身是升序排列，而且题目还明确要求时间复杂度必须是 `O(log n)`，那就不用多想了，直接上二分查找。

### 二分查找该怎么写

我们用两个指针维护当前的查找区间：

- `left` 指向左边界
- `right` 指向右边界

每次取中间位置 `mid`，比较 `nums[mid]` 和 `target`：

- 如果相等，说明已经找到，直接返回 `mid`
- 如果 `nums[mid] < target`，说明目标值只可能在右半边，更新 `left = mid + 1`
- 如果 `nums[mid] > target`，说明目标值只可能在左半边，更新 `right = mid - 1`

这部分本身就是最标准的二分写法，没有什么绕的地方。

真正容易让人停一下的地方只有一个：**如果最后没找到，为什么直接返回 `left` 就对了？**

### 为什么最后返回 `left`

这个结论不是硬记的，它其实是二分过程自然推出来的。

在整个查找过程中，我们一直在缩小区间：

- `left` 左边的元素，一定都小于 `target`
- `right` 右边的元素，一定都大于 `target`

当循环结束时，条件是 `left > right`。  
这意味着原来的查找区间已经被压缩到空了，但边界信息还在：

- `right` 停在最后一个小于 `target` 的位置
- `left` 停在第一个大于等于 `target` 的位置

而这个位置，恰好就是题目要求的答案。

所以这题找不到目标值时，不需要额外讨论“插入到哪儿”，因为二分结束后，`left` 本身就已经站在那个位置上了。

### 用例子走一遍就更清楚了

比如：

```text
nums = [1,3,5,6], target = 2
```

初始时：

- `left = 0`
- `right = 3`

先取中点：

- `mid = 1`
- `nums[mid] = 3`

因为 `3 > 2`，说明答案只能在左边，于是更新：

- `right = mid - 1 = 0`

现在区间变成 `[0, 0]`。

再取中点：

- `mid = 0`
- `nums[mid] = 1`

因为 `1 < 2`，说明答案只能在右边，于是更新：

- `left = mid + 1 = 1`

这时循环结束，因为 `left = 1`，`right = 0`，已经满足 `left > right`。

最后返回 `left = 1`，也就是 `2` 应该插入的位置。

你会发现，这个结果不是额外算出来的，而是二分过程自己“收缩”出来的。

### 代码

```java
public int searchInsert(int[] nums, int target) {
    int left = 0;
    int right = nums.length - 1;

    while (left <= right) {
        int mid = left + (right - left) / 2;

        if (nums[mid] == target) {
            return mid;
        } else if (nums[mid] < target) {
            left = mid + 1;
        } else {
            right = mid - 1;
        }
    }

    return left;
}
```

### 复杂度分析

时间复杂度是 $O(\log n)$。  
因为每一次比较后，搜索区间都会缩小一半。

空间复杂度是 $O(1)$。  
整个过程中只用了几个额外变量。

### 这题要记住的点

这题本身不难，关键是要把思路想顺。

很多人一开始会把它拆成两问：

1. 找到目标值怎么办
2. 找不到目标值怎么办

但其实没必要这么分。

更自然的理解方式是：

**无论目标值存不存在，我们始终都在找第一个大于等于 `target` 的位置。**

一旦这样想，这题就会变得很统一：

- 找到了，返回它的下标
- 没找到，返回插入位置
- 而这两个结果，本质上就是同一个位置

以后再遇到这类“有序数组 + 查找边界位置”的题，直接往这个方向想，很多题都会顺手很多。

### 完整代码

```java
import java.util.Arrays;

public class Main {

    public static int searchInsert(int[] nums, int target) {
        int left = 0;
        int right = nums.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return left;
    }

    public static void main(String[] args) {
        int[] nums1 = {1, 3, 5, 6};
        int target1 = 5;
        System.out.println("示例 1 输入: nums = " + Arrays.toString(nums1) + ", target = " + target1);
        System.out.println("示例 1 输出: " + searchInsert(nums1, target1));
        System.out.println("-----------------------------");

        int[] nums2 = {1, 3, 5, 6};
        int target2 = 2;
        System.out.println("示例 2 输入: nums = " + Arrays.toString(nums2) + ", target = " + target2);
        System.out.println("示例 2 输出: " + searchInsert(nums2, target2));
        System.out.println("-----------------------------");

        int[] nums3 = {1, 3, 5, 6};
        int target3 = 7;
        System.out.println("示例 3 输入: nums = " + Arrays.toString(nums3) + ", target = " + target3);
        System.out.println("示例 3 输出: " + searchInsert(nums3, target3));
        System.out.println("-----------------------------");

        int[] nums4 = {1, 3, 5, 6};
        int target4 = 0;
        System.out.println("示例 4 输入: nums = " + Arrays.toString(nums4) + ", target = " + target4);
        System.out.println("示例 4 输出: " + searchInsert(nums4, target4));
    }
}
```
