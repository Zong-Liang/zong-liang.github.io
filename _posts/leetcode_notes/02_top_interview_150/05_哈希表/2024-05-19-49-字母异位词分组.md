## 题目

**描述：**

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词** 是由相同字母按不同顺序排列形成的字符串。

**示例 1:**

- **输入:** `strs = ["eat","tea","tan","ate","nat","bat"]`
- **输出:** `[["bat"],["nat","tan"],["ate","eat","tea"]]`

**示例 2:**

- **输入:** `strs = [""]`
- **输出:** `[[""]]`

**示例 3:**

- **输入:** `strs = ["a"]`
- **输出:** `[["a"]]`

**提示:**

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅包含小写字母

## 题解

这题如果换个说法，其实是在做“归类”：把那些字符组成相同、只是顺序不同的字符串放到同一组里。所以它的关键并不是两两比较谁和谁相等，而是先为每个字符串造出一个统一的“标准表示”，然后按这个标准表示去分桶。

对于异位词来说，最自然的标准表示就是排序后的字符串。比如 `"eat"`、`"tea"`、`"ate"`，原始顺序都不一样，但排完序以后都会变成 `"aet"`。一旦有了这个标准形式，分组就很顺了：遍历每个字符串，把它排序后的结果当成 key，原字符串本身当成 value 加进对应列表里。最后哈希表里每个 value 列表，就是一组异位词。

这个思路之所以稳，是因为排序后的 key 会同时保留字符种类和字符频次。像 `"abb"` 和 `"aab"` 这种字符串，虽然字符种类看起来都只有 `a`、`b`，但排序后分别是 `"abb"` 和 `"aab"`，不会被错误地放进同一组。所以这里的 key 不是随便构造的，而是确实能把“本质相同”和“本质不同”分开。

拿 `strs = ["eat", "tea", "tan", "ate", "nat", "bat"]` 来看，`eat/tea/ate` 的 key 都是 `aet`，`tan/nat` 的 key 都是 `ant`，`bat` 的 key 是 `abt`。于是哈希表里自然会形成三组，这比拿每个字符串去和其他字符串一一比较要顺得多。

当然，这题也可以用字符频次数组去构造 key，比如把每个字符串压成 `#1#0#0...` 这种编码形式。不过从直观程度和教学表达来看，排序法更容易一眼看明白：异位词排序后相同，所以可以直接拿排序结果来分组。

代码如下：

```java
public List<List<String>> groupAnagrams(String[] strs) {
    if (strs == null || strs.length == 0) {
        return new ArrayList<>();
    }

    Map<String, List<String>> map = new HashMap<>();

    for (String s : strs) {
        char[] chars = s.toCharArray();
        Arrays.sort(chars);
        String key = new String(chars);

        if (!map.containsKey(key)) {
            map.put(key, new ArrayList<>());
        }

        map.get(key).add(s);
    }

    return new ArrayList<>(map.values());
}
```

假设一共有 `N` 个字符串，每个字符串平均长度是 `K`，那么时间复杂度是 `O(N * K log K)`，因为每个字符串都要排序一次；空间复杂度主要来自哈希表中保存的分组结果。

这题比较适合记住的，不只是“哈希表 + 排序”，而是那个更通用的拆法：当题目在要求你把“本质相同、表现形式不同”的对象归类时，先想能不能为每个对象构造一个统一的标准表示。有了这个表示，后面的分组往往就只是普通哈希表操作了。

完整代码如下：

```java
import java.util.*;

public class Main {

    public static List<List<String>> groupAnagrams(String[] strs) {
        if (strs == null || strs.length == 0) {
            return new ArrayList<>();
        }

        Map<String, List<String>> map = new HashMap<>();

        for (String s : strs) {
            char[] chars = s.toCharArray();
            Arrays.sort(chars);
            String key = new String(chars);

            if (!map.containsKey(key)) {
                map.put(key, new ArrayList<>());
            }

            map.get(key).add(s);
        }

        return new ArrayList<>(map.values());
    }

    public static void main(String[] args) {
        String[] strs1 = {"eat", "tea", "tan", "ate", "nat", "bat"};
        System.out.println("示例 1 输入: " + Arrays.toString(strs1));
        System.out.println("示例 1 输出: " + groupAnagrams(strs1));
        System.out.println("-----------------------------");

        String[] strs2 = {""};
        System.out.println("示例 2 输入: " + Arrays.toString(strs2));
        System.out.println("示例 2 输出: " + groupAnagrams(strs2));
        System.out.println("-----------------------------");

        String[] strs3 = {"a"};
        System.out.println("示例 3 输入: " + Arrays.toString(strs3));
        System.out.println("示例 3 输出: " + groupAnagrams(strs3));
    }
}
```
