## 题目

**描述：**

给你两个字符串：`ransomNote` 和 `magazine` ，判断 `ransomNote` 能不能由 `magazine` 里面的字符构成。

如果可以，返回 `true` ；否则返回 `false` 。

`magazine` 中的每个字符只能在 `ransomNote` 中使用一次。

**示例 1:**

- **输入:** `ransomNote = "a"`, `magazine = "b"`
- **输出:** `false`

**示例 2:**

- **输入:** `ransomNote = "aa"`, `magazine = "ab"`
- **输出:** `false`

**示例 3:**

- **输入:** `ransomNote = "aa"`, `magazine = "aab"`
- **输出:** `true`

**提示:**

- `1 <= ransomNote.length, magazine.length <= 10^5`
- `ransomNote` 和 `magazine` 由小写英文字母组成

## 题解

这题表面上是在比较两个字符串，实际上更像是在做一次很简单的“库存够不够”判断。`magazine` 里每个字符都是库存，`ransomNote` 里每个字符都是需求。题目问的不是某个字符在不在，而是每种字符的数量，能不能把赎金信里的需求全部覆盖掉。

这一点很容易用一个小例子看出来。`ransomNote = "aa"`，`magazine = "ab"` 时，虽然 `magazine` 里确实有字符 `a`，但只有一个，显然不够拼出两个 `a`。所以这题不能只判断“出现过没有”，必须比较频次。

最顺手的写法就是两步：先统计 `magazine` 里每个字符出现了多少次，再去遍历 `ransomNote`，每遇到一个字符，就把对应库存减一。如果某次减完以后变成负数，说明这个字符已经不够用了，可以立刻返回 `false`。整个过程并不复杂，关键只是先把题目翻译成“补货”和“消耗”这两个动作。

因为题目已经限定字符串只由小写字母组成，所以这里没必要用 `HashMap`，直接开一个长度为 `26` 的数组就够了。`cnt[0]` 对应 `'a'`，`cnt[25]` 对应 `'z'`，字符到下标的映射就是 `c - 'a'`。在这种字符集固定的场景下，数组会比哈希表更短、更直接。

代码如下：

```java
public boolean canConstruct(String ransomNote, String magazine) {
    if (ransomNote.length() > magazine.length()) {
        return false;
    }

    int[] cnt = new int[26];

    for (char c : magazine.toCharArray()) {
        cnt[c - 'a']++;
    }

    for (char c : ransomNote.toCharArray()) {
        cnt[c - 'a']--;
        if (cnt[c - 'a'] < 0) {
            return false;
        }
    }

    return true;
}
```

时间复杂度是 `O(m + n)`，其中 `m` 是 `magazine` 的长度，`n` 是 `ransomNote` 的长度；空间复杂度是 `O(1)`，因为只用了一个固定长度为 `26` 的数组。

这题比较适合记住的，不是“赎金信”这个题面，而是那个判断模型：当题目本质上是在问“一个集合的频次能不能覆盖另一个集合的频次”时，优先考虑做计数统计。字符范围固定时用数组，字符范围不固定时再换成哈希表。

完整代码如下：

```java
public class Main {

    public static boolean canConstruct(String ransomNote, String magazine) {
        if (ransomNote.length() > magazine.length()) {
            return false;
        }

        int[] cnt = new int[26];

        for (char c : magazine.toCharArray()) {
            cnt[c - 'a']++;
        }

        for (char c : ransomNote.toCharArray()) {
            cnt[c - 'a']--;
            if (cnt[c - 'a'] < 0) {
                return false;
            }
        }

        return true;
    }

    public static void main(String[] args) {
        String ransomNote1 = "a";
        String magazine1 = "b";
        System.out.println("示例 1 输入: ransomNote = \"" + ransomNote1 + "\", magazine = \"" + magazine1 + "\"");
        System.out.println("示例 1 输出: " + canConstruct(ransomNote1, magazine1));
        System.out.println("-----------------------------");

        String ransomNote2 = "aa";
        String magazine2 = "ab";
        System.out.println("示例 2 输入: ransomNote = \"" + ransomNote2 + "\", magazine = \"" + magazine2 + "\"");
        System.out.println("示例 2 输出: " + canConstruct(ransomNote2, magazine2));
        System.out.println("-----------------------------");

        String ransomNote3 = "aa";
        String magazine3 = "aab";
        System.out.println("示例 3 输入: ransomNote = \"" + ransomNote3 + "\", magazine = \"" + magazine3 + "\"");
        System.out.println("示例 3 输出: " + canConstruct(ransomNote3, magazine3));
    }
}
```
