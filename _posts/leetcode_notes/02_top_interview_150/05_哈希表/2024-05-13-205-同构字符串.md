## 题目

**描述：**

给定两个字符串 `s` 和 `t` ，判断它们是否是同构的。

如果 `s` 中的字符可以按某种映射关系替换得到 `t` ，那么这两个字符串是同构的。

每个出现的字符都应当映射到另一个字符，同时不改变字符的顺序。不同字符不能映射到同一个字符上，相同字符只能映射到同一个字符上，字符可以映射到自己本身。

**示例 1:**

- **输入:** `s = "egg"`, `t = "add"`
- **输出:** `true`

**示例 2:**

- **输入:** `s = "foo"`, `t = "bar"`
- **输出:** `false`

**示例 3:**

- **输入:** `s = "paper"`, `t = "title"`
- **输出:** `true`

**提示:**

- `1 <= s.length <= 5 * 10^4`
- `t.length == s.length`
- `s` 和 `t` 由任意有效的 ASCII 字符组成

## 题解

这题名字看起来有点抽象，其实问的是一件很具体的事：`s` 里的字符，能不能和 `t` 里的字符建立起一种稳定的一一对应关系。所谓“稳定”，就是说同一个字符每次都得映射到同一个目标字符；所谓“一一对应”，就是说不同字符又不能挤到同一个目标字符上。

所以这题不能只检查单向映射。像 `s = "ab"`、`t = "aa"` 这种例子，如果你只看 `s -> t`，会得到 `a -> a`、`b -> a`，看起来好像都能记下来；但它明显不合法，因为两个不同字符映射到了同一个字符上。问题的关键不在“能不能对应上一次”，而在“整个对应关系会不会乱”。

这题有个很巧的观察：如果两个字符串是同构的，那么在同一个位置上的两个字符，它们之前出现的位置模式应该完全一致。比如 `"egg"` 和 `"add"`，第一个位置都是第一次出现，第二个位置也都是第一次出现，第三个位置里 `g` 和 `d` 都是在下标 `1` 出现过一次之后再次出现，所以模式是同步的。反过来，`"foo"` 和 `"bar"` 到第三个位置时就对不上了，因为 `o` 是重复出现，而 `r` 是第一次出现。

顺着这个观察，就可以不显式维护 `a -> x` 这样的映射，而是分别记录每个字符上一次出现的位置。遍历到位置 `i` 时，如果 `s[i]` 和 `t[i]` 之前出现的位置不相同，就说明这两个字符串的重复模式已经不一致，可以直接返回 `false`。题目字符范围是有效 ASCII，所以这里直接开两个长度为 `256` 的数组就够了。

数组里通常存 `i + 1`，而不是直接存 `i`。原因是数组初始值默认是 `0`，如果直接存下标，就分不清“这个字符从来没出现过”和“它上一次出现在下标 `0`”这两种情况。存成 `i + 1` 以后，`0` 就专门表示“从未出现过”。

代码如下：

```java
public boolean isIsomorphic(String s, String t) {
    int[] mapS = new int[256];
    int[] mapT = new int[256];

    for (int i = 0; i < s.length(); i++) {
        char c1 = s.charAt(i);
        char c2 = t.charAt(i);

        if (mapS[c1] != mapT[c2]) {
            return false;
        }

        mapS[c1] = i + 1;
        mapT[c2] = i + 1;
    }

    return true;
}
```

时间复杂度是 `O(n)`，因为只遍历了一遍字符串；空间复杂度是 `O(1)`，因为数组大小固定。

这题很值得顺手记住一个判断角度：有些题看似在比较“值能不能映射”，其实更适合比较“出现模式是不是一致”。一旦能把值本身抽掉，只盯着结构，代码反而会更短也更稳。

完整代码如下：

```java
public class Main {

    public static boolean isIsomorphic(String s, String t) {
        if (s.length() != t.length()) {
            return false;
        }

        int[] mapS = new int[256];
        int[] mapT = new int[256];

        for (int i = 0; i < s.length(); i++) {
            char c1 = s.charAt(i);
            char c2 = t.charAt(i);

            if (mapS[c1] != mapT[c2]) {
                return false;
            }

            mapS[c1] = i + 1;
            mapT[c2] = i + 1;
        }

        return true;
    }

    public static void main(String[] args) {
        String s1 = "egg";
        String t1 = "add";
        System.out.println("示例 1 输入: s = \"" + s1 + "\", t = \"" + t1 + "\"");
        System.out.println("示例 1 输出: " + isIsomorphic(s1, t1));
        System.out.println("-----------------------------");

        String s2 = "foo";
        String t2 = "bar";
        System.out.println("示例 2 输入: s = \"" + s2 + "\", t = \"" + t2 + "\"");
        System.out.println("示例 2 输出: " + isIsomorphic(s2, t2));
        System.out.println("-----------------------------");

        String s3 = "paper";
        String t3 = "title";
        System.out.println("示例 3 输入: s = \"" + s3 + "\", t = \"" + t3 + "\"");
        System.out.println("示例 3 输出: " + isIsomorphic(s3, t3));
    }
}
```
