## 题目

**描述：**

给定两个字符串 `s` 和 `t` ，编写一个函数来判断 `t` 是否是 `s` 的字母异位词。

**字母异位词** 是由相同字母按不同顺序排列形成的字符串。

**示例 1:**

- **输入:** `s = "anagram"`, `t = "nagaram"`
- **输出:** `true`

**示例 2:**

- **输入:** `s = "rat"`, `t = "car"`
- **输出:** `false`

**提示:**

- `1 <= s.length, t.length <= 5 * 10^4`
- `s` 和 `t` 仅包含小写字母

**进阶:** 如果输入字符串包含 unicode 字符怎么办？你能否调整你的解法来应对这种情况？

## 题解

这题如果先不管复杂度，最直观的办法就是把两个字符串都排序，再比较排序结果是否相同。这个思路当然能做，因为异位词只是在顺序上打乱了，排序以后自然会被拉回同一种样子。但排序的代价是 `O(n log n)`，而这题其实有个更贴题的角度：异位词比较的不是顺序，而是组成它的每个字符出现了多少次。

一旦换成频次视角，事情就简单很多了。对两个字符串来说，只要每个字母出现的次数完全一致，它们就是异位词；只要有任何一个字母的次数不同，就不是。题目原始条件里已经限定只包含小写字母，所以这里最适合的工具不是哈希表，而是一个长度为 `26` 的整型数组。

做法也很直接。先判断长度，如果长度不同，直接返回 `false`。然后遍历两个字符串的同一位置：`s` 当前字符对应的计数加一，`t` 当前字符对应的计数减一。最后如果数组里所有位置都回到了 `0`，就说明两个字符串在每个字符上的贡献刚好抵消完，频次完全一致。

这种写法的好处是很干净：既不需要额外排序，也不需要开两张表分别统计，整个数组本身就同时在记录“谁多了多少”和“谁少了多少”。

代码如下：

```java
public boolean isAnagram(String s, String t) {
    if (s.length() != t.length()) {
        return false;
    }

    int[] table = new int[26];

    for (int i = 0; i < s.length(); i++) {
        table[s.charAt(i) - 'a']++;
        table[t.charAt(i) - 'a']--;
    }

    for (int count : table) {
        if (count != 0) {
            return false;
        }
    }

    return true;
}
```

在题目原始条件下，时间复杂度是 `O(n)`，空间复杂度是 `O(1)`。

题目最后还追问了 Unicode 的情况，这时候就不能再直接用长度为 `26` 的数组了。更准确的做法，是按 Unicode 码点遍历，再用 `Map<Integer, Integer>` 统计频次。这里要注意，Java 的 `char` 只是 UTF-16 的代码单元，并不总能完整表示一个字符，所以如果真的想兼容完整 Unicode，最好直接按 `codePoints()` 处理。

完整代码如下：

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    // 适用于题目原始条件：仅包含小写字母
    public static boolean isAnagram(String s, String t) {
        if (s.length() != t.length()) {
            return false;
        }

        int[] table = new int[26];

        for (int i = 0; i < s.length(); i++) {
            table[s.charAt(i) - 'a']++;
            table[t.charAt(i) - 'a']--;
        }

        for (int count : table) {
            if (count != 0) {
                return false;
            }
        }

        return true;
    }

    // 适用于完整 Unicode 字符
    public static boolean isAnagramUnicode(String s, String t) {
        if (s.codePointCount(0, s.length()) != t.codePointCount(0, t.length())) {
            return false;
        }

        Map<Integer, Integer> map = new HashMap<>();

        s.codePoints().forEach(cp -> map.put(cp, map.getOrDefault(cp, 0) + 1));

        for (int cp : t.codePoints().toArray()) {
            int count = map.getOrDefault(cp, 0);
            if (count == 0) {
                return false;
            }
            map.put(cp, count - 1);
        }

        return true;
    }

    public static void main(String[] args) {
        String s1 = "anagram";
        String t1 = "nagaram";
        System.out.println("示例 1 输入: s = \"" + s1 + "\", t = \"" + t1 + "\"");
        System.out.println("示例 1 输出: " + isAnagram(s1, t1));
        System.out.println("-----------------------------");

        String s2 = "rat";
        String t2 = "car";
        System.out.println("示例 2 输入: s = \"" + s2 + "\", t = \"" + t2 + "\"");
        System.out.println("示例 2 输出: " + isAnagram(s2, t2));
        System.out.println("-----------------------------");

        String s3 = "你好世界";
        String t3 = "世界你好";
        System.out.println("进阶示例 输入: s = \"" + s3 + "\", t = \"" + t3 + "\"");
        System.out.println("进阶示例 输出: " + isAnagramUnicode(s3, t3));
        System.out.println("-----------------------------");

        String s4 = "a😊b";
        String t4 = "😊ab";
        System.out.println("补充示例 输入: s = \"" + s4 + "\", t = \"" + t4 + "\"");
        System.out.println("补充示例 输出: " + isAnagramUnicode(s4, t4));
    }
}
```
