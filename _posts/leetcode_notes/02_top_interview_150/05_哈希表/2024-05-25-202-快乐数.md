## 题目

**描述：**

编写一个算法来判断一个数 `n` 是不是快乐数。

**「快乐数」** 定义为：

- 对于一个正整数，每一次将该数替换为它每个位置上的数字的平方和。
- 然后重复这个过程直到这个数变为 1，也可能是 **无限循环** 但始终变不到 1。
- 如果这个过程 **结果为 1**，那么这个数就是快乐数。

如果 `n` 是快乐数就返回 `true` ；不是，则返回 `false` 。

**示例 1:**

- **输入:** `n = 19`
- **输出:** `true`
- **解释:**
  - 1² + 9² = 82
  - 8² + 2² = 68
  - 6² + 8² = 100
  - 1² + 0² + 0² = 1

**示例 2:**

- **输入:** `n = 2`
- **输出:** `false`

**提示:**

- `1 <= n <= 2^31 - 1`

## 题解

这题看起来像一道纯数学模拟题，但如果只是一轮一轮地往下算，很容易漏掉一个关键问题：这个过程到底什么时候会停。一个数不断替换成“各位平方和”以后，只会有两种结局——要么最终到 `1`，要么陷进某个循环里再也出不来。所以这题的难点不是怎么算下一步，而是怎么判断有没有成环。

先把状态转移看清楚会很有帮助。当前数字 `n` 会唯一地变成下一个数字 `getNext(n)`，其中 `getNext` 的规则就是把每一位拆出来平方再求和。只要这个“当前状态唯一指向下一个状态”的结构成立，整件事就很像一条隐式链表：如果最后走到 `1`，说明这是快乐数；如果走着走着回到了之前某个状态，说明进入了环，就不是快乐数。

最直观的判环办法，是用一个集合把出现过的状态记下来。每算出一个新数，就看它之前有没有出现过；出现过就说明成环了，没有就继续。这种写法很好理解。但这题还有一个更省空间的版本，就是 Floyd 判圈法，也就是快慢指针。

用快慢指针时，`slow` 每次走一步，`fast` 每次走两步。如果这个状态链最后会落到 `1`，那么快指针总会先到 `1`；如果不会到 `1`，它们迟早会在环里相遇。这里虽然看不到一条真实的链表，但状态转移关系已经足够让判圈思路成立了。

拿 `19` 来看，变化序列会是 `19 -> 82 -> 68 -> 100 -> 1`，最后能到 `1`，所以它是快乐数。而像 `2` 这样的数，会反复掉进某个固定循环里，快慢指针最终会在环中相遇。

代码如下：

```java
public boolean isHappy(int n) {
    int slow = n;
    int fast = getNext(n);

    while (fast != 1 && slow != fast) {
        slow = getNext(slow);
        fast = getNext(getNext(fast));
    }

    return fast == 1;
}

private int getNext(int n) {
    int sum = 0;
    while (n > 0) {
        int digit = n % 10;
        n /= 10;
        sum += digit * digit;
    }
    return sum;
}
```

单次计算 `getNext` 需要按位拆数字，所以时间主要花在状态转移上；空间复杂度方面，快慢指针写法是 `O(1)`，这也是它比哈希集合法更常被当作标准答案的原因。

这题比较值得带走的，不是“快乐数”这个名字，而是那种抽象方式：只要问题能被看成“当前状态唯一指向下一个状态”，就可以问自己它最后是终止，还是成环。这个视角在很多看起来不像链表的题里都很有用。

完整代码如下：

```java
import java.util.HashSet;
import java.util.Set;

public class Main {

    private static int getNext(int n) {
        int sum = 0;
        while (n > 0) {
            int digit = n % 10;
            n /= 10;
            sum += digit * digit;
        }
        return sum;
    }

    public static boolean isHappy(int n) {
        int slow = n;
        int fast = getNext(n);

        while (fast != 1 && slow != fast) {
            slow = getNext(slow);
            fast = getNext(getNext(fast));
        }

        return fast == 1;
    }

    public static boolean isHappySet(int n) {
        Set<Integer> seen = new HashSet<>();

        while (n != 1 && !seen.contains(n)) {
            seen.add(n);
            n = getNext(n);
        }

        return n == 1;
    }

    public static void main(String[] args) {
        int n1 = 19;
        System.out.println("示例 1 输入: n = " + n1);
        System.out.println("示例 1 输出: " + isHappy(n1));
        System.out.println("-----------------------------");

        int n2 = 2;
        System.out.println("示例 2 输入: n = " + n2);
        System.out.println("示例 2 输出: " + isHappy(n2));
    }
}
```
