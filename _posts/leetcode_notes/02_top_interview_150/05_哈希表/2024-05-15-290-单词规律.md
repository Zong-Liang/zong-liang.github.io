## 题目

**描述：**

给定一种规律 `pattern` 和一个字符串 `s` ，判断 `s` 是否遵循相同的规律。

这里的 **遵循** 指完全匹配，例如，`pattern` 里的每个字母和字符串 `s` 中的每个非空单词之间存在着双向连接的对应规律。具体来说：

- `pattern` 中的每个字母都 **恰好** 映射到 `s` 中的一个唯一单词。
- `s` 中的每个唯一单词都 **恰好** 映射到 `pattern` 中的一个字母。
- 没有两个字母映射到同一个单词，也没有两个单词映射到同一个字母。

**示例 1:**

- **输入:** `pattern = "abba"`, `s = "dog cat cat dog"`
- **输出:** `true`

**示例 2:**

- **输入:** `pattern = "abba"`, `s = "dog cat cat fish"`
- **输出:** `false`

**示例 3:**

- **输入:** `pattern = "aaaa"`, `s = "dog cat cat dog"`
- **输出:** `false`

**提示:**

- `1 <= pattern.length <= 300`
- `pattern` 只包含小写英文字母
- `1 <= s.length <= 3000`
- `s` 只包含小写英文字母和 `' '`
- `s` **不包含** 任何前导或尾随空格
- `s` 中每个单词都被 **单个空格** 分隔

## 题解

这题和上一题“同构字符串”其实是同一个骨架，只不过把其中一边从字符换成了单词。它要求的仍然不是“差不多能对上”，而是能不能建立起一套稳定的双向映射：`pattern` 里的同一个字符必须始终对应同一个单词，而 `s` 里的同一个单词也不能被两个不同字符占用。

所以第一步很自然：先把 `s` 按空格拆成单词数组。如果单词数量和 `pattern` 长度都对不上，那后面根本不存在逐位匹配的可能，直接返回 `false` 就行。长度这一关看着简单，但最好早点做掉，不然后面的映射逻辑会很别扭。

主体还是双向检查。遍历到位置 `i` 时，拿到当前字符 `pattern[i]` 和当前单词 `words[i]`。如果这个字符以前出现过，就必须确认它之前映射的还是这个单词；如果这个字符是新的，那也得先确认这个单词没被别的字符占用。只有两边都不冲突，才能把这组新关系记进去。

这里最容易漏掉的是“新字符 + 老单词”这种冲突。比如 `pattern = "ab"`，`s = "dog dog"`，第二个字符 `b` 虽然是新的，但单词 `dog` 已经被 `a` 用过了，所以这不是新映射，而是映射冲突。也正因为如此，这题不能只维护一张 `char -> word` 的表，还必须同步检查 `word -> char`。

代码如下：

```java
public boolean wordPattern(String pattern, String s) {
    String[] words = s.split(" ");

    if (pattern.length() != words.length) {
        return false;
    }

    Map<Character, String> charToWord = new HashMap<>();
    Map<String, Character> wordToChar = new HashMap<>();

    for (int i = 0; i < pattern.length(); i++) {
        char c = pattern.charAt(i);
        String w = words[i];

        if (charToWord.containsKey(c)) {
            if (!charToWord.get(c).equals(w)) {
                return false;
            }
        } else {
            if (wordToChar.containsKey(w)) {
                return false;
            }

            charToWord.put(c, w);
            wordToChar.put(w, c);
        }
    }

    return true;
}
```

时间复杂度是 `O(N + M)`，其中 `N` 是 `pattern` 的长度，`M` 是字符串 `s` 的长度；空间复杂度主要来自拆分后的单词数组和两张哈希表。

这题比较适合记住的，不是“单词规律”这个名字，而是那个更一般的判断：当题目要比较两个序列之间能不能形成一一对应关系时，别只看单向能不能对上，还要防止另一边被重复占用。很多模式匹配题，难点都在这里。

完整代码如下：

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    public static boolean wordPattern(String pattern, String s) {
        String[] words = s.split(" ");

        if (pattern.length() != words.length) {
            return false;
        }

        Map<Character, String> charToWord = new HashMap<>();
        Map<String, Character> wordToChar = new HashMap<>();

        for (int i = 0; i < pattern.length(); i++) {
            char c = pattern.charAt(i);
            String w = words[i];

            if (charToWord.containsKey(c)) {
                if (!charToWord.get(c).equals(w)) {
                    return false;
                }
            } else {
                if (wordToChar.containsKey(w)) {
                    return false;
                }

                charToWord.put(c, w);
                wordToChar.put(w, c);
            }
        }

        return true;
    }

    public static void main(String[] args) {
        String pattern1 = "abba";
        String s1 = "dog cat cat dog";
        System.out.println("示例 1 输入: pattern = \"" + pattern1 + "\", s = \"" + s1 + "\"");
        System.out.println("示例 1 输出: " + wordPattern(pattern1, s1));
        System.out.println("-----------------------------");

        String pattern2 = "abba";
        String s2 = "dog cat cat fish";
        System.out.println("示例 2 输入: pattern = \"" + pattern2 + "\", s = \"" + s2 + "\"");
        System.out.println("示例 2 输出: " + wordPattern(pattern2, s2));
        System.out.println("-----------------------------");

        String pattern3 = "aaaa";
        String s3 = "dog cat cat dog";
        System.out.println("示例 3 输入: pattern = \"" + pattern3 + "\", s = \"" + s3 + "\"");
        System.out.println("示例 3 输出: " + wordPattern(pattern3, s3));
        System.out.println("-----------------------------");

        String pattern4 = "aaa";
        String s4 = "aa aa aa aa";
        System.out.println("额外示例 输入: pattern = \"" + pattern4 + "\", s = \"" + s4 + "\"");
        System.out.println("额外示例 输出: " + wordPattern(pattern4, s4));
    }
}
```
