## 题目

**描述：**

给你一个字符串表达式 `s` ，请你实现一个基本计算器来计算并返回它的值。

注意: 不允许使用任何将字符串作为数学表达式计算的内置函数，比如 `eval()` 。

**示例 1:**

- **输入:** `s = "1 + 1"`
- **输出:** 2

**示例 2:**

- **输入:** `s = " 2-1 + 2 "`
- **输出:** 3

**示例 3:**

- **输入:** `s = "(1+(4+5+2)-3)+(6+8)"`
- **输出:** 23

**提示:**

- `1 <= s.length <= 3 * 10^5`
- `s` 由数字、`'+'`、`'-'`、`'('`、`')'`、和 `' '` 组成
- `s` 表示一个有效的表达式
- `'+'` 不能用作一元运算 (例如， `"+1"` 和 `"+(2 + 3)"` 无效)
- `'-'` 可以用作一元运算 (即 `"-1"` 和 `"-(2 + 3)"` 是有效的)
- 输入中不存在两个连续的操作符
- 每个数字和运行的计算将适合于一个有符号的 32 位整数

## 题解

这题难的不是加减法本身，而是括号会不断切换当前这一层的计算环境。如果表达式里只有数字、加号和减号，没有括号，那其实很好处理：顺着扫过去，维护一个当前结果 `res` 和一个当前符号 `sign` 就够了。遇到数字时做 `res += sign * num`，遇到 `+` 就把 `sign` 改成 `1`，遇到 `-` 就把 `sign` 改成 `-1`。因为题目里没有乘除，也就不存在更高优先级的问题。

真正让这题复杂起来的，是括号。括号一出现，等于在原表达式里临时开出了一层新的子表达式；而这层子表达式算完以后，又得带着括号前面的符号回到外层继续往下算。所以这题要解决的关键其实是：进入括号之前，怎么把外层状态存起来；括号结束以后，又怎么把里面的结果接回到外面。

这里最适合保存的状态有两个：当前层已经累计出来的结果 `res`，以及当前括号前面的符号 `sign`。每次遇到左括号 `(`，就先把这两个值压进栈里，然后把当前环境重置成一层新的计算：`res = 0`，`sign = 1`。这一步本质上就是“先把外层暂停，进去把括号里面单独算完”。

等遇到右括号 `)` 时，当前 `res` 已经表示括号内部的结果了。这时从栈里先弹出括号前的符号 `prevSign`，再弹出括号前已经算到一半的结果 `prevRes`，然后用 `prevRes + prevSign * res` 把内层结果接回外层。比如表达式 `1 - (2 + 3)`，括号里算完以后 `res = 5`，而外层保存下来的状态是 `prevRes = 1`、`prevSign = -1`，接回去以后就是 `1 + (-1) * 5 = -4`。

这也是为什么这题虽然叫“计算器”，其实并不需要操作符优先级栈，也不需要构造表达式树。题目只涉及加减和括号，所以线性扫描加一个状态栈就够了。唯一别忘记的细节，是数字可能不止一位，因此遇到数字字符时，要把后面连续的数字一起读出来，不能把 `123` 错拆成三个独立数字。

拿 `"(1+(4+5+2)-3)+(6+8)"` 来看，整个过程其实就是在不同括号层级之间不断切换和回退。每次进括号，就把上一层的 `res` 和 `sign` 存起来；每次出括号，就把当前层的结果乘上进入时的符号，再加回到外层结果里。只要这个状态切换关系想清楚了，代码本身并不绕。

代码如下：

```java
public int calculate(String s) {
    Deque<Integer> stack = new ArrayDeque<>();
    int sign = 1;
    int res = 0;
    int n = s.length();

    for (int i = 0; i < n; i++) {
        char ch = s.charAt(i);

        if (Character.isDigit(ch)) {
            int num = 0;
            while (i < n && Character.isDigit(s.charAt(i))) {
                num = num * 10 + (s.charAt(i) - '0');
                i++;
            }
            res += sign * num;
            i--;
        } else if (ch == '+') {
            sign = 1;
        } else if (ch == '-') {
            sign = -1;
        } else if (ch == '(') {
            stack.push(res);
            stack.push(sign);
            res = 0;
            sign = 1;
        } else if (ch == ')') {
            int prevSign = stack.pop();
            int prevRes = stack.pop();
            res = prevRes + prevSign * res;
        }
    }

    return res;
}
```

时间复杂度是 `O(n)`，因为整个字符串只会被顺着扫描一遍；空间复杂度是 `O(n)`，最坏情况下如果括号嵌套很深，栈里需要保存很多层状态。

这题更值得记住的，是“括号在切换计算环境”这个视角。只要题目可以被拆成一层一层的子表达式，而且某一层结束后还要回到上一层继续算，就可以考虑把当前层状态压栈保存，等子问题结束后再恢复回来。本题里被保存的状态，就是外层的结果和符号。

完整代码如下：

```java
import java.util.ArrayDeque;
import java.util.Deque;
public class Main {
    public static int calculate(String s) {
        Deque<Integer> stack = new ArrayDeque<>();
        int sign = 1;
        int res = 0;
        int length = s.length();
        for (int i = 0; i < length; i++) {
            char ch = s.charAt(i);
            if (Character.isDigit(ch)) {
                int num = 0;
                while (i < length && Character.isDigit(s.charAt(i))) {
                    num = num * 10 + (s.charAt(i) - '0');
                    i++;
                }
                res += sign * num;
                i--;
            } else if (ch == '+') {
                sign = 1;
            } else if (ch == '-') {
                sign = -1;
            } else if (ch == '(') {
                stack.push(res);
                stack.push(sign);
                res = 0;
                sign = 1;
            } else if (ch == ')') {
                int prevSign = stack.pop();
                int prevRes = stack.pop();
                res = prevRes + prevSign * res;
            }
        }
        return res;
    }
    public static void main(String[] args) {
        String s1 = "1 + 1";
        System.out.println("示例 1 输入: \"" + s1 + "\"");
        int res1 = calculate(s1);
        System.out.println("示例 1 输出: " + res1);
        System.out.println("-----------------------------");
        String s2 = " 2-1 + 2 ";
        System.out.println("示例 2 输入: \"" + s2 + "\"");
        int res2 = calculate(s2);
        System.out.println("示例 2 输出: " + res2);
        System.out.println("-----------------------------");
        String s3 = "(1+(4+5+2)-3)+(6+8)";
        System.out.println("示例 3 输入: \"" + s3 + "\"");
        int res3 = calculate(s3);
        System.out.println("示例 3 输出: " + res3);
        System.out.println("-----------------------------");
        String s4 = "- (3 + (4 - 5))";
        System.out.println("示例 4 输入: \"" + s4 + "\"");
        int res4 = calculate(s4);
        System.out.println("示例 4 输出: " + res4);
    }
}
```
