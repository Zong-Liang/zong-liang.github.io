## 题目

**描述：**

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足：

1.  左括号必须用相同类型的右括号闭合。
2.  左括号必须以正确的顺序闭合。
3.  每个右括号都有一个对应的相同类型的左括号。

**示例 1:**

- **输入:** `s = "()"`
- **输出:** `true`

**示例 2:**

- **输入:** `s = "()[]{}"`
- **输出:** `true`

**示例 3:**

- **输入:** `s = "(]"`
- **输出:** `false`

**示例 4:**

- **输入:** `s = "([)]"`
- **输出:** `false`

**示例 5:**

- **输入:** `s = "{[]}"`
- **输出:** `true`

**提示:**

- `1 <= s.length <= 10^4`
- `s` 仅由括号 `'()[]{}'` 组成

## 题解

这题最容易让人误判的地方，是以为它只是在检查括号有没有成对出现。其实数量对不对只是最表面的一层，真正决定答案的，是关闭顺序对不对。像 `"([)]"` 这种字符串，每种括号的数量都没问题，但它依然是错的，因为 `(` 打开以后，本来应该在 `[` 之后再被关闭，结果中途先遇到了 `)`，层级一下就乱了。

一旦把问题看成“最后打开的括号，必须最先被关掉”，数据结构就很自然了。这正是标准的后进先出关系，所以最适合用栈。顺着字符串往后扫，左括号出现时，说明有一个新的未完成结构被打开；右括号出现时，说明现在要去关闭最近打开、但还没关闭的那个结构。

这题有两种常见写法。最朴素的写法，是左括号入栈，遇到右括号时再和栈顶做映射匹配；更顺手一点的写法，是栈里不存左括号，而是直接存“它未来应该遇到的右括号”。比如读到 `(` 时，就把 `)` 压栈；读到 `[` 时，就把 `]` 压栈。这样后面真的遇到右括号时，只需要检查两件事：栈是不是空的，以及栈顶字符是不是刚好等于当前字符。逻辑会更直一些，因为栈顶始终表示“我现在最期待看到哪个右括号”。

拿 `"{[]}"` 来看会很直观。顺着读进去时，先看到 `{`，再看到 `[`，所以最近打开的是 `[`。接下来遇到 `]`，它就应该先去关闭 `[`；再遇到 `}`，才轮到外层的 `{`。整个过程完全符合栈的行为。再看 `"([)]"`，当我们读到 `(` 后压入 `)`，读到 `[` 后压入 `]`，这时栈顶期待的是 `]`，但下一个字符却是 `)`，于是当场就能判错，不需要继续往后看。

代码里还有一个很简单但很实用的剪枝：如果字符串长度是奇数，那一定不可能完全匹配，因为括号必须成对出现，总长度必然是偶数。这个判断不是必须的，即使不写，后面的栈逻辑也能得出结论；不过提早排掉这些明显不可能的情况，会让代码更利落一点。

代码如下：

```java
public boolean isValid(String s) {
    if (s.length() % 2 != 0) {
        return false;
    }

    Deque<Character> stack = new ArrayDeque<>();

    for (char c : s.toCharArray()) {
        if (c == '(') {
            stack.push(')');
        } else if (c == '[') {
            stack.push(']');
        } else if (c == '{') {
            stack.push('}');
        } else {
            if (stack.isEmpty() || stack.pop() != c) {
                return false;
            }
        }
    }

    return stack.isEmpty();
}
```

时间复杂度是 `O(n)`，因为整个字符串只遍历一遍；空间复杂度是 `O(n)`，最坏情况下所有字符都是左括号，栈里会压入 `n` 个元素。

这题比较值得记的，不只是“括号匹配用栈”这句结论，而是后面的判断模式：只要题目里带着明显的“最后开始的事情必须最先结束”的味道，就应该优先想到栈。括号、嵌套结构、路径回退、成对消除，本质上都在用同一种后进先出关系。

完整代码如下：

```java
import java.util.ArrayDeque;
import java.util.Deque;
public class Main {
    public static boolean isValid(String s) {
        if (s.length() % 2 != 0) {
            return false;
        }
        Deque<Character> stack = new ArrayDeque<>();
        for (char c : s.toCharArray()) {
            if (c == '(') {
                stack.push(')');
            } else if (c == '[') {
                stack.push(']');
            } else if (c == '{') {
                stack.push('}');
            } else {
                if (stack.isEmpty() || stack.pop() != c) {
                    return false;
                }
            }
        }
        return stack.isEmpty();
    }
    public static void main(String[] args) {
        String s1 = "()";
        System.out.println("示例 1 输入: " + s1);
        boolean res1 = isValid(s1);
        System.out.println("示例 1 输出: " + res1);
        System.out.println("-----------------------------");
        String s2 = "()[]{}";
        System.out.println("示例 2 输入: " + s2);
        boolean res2 = isValid(s2);
        System.out.println("示例 2 输出: " + res2);
        System.out.println("-----------------------------");
        String s3 = "(]";
        System.out.println("示例 3 输入: " + s3);
        boolean res3 = isValid(s3);
        System.out.println("示例 3 输出: " + res3);
        System.out.println("-----------------------------");
        String s4 = "([)]";
        System.out.println("示例 4 输入: " + s4);
        boolean res4 = isValid(s4);
        System.out.println("示例 4 输出: " + res4);
        System.out.println("-----------------------------");
        String s5 = "{[]}";
        System.out.println("示例 5 输入: " + s5);
        boolean res5 = isValid(s5);
        System.out.println("示例 5 输出: " + res5);
    }
}
```
