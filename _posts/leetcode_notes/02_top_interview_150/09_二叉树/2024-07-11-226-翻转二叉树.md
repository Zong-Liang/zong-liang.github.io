## 题目

**描述：**

给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。

**示例 1:**

- **输入:** `root = [4,2,7,1,3,6,9]`
- **输出:** `[4,7,2,9,6,3,1]`

**示例 2:**

- **输入:** `root = [2,1,3]`
- **输出:** `[2,3,1]`

**示例 3:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中节点数目范围在 `[0, 100]` 内
- `-100 <= Node.val <= 100`

## 题解

### 这题本质上只做一件事

翻转二叉树，听起来像是在做一个挺复杂的结构变换，但这题真正要做的事情其实非常简单：

**把每个节点的左孩子和右孩子交换一下。**

就这么一句话，已经把整道题说完了。

因为一棵二叉树是由很多个节点连起来的，只要每个节点都完成了左右交换，整棵树自然就翻转过来了。

所以这题的关键不在于“怎么翻转整棵树”，而在于：

**怎么保证每个节点都被访问到，并完成一次左右交换。**

### 递归是最顺手的写法

树这种结构，递归一般都很好用。

对于当前节点 `root` 来说，我们只需要做两步：

1. 交换它的左子树和右子树
2. 继续去处理它的左右子树

只要对每个节点都这样做一遍，整棵树就翻转完成了。

这里最容易想明白的一点是：  
翻转整棵树，并不需要你站在全局去“重建结构”，而是只要把每个局部都处理好，整体结果自然就对了。

### 为什么这样做一定是对的

假设当前节点长这样：

```text
    root
   /    \
 left   right
```

如果要把这棵树翻转，那么翻转后的样子一定是：

```text
    root
   /    \
right   left
```

也就是说，当前节点这一层要做的事就是交换左右孩子。

交换完以后，原来的左子树和右子树虽然位置变了，但它们自己内部也还要继续翻转。  
所以接下来再递归处理新的左子树和新的右子树就可以了。

这个思路其实非常直接：  
**先把当前节点翻过来，再把下面的子树也翻过来。**

### 先把边界情况说清楚

递归题都一样，先看什么时候停。

如果当前节点是 `null`，说明这里已经没有东西可翻了，直接返回 `null` 就行。

这就是递归终止条件。

除了这个边界之外，其他非空节点都要完成一次左右交换。

### 结合示例看一下

比如示例 1：

```text
root = [4,2,7,1,3,6,9]
```

它原来的结构是：

```text
        4
      /   \
     2     7
    / \   / \
   1   3 6   9
```

先看根节点 `4`，交换左右子树之后变成：

```text
        4
      /   \
     7     2
    / \   / \
   6   9 1   3
```

但这还只是第一层的交换。

接着继续递归处理 `7` 和 `2`，其实它们下面的左右孩子也都各自交换了一次。  
因为这棵树本身比较规整，所以最终结果看起来就是整棵树左右镜像了一下：

```text
[4,7,2,9,6,3,1]
```

### 递归写法

```java
public TreeNode invertTree(TreeNode root) {
    if (root == null) {
        return null;
    }

    TreeNode temp = root.left;
    root.left = root.right;
    root.right = temp;

    invertTree(root.left);
    invertTree(root.right);

    return root;
}
```

这段代码其实就是把刚才的思路直接翻译成程序：

- 空节点直接返回
- 当前节点交换左右孩子
- 继续递归处理左右子树
- 最后返回根节点

### 复杂度分析

时间复杂度是 $O(n)$，因为每个节点都会被访问一次，也都会交换一次左右孩子。

空间复杂度是 $O(h)$，这里的 `h` 是树的高度。  
这是递归调用栈带来的开销。

如果树比较平衡，递归深度通常是 $O(\log n)$；  
如果树退化成链表，最坏情况就是 $O(n)$。

### 迭代也可以做

这题不一定非要递归，迭代同样能做。

既然目标是“访问每个节点，并交换它的左右孩子”，那我们也可以用队列做层序遍历。每次从队列里取出一个节点，交换它的左右孩子，再把它的孩子继续放进队列。

一直处理到队列为空，就说明整棵树都翻转完了。

这个思路和递归本质上是一样的，只是一个靠系统栈往下走，一个靠队列一层一层扫。

### 迭代写法

```java
public TreeNode invertTree(TreeNode root) {
    if (root == null) {
        return null;
    }

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        TreeNode current = queue.poll();

        TreeNode temp = current.left;
        current.left = current.right;
        current.right = temp;

        if (current.left != null) {
            queue.offer(current.left);
        }
        if (current.right != null) {
            queue.offer(current.right);
        }
    }

    return root;
}
```

### 这题该怎么理解才算真的会了

这题本身不难，真正值得记住的，不是代码有多短，而是这个思路：

**树上的很多操作，看起来是在改整棵树，实际上都可以拆成“每个节点做一次局部处理”。**

这题里的局部处理，就是交换左右孩子。

只要你能建立这种意识，后面再碰到类似的树题，思路就会清楚很多。

### 递归和迭代怎么选

如果只是做这道题，递归通常更自然，代码也更短。

因为“翻转当前节点，再翻转左右子树”这个过程，本来就和树的递归结构非常贴合。

迭代写法也没问题，只是代码会稍微啰嗦一点，更适合你想顺手复习 BFS 的时候写。

所以这题一般推荐先把递归写法吃透。

### 完整代码

```java
import java.util.LinkedList;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static TreeNode invertTree(TreeNode root) {
        if (root == null) {
            return null;
        }

        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;

        invertTree(root.left);
        invertTree(root.right);

        return root;
    }

    public static void printTree(TreeNode root) {
        if (root == null) {
            System.out.println("[]");
            return;
        }

        StringBuilder sb = new StringBuilder();
        sb.append("[");
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            TreeNode node = queue.poll();
            if (node == null) {
                continue;
            }

            sb.append(node.val).append(", ");

            if (node.left != null || node.right != null) {
                queue.offer(node.left);
                queue.offer(node.right);
            }
        }

        if (sb.lastIndexOf(", ") > 0) {
            sb.setLength(sb.length() - 2);
        }

        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(4);
        root1.left = new TreeNode(2, new TreeNode(1), new TreeNode(3));
        root1.right = new TreeNode(7, new TreeNode(6), new TreeNode(9));

        System.out.println("示例 1 原树 (层序): [4, 2, 7, 1, 3, 6, 9]");

        TreeNode res1 = invertTree(root1);

        System.out.print("示例 1 翻转后: ");
        printTree(res1);
        System.out.println("-----------------------------");

        TreeNode root2 = new TreeNode(2, new TreeNode(1), new TreeNode(3));

        System.out.println("示例 2 原树 (层序): [2, 1, 3]");

        TreeNode res2 = invertTree(root2);

        System.out.print("示例 2 翻转后: ");
        printTree(res2);
    }
}
```
