## 题目

**描述：**

给你一棵二叉树的根节点 `root` ，翻转这棵二叉树，并返回其根节点。

**示例 1:**

- **输入:** `root = [4,2,7,1,3,6,9]`
- **输出:** `[4,7,2,9,6,3,1]`

**示例 2:**

- **输入:** `root = [2,1,3]`
- **输出:** `[2,3,1]`

**示例 3:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中节点数目范围在 `[0, 100]` 内
- `-100 <= Node.val <= 100`

## 题解

翻转二叉树其实就是一件局部操作：每到一个节点，就交换它的左右孩子。全树翻转并不需要额外的全局技巧，局部都做对了，整体自然就对。

实现上用递归最顺手：当前节点先交换左右指针，然后递归处理新的左子树和右子树。空节点直接返回，避免空指针问题。

你也可以用队列或栈做迭代，本质仍然一样：保证每个节点都被访问一次，并执行一次左右交换。

时间复杂度是 $O(n)$，因为每个节点只处理一次。递归版的空间复杂度是 $O(h)$，`h` 为树高。

这题值得沉淀的经验是：很多树题看着像“大改结构”，实际是“定义一个节点级动作 + 覆盖所有节点”。先把节点级动作写清，难度会明显下降。

**Java 实现：**

```java
import java.util.LinkedList;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static TreeNode invertTree(TreeNode root) {
        if (root == null) {
            return null;
        }

        TreeNode temp = root.left;
        root.left = root.right;
        root.right = temp;

        invertTree(root.left);
        invertTree(root.right);

        return root;
    }

    public static void printTree(TreeNode root) {
        if (root == null) {
            System.out.println("[]");
            return;
        }

        StringBuilder sb = new StringBuilder();
        sb.append("[");
        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            TreeNode node = queue.poll();
            if (node == null) {
                continue;
            }

            sb.append(node.val).append(", ");

            if (node.left != null || node.right != null) {
                queue.offer(node.left);
                queue.offer(node.right);
            }
        }

        if (sb.lastIndexOf(", ") > 0) {
            sb.setLength(sb.length() - 2);
        }

        sb.append("]");
        System.out.println(sb.toString());
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(4);
        root1.left = new TreeNode(2, new TreeNode(1), new TreeNode(3));
        root1.right = new TreeNode(7, new TreeNode(6), new TreeNode(9));

        System.out.println("示例 1 原树 (层序): [4, 2, 7, 1, 3, 6, 9]");

        TreeNode res1 = invertTree(root1);

        System.out.print("示例 1 翻转后: ");
        printTree(res1);
        System.out.println("-----------------------------");

        TreeNode root2 = new TreeNode(2, new TreeNode(1), new TreeNode(3));

        System.out.println("示例 2 原树 (层序): [2, 1, 3]");

        TreeNode res2 = invertTree(root2);

        System.out.print("示例 2 翻转后: ");
        printTree(res2);
    }
}
```
