## 题目

**描述：**

给你一棵 **完全二叉树** 的根节点 `root`，求出该树的节点个数。

**完全二叉树** 的定义如下：在完全二叉树中，除了最底层节点可能没填满外，其余每层节点数都达到最大值，并且最下面一层的节点都集中在该层最左边的若干位置。若最底层为第 `h` 层（从第 0 层开始），则该层包含 `1`~`2^h` 个节点。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/01/14/complete.jpg)

- **输入:** `root = [1,2,3,4,5,6]`
- **输出:** 6

**示例 2:**

- **输入:** `root = []`
- **输出:** 0

**示例 3:**

- **输入:** `root = [1]`
- **输出:** 1

**提示:**

- 树中节点的数目范围是 `[0, 5 * 10^4]`
- `0 <= Node.val <= 5 * 10^4`
- 题目数据保证输入的树是 **完全二叉树**

**进阶：** 遍历树来统计节点是一种时间复杂度为 `O(n)` 的简单解决方案。你可以设计一个更快的算法吗？

## 题解

把它当普通二叉树递归统计当然能做，但题目给了“完全二叉树”这个条件，最好把这个结构性质利用起来。

对于某个子树，分别沿最左链和最右链向下走，得到左高和右高。如果两者相等，说明这是一棵满二叉树，节点数可以直接用公式 `2^h - 1` 算出，不必继续递归。

只有在左右高度不等时，才递归统计左右子树。这样就把很多完整子树一次性剪掉了。

优化写法时间复杂度是 $O((\log n)^2)$，空间复杂度 $O(\log n)$（递归深度）。如果用朴素 DFS，则是 $O(n)$ 时间。

这题的思路很典型：当输入结构有“接近规则形态”时，先看能不能做整块计算，再决定是否下钻。

**Java 实现：**

```java
import java.util.*;

// 二叉树节点定义
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {}

    TreeNode(int val) {
        this.val = val;
    }

    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

public class Main {

    public static int countNodes(TreeNode root) {
        if (root == null) {
            return 0;
        }

        int leftH = getHeight(root.left);
        int rightH = getHeight(root.right);

        if (leftH == rightH) {
            return (1 << leftH) + countNodes(root.right);
        } else {
            return (1 << rightH) + countNodes(root.left);
        }
    }

    private static int getHeight(TreeNode node) {
        int height = 0;
        while (node != null) {
            height++;
            node = node.left;
        }
        return height;
    }

    public static TreeNode buildTree(Integer[] arr) {
        if (arr.length == 0) {
            return null;
        }

        TreeNode root = new TreeNode(arr[0]);
        Queue<TreeNode> q = new LinkedList<>();
        q.add(root);
        int i = 1;

        while (i < arr.length) {
            TreeNode curr = q.poll();

            if (i < arr.length && arr[i] != null) {
                curr.left = new TreeNode(arr[i]);
                q.add(curr.left);
            }
            i++;

            if (i < arr.length && arr[i] != null) {
                curr.right = new TreeNode(arr[i]);
                q.add(curr.right);
            }
            i++;
        }

        return root;
    }

    public static void main(String[] args) {
        Integer[] input1 = {1, 2, 3, 4, 5, 6};
        TreeNode root1 = buildTree(input1);

        System.out.println("示例 1 输入: root = [1,2,3,4,5,6]");
        int res1 = countNodes(root1);
        System.out.println("示例 1 输出: " + res1);
        System.out.println("-----------------------------");

        Integer[] input2 = {};
        TreeNode root2 = buildTree(input2);

        System.out.println("示例 2 输入: root = []");
        int res2 = countNodes(root2);
        System.out.println("示例 2 输出: " + res2);
        System.out.println("-----------------------------");

        Integer[] input3 = {1};
        TreeNode root3 = buildTree(input3);

        System.out.println("示例 3 输入: root = [1]");
        int res3 = countNodes(root3);
        System.out.println("示例 3 输出: " + res3);

        Integer[] input4 = {1, 2, 3, 4, 5};
        TreeNode root4 = buildTree(input4);
        System.out.println("-----------------------------");
        System.out.println("额外测试输入: root = [1,2,3,4,5]");
        System.out.println("额外测试输出: " + countNodes(root4));
    }
}
```
