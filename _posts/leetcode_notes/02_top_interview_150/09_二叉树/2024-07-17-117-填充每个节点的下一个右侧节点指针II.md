## 题目

**描述：**

给定一个二叉树：

```c
struct Node {
  int val;
  Node *left;
  Node *right;
  Node *next;
}
```

填充它的每个 `next` 指针，让这个指针指向其下一个右侧节点。如果找不到下一个右侧节点，则将 `next` 指针设置为 `NULL`。

初始状态下，所有 `next` 指针都被设置为 `NULL`。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2019/02/15/117_sample.png)

- **输入:** `root = [1,2,3,4,5,null,7]`
- **输出:** `[1,#,2,3,#,4,5,7,#]`
- **解释:** 给定二叉树如图 A 所示，你的函数应该填充它的每个 `next` 指针，以指向其下一个右侧节点，如图 B 所示。序列化输出按层序遍历顺序（由 `next` 指针连接），`'#'` 表示每层的末尾。

**示例 2:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中的节点数在范围 `[0, 6000]` 内
- `-100 <= Node.val <= 100`

**进阶:**

- 你只能使用常量级额外空间。
- 使用递归解题也符合要求，本题中递归程序的隐式栈空间不计入额外空间复杂度。

## 题解

### 先看这题到底要做什么

这题不是普通的层序遍历题，它要求你把每一层的节点从左到右连起来。

也就是说，同一层里：

- 每个节点的 `next` 都要指向它右边的那个节点
- 如果它已经是这一层最右边的节点，那么 `next` 就保持为 `null`

所以这题本质上是在做一件事：

**把二叉树的每一层，连成一条链表。**

理解成这一点之后，思路就会清楚很多。

### 最直接的做法：层序遍历

最容易想到的办法，就是按层遍历整棵树。

因为题目本来就是要求把“同一层”的节点连起来，所以用队列做 BFS 非常自然：

1. 每次取出当前层的所有节点
2. 按顺序从左到右处理
3. 让当前节点的 `next` 指向这一层里的下一个节点

比如当前层有这些节点：

```text
2   3   5   7
```

那就连成：

```text
2 -> 3 -> 5 -> 7 -> null
```

这个思路很好理解，也最好写。

### 队列法为什么成立

层序遍历有个天然优势：  
队列里存的本来就是“按层从左到右”的节点顺序。

所以当我们处理一层的时候，只要知道这一层一共有多少个节点，就能判断：

- 如果当前节点不是这一层最后一个节点，那它的 `next` 应该指向本层下一个节点
- 如果它是最后一个节点，那它的 `next` 就不需要再改

也就是说，队列已经帮我们把“谁和谁应该相连”这件事排好了。

### 队列法写法

```java
public Node connect(Node root) {
    if (root == null) {
        return null;
    }

    Queue<Node> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size();

        for (int i = 0; i < size; i++) {
            Node node = queue.poll();

            if (i < size - 1) {
                node.next = queue.peek();
            }

            if (node.left != null) {
                queue.offer(node.left);
            }
            if (node.right != null) {
                queue.offer(node.right);
            }
        }
    }

    return root;
}
```

### 这个做法的问题在哪里

如果只是为了把题做出来，队列法已经够用了。

但这题的进阶要求是：  
**只能使用常量级额外空间。**

而队列法在最坏情况下，需要存下一整层的节点，所以空间复杂度是 $O(n)$，不满足进阶要求。

于是这题真正值得学的，其实不是队列法，而是：

**能不能不借助队列，直接把下一层串起来？**

### 关键转折：利用已经建好的 `next`

这题最巧的地方就在这里。

当我们已经把某一层的 `next` 指针连好之后，这一层本身其实就已经是一条链表了。  
既然它已经是一条链表，我们就不需要再用队列遍历这一层，而是可以直接顺着 `next` 往右走。

换句话说：

- 当前层，靠 `next` 来遍历
- 下一层，在遍历当前层的过程中顺手连出来

这就是进阶解法的核心。

### 这一层怎么帮下一层连线

假设我们现在正在处理某一层，当前层已经能通过 `next` 串起来了，比如：

```text
2 -> 3 -> 5 -> null
```

那我们就顺着这条链，从左到右扫描当前层的每个节点。  
扫描到一个节点时：

- 如果它有左孩子，就把这个左孩子接到“下一层链表”的尾部
- 如果它有右孩子，就把这个右孩子接到“下一层链表”的尾部

这样当前层扫完之后，下一层就被我们按从左到右的顺序完整串好了。

所以这题的本质其实变成了：

**一边沿着当前层横着走，一边把下一层从左到右接出来。**

### 为什么要用 dummy 和 tail

实现这个过程时，通常会用两个辅助指针：

- `dummy`：下一层链表的虚拟头节点
- `tail`：下一层链表当前的尾巴

作用很简单：

- `dummy.next` 永远指向下一层的第一个真实节点
- 每找到一个新的孩子节点，就接到 `tail.next` 后面，再把 `tail` 往后移

这样无论下一层第一个出现的是左孩子还是右孩子，都很好处理，不需要单独写很多分支。

这是链表题里很常见的写法，放在这里也特别合适。

### 整个过程怎么走

可以把整个流程理解成下面这样：

1. `cur` 指向当前层的第一个节点
2. 新建一个 `dummy`，准备去串下一层
3. 顺着当前层的 `next` 一路往右扫描
4. 遇到孩子节点就接到 `tail` 后面
5. 当前层扫完后，让 `cur = dummy.next`
6. 这样就切换到下一层了

然后重复这个过程，直到没有下一层为止。

### 用示例看一下

还是看题目的例子：

```text
root = [1,2,3,4,5,null,7]
```

树的结构大致是：

```text
        1
      /   \
     2     3
    / \     \
   4   5     7
```

先处理第一层：

- 当前层只有 `1`
- 扫到 `1` 时，把它的左孩子 `2` 和右孩子 `3` 接到下一层
- 于是第二层变成：

```text
2 -> 3 -> null
```

再处理第二层：

- 顺着 `2 -> 3` 这条链往右走
- `2` 的孩子有 `4` 和 `5`，先接进去
- `3` 的孩子有 `7`，再接进去

于是第三层变成：

```text
4 -> 5 -> 7 -> null
```

到这里，所有层就都连好了。

### 进阶写法

```java
public Node connect(Node root) {
    Node cur = root;

    while (cur != null) {
        Node dummy = new Node(0);
        Node tail = dummy;

        while (cur != null) {
            if (cur.left != null) {
                tail.next = cur.left;
                tail = tail.next;
            }

            if (cur.right != null) {
                tail.next = cur.right;
                tail = tail.next;
            }

            cur = cur.next;
        }

        cur = dummy.next;
    }

    return root;
}
```

### 这段代码真正该怎么理解

这段代码表面上看是在操作很多指针，但核心其实只有一句话：

**用当前层的链表，去生成下一层的链表。**

其中：

- `cur` 负责遍历当前层
- `dummy.next` 负责记录下一层的开头
- `tail` 负责把下一层一个个接起来

当前层处理完，就跳到下一层继续做同样的事。

只要把这个大逻辑看清，代码就不难记。

### 复杂度分析

如果用队列法：

- 时间复杂度是 $O(n)$
- 空间复杂度是 $O(n)$

如果用 `next` 指针的进阶写法：

- 时间复杂度还是 $O(n)$
- 空间复杂度降为 $O(1)$

因为每个节点都只会被访问有限次，而整个过程中只用了少量指针变量。

### 这题最容易卡住的点

这题一般不是难在代码量，而是难在思维切换。

很多人会停留在“按层遍历”这一步，但进阶解法真正厉害的地方在于：

**它不再把某一层当成树的一部分去遍历，而是把这一层当成一条已经存在的链表来遍历。**

一旦这个视角切过来，后面的 `dummy` 和 `tail` 都会顺很多。

### 这题真正值得记住的点

这题最值得记住的，不是某几个变量名，而是这种处理方式：

**当题目让你按层处理树节点时，不一定非要靠队列；如果层与层之间已经能建立横向连接，就可以直接把每一层当链表来处理。**

这是这题最有价值的地方。

### 完整代码

```java
import java.util.*;

// 节点定义
class Node {
    public int val;
    public Node left;
    public Node right;
    public Node next;

    public Node() {}

    public Node(int _val) {
        val = _val;
    }

    public Node(int _val, Node _left, Node _right, Node _next) {
        val = _val;
        left = _left;
        right = _right;
        next = _next;
    }
}

public class Main {

    public static Node connect(Node root) {
        Node cur = root;

        while (cur != null) {
            Node dummy = new Node(0);
            Node tail = dummy;

            while (cur != null) {
                if (cur.left != null) {
                    tail.next = cur.left;
                    tail = tail.next;
                }

                if (cur.right != null) {
                    tail.next = cur.right;
                    tail = tail.next;
                }

                cur = cur.next;
            }

            cur = dummy.next;
        }

        return root;
    }

    public static void printTreeNextPointers(Node root) {
        if (root == null) {
            System.out.println("[]");
            return;
        }

        Node curLevel = root;
        List<String> result = new ArrayList<>();

        while (curLevel != null) {
            Node cur = curLevel;
            Node nextLevelHead = null;

            while (cur != null) {
                result.add(String.valueOf(cur.val));

                if (nextLevelHead == null) {
                    if (cur.left != null) {
                        nextLevelHead = cur.left;
                    } else if (cur.right != null) {
                        nextLevelHead = cur.right;
                    }
                }

                cur = cur.next;
            }

            result.add("#");
            curLevel = nextLevelHead;
        }

        System.out.println(result);
    }

    public static void main(String[] args) {
        Node root = new Node(1);
        root.left = new Node(2);
        root.right = new Node(3);
        root.left.left = new Node(4);
        root.left.right = new Node(5);
        root.right.right = new Node(7);

        System.out.println("示例 1 输入: root = [1,2,3,4,5,null,7]");
        connect(root);
        System.out.print("示例 1 输出: ");
        printTreeNextPointers(root);
        System.out.println("-----------------------------");

        Node root2 = null;
        System.out.println("示例 2 输入: root = []");
        connect(root2);
        System.out.print("示例 2 输出: ");
        printTreeNextPointers(root2);
    }
}
```
