## 题目

**描述**

给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

**示例 1:**

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** 3

**示例 2:**

- **输入:** `root = [1,null,2]`
- **输出:** 2

**提示:**

- 树中节点的数量在 `[0, 10^4]` 区间内。
- `-100 <= Node.val <= 100`

## 题解

### 先把题意理解透

这题要求的是二叉树的最大深度。

说白了，就是从根节点出发，一直往下走，直到某个叶子节点为止，能走出的最长那条路径一共有多少个节点。

所以这题虽然看起来是在问“深度”，本质上其实是在问：

**一棵树的高度是多少。**

一旦把这一点想清楚，思路就比较自然了。

### 递归为什么最顺手

对于任意一个节点来说，它的最大深度其实取决于左右子树。

如果左子树更深，那当前节点的最大深度就是“左子树深度 + 1”；  
如果右子树更深，那就是“右子树深度 + 1”。

也就是说，当前节点的答案可以直接写成：

```text
当前节点的最大深度 = max(左子树最大深度, 右子树最大深度) + 1
```

这里加上的这个 `1`，就是当前节点自己这一层。

于是这题最自然的写法就是递归。

### 递归是怎么一步步算出来的

递归里最重要的是先想清楚终止条件。

如果当前节点是 `null`，说明这条路已经走到底了，那它的深度就是 `0`。

如果当前节点不为空，就分别去算：

- 左子树的最大深度
- 右子树的最大深度

然后取两者较大值，再加上当前节点这一层。

整个过程其实就是一个很典型的后序思路：  
**先算孩子，再算自己。**

### 用示例看一眼

比如这棵树：

```text
root = [3,9,20,null,null,15,7]
```

它的结构大致是：

```text
    3
   / \
  9  20
    /  \
   15   7
```

从叶子节点开始看：

- 节点 `9` 的深度是 `1`
- 节点 `15` 的深度是 `1`
- 节点 `7` 的深度是 `1`

再往上一层：

- 节点 `20` 的最大深度是 `max(1, 1) + 1 = 2`

最后回到根节点：

- 节点 `3` 的最大深度是 `max(1, 2) + 1 = 3`

所以答案就是 `3`。

### 递归写法

```java
public int maxDepth(TreeNode root) {
    if (root == null) {
        return 0;
    }
    int leftDepth = maxDepth(root.left);
    int rightDepth = maxDepth(root.right);
    return Math.max(leftDepth, rightDepth) + 1;
}
```

### 复杂度分析

时间复杂度是 $O(n)$，因为每个节点都会被访问一次。

空间复杂度是 $O(h)$，这里的 `h` 是树的高度。  
因为递归调用需要栈空间，最坏情况下如果整棵树退化成链表，空间复杂度就是 $O(n)$；如果树比较平衡，那一般是 $O(\log n)$。

### 迭代也能做

除了递归，这题还可以用层序遍历来做。

因为“最大深度”本质上也可以理解成这棵树一共有多少层，所以我们可以一层一层往下扫。每处理完一整层，深度就加一。

这个思路本质上就是广度优先搜索，也就是 BFS。

做法并不复杂：

- 先把根节点放进队列
- 每次取出当前层的所有节点
- 再把下一层的节点加入队列
- 当前层处理完之后，深度加一

队列空了，说明整棵树都遍历完了，这时候得到的层数就是最大深度。

### 迭代写法

```java
public int maxDepth(TreeNode root) {
    if (root == null) {
        return 0;
    }

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    int depth = 0;

    while (!queue.isEmpty()) {
        int size = queue.size();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            if (node.left != null) {
                queue.offer(node.left);
            }
            if (node.right != null) {
                queue.offer(node.right);
            }
        }

        depth++;
    }

    return depth;
}
```

### 这两种写法怎么选

如果只是为了做这道题，递归写法通常更直接，也更贴合“树的高度”这个定义，代码会更短一些。

而层序遍历的好处在于思路也很清楚：一层一层数过去，最后得到层数。  
如果你本来就对 BFS 更熟，这种写法也完全没问题。

这题本身并不在于选哪一种，而在于你能不能意识到：

- 递归是在利用“当前节点的答案由左右子树决定”这个性质
- 迭代是在利用“最大深度就是层数”这个性质

能把这两层想明白，这题就算吃透了。

### 完整代码

```java
import java.util.LinkedList;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static int maxDepthRecursive(TreeNode root) {
        if (root == null) {
            return 0;
        }
        return Math.max(maxDepthRecursive(root.left), maxDepthRecursive(root.right)) + 1;
    }

    public static int maxDepthIterative(TreeNode root) {
        if (root == null) {
            return 0;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        int depth = 0;

        while (!queue.isEmpty()) {
            int size = queue.size();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }
            depth++;
        }

        return depth;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("DFS 结果: " + maxDepthRecursive(root1));
        System.out.println("BFS 结果: " + maxDepthIterative(root1));
        System.out.println("-----------------------------");

        TreeNode root2 = new TreeNode(1);
        root2.right = new TreeNode(2);

        System.out.println("示例 2 输入: [1,null,2]");
        System.out.println("DFS 结果: " + maxDepthRecursive(root2));
        System.out.println("BFS 结果: " + maxDepthIterative(root2));
        System.out.println("-----------------------------");

        TreeNode root3 = null;
        System.out.println("示例 3 输入: []");
        System.out.println("DFS 结果: " + maxDepthRecursive(root3));
        System.out.println("BFS 结果: " + maxDepthIterative(root3));
    }
}
```
