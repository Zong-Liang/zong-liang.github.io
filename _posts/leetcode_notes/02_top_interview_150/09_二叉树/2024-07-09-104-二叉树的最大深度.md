## 题目

**描述**

给定一个二叉树 `root` ，返回其最大深度。

二叉树的 **最大深度** 是指从根节点到最远叶子节点的最长路径上的节点数。

**示例 1:**

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** 3

**示例 2:**

- **输入:** `root = [1,null,2]`
- **输出:** 2

**提示:**

- 树中节点的数量在 `[0, 10^4]` 区间内。
- `-100 <= Node.val <= 100`

## 题解

这题不难，但很适合用来把“树上的递归定义”练扎实。所谓最大深度，本质就是在问：以某个节点为根时，这棵子树的高度是多少。

递归写法非常顺：空节点深度是 `0`，非空节点的深度等于 `max(左子树深度, 右子树深度) + 1`。这就是典型的“先算孩子，再合成当前节点答案”。

如果你更喜欢按层思考，也可以用 BFS：每处理完一层，深度加一。代码里两种都给了，面试时任选你最稳的写法即可。

复杂度上，DFS 和 BFS 的时间都是 $O(n)$。DFS 的额外空间是递归栈 $O(h)$，BFS 的额外空间是队列，最宽层可能到 $O(n)$。

这题做完可以记住一句话：遇到“树高/最大深度/最小深度”这类词，第一反应就是把答案定义在“子树”上，再决定递归还是按层遍历。

**Java 实现：**

```java
import java.util.LinkedList;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static int maxDepthRecursive(TreeNode root) {
        if (root == null) {
            return 0;
        }
        return Math.max(maxDepthRecursive(root.left), maxDepthRecursive(root.right)) + 1;
    }

    public static int maxDepthIterative(TreeNode root) {
        if (root == null) {
            return 0;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        int depth = 0;

        while (!queue.isEmpty()) {
            int size = queue.size();
            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }
            depth++;
        }

        return depth;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("DFS 结果: " + maxDepthRecursive(root1));
        System.out.println("BFS 结果: " + maxDepthIterative(root1));
        System.out.println("-----------------------------");

        TreeNode root2 = new TreeNode(1);
        root2.right = new TreeNode(2);

        System.out.println("示例 2 输入: [1,null,2]");
        System.out.println("DFS 结果: " + maxDepthRecursive(root2));
        System.out.println("BFS 结果: " + maxDepthIterative(root2));
        System.out.println("-----------------------------");

        TreeNode root3 = null;
        System.out.println("示例 3 输入: []");
        System.out.println("DFS 结果: " + maxDepthRecursive(root3));
        System.out.println("BFS 结果: " + maxDepthIterative(root3));
    }
}
```
