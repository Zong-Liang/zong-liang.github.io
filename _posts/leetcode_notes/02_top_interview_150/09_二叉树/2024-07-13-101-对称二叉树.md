## 题目

**描述：**

给你一个二叉树的根节点 `root`，检查它是否轴对称。

**示例 1:**

![示例1图片](https://pic.leetcode.cn/1698026966-JDYPDU-image.png)

- **输入:** `root = [1,2,2,3,4,4,3]`
- **输出:** `true`

**示例 2:**

![示例2图片](https://pic.leetcode.cn/1698027008-nPFLbM-image.png)

- **输入:** `root = [1,2,2,null,3,null,3]`
- **输出:** `false`

**提示:**

- 树中节点数目在范围 `[1, 1000]` 内
- `-100 <= Node.val <= 100`

**进阶：** 你可以运用递归和迭代两种方法解决这个问题吗？

## 题解

### 先想清楚，对称到底在比什么

这题问的是一棵二叉树是不是对称。

这里的“对称”，不是说左子树和右子树完全一样，而是说它们要**互为镜像**。

也就是说，拿根节点中间那条竖线当作对称轴去看：

- 左子树的左边，要和右子树的右边对应
- 左子树的右边，要和右子树的左边对应
- 而且对应位置上的节点值也必须相同

所以这题本质上不是在判断“两棵子树是否一样”，而是在判断：

**左子树和右子树能不能一左一右地完全对上。**

### 这题为什么不能直接比较左右子树是否相同

很多人第一次做这题时，会下意识想到上一题“相同的树”，然后去比较：

- `root.left` 和 `root.right` 是否相同

但这里不能这么比。

因为对称看的是镜像关系，不是同向关系。

比如这棵树：

```text
    1
   / \
  2   2
 /     \
3       3
```

它显然是对称的。  
但如果你直接拿左子树和右子树按“相同的树”那种方式去比较，就会发现：

- 左子树的左节点是 `3`
- 右子树的左节点是 `null`

这样就会误判。

所以这里真正该比较的是：

- 左子树的左节点 对 右子树的右节点
- 左子树的右节点 对 右子树的左节点

这才是镜像。

### 递归该怎么想

既然判断的是镜像关系，那我们可以写一个辅助函数 `check(p, q)`，表示：

**以 `p` 和 `q` 为根的两棵树，是否互为镜像。**

那对于 `p` 和 `q` 来说，要想镜像成立，必须同时满足三件事：

- `p` 和 `q` 当前节点值相同
- `p.left` 和 `q.right` 互为镜像
- `p.right` 和 `q.left` 互为镜像

这个对应关系一定不能写反。  
这题最核心的点，其实就在这里。

### 先把边界情况分清楚

递归题最怕边界没想透，这题也一样。

#### 1. 两个节点都为空

说明这两个位置都没有节点，而且目前没有发现问题，这一部分当然是对称的，返回 `true`。

#### 2. 只有一个节点为空

这说明结构已经对不上了，直接返回 `false`。

#### 3. 两个节点都不为空，但值不同

结构虽然对上了，但值不一样，也不是对称的，返回 `false`。

只有这三种情况都排除之后，才有继续往下递归的必要。

### 真正的递归关系

当 `p` 和 `q` 都不为空，且 `p.val == q.val` 时，接下来要检查的就是：

- `check(p.left, q.right)`
- `check(p.right, q.left)`

只有这两部分都成立，`p` 和 `q` 这两棵树才互为镜像。

所以整道题的递归关系其实非常清楚：

```text
两棵树互为镜像
=
当前节点值相同
并且外侧对子树互为镜像
并且内侧对子树互为镜像
```

这里的“外侧”和“内侧”这个理解会很好用。

### 结合示例看一下

先看示例 1：

```text
root = [1,2,2,3,4,4,3]
```

它的结构可以理解成：

```text
        1
      /   \
     2     2
    / \   / \
   3   4 4   3
```

从根节点往下看：

- 左边的 `2` 和右边的 `2` 相同
- 再看外侧：左边的 `3` 和右边的 `3` 相同
- 再看内侧：左边的 `4` 和右边的 `4` 相同

一路都能对上，所以这棵树是对称的。

再看示例 2：

```text
root = [1,2,2,null,3,null,3]
```

结构大致是：

```text
        1
      /   \
     2     2
      \     \
       3     3
```

问题就出在镜像位置没对上：

- 左子树的左边是 `null`
- 右子树的右边是 `3`

这两个位置一个为空，一个不为空，所以不对称。

### 递归写法

```java
public boolean isSymmetric(TreeNode root) {
    if (root == null) {
        return true;
    }
    return check(root.left, root.right);
}

private boolean check(TreeNode p, TreeNode q) {
    if (p == null && q == null) {
        return true;
    }
    if (p == null || q == null || p.val != q.val) {
        return false;
    }
    return check(p.left, q.right) && check(p.right, q.left);
}
```

这段代码本质上就是把刚才那套镜像判断原样写出来。

先比较当前节点，再按“外侧对外侧、内侧对内侧”的方式继续递归。

### 复杂度分析

时间复杂度是 $O(n)$，因为每个节点最多都会被访问一次。

空间复杂度是 $O(h)$，这里的 `h` 是树的高度。  
这是递归调用栈带来的开销。

如果树比较平衡，通常是 $O(\log n)$；  
最坏情况下如果树退化得很厉害，也可能达到 $O(n)$。

### 迭代为什么也能做

这题不一定非要递归，迭代同样可以。

关键在于你得记住：  
我们不是在单独遍历一个节点，而是在**成对比较两个应该互为镜像的位置**。

所以队列里也不能随便放节点，而是要把需要比较的两个节点一起放进去。

比如一开始就放：

- `root.left`
- `root.right`

之后每次从队列里取出两个节点 `u` 和 `v`，检查它们是否匹配。

如果匹配，再继续把它们下一层需要对应的位置放进去：

- `u.left` 和 `v.right`
- `u.right` 和 `v.left`

这样队列就等于是在模拟递归里的镜像比较过程。

### 迭代写法

```java
public boolean isSymmetricIterative(TreeNode root) {
    if (root == null) {
        return true;
    }

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root.left);
    queue.offer(root.right);

    while (!queue.isEmpty()) {
        TreeNode u = queue.poll();
        TreeNode v = queue.poll();

        if (u == null && v == null) {
            continue;
        }

        if (u == null || v == null || u.val != v.val) {
            return false;
        }

        queue.offer(u.left);
        queue.offer(v.right);

        queue.offer(u.right);
        queue.offer(v.left);
    }

    return true;
}
```

### 这题真正容易错的地方

这题本身不难，但很容易写错比较顺序。

最常见的错误有两个：

- 把它写成比较左右子树是否“相同”，而不是是否“镜像”
- 递归或入队时，把对应关系写成了 `left-left`、`right-right`

一旦这么写，思路就偏了。

所以做这题时，脑子里最好一直记着一句话：

**对称看的是镜像位置，不是同一侧位置。**

### 这题真正值得记住的点

这题和“相同的树”放在一起看，会特别有帮助。

- “相同的树”比较的是：左对左，右对右
- “对称二叉树”比较的是：左对右，右对左

看起来只差一点点，但本质上比的是两种完全不同的对应关系。

只要把这个区别彻底想明白，后面这类树的判断题就不容易混了。

### 完整代码

```java
import java.util.*;

// 二叉树节点定义
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;

    TreeNode() {}

    TreeNode(int val) {
        this.val = val;
    }

    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

public class Main {

    public static boolean isSymmetricRecursion(TreeNode root) {
        if (root == null) {
            return true;
        }
        return check(root.left, root.right);
    }

    private static boolean check(TreeNode p, TreeNode q) {
        if (p == null && q == null) {
            return true;
        }
        if (p == null || q == null || p.val != q.val) {
            return false;
        }
        return check(p.left, q.right) && check(p.right, q.left);
    }

    public static boolean isSymmetricIterative(TreeNode root) {
        if (root == null) {
            return true;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root.left);
        queue.offer(root.right);

        while (!queue.isEmpty()) {
            TreeNode u = queue.poll();
            TreeNode v = queue.poll();

            if (u == null && v == null) {
                continue;
            }
            if (u == null || v == null || u.val != v.val) {
                return false;
            }

            queue.offer(u.left);
            queue.offer(v.right);

            queue.offer(u.right);
            queue.offer(v.left);
        }

        return true;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(1);
        root1.left = new TreeNode(2);
        root1.right = new TreeNode(2);
        root1.left.left = new TreeNode(3);
        root1.left.right = new TreeNode(4);
        root1.right.left = new TreeNode(4);
        root1.right.right = new TreeNode(3);

        System.out.println("示例 1 输入: root = [1,2,2,3,4,4,3]");
        System.out.println("递归法输出: " + isSymmetricRecursion(root1));
        System.out.println("迭代法输出: " + isSymmetricIterative(root1));
        System.out.println("-----------------------------");

        TreeNode root2 = new TreeNode(1);
        root2.left = new TreeNode(2);
        root2.right = new TreeNode(2);
        root2.left.right = new TreeNode(3);
        root2.right.right = new TreeNode(3);

        System.out.println("示例 2 输入: root = [1,2,2,null,3,null,3]");
        System.out.println("递归法输出: " + isSymmetricRecursion(root2));
        System.out.println("迭代法输出: " + isSymmetricIterative(root2));
    }
}
```
