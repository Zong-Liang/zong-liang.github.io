## 题目

**描述：**

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例 1:**

- **输入:** `n = 2`
- **输出:** 2
- **解释:** 有两种方法可以爬到楼顶。
  1.  1 阶 + 1 阶
  2.  2 阶

**示例 2:**

- **输入:** `n = 3`
- **输出:** 3
- **解释:** 有三种方法可以爬到楼顶。
  1.  1 阶 + 1 阶 + 1 阶
  2.  1 阶 + 2 阶
  3.  2 阶 + 1 阶

**提示:**

- `1 <= n <= 45`

## 题解
把这题当成“到达第 `n` 阶有多少种走法”就很自然了。最后一步只有两种可能：从 `n-1` 迈 1 阶，或者从 `n-2` 迈 2 阶。

因此状态转移是 `f(n) = f(n-1) + f(n-2)`。这和斐波那契同构，只是初始条件按题意设置。顺着这个关系往后推即可得到答案。

实现上不一定要开数组，因为每次只依赖前两项。用两个变量滚动更新就能把空间降到常数，代码也更简洁。

代码如下：

```java
public class Main {

    public static int climbStairs(int n) {
        if (n <= 1) return 1;
        if (n == 2) return 2;

        int first = 1;
        int second = 2;

        for (int i = 3; i <= n; i++) {
            int third = first + second;
            first = second;
            second = third;
        }

        return second;
    }

    public static void main(String[] args) {
        int n1 = 2;
        System.out.println("示例 1 输入: n = " + n1);
        System.out.println("示例 1 输出: " + climbStairs(n1));
        System.out.println("-----------------------------");

        int n2 = 3;
        System.out.println("示例 2 输入: n = " + n2);
        System.out.println("示例 2 输出: " + climbStairs(n2));
        System.out.println("-----------------------------");

        int n3 = 5;
        System.out.println("额外示例 输入: n = " + n3);
        System.out.println("额外示例 输出: " + climbStairs(n3));
    }
}
```

复杂度分析：时间复杂度 `O(n)`，空间复杂度 `O(1)`。

这题是动态规划入门模板：先盯“最后一步”，状态转移往往就自己出来了。
