## 题目

**描述：**

给你一个二叉搜索树的根节点 `root`，返回 **树中任意两不同节点值之间的最小差值**。

差值是一个正数，其数值等于两值之差的绝对值。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/02/05/bst1.jpg)

- **输入:** `root = [4,2,6,1,3]`
- **输出:** 1

**示例 2:**

![示例2图片](https://assets.leetcode.com/uploads/2021/02/05/bst2.jpg)

- **输入:** `root = [1,0,48,null,null,12,49]`
- **输出:** 1

**提示:**

- 树中节点的数目范围是 `[2, 10^4]`
- `0 <= Node.val <= 10^5`

**注意：** 本题与 783. 二叉搜索树节点间的最小距离 相同。

## 题解

### 这题的关键不在“遍历整棵树”，而在“这是二叉搜索树”

如果这题给的是普通二叉树，那我们确实只能把所有节点值取出来，再去想怎么比较差值。

但这里给的是 **二叉搜索树**。

二叉搜索树最重要的性质是：

- 左子树所有节点值都小于当前节点
- 右子树所有节点值都大于当前节点

而这个性质一旦和 **中序遍历** 结合起来，就会得到一个非常重要的结论：

**二叉搜索树的中序遍历结果一定是递增序列。**

这才是这题真正该抓住的地方。

### 为什么有了递增序列，问题就简单了

题目要找的是任意两个不同节点值之间的最小绝对差。

很多人第一反应会想：  
既然是任意两个，那是不是要两两比较？

其实不用。

因为一旦所有节点值排成了递增序列，最小差值一定只会出现在**相邻两个数之间**。

比如有序序列：

```text
[1, 3, 8, 10]
```

如果最小差值不在相邻元素之间，而出现在 `1` 和 `8` 之间，那中间的 `3` 一定会让差值更小，这就矛盾了。

所以这题一旦转化成“中序遍历得到有序序列”，剩下的事情就只剩一件：

**遍历时不断比较当前节点和前一个节点的差值，取最小值。**

### 这题真正的思路

我们不需要真的把整棵树转成数组再处理。

因为中序遍历本来就是按从小到大的顺序访问节点，所以在遍历过程中，维护两个变量就够了：

- `prev`：记录中序遍历时上一个访问到的节点值
- `minDiff`：记录目前为止的最小差值

每访问到一个新节点时：

- 如果 `prev` 不为空，就计算 `当前值 - prev`
- 用这个差值去更新 `minDiff`
- 然后把 `prev` 更新为当前节点值

这样整棵树遍历完，答案也就出来了。

### 为什么这里直接用减法就行

题目里说的是“最小绝对差”，但在代码里我们并没有写 `Math.abs(...)`。

原因就在于中序遍历后的值是递增的。

也就是说，当前访问到的节点值一定大于等于前一个节点值，所以：

```text
当前值 - prev >= 0
```

既然差值天然不会是负数，那这里直接相减就行，不需要再取绝对值。

这个细节虽然不大，但说明你是真的抓住了 BST 的性质，而不是机械套模板。

### 用示例看一遍

比如示例 1：

```text
root = [4,2,6,1,3]
```

这棵 BST 中序遍历的结果是：

```text
[1,2,3,4,6]
```

相邻差值分别是：

- `2 - 1 = 1`
- `3 - 2 = 1`
- `4 - 3 = 1`
- `6 - 4 = 2`

最小值就是 `1`。

这道题本质上就是在做这件事，只不过我们不需要真的把这个有序序列完整存下来，而是在遍历过程中顺手比较。

### 代码

```java
int minDiff = Integer.MAX_VALUE;
Integer prev = null;

public int getMinimumDifference(TreeNode root) {
    inorder(root);
    return minDiff;
}

private void inorder(TreeNode node) {
    if (node == null) return;

    inorder(node.left);

    if (prev != null) {
        minDiff = Math.min(minDiff, node.val - prev);
    }
    prev = node.val;

    inorder(node.right);
}
```

### 这段代码最关键的地方

关键其实不是“递归”本身，而是中间这两句：

```java
if (prev != null) {
    minDiff = Math.min(minDiff, node.val - prev);
}
prev = node.val;
```

它们的含义是：

- 当前节点是中序序列里的下一个数
- `prev` 是中序序列里的前一个数
- 比较这两个相邻值的差，就够了

也就是说，这题真正的核心不是“在树上找最小差值”，而是：

**借助 BST 的中序有序性，把问题压缩成“比较有序序列中相邻元素的差”。**

一旦想清楚这一层，代码就只是顺手写出来的结果。

### 为什么不用把所有节点先存进数组

当然也可以这么做：

1. 先中序遍历整棵树
2. 把结果存进列表
3. 再遍历列表，比较相邻元素差值

这样也能做对。

但既然中序遍历本来就是按顺序访问节点，那我们完全可以边遍历边比较，没有必要额外再开一个数组把所有值存下来。

所以直接维护 `prev` 和 `minDiff`，写法会更紧凑，空间上也更省。

### 再补一种写法：迭代中序遍历

如果你不想写递归，也可以用栈手动模拟中序遍历。

思路完全一样：

- 一路向左入栈
- 弹出当前节点
- 处理当前值和前一个值的差
- 再转向右子树

区别只在于，递归系统帮你维护调用栈；迭代则是你自己显式写一个栈出来。

### 迭代写法为什么和递归本质一样

因为中序遍历的访问顺序没有变，仍然是：

**左 -> 中 -> 右**

既然访问顺序完全一样，那么“当前节点”和“前一个节点”之间的关系也完全一样。

所以无论递归还是迭代，真正起作用的都不是写法形式，而是这件事：

**我们始终在按递增顺序访问 BST 的节点值。**

### 迭代代码

```java
public int getMinimumDifference(TreeNode root) {
    int minDiff = Integer.MAX_VALUE;
    Integer prev = null;
    Deque<TreeNode> stack = new ArrayDeque<>();
    TreeNode cur = root;

    while (cur != null || !stack.isEmpty()) {
        while (cur != null) {
            stack.push(cur);
            cur = cur.left;
        }

        cur = stack.pop();

        if (prev != null) {
            minDiff = Math.min(minDiff, cur.val - prev);
        }
        prev = cur.val;

        cur = cur.right;
    }

    return minDiff;
}
```

### 这题该怎么选写法

如果只是为了把题做对，而且你对递归不排斥，那这题直接写递归版中序遍历就够了。

原因很简单：

- 代码短
- 逻辑顺
- 题目的核心能体现得最清楚

迭代版当然也很好，尤其是在你想顺手练中序遍历模板的时候。  
但就这题本身来说，递归版已经足够自然了。

### 复杂度分析

无论递归还是迭代，每个节点都只访问一次，所以时间复杂度都是 $O(n)$，其中 $n$ 是节点总数。

空间复杂度取决于树的高度：

- 递归版的空间复杂度是 $O(h)$，来自递归调用栈
- 迭代版的空间复杂度也是 $O(h)$，来自显式栈

其中 $h$ 表示树的高度。  
如果树比较平衡，通常是 $O(\log n)$；最坏情况下退化成链表，则是 $O(n)$。

### 这题真正值得记住的地方

这题最值得记住的，不是“再背一个最小差值模板”，而是这个转换过程：

**只要题目给的是二叉搜索树，就先想中序遍历能不能把问题变成有序序列问题。**

很多 BST 题真正的突破口都在这里：

- 第 k 小元素
- 最小绝对差
- 众数
- 是否有序

本质上都和“中序遍历后有序”有关。

这题只是其中最典型的一种。

### 完整代码

```java
import java.util.ArrayDeque;
import java.util.Deque;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    static int minDiff;
    static Integer prev;

    public static int getMinimumDifference(TreeNode root) {
        minDiff = Integer.MAX_VALUE;
        prev = null;
        inorder(root);
        return minDiff;
    }

    private static void inorder(TreeNode node) {
        if (node == null) {
            return;
        }

        inorder(node.left);

        if (prev != null) {
            minDiff = Math.min(minDiff, node.val - prev);
        }
        prev = node.val;

        inorder(node.right);
    }

    public static int getMinimumDifferenceIterative(TreeNode root) {
        int result = Integer.MAX_VALUE;
        Integer prevVal = null;
        Deque<TreeNode> stack = new ArrayDeque<>();
        TreeNode cur = root;

        while (cur != null || !stack.isEmpty()) {
            while (cur != null) {
                stack.push(cur);
                cur = cur.left;
            }

            cur = stack.pop();

            if (prevVal != null) {
                result = Math.min(result, cur.val - prevVal);
            }
            prevVal = cur.val;

            cur = cur.right;
        }

        return result;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(4);
        root1.left = new TreeNode(2);
        root1.right = new TreeNode(6);
        root1.left.left = new TreeNode(1);
        root1.left.right = new TreeNode(3);

        System.out.println("示例 1 输入: [4,2,6,1,3]");
        System.out.println("示例 1 输出: " + getMinimumDifference(root1));

        TreeNode root2 = new TreeNode(1);
        root2.left = new TreeNode(0);
        root2.right = new TreeNode(48);
        root2.right.left = new TreeNode(12);
        root2.right.right = new TreeNode(49);

        System.out.println("示例 2 输入: [1,0,48,null,null,12,49]");
        System.out.println("示例 2 输出: " + getMinimumDifference(root2));

        TreeNode root3 = new TreeNode(236);
        root3.left = new TreeNode(104);
        root3.right = new TreeNode(701);
        root3.left.right = new TreeNode(227);
        root3.right.right = new TreeNode(911);

        System.out.println("示例 3 输入: [236,104,701,null,227,null,911]");
        System.out.println("示例 3 输出: " + getMinimumDifferenceIterative(root3));
    }
}
```
