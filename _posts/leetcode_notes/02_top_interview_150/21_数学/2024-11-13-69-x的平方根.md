## 题目

**描述：**

给你一个非负整数 `x`，计算并返回 `x` 的 **算术平方根**。

由于返回类型是整数，结果只保留 **整数部分**，小数部分将被 **舍去**。

**注意：** 不允许使用任何内置指数函数和算符，例如 `pow(x, 0.5)` 或者 `x ** 0.5`。

**示例 1:**

- **输入:** `x = 4`
- **输出:** 2

**示例 2:**

- **输入:** `x = 8`
- **输出:** 2
- **解释:** 8 的算术平方根是 2.82842...，由于返回类型是整数，小数部分将被舍去。

**提示:**

- `0 <= x <= 2^31 - 1`

## 题解
题目要的是整数部分平方根，也就是找最大的 `k`，满足 `k*k <= x`。这句话本身就很适合二分：可行解区间是连续的。

二分时如果 `mid*mid <= x`，说明 `mid` 可行，先记为当前答案并尝试向右找更大可行值；否则说明太大，向左收缩。循环结束后保存下来的就是最终答案。

同一个文件里还给了牛顿迭代的写法。它通过迭代公式快速逼近平方根，实践里收敛很快，属于数值计算常用技巧。面试里先写二分最稳，熟悉后再补牛顿法会更完整。

代码如下：

```java
public class Main {

    public static int mySqrt(int x) {
        int left = 0;
        int right = x;
        int ans = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if ((long) mid * mid <= x) {
                ans = mid;
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return ans;
    }

    public static int mySqrtNewton(int x) {
        if (x == 0) {
            return 0;
        }

        long k = x;
        while (k * k > x) {
            k = (k + x / k) / 2;
        }

        return (int) k;
    }

    public static void main(String[] args) {
        int x1 = 4;
        System.out.println("示例 1 输入: x = " + x1);
        System.out.println("示例 1 输出: " + mySqrt(x1));
        System.out.println("-----------------------------");

        int x2 = 8;
        System.out.println("示例 2 输入: x = " + x2);
        System.out.println("示例 2 输出: " + mySqrt(x2));
        System.out.println("-----------------------------");

        int x3 = 0;
        System.out.println("示例 3 输入: x = " + x3);
        System.out.println("示例 3 输出: " + mySqrt(x3));
        System.out.println("-----------------------------");

        int x4 = 1;
        System.out.println("示例 4 输入: x = " + x4);
        System.out.println("示例 4 输出: " + mySqrt(x4));
        System.out.println("-----------------------------");

        int x5 = 2147395600;
        System.out.println("示例 5 输入: x = " + x5);
        System.out.println("示例 5 输出: " + mySqrt(x5));
    }
}
```

复杂度分析：二分法时间复杂度 `O(log x)`、空间复杂度 `O(1)`；牛顿法同样是常数空间，迭代次数通常更少。

这题的共通点是“不要硬求精确值，先抓单调性或收敛性”。
