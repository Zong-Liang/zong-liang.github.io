## 题目

**描述：**

给定一个表示 **大整数** 的整数数组 `digits`，其中 `digits[i]` 是整数的第 `i` 位数字。这些数字按从左到右，从最高位到最低位排列。这个大整数不包含任何前导 0。

将大整数加 1，并返回结果的数字数组。

**示例 1:**

- **输入:** `digits = [1,2,3]`
- **输出:** `[1,2,4]`
- **解释:** 输入数组表示数字 123。加 1 后得到 123 + 1 = 124。因此，结果应该是 `[1,2,4]`。

**示例 2:**

- **输入:** `digits = [4,3,2,1]`
- **输出:** `[4,3,2,2]`
- **解释:** 输入数组表示数字 4321。加 1 后得到 4321 + 1 = 4322。因此，结果应该是 `[4,3,2,2]`。

**示例 3:**

- **输入:** `digits = [9]`
- **输出:** `[1,0]`
- **解释:** 输入数组表示数字 9。加 1 后得到 9 + 1 = 10。因此，结果应该是 `[1,0]`。

**提示:**

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` 不包含任何前导 0。

## 题解
这题的本质就是手算加法里的“进位传递”。从最低位开始，如果当前位不是 `9`，直接加一就结束；只有遇到 `9` 才会把当前位置清零并把进位继续往前传。

所以算法可以从后往前扫一遍数组。只要某一位加一后不再进位，立刻返回原数组；如果一路都在进位，说明原数组全是 `9`，最后新建一个长度 `n+1` 的数组，把首位设成 `1`。

这种写法和真实竖式完全一致，边界也自然覆盖：例如 `[1,2,3]` 会在最后一位停止，`[9,9,9]` 会触发扩容。

代码如下：

```java
import java.util.Arrays;

public class Main {

    public static int[] plusOne(int[] digits) {
        int n = digits.length;

        for (int i = n - 1; i >= 0; i--) {
            if (digits[i] < 9) {
                digits[i]++;
                return digits;
            }

            digits[i] = 0;
        }

        int[] newNumber = new int[n + 1];
        newNumber[0] = 1;
        return newNumber;
    }

    public static void main(String[] args) {
        int[] digits1 = {1, 2, 3};
        System.out.println("示例 1 输入: " + Arrays.toString(digits1));
        int[] res1 = plusOne(digits1);
        System.out.println("示例 1 输出: " + Arrays.toString(res1));
        System.out.println("-----------------------------");

        int[] digits2 = {4, 3, 2, 1};
        System.out.println("示例 2 输入: " + Arrays.toString(digits2));
        int[] res2 = plusOne(digits2);
        System.out.println("示例 2 输出: " + Arrays.toString(res2));
        System.out.println("-----------------------------");

        int[] digits3 = {9};
        System.out.println("示例 3 输入: " + Arrays.toString(digits3));
        int[] res3 = plusOne(digits3);
        System.out.println("示例 3 输出: " + Arrays.toString(res3));
        System.out.println("-----------------------------");

        int[] digits4 = {1, 9, 9};
        System.out.println("额外示例 输入: " + Arrays.toString(digits4));
        int[] res4 = plusOne(digits4);
        System.out.println("额外示例 输出: " + Arrays.toString(res4));
    }
}
```

复杂度分析：时间复杂度 `O(n)`，空间复杂度 `O(1)`；仅在全 `9` 情况下需要返回新数组，额外空间为 `O(n)`（输出所需）。

以后碰到“逐位进位/借位”题，先从末位往前想，通常会比从高位正推更直观。
