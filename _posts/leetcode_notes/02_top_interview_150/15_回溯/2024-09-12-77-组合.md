## 题目

**描述：**

给定两个整数 `n` 和 `k`，返回范围 `[1, n]` 中所有可能的 `k` 个数的组合。

你可以按 **任何顺序** 返回答案。**示例 1:**

- **输入:** `n = 4`, `k = 2`
- **输出:**

  ```
  [
    [2,4],
    [3,4],
    [2,3],
    [1,2],
    [1,3],
    [1,4],
  ]
  ```

**示例 2:**

- **输入:** `n = 1`, `k = 1`
- **输出:** `[[1]]`**提示:**

- `1 <= n <= 20`
- `1 <= k <= n`

## 题解


这题要求从 `1..n` 里选出 `k` 个数，最自然的建模就是回溯：路径里放“已经选了哪些数”，递归参数里放“下一次从哪里开始选”。

关键点是 `startIndex`。它保证后续只能选更大的数，所以不会出现 `[1,2]` 和 `[2,1]` 这种顺序不同但本质重复的结果。每次把一个数加入路径，递归下一层，返回后再弹出，继续试下一个候选。

这份代码还做了一个实用剪枝：当前路径长度是 `path.size()`，还需要补 `k - path.size()` 个数，那么循环上界可以直接收紧到 `n - (k - path.size()) + 1`。这样能提前跳过“剩余元素个数不够”的分支。

时间复杂度可以写成 $O(C(n,k)\cdot k)$，因为一共会生成 `C(n,k)` 组结果，每组长度是 `k`；空间复杂度是 $O(k)$（递归深度和路径长度，不含结果集）。

这题的迁移价值在于：组合类题目通常都要靠“起点单调前进 + 回溯撤销”来同时解决不重不漏和去重问题。

完整代码如下：

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;

public class Main {

    static List<List<Integer>> result = new ArrayList<>();
    static LinkedList<Integer> path = new LinkedList<>();

    public static List<List<Integer>> combine(int n, int k) {
        result.clear();
        path.clear();
        backtracking(n, k, 1);
        return result;
    }

    private static void backtracking(int n, int k, int startIndex) {
        if (path.size() == k) {
            result.add(new ArrayList<>(path));
            return;
        }

        int limit = n - (k - path.size()) + 1;
        for (int i = startIndex; i <= limit; i++) {
            path.add(i);
            backtracking(n, k, i + 1);
            path.removeLast();
        }
    }

    public static void main(String[] args) {
        int n1 = 4;
        int k1 = 2;
        System.out.println("示例 1 输入: n = " + n1 + ", k = " + k1);
        List<List<Integer>> res1 = combine(n1, k1);
        System.out.println("示例 1 输出: " + res1);
        System.out.println("-----------------------------");

        int n2 = 1;
        int k2 = 1;
        System.out.println("示例 2 输入: n = " + n2 + ", k = " + k2);
        List<List<Integer>> res2 = combine(n2, k2);
        System.out.println("示例 2 输出: " + res2);
        System.out.println("-----------------------------");

        int n3 = 5;
        int k3 = 4;
        System.out.println("示例 3 输入: n = " + n3 + ", k = " + k3);
        List<List<Integer>> res3 = combine(n3, k3);
        System.out.println("示例 3 输出: " + res3);
    }
}
```
