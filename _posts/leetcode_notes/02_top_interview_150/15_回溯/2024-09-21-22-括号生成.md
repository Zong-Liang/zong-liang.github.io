## 题目

**描述：**

数字 `n` 代表生成括号的对数，请你设计一个函数，用于能够生成所有可能的并且 **有效** 的括号组合。**示例 1:**

- **输入:** `n = 3`
- **输出:** `["((()))","(()())","(())()","()(())","()()()"]`**示例 2:**

- **输入:** `n = 1`
- **输出:** `["()"]`**提示:**

- `1 <= n <= 8`

## 题解


这题看起来是字符串生成，实质是受约束的回溯。

递归里维护三个量：当前字符串 `cur`、已放左括号数 `open`、已放右括号数 `close`。合法扩展规则只有两条：`open < n` 时可以继续放 `'('`，`close < open` 时可以放 `')'`。

第二条约束非常关键，它保证了任意前缀都不会出现右括号多于左括号的非法状态。等 `cur.length() == 2*n` 时，当前字符串必然合法，可以直接加入答案。

时间复杂度和答案规模同阶，可写成 $O(C_n \cdot n)$（`C_n` 是第 `n` 个 Catalan 数）；空间复杂度是 $O(n)$（递归深度，不含结果集）。

这题可以当作回溯剪枝的标准例子：不是先生成再过滤，而是在扩展阶段就把非法分支掐掉。

完整代码如下：

```java
import java.util.ArrayList;
import java.util.List;

public class Main {

    public static List<String> generateParenthesis(int n) {
        List<String> result = new ArrayList<>();
        if (n <= 0) {
            return result;
        }
        backtrack(result, "", 0, 0, n);
        return result;
    }

    private static void backtrack(List<String> result, String cur, int open, int close, int n) {
        if (cur.length() == 2 * n) {
            result.add(cur);
            return;
        }

        if (open < n) {
            backtrack(result, cur + "(", open + 1, close, n);
        }

        if (close < open) {
            backtrack(result, cur + ")", open, close + 1, n);
        }
    }

    public static void main(String[] args) {
        int n1 = 3;
        System.out.println("示例 1 输入: n = " + n1);
        List<String> result1 = generateParenthesis(n1);
        System.out.println("示例 1 输出: " + result1);
        System.out.println("-----------------------------");

        int n2 = 1;
        System.out.println("示例 2 输入: n = " + n2);
        List<String> result2 = generateParenthesis(n2);
        System.out.println("示例 2 输出: " + result2);
    }
}
```
