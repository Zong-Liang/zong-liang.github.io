## 题目

**描述：****n 皇后问题** 研究的是如何将 `n` 个皇后放置在 `n x n` 的棋盘上，并且使皇后彼此之间不能相互攻击。

给你一个整数 `n`，返回 **n 皇后问题** 不同的解决方案的数量。

(注：皇后可以攻击同一行、同一列、以及两条对角线上的任何棋子。)

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2020/11/13/queens.jpg)

- **输入:** `n = 4`
- **输出:** 2
- **解释:** 如上图所示，4 皇后问题存在两个不同的解法。**示例 2:**

- **输入:** `n = 1`
- **输出:** 1**提示:**

- `1 <= n <= 9`

## 题解


N 皇后 II 只要计数，不需要把棋盘字符串都存下来，所以写法可以更轻。

思路是按行放皇后：第 `row` 行枚举每一列 `col`，只在“列、主对角线、副对角线”都没被占用时落子。代码用三个布尔数组维护这三类约束，判断和撤销都能做到 $O(1)$。

当 `row == n` 时，说明前面每一行都合法放置过，形成一个完整解，返回 `1`；否则继续尝试当前行所有可放列，把下一层返回的解数累加起来。

时间复杂度通常按回溯搜索估算为 $O(n!)$ 量级；空间复杂度是 $O(n)$（递归栈和三个标记数组，严格写可看作 $O(n)$ 到 $O(2n)$ 级别）。

这题的要点不是“如何画棋盘”，而是把约束转成常数时间可查询的状态数组，回溯才会跑得动。

完整代码如下：

```java
public class Main {

    public static int totalNQueens(int n) {
        boolean[] cols = new boolean[n];
        boolean[] diag1 = new boolean[2 * n];
        boolean[] diag2 = new boolean[2 * n];

        return backtrack(0, n, cols, diag1, diag2);
    }

    private static int backtrack(int row, int n, boolean[] cols, boolean[] diag1, boolean[] diag2) {
        if (row == n) {
            return 1;
        }

        int count = 0;
        for (int col = 0; col < n; col++) {
            int d1 = row - col + n;
            int d2 = row + col;

            if (cols[col] || diag1[d1] || diag2[d2]) {
                continue;
            }

            cols[col] = true;
            diag1[d1] = true;
            diag2[d2] = true;

            count += backtrack(row + 1, n, cols, diag1, diag2);

            cols[col] = false;
            diag1[d1] = false;
            diag2[d2] = false;
        }

        return count;
    }

    public static void main(String[] args) {
        int n1 = 4;
        System.out.println("示例 1 输入: n = " + n1);
        int result1 = totalNQueens(n1);
        System.out.println("示例 1 输出: " + result1);
        System.out.println("-----------------------------");

        int n2 = 1;
        System.out.println("示例 2 输入: n = " + n2);
        int result2 = totalNQueens(n2);
        System.out.println("示例 2 输出: " + result2);
        System.out.println("-----------------------------");

        int n3 = 8;
        System.out.println("额外测试 输入: n = " + n3);
        int result3 = totalNQueens(n3);
        System.out.println("额外测试 输出: " + result3);
    }
}
```
