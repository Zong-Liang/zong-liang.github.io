## 题目

**描述：**

给你无向 **连通** 图中一个节点的引用，请你返回该图的 **深拷贝**（克隆）。

图中的每个节点都包含它的值 `val` (`int`) 和其邻居的列表 (`list[Node]`)。

```java
class Node {
    public int val;
    public List<Node> neighbors;
}
```

**测试用例格式：**

简单起见，每个节点的值都和它的索引相同。例如，第一个节点值为 1 (`val = 1`)，第二个节点值为 2 (`val = 2`)，以此类推。该图在测试用例中使用邻接列表表示。

**邻接列表** 是用于表示有限图的无序列表的集合。每个列表都描述了图中节点的邻居集。

给定节点将始终是图中的第一个节点（值为 1）。你必须将 **给定节点的拷贝** 作为对克隆图的引用返回。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2019/11/04/133_clone_graph_question.png)

- **输入:** `adjList = [[2,4],[1,3],[2,4],[1,3]]`
- **输出:** `[[2,4],[1,3],[2,4],[1,3]]`

**示例 2:**

- **输入:** `adjList = [[]]`
- **输出:** `[[]]`

**提示:**

- 这张图中的节点数在 `[0, 100]` 之间。
- `1 <= Node.val <= 100`
- 每个节点值 `Node.val` 都是唯一的。
- 图中没有重复的边，也没有自环。
- 图是连通图，你可以从给定节点访问到所有节点。

## 题解

### 这题难的不是“复制”，而是“不能重复复制”

这题表面上是在做图的深拷贝，但真正容易出问题的地方，其实是图里可能有环。

比如节点 `1` 连着节点 `2`，节点 `2` 又连回节点 `1`。  
如果你只是顺着邻居一直递归下去，不记录哪些节点已经处理过，就会在两个节点之间来回打转，最后陷入死循环。

所以这题最关键的不是“怎么新建节点”，而是先想清楚：

**原图里的每个节点，只能克隆一次。**

一旦这个前提抓住了，后面的写法就顺了。

### 为什么一定要用哈希表

假设我们正在克隆某个节点 `node`，有两件事必须同时保证：

1. 这个节点对应的克隆节点只能创建一次
2. 后面如果再次走到这个原节点，必须直接拿到之前已经创建好的克隆节点

这正好就是哈希表最适合做的事。

我们可以用一个 `Map<Node, Node>` 来保存映射关系：

- key：原图中的节点
- value：克隆出来的节点

这样一来，每次处理一个节点时，先查表：

- 如果已经克隆过，直接返回对应的克隆节点
- 如果还没克隆过，就先创建节点，再把映射关系存进去

这个“先存映射，再递归处理邻居”的顺序非常重要。  
因为只有先把当前节点记下来，后面遇到环的时候，才知道该回到哪里。

### 用 DFS 写最自然

DFS 的思路可以概括成一句话：

**先克隆当前节点，再递归克隆它的所有邻居。**

具体过程是这样的：

1. 如果当前节点是 `null`，直接返回 `null`
2. 如果当前节点已经在哈希表里，说明之前克隆过了，直接返回哈希表中的结果
3. 如果没有克隆过，就先创建一个新节点
4. 把“原节点 -> 克隆节点”的映射先存进哈希表
5. 遍历原节点的所有邻居，递归克隆后加入当前克隆节点的邻居列表

这里最容易写错的点，就是第 4 步。

一定要先把当前节点放进哈希表，再去递归邻居。  
否则一旦图里有环，递归绕回来时，程序根本不知道这个节点已经在处理过程中，就会重复创建，甚至死循环。

### 用一个最简单的环来理解

比如示例 1 里的图，本质上是一个四个节点组成的环。

假设从节点 `1` 开始克隆：

- 先创建 `1'`
- 然后去克隆 `2`
- 创建 `2'`
- 再去克隆 `1`

这时候如果没有哈希表，程序会以为 `1` 还没处理过，又去新建一个新的 `1'`，整个结构立刻就乱了。

但如果我们在第一次处理 `1` 时，就已经把 `1 -> 1'` 存进了哈希表，那么后面从 `2` 再走回 `1` 时，直接把原来那个 `1'` 拿出来用就行，不会重复创建。

所以哈希表在这里起的作用，不只是“防止死循环”，更是“保证同一个原节点只对应同一个克隆节点”。

### DFS 写法

```java
private Map<Node, Node> visited = new HashMap<>();

public Node cloneGraph(Node node) {
    if (node == null) {
        return null;
    }

    if (visited.containsKey(node)) {
        return visited.get(node);
    }

    Node cloneNode = new Node(node.val, new ArrayList<>());
    visited.put(node, cloneNode);

    for (Node neighbor : node.neighbors) {
        cloneNode.neighbors.add(cloneGraph(neighbor));
    }

    return cloneNode;
}
```

### 这段代码到底在做什么

可以把它理解成两层工作。

第一层，是“复制节点本身”。  
也就是把 `val` 复制过去，并且创建一个新的邻居列表。

第二层，是“重建节点之间的连接关系”。  
遍历原图中当前节点的所有邻居，把它们各自克隆出来，再挂到当前克隆节点的邻居列表里。

这样递归下去，整个图就被完整复制出来了。

这里要注意一件事：  
我们不是只复制节点值，而是连图里的连接关系一起复制。  
这也是为什么题目强调的是**深拷贝**，而不是简单地新建几个值相同的节点。

### 再看复杂度

时间复杂度是 $O(N + E)$。

因为每个节点最多只会真正克隆一次，每条边也只会在遍历邻居时处理一次。  
这里的 $N$ 是节点数，$E$ 是边数。

空间复杂度是 $O(N)$。

主要来自两部分：

- 哈希表要保存所有原节点和克隆节点的映射
- 递归调用栈在最坏情况下可能达到 $O(N)$

所以整体空间复杂度记作 $O(N)$。

### BFS 也能做

如果不想写递归，也可以用 BFS。

思路其实没变，还是一样要用哈希表记录“原节点 -> 克隆节点”的映射。  
只不过 DFS 是递归一路往下走，BFS 则是用队列一层一层处理。

BFS 的过程可以理解成：

1. 先克隆起点，并放进队列
2. 每次从队列里取出一个原节点
3. 遍历它的邻居
4. 如果某个邻居还没克隆过，就先创建出来并入队
5. 然后把这个邻居对应的克隆节点，加入当前克隆节点的邻居列表

写起来也不难，只是代码会比 DFS 稍微绕一点。

### BFS 写法

```java
public Node cloneGraph(Node node) {
    if (node == null) {
        return null;
    }

    Map<Node, Node> visited = new HashMap<>();
    Queue<Node> queue = new LinkedList<>();

    Node cloneNode = new Node(node.val, new ArrayList<>());
    visited.put(node, cloneNode);
    queue.offer(node);

    while (!queue.isEmpty()) {
        Node curr = queue.poll();

        for (Node neighbor : curr.neighbors) {
            if (!visited.containsKey(neighbor)) {
                visited.put(neighbor, new Node(neighbor.val, new ArrayList<>()));
                queue.offer(neighbor);
            }
            visited.get(curr).neighbors.add(visited.get(neighbor));
        }
    }

    return visited.get(node);
}
```

### 这题真正要记住的是什么

这题表面上是图论题，但最值得记住的，其实是一个很通用的处理方式：

**当题目要求“复制一个带引用关系的结构”时，往往都需要先建立“原对象 -> 新对象”的映射。**

图是这样，链表随机指针那类题也是这样。  
因为只要结构里可能出现回头路、共享节点或者重复访问，单纯递归创建就很容易出问题。

所以以后再碰到类似“深拷贝复杂结构”的题，可以先问自己一句：

**我是不是应该先建一张映射表？**

很多时候，关键就卡在这里。

### 完整代码

```java
import java.util.*;

public class Main {

    static class Node {
        public int val;
        public List<Node> neighbors;

        public Node() {
            val = 0;
            neighbors = new ArrayList<Node>();
        }

        public Node(int _val) {
            val = _val;
            neighbors = new ArrayList<Node>();
        }

        public Node(int _val, ArrayList<Node> _neighbors) {
            val = _val;
            neighbors = _neighbors;
        }
    }

    private Map<Node, Node> visited = new HashMap<>();

    public Node cloneGraph(Node node) {
        if (node == null) {
            return null;
        }

        if (visited.containsKey(node)) {
            return visited.get(node);
        }

        Node cloneNode = new Node(node.val, new ArrayList<>());
        visited.put(node, cloneNode);

        for (Node neighbor : node.neighbors) {
            cloneNode.neighbors.add(cloneGraph(neighbor));
        }

        return cloneNode;
    }

    public static void printGraph(Node node) {
        if (node == null) {
            return;
        }

        Set<Node> visitedPrint = new HashSet<>();
        Queue<Node> queue = new LinkedList<>();
        queue.offer(node);
        visitedPrint.add(node);

        while (!queue.isEmpty()) {
            Node curr = queue.poll();
            System.out.print("Node " + curr.val + " neighbors: [");
            for (int i = 0; i < curr.neighbors.size(); i++) {
                System.out.print(curr.neighbors.get(i).val);
                if (i < curr.neighbors.size() - 1) {
                    System.out.print(", ");
                }
            }
            System.out.println("]");

            for (Node neighbor : curr.neighbors) {
                if (!visitedPrint.contains(neighbor)) {
                    visitedPrint.add(neighbor);
                    queue.offer(neighbor);
                }
            }
        }
    }

    public static void main(String[] args) {
        Main solution = new Main();

        Node n1 = new Node(1);
        Node n2 = new Node(2);
        Node n3 = new Node(3);
        Node n4 = new Node(4);

        n1.neighbors.add(n2);
        n1.neighbors.add(n4);

        n2.neighbors.add(n1);
        n2.neighbors.add(n3);

        n3.neighbors.add(n2);
        n3.neighbors.add(n4);

        n4.neighbors.add(n1);
        n4.neighbors.add(n3);

        System.out.println("示例 1 原图结构:");
        printGraph(n1);

        Node clonedGraph1 = solution.cloneGraph(n1);

        System.out.println();
        System.out.println("示例 1 克隆图结构:");
        printGraph(clonedGraph1);

        System.out.println();
        System.out.println("验证是否为深拷贝:");
        System.out.println("n1 == clonedGraph1 ? " + (n1 == clonedGraph1));
        System.out.println("-----------------------------");

        Main solution2 = new Main();
        Node emptyResult = solution2.cloneGraph(null);
        System.out.println("示例 2 输出: " + emptyResult);
    }
}
```
