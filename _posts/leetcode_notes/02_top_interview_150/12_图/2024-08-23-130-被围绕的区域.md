## 题目

**描述：**

给你一个 `m x n` 的矩阵 `board`，由若干字符 `'X'` 和 `'O'` 组成，捕获所有 **被围绕的区域**：

- **连接：** 一个单元格与水平或垂直方向上相邻的单元格连接。
- **区域：** 连接所有 `'O'` 的单元格来形成一个区域。
- **围绕：** 如果您可以用 `'X'` 单元格 **连接** 这个区域，并且区域中没有任何单元格位于 `board` **边缘**，则该区域被 `'X'` 单元格围绕。

通过 **原地** 将输入矩阵中的所有 `'O'` 替换为 `'X'` 来捕获被围绕的区域。你不需要返回任何值。

**示例 1:**![示例1图片](https://assets.leetcode.com/uploads/2021/02/19/xogrid.jpg)

- **输入:** `board = [["X","X","X","X"],["X","O","O","X"],["X","X","O","X"],["X","O","X","X"]]`
- **输出:** `[["X","X","X","X"],["X","X","X","X"],["X","X","X","X"],["X","O","X","X"]]`

**示例 2:**

- **输入:** `board = [["X"]]`
- **输出:** `[["X"]]`

**提示:**

- `m == board.length`
- `n == board[i].length`
- `1 <= m, n <= 200`
- `board[i][j]` 为 `'X'` 或 `'O'`

## 题解

### 这题别顺着做，要反过来想

这题乍一看，很容易想到去找哪些 `'O'` 被 `'X'` 包围了，然后把它们改成 `'X'`。

但真这么做，其实不太好下手。因为你很难只看某一个 `'O'`，就立刻判断它所在的整块区域到底会不会连到边界。

这题真正好做的地方在于：

**所有和边界连通的 `'O'`，一定不能被改掉。**

换句话说，真正要找的不是“哪些 `'O'` 会被填掉”，而是“哪些 `'O'` 一定安全”。

只要先把这些安全的 `'O'` 全部找出来，剩下那些还没被标记到的 `'O'`，自然就是被围住的区域。

### 为什么从边界出发就够了

题目里说得很清楚：如果一个区域中没有任何格子位于边缘，并且整个区域都被 `'X'` 包住，那它才算被围绕。

这句话可以换个说法理解：

- 只要一个 `'O'` 能直接或间接走到边界，它就不会被替换
- 只有那些完全走不到边界的 `'O'`，最后才会变成 `'X'`

所以思路就很明确了：

1. 先从四条边出发，把所有和边界连通的 `'O'` 都找出来
2. 这些 `'O'` 先做个临时标记
3. 最后再扫一遍整张图：
   - 还保持 `'O'` 的，说明它不连边界，要改成 `'X'`
   - 临时标记过的，说明它是安全的，再改回 `'O'`

这就是这题最关键的一步。

### 用示例 1 看一下

示例 1 是：

```text
board = [
["X","X","X","X"],
["X","O","O","X"],
["X","X","O","X"],
["X","O","X","X"]
]
```

先看边界上的 `'O'`。

这里边界上只有最后一行第二列那个 `'O'`。它和边界直接相连，所以它一定不能被改掉。  
从它出发继续搜索，会发现它周围并没有别的 `'O'` 连在一起，所以这一块安全区域只有它自己。

再看中间那三个 `'O'`：

```text
["X","O","O","X"]
["X","X","O","X"]
```

它们虽然彼此连通，但整块都被包在内部，走不到边界，所以最后都要改成 `'X'`。

因此最终结果就是：

```text
[
["X","X","X","X"],
["X","X","X","X"],
["X","X","X","X"],
["X","O","X","X"]
]
```

你会发现，这题一旦转成“先找安全区”，逻辑就顺很多了。

### 做法其实只有两步

整道题可以拆成很清楚的两步。

第一步，从边界出发，把所有安全的 `'O'` 找出来。  
为了不和原来的 `'O'` 混在一起，可以先把它们改成一个临时字符，比如 `'A'`。

第二步，遍历整张图：

- 遇到 `'O'`，说明它没有连到边界，改成 `'X'`
- 遇到 `'A'`，说明它本来是安全的，再还原成 `'O'`

这样处理完之后，答案就出来了。

### DFS 为什么正好适合这题

从某个边界上的 `'O'`` 出发，我们需要把和它连通的整块区域都找出来。  
这种“从一个点出发，把整片连通块全部走完”的事，本来就是 DFS 最擅长的场景。

DFS 的终止条件也很简单：

- 越界了，返回
- 当前不是 `'O'`，返回

只要当前位置还是 `'O'`，就先把它标记成 `'A'`，再去递归访问上下左右四个方向。

### DFS 写法

```java
public void solve(char[][] board) {
    if (board == null || board.length == 0) {
        return;
    }

    int m = board.length;
    int n = board[0].length;

    for (int i = 0; i < m; i++) {
        dfs(board, i, 0);
        dfs(board, i, n - 1);
    }

    for (int j = 1; j < n - 1; j++) {
        dfs(board, 0, j);
        dfs(board, m - 1, j);
    }

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (board[i][j] == 'O') {
                board[i][j] = 'X';
            } else if (board[i][j] == 'A') {
                board[i][j] = 'O';
            }
        }
    }
}

private void dfs(char[][] board, int r, int c) {
    if (r < 0 || c < 0 || r >= board.length || c >= board[0].length || board[r][c] != 'O') {
        return;
    }

    board[r][c] = 'A';

    dfs(board, r - 1, c);
    dfs(board, r + 1, c);
    dfs(board, r, c - 1);
    dfs(board, r, c + 1);
}
```

### 这段代码在做什么

外层的两组循环只做一件事：  
**枚举所有边界位置。**

只要边界上有 `'O'`，就从那里启动 DFS，把和它连通的整块安全区域全部标成 `'A'`。

等所有边界都处理完之后，再扫一遍整张图：

- 没被标记过的 `'O'`，说明它不安全，直接变成 `'X'`
- 被标记过的 `'A'`，说明它是安全区的一部分，还原成 `'O'`

整个过程没有真的去判断“这一块是不是被围住”，而是用“能不能从边界走到它”这个条件，把问题换了个更容易处理的角度。

### 再看复杂度

时间复杂度是 $O(m \times n)$。

因为每个格子最多只会被处理有限次：  
要么在边界搜索时被标记一次，要么在最后统一遍历时被改一次。

空间复杂度在最坏情况下是 $O(m \times n)$。  
这个主要来自递归调用栈。比如整张图全是 `'O'`，DFS 递归深度就可能很大。

### BFS 也可以做

这题用 BFS 完全没问题，思路和 DFS 一模一样。

区别只是：

- DFS 用递归，把连通区域一口气往下走
- BFS 用队列，一层一层往外扩展

本质上做的事情没有变，都是从边界上的 `'O'` 出发，把所有安全区域先标记出来。

如果你担心递归层数太深，BFS 会更稳一些；如果只是写题，DFS 一般更直接。

### 这题真正要记住的点

这题最值得记住的，不是“把 `'O'` 改成 `'X'`”这个操作本身，而是这种想法：

**当正面判断很难时，可以反过来找那些一定不满足条件的部分。**

这道题里：

- 直接找“被围住的区域”不太好找
- 但找“和边界连通的安全区域”却很容易

一旦把安全区先保护起来，剩下的部分就不用再判断了，直接处理掉就行。

以后再遇到这种“图中保留一部分、删除一部分”的题，可以先想一想：  
是不是也能从“必然保留的部分”入手，反着做。

### 完整代码

```java
import java.util.Arrays;

public class Main {

    public static void solve(char[][] board) {
        if (board == null || board.length == 0) {
            return;
        }

        int m = board.length;
        int n = board[0].length;

        for (int i = 0; i < m; i++) {
            dfs(board, i, 0);
            dfs(board, i, n - 1);
        }

        for (int j = 1; j < n - 1; j++) {
            dfs(board, 0, j);
            dfs(board, m - 1, j);
        }

        for (int i = 0; i < m; i++) {
            for (int j = 0; j < n; j++) {
                if (board[i][j] == 'O') {
                    board[i][j] = 'X';
                } else if (board[i][j] == 'A') {
                    board[i][j] = 'O';
                }
            }
        }
    }

    private static void dfs(char[][] board, int r, int c) {
        if (r < 0 || c < 0 || r >= board.length || c >= board[0].length || board[r][c] != 'O') {
            return;
        }

        board[r][c] = 'A';

        dfs(board, r - 1, c);
        dfs(board, r + 1, c);
        dfs(board, r, c - 1);
        dfs(board, r, c + 1);
    }

    public static void printBoard(char[][] board) {
        for (char[] row : board) {
            System.out.println(Arrays.toString(row));
        }
    }

    public static void main(String[] args) {
        char[][] board1 = {
            {'X', 'X', 'X', 'X'},
            {'X', 'O', 'O', 'X'},
            {'X', 'X', 'O', 'X'},
            {'X', 'O', 'X', 'X'}
        };

        System.out.println("示例 1 输入:");
        printBoard(board1);

        solve(board1);

        System.out.println("示例 1 输出:");
        printBoard(board1);
        System.out.println("-----------------------------");

        char[][] board2 = {
            {'X'}
        };

        System.out.println("示例 2 输入:");
        printBoard(board2);

        solve(board2);

        System.out.println("示例 2 输出:");
        printBoard(board2);
        System.out.println("-----------------------------");

        char[][] board3 = {
            {'O', 'O', 'O'},
            {'O', 'O', 'O'},
            {'O', 'O', 'O'}
        };

        System.out.println("示例 3 输入:");
        printBoard(board3);

        solve(board3);

        System.out.println("示例 3 输出:");
        printBoard(board3);
    }
}
```
