## 题目

**描述：**

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

**示例 1:**

- **输入:** `s = "abcabcbb"`
- **输出:** 3
- **解释:** 因为无重复字符的最长子串是 `"abc"`，所以其长度为 3。注意 `"bca"` 和 `"cab"` 也是正确答案。

**示例 2:**

- **输入:** `s = "bbbbb"`
- **输出:** 1
- **解释:** 因为无重复字符的最长子串是 `"b"`，所以其长度为 1。

**示例 3:**

- **输入:** `s = "pwwkew"`
- **输出:** 3
- **解释:** 因为无重复字符的最长子串是 `"wke"`，所以其长度为 3。请注意，你的答案必须是 **子串** 的长度，`"pwke"` 是一个子序列，不是子串。

**提示:**

- `0 <= s.length <= 5 * 10^4`
- `s` 由英文字母、数字、符号和空格组成

## 题解

这题先抓住两件事：它找的是“最长”，而且找的是“子串”。“子串”意味着答案必须是一段连续区间，所以它天然就很像滑动窗口题；难点不在于要不要开窗口，而在于窗口里一旦出现重复字符，怎么又快又准地把它修回合法状态。

如果用最直白的方法做，当然可以枚举每个起点，再不断向右扩，直到遇到重复字符为止。问题是这样会反复检查很多已经看过的字符，尤其当字符串很长时，重复劳动会非常多。更顺手的写法，是始终维护一个“当前没有重复字符的窗口”，让这个窗口在字符串上持续向右滑。

于是我们用 `left` 和 `right` 表示当前窗口，再用一个哈希表记录每个字符最近一次出现的位置。`right` 每向右走一步，都会读入一个新字符 `c`。如果 `c` 之前没有在当前窗口里出现过，那窗口仍然合法；如果出现过，问题就出在它上一次出现的位置上，这时左边界不需要一格一格地慢慢试，而是可以直接跳到 `c` 上一次出现位置的后一位。

这里最关键的细节，是左边界只能向右走，不能后退。所以更新时一定要写成 `left = Math.max(left, map.get(c) + 1)`。`"abba"` 是最容易看出这个点的例子：当前面那个 `a` 早就已经被窗口甩掉以后，最后再遇到 `a`，你不能因为它在哈希表里出现过，就把 `left` 拉回去。`Math.max` 的作用就是保证窗口只会继续向右滑，不会倒退。

拿 `"abcabcbb"` 来看，窗口一开始会顺利扩成 `"abc"`；当 `right` 走到第二个 `a` 时，说明 `a` 重复了，于是 `left` 直接跳到第一个 `a` 后面，窗口变成 `"bca"`。后面每次遇到重复字符，都是同样的处理逻辑：右边继续扩，左边在必要时直接跳到冲突字符的后一位，窗口始终保持无重复。

代码如下：

```java
public int lengthOfLongestSubstring(String s) {
    Map<Character, Integer> map = new HashMap<>();
    int maxLen = 0;
    int left = 0;

    for (int right = 0; right < s.length(); right++) {
        char c = s.charAt(right);

        if (map.containsKey(c)) {
            left = Math.max(left, map.get(c) + 1);
        }

        map.put(c, right);
        maxLen = Math.max(maxLen, right - left + 1);
    }

    return maxLen;
}
```

时间复杂度是 `O(n)`，因为左右指针都只会单向移动，哈希表查询和更新平均是 `O(1)`；空间复杂度是 `O(Σ)`，其中 `Σ` 表示字符集大小。

这题值得顺手记住的，不只是“滑动窗口 + 哈希表”，而是窗口修复的那种方式：一旦知道冲突点在哪里，左边界就没必要一点点试，可以直接跳过去。这种“窗口失效后直接定位冲突位置”的思路，在很多字符串题里都非常常见。

完整代码如下：

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    public static int lengthOfLongestSubstring(String s) {
        Map<Character, Integer> map = new HashMap<>();
        int maxLen = 0;
        int left = 0;

        for (int right = 0; right < s.length(); right++) {
            char c = s.charAt(right);

            if (map.containsKey(c)) {
                left = Math.max(left, map.get(c) + 1);
            }

            map.put(c, right);
            maxLen = Math.max(maxLen, right - left + 1);
        }

        return maxLen;
    }

    public static void main(String[] args) {
        String s1 = "abcabcbb";
        System.out.println("示例 1 输入: s = \"" + s1 + "\"");
        System.out.println("示例 1 输出: " + lengthOfLongestSubstring(s1));
        System.out.println("-----------------------------");

        String s2 = "bbbbb";
        System.out.println("示例 2 输入: s = \"" + s2 + "\"");
        System.out.println("示例 2 输出: " + lengthOfLongestSubstring(s2));
        System.out.println("-----------------------------");

        String s3 = "pwwkew";
        System.out.println("示例 3 输入: s = \"" + s3 + "\"");
        System.out.println("示例 3 输出: " + lengthOfLongestSubstring(s3));
    }
}
```
