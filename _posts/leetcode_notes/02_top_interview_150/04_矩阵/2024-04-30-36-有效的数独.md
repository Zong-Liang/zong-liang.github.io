## 题目

**描述：**

请你判断一个 `9 x 9` 的数独是否有效。只需要 **根据以下规则** ，验证已经填入的数字是否有效即可。

1.  数字 `1-9` 在每一行只能出现一次。
2.  数字 `1-9` 在每一列只能出现一次。
3.  数字 `1-9` 在每一个以粗实线分隔的 `3x3` 宫内只能出现一次。(请参考示例图)

**注意：**

- 一个有效的数独（部分已被填充）不一定是可解的。
- 只需要根据以上规则，验证已经填入的数字是否有效即可。
- 空白格用 `'.'` 表示。

**示例 1:**

- **输入:** `board` (一个有效的 9x9 数独板)
- **输出:** `true`

**示例 2:**

- **输入:** `board` (一个无效的 9x9 数独板，左上角 3x3 宫内有两个 8)
- **输出:** `false`

**提示：**

- `board.length == 9`
- `board[i].length == 9`
- `board[i][j]` 是一位数字（`1-9`）或者 `.`

## 题解

这题虽然名字里带着“数独”，但它和回溯求解其实没什么关系。题目并没有让我们把空位填出来，也不关心这盘数独最后有没有解，它只是在问：当前已经填进去的这些数字，彼此之间有没有冲突。把这点先想清楚，这题就会从“搜索题”一下子变成很直接的状态记录题。

所谓合法，检查的也只有三件事：同一行里不能重复、同一列里不能重复、同一个 `3 x 3` 宫里不能重复。所以最自然的写法，就是一边扫描棋盘，一边把已经出现过的数字记下来。为此可以准备三张表：`rows[i][num]` 表示数字 `num` 是否在第 `i` 行出现过，`cols[j][num]` 表示是否在第 `j` 列出现过，`boxes[k][num]` 表示是否在第 `k` 个小宫里出现过。

这样一来，遍历到某个已填数字时，先看它在对应的行、列、宫里是不是已经出现过；只要有一个地方出现过，当前棋盘就不合法，可以立刻返回 `false`。如果三个位置都没出现过，再把它标进去，继续往后扫。整道题的节奏其实很简单：看到一个数，就检查它会不会和前面已经出现过的数撞车。

这里唯一需要停一下的是 `3 x 3` 宫怎么编号。最常用的写法是 `boxIndex = (i / 3) * 3 + (j / 3)`。`i / 3` 决定它落在第几排宫，`j / 3` 决定它落在第几列宫，把这两个量合起来，就能把九个小宫编号成 `0` 到 `8`。比如 `(4, 7)` 这个位置，`i / 3 = 1`，`j / 3 = 2`，所以它属于编号 `5` 的那个宫。

代码如下：

```java
public boolean isValidSudoku(char[][] board) {
    boolean[][] rows = new boolean[9][9];
    boolean[][] cols = new boolean[9][9];
    boolean[][] boxes = new boolean[9][9];

    for (int i = 0; i < 9; i++) {
        for (int j = 0; j < 9; j++) {
            if (board[i][j] == '.') {
                continue;
            }

            int num = board[i][j] - '1';
            int boxIndex = (i / 3) * 3 + j / 3;

            if (rows[i][num] || cols[j][num] || boxes[boxIndex][num]) {
                return false;
            }

            rows[i][num] = true;
            cols[j][num] = true;
            boxes[boxIndex][num] = true;
        }
    }

    return true;
}
```

如果把棋盘大小视为固定的 `9 x 9`，时间复杂度和空间复杂度都可以看成 `O(1)`；如果抽象成一般的 `N x N` 棋盘，则时间复杂度是 `O(N^2)`。

这题比较适合留下来的，不是某个数独技巧，而是一种判断方式：当规则本质上是在检查“一个元素是否在多个维度上重复出现”时，往往可以为每个维度各准备一份记录表，然后边扫描边判重。题目一下就会从“推理题”变成很朴素的实现题。

完整代码如下：

```java
public class Main {

    public static boolean isValidSudoku(char[][] board) {
        boolean[][] rows = new boolean[9][9];
        boolean[][] cols = new boolean[9][9];
        boolean[][] boxes = new boolean[9][9];

        for (int i = 0; i < 9; i++) {
            for (int j = 0; j < 9; j++) {
                char c = board[i][j];

                if (c == '.') {
                    continue;
                }

                int num = c - '1';
                int boxIndex = (i / 3) * 3 + j / 3;

                if (rows[i][num] || cols[j][num] || boxes[boxIndex][num]) {
                    return false;
                }

                rows[i][num] = true;
                cols[j][num] = true;
                boxes[boxIndex][num] = true;
            }
        }

        return true;
    }

    public static void main(String[] args) {
        char[][] board1 = {
            {'5','3','.','.','7','.','.','.','.'},
            {'6','.','.','1','9','5','.','.','.'},
            {'.','9','8','.','.','.','.','6','.'},
            {'8','.','.','.','6','.','.','.','3'},
            {'4','.','.','8','.','3','.','.','1'},
            {'7','.','.','.','2','.','.','.','6'},
            {'.','6','.','.','.','.','2','8','.'},
            {'.','.','.','4','1','9','.','.','5'},
            {'.','.','.','.','8','.','.','7','9'}
        };

        System.out.println("示例 1 输出: " + isValidSudoku(board1));
        System.out.println("-----------------------------");

        char[][] board2 = {
            {'5','3','.','.','7','.','.','.','5'},
            {'6','.','.','1','9','5','.','.','.'},
            {'.','9','8','.','.','.','.','6','.'},
            {'8','.','.','.','6','.','.','.','3'},
            {'4','.','.','8','.','3','.','.','1'},
            {'7','.','.','.','2','.','.','.','6'},
            {'.','6','.','.','.','.','2','8','.'},
            {'.','.','.','4','1','9','.','.','5'},
            {'.','.','.','.','8','.','.','7','9'}
        };

        System.out.println("示例 2 输出: " + isValidSudoku(board2));
    }
}
```
