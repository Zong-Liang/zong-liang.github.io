## 题目

**描述：**

给定一个 `m x n` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 **原地** 算法。

**示例 1:**

- **输入:** `matrix = [[1,1,1],[1,0,1],[1,1,1]]`
- **输出:** `[[1,0,1],[0,0,0],[1,0,1]]`

**示例 2:**

- **输入:** `matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]`
- **输出:** `[[0,0,0,0],[0,4,5,0],[0,3,1,0]]`

**提示:**

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

**进阶:**

- 一个直观的解决方案是使用 `O(mn)` 的额外空间，但这并不是一个好的解决方案。
- 一个简单的改进方案是使用 `O(m + n)` 的额外空间，但这仍然不是最好的解决方案。
- 你能想出一个仅使用常量空间的解决方案吗？

## 题解

这题最容易踩的坑，其实不在“置零”这个动作本身，而在于一旦你边扫边改，后面的判断就会被自己刚写进去的 `0` 污染。比如某个位置本来不是 `0`，只是因为它所在的行列之后需要清零才被改成了 `0`，那你后面再扫到它时，就会误以为这里原本也有一个 `0`，从而把更多不该清零的行列一起带进去。

所以这题第一步要先想清楚：原始的 `0` 和后来补出来的 `0` 必须分开。最直观的写法，是先用两个辅助数组把“哪些行要清零、哪些列要清零”记下来，等第一遍扫描结束，再做第二遍统一置零。这个版本思路最顺，但额外空间是 `O(m + n)`。

题目后面追问常数空间，关键就在于：能不能把这两张标记表塞回矩阵自己。比较巧的办法是，拿第一行当列标记区，拿第一列当行标记区。扫描内部区域时，只要发现 `matrix[i][j] == 0`，就先把 `matrix[i][0]` 和 `matrix[0][j]` 置成 `0`，相当于记下“第 `i` 行要清零”和“第 `j` 列要清零”。

这里唯一需要额外处理的是 `matrix[0][0]`。因为它同时属于第一行和第一列，如果只靠这一个格子，就没法分清到底是第一行要清零、第一列要清零，还是两者都要。所以要先单独用两个布尔变量记住：第一行原本有没有 `0`，第一列原本有没有 `0`。注意这里说的是“原本”，因为后面第一行和第一列会被当成标记区使用。

整个原地过程其实分得很清楚。先扫描第一行、第一列，记录 `firstRowHasZero` 和 `firstColHasZero`；再扫描内部区域，把需要清零的行列信息写回第一列和第一行；然后根据这些标记去处理内部元素；最后再看那两个布尔变量，决定第一行和第一列本身要不要清零。这个顺序不能乱，因为如果太早把第一行或第一列清掉，前面存的标记也就一起没了。

拿 `[[1,1,1],[1,0,1],[1,1,1]]` 这个例子来看，内部的 `0` 会先把它所在的行首和列首都标成 `0`，也就是把“第二行要清零”“第二列要清零”这两个信息存起来。后面再按标记去清理内部区域，最后得到的就是 `[[1,0,1],[0,0,0],[1,0,1]]`。

代码如下：

```java
public void setZeroes(int[][] matrix) {
    int m = matrix.length;
    int n = matrix[0].length;

    boolean firstRowHasZero = false;
    boolean firstColHasZero = false;

    for (int j = 0; j < n; j++) {
        if (matrix[0][j] == 0) {
            firstRowHasZero = true;
            break;
        }
    }

    for (int i = 0; i < m; i++) {
        if (matrix[i][0] == 0) {
            firstColHasZero = true;
            break;
        }
    }

    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            if (matrix[i][j] == 0) {
                matrix[i][0] = 0;
                matrix[0][j] = 0;
            }
        }
    }

    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                matrix[i][j] = 0;
            }
        }
    }

    if (firstRowHasZero) {
        for (int j = 0; j < n; j++) {
            matrix[0][j] = 0;
        }
    }

    if (firstColHasZero) {
        for (int i = 0; i < m; i++) {
            matrix[i][0] = 0;
        }
    }
}
```

时间复杂度是 `O(mn)`，因为本质上还是扫矩阵若干遍；空间复杂度是 `O(1)`，因为额外只用了两个布尔变量。

这题很适合顺手记住一种很常见的“原地标记”思路：当题目要求原地处理，但中途又需要保存标记信息时，可以先看看输入本身有没有某一部分位置能够借出来当标记区。这题就是把两张辅助表折叠进了第一行和第一列。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        boolean firstRowHasZero = false;
        boolean firstColHasZero = false;

        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                firstRowHasZero = true;
                break;
            }
        }

        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                firstColHasZero = true;
                break;
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        if (firstRowHasZero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }

        if (firstColHasZero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }

    public static void main(String[] args) {
        int[][] matrix1 = {
            {1, 1, 1},
            {1, 0, 1},
            {1, 1, 1}
        };

        System.out.println("示例 1 输入:");
        printMatrix(matrix1);

        setZeroes(matrix1);

        System.out.println("示例 1 输出:");
        printMatrix(matrix1);
        System.out.println("-----------------------------");

        int[][] matrix2 = {
            {0, 1, 2, 0},
            {3, 4, 5, 2},
            {1, 3, 1, 5}
        };

        System.out.println("示例 2 输入:");
        printMatrix(matrix2);

        setZeroes(matrix2);

        System.out.println("示例 2 输出:");
        printMatrix(matrix2);
    }

    private static void printMatrix(int[][] matrix) {
        System.out.println("[");
        for (int[] row : matrix) {
            System.out.println("  " + Arrays.toString(row));
        }
        System.out.println("]");
    }
}
```
