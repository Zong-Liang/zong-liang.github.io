## 题目

**描述：**

给定一个 `m x n` 的矩阵，如果一个元素为 **0** ，则将其所在行和列的所有元素都设为 **0** 。请使用 **原地** 算法。

**示例 1:**

- **输入:** `matrix = [[1,1,1],[1,0,1],[1,1,1]]`
- **输出:** `[[1,0,1],[0,0,0],[1,0,1]]`

**示例 2:**

- **输入:** `matrix = [[0,1,2,0],[3,4,5,2],[1,3,1,5]]`
- **输出:** `[[0,0,0,0],[0,4,5,0],[0,3,1,0]]`

**提示:**

- `m == matrix.length`
- `n == matrix[0].length`
- `1 <= m, n <= 200`
- `-2^31 <= matrix[i][j] <= 2^31 - 1`

**进阶:**

- 一个直观的解决方案是使用 `O(mn)` 的额外空间，但这并不是一个好的解决方案。
- 一个简单的改进方案是使用 `O(m + n)` 的额外空间，但这仍然不是最好的解决方案。
- 你能想出一个仅使用常量空间的解决方案吗？

## 题解

### 这题最容易踩的坑，是边遍历边置零

这题表面看起来不难：只要遇到一个 `0`，就把它所在行和所在列全部改成 `0`。

但真要直接这么写，很快就会出问题。

因为你一旦在遍历过程中立刻把一整行、一整列改成 `0`，后面再扫到这些新产生的 `0` 时，就分不清它们到底是原来就有的，还是你刚刚改出来的。这样会把本来不该清零的位置也连带污染掉。

所以这题真正的关键不是“怎么置零”，而是先把另一件事想清楚：

**哪些行要清零，哪些列要清零，必须先记录下来，再统一处理。**

### 最自然的想法，是先把要清零的行和列记下来

最直接的做法，是准备两个辅助数组：

- `rowFlags[i]` 表示第 `i` 行是否需要置零
- `colFlags[j]` 表示第 `j` 列是否需要置零

先完整扫一遍矩阵：

- 如果 `matrix[i][j] == 0`，就把 `rowFlags[i]` 和 `colFlags[j]` 标记为 `true`

然后再扫第二遍：

- 只要当前元素所在行或所在列被标记过，就把它置成 `0`

这个写法其实很顺，思路也最清楚，因为它把“找哪些地方要清零”和“真正去清零”完全分开了。

### 先把基础解法写明白

```java
public void setZeroes(int[][] matrix) {
    int m = matrix.length;
    int n = matrix[0].length;
    boolean[] rowFlags = new boolean[m];
    boolean[] colFlags = new boolean[n];

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (matrix[i][j] == 0) {
                rowFlags[i] = true;
                colFlags[j] = true;
            }
        }
    }

    for (int i = 0; i < m; i++) {
        for (int j = 0; j < n; j++) {
            if (rowFlags[i] || colFlags[j]) {
                matrix[i][j] = 0;
            }
        }
    }
}
```

这个方法的时间复杂度是 $O(mn)$，空间复杂度是 $O(m+n)$。

如果只是为了把题做出来，这已经完全够用了。  
但题目后面明确问了进阶：能不能做到常数空间？

这就意味着，连这两个辅助数组最好也别用。

### 真正的难点，是怎么把标记“塞回矩阵自己”

既然不能额外开 `rowFlags` 和 `colFlags`，那就得想办法把“某一行要不要清零”“某一列要不要清零”这些信息，直接存在原矩阵里。

最自然的位置，其实就是：

- 用每一行的第一个元素，记录这一行是否需要清零
- 用每一列的第一个元素，记录这一列是否需要清零

也就是：

- `matrix[i][0]` 当作第 `i` 行的标记位
- `matrix[0][j]` 当作第 `j` 列的标记位

这样一来，如果某个位置 `matrix[i][j] == 0`，我们就不急着立刻清零整行整列，而是先做两件事：

```text
matrix[i][0] = 0
matrix[0][j] = 0
```

意思就是：第 `i` 行以后要清零，第 `j` 列以后也要清零。

### 这里有个关键冲突：第一行和第一列共用了一个格子

这个思路看起来很好，但马上会遇到一个问题：

```text
matrix[0][0]
```

这个位置既属于第一行，又属于第一列。  
如果它被改成了 `0`，你就没法分清楚：

- 是第一行原本应该清零
- 还是第一列原本应该清零
- 还是两者都应该清零

所以第一行和第一列不能只靠 `matrix[0][0]` 这一个格子判断，必须额外单独记录。

通常的做法是再准备两个布尔变量：

- `firstRowHasZero`：第一行原本是否有 `0`
- `firstColHasZero`：第一列原本是否有 `0`

注意这里强调的是“原本”。  
因为后面第一行、第一列会被拿来当标记区使用，所以必须在改动之前，先把这两个信息记下来。

### 整个过程其实分成四步

#### 第一步：先判断第一行和第一列原本有没有 0

这是为了避免后面标记过程把它们原始状态覆盖掉。

#### 第二步：扫描除第一行、第一列之外的区域

如果 `matrix[i][j] == 0`，就把：

- `matrix[i][0] = 0`
- `matrix[0][j] = 0`

把对应行和列标记下来。

#### 第三步：根据第一行和第一列里的标记，处理内部元素

这里处理的是从 `(1,1)` 开始的内部区域。  
只要：

- `matrix[i][0] == 0`
- 或者 `matrix[0][j] == 0`

就说明当前元素所在行或所在列应该被清零，于是把 `matrix[i][j] = 0`。

#### 第四步：最后再单独处理第一行和第一列

因为它们一开始被拿去当标记区了，所以必须等内部处理完之后，再根据 `firstRowHasZero` 和 `firstColHasZero` 决定它们自己要不要整行、整列清零。

这个顺序不能乱。  
如果太早把第一行或第一列清掉，前面存的标记就没了。

### 用示例走一遍会更清楚

看示例 1：

```text
matrix = [
  [1,1,1],
  [1,0,1],
  [1,1,1]
]
```

先看第一行和第一列，都没有 `0`，所以：

- `firstRowHasZero = false`
- `firstColHasZero = false`

接着扫描内部区域，发现 `matrix[1][1] == 0`。  
于是把它对应的行和列先标记出来：

```text
matrix[1][0] = 0
matrix[0][1] = 0
```

这时矩阵里的第一列和第一行已经存下了“第 1 行要清零”和“第 1 列要清零”这两个信息。

然后再扫描内部区域，只要某个位置所在行或所在列被标记过，就置零。  
最终得到：

```text
[
  [1,0,1],
  [0,0,0],
  [1,0,1]
]
```

这正是答案。

### 代码

```java
public void setZeroes(int[][] matrix) {
    int m = matrix.length;
    int n = matrix[0].length;

    boolean firstRowHasZero = false;
    boolean firstColHasZero = false;

    for (int j = 0; j < n; j++) {
        if (matrix[0][j] == 0) {
            firstRowHasZero = true;
            break;
        }
    }

    for (int i = 0; i < m; i++) {
        if (matrix[i][0] == 0) {
            firstColHasZero = true;
            break;
        }
    }

    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            if (matrix[i][j] == 0) {
                matrix[i][0] = 0;
                matrix[0][j] = 0;
            }
        }
    }

    for (int i = 1; i < m; i++) {
        for (int j = 1; j < n; j++) {
            if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                matrix[i][j] = 0;
            }
        }
    }

    if (firstRowHasZero) {
        for (int j = 0; j < n; j++) {
            matrix[0][j] = 0;
        }
    }

    if (firstColHasZero) {
        for (int i = 0; i < m; i++) {
            matrix[i][0] = 0;
        }
    }
}
```

时间复杂度是 $O(mn)$，因为本质上还是扫描矩阵若干遍。  
空间复杂度是 $O(1)$，因为额外只用了两个布尔变量，标记信息都借助矩阵本身存下来了。

### 这题真正值得记住的地方

这题最值得记住的，不只是“首行首列做标记”这个技巧，而是它背后的思路：

**当题目要求原地处理，又需要额外标记信息时，可以优先想一想，能不能把输入本身的一部分位置借出来当标记区。**

这类题难的地方，往往不是操作本身，而是怎么在不额外开空间的前提下，把“中间状态”存下来。

这题就是一个很典型的例子：

- 直观解法是开两个数组记录行和列
- 进阶解法是把这两个数组“折叠”进原矩阵的第一行和第一列

一旦把这个思路想通，代码其实并不复杂。

### 完整代码

```java
import java.util.Arrays;

public class Main {

    public static void setZeroes(int[][] matrix) {
        int m = matrix.length;
        int n = matrix[0].length;
        boolean firstRowHasZero = false;
        boolean firstColHasZero = false;

        for (int j = 0; j < n; j++) {
            if (matrix[0][j] == 0) {
                firstRowHasZero = true;
                break;
            }
        }

        for (int i = 0; i < m; i++) {
            if (matrix[i][0] == 0) {
                firstColHasZero = true;
                break;
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][j] == 0) {
                    matrix[i][0] = 0;
                    matrix[0][j] = 0;
                }
            }
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                if (matrix[i][0] == 0 || matrix[0][j] == 0) {
                    matrix[i][j] = 0;
                }
            }
        }

        if (firstRowHasZero) {
            for (int j = 0; j < n; j++) {
                matrix[0][j] = 0;
            }
        }

        if (firstColHasZero) {
            for (int i = 0; i < m; i++) {
                matrix[i][0] = 0;
            }
        }
    }

    public static void main(String[] args) {
        int[][] matrix1 = {
            {1, 1, 1},
            {1, 0, 1},
            {1, 1, 1}
        };

        System.out.println("示例 1 输入:");
        printMatrix(matrix1);

        setZeroes(matrix1);

        System.out.println("示例 1 输出:");
        printMatrix(matrix1);
        System.out.println("-----------------------------");

        int[][] matrix2 = {
            {0, 1, 2, 0},
            {3, 4, 5, 2},
            {1, 3, 1, 5}
        };

        System.out.println("示例 2 输入:");
        printMatrix(matrix2);

        setZeroes(matrix2);

        System.out.println("示例 2 输出:");
        printMatrix(matrix2);
    }

    private static void printMatrix(int[][] matrix) {
        System.out.println("[");
        for (int[] row : matrix) {
            System.out.println("  " + Arrays.toString(row));
        }
        System.out.println("]");
    }
}
```
