## 题目

**描述：**

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：** 答案中不可以包含重复的三元组。

**示例 1:**

- **输入:** `nums = [-1,0,1,2,-1,-4]`
- **输出:** `[[-1,-1,2],[-1,0,1]]`
- **解释:**
  - `nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0` 。
  - `nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0` 。
  - `nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0` 。
  - 不同的三元组是 `[-1,0,1]` 和 `[-1,-1,2]` 。
  - 注意，输出的顺序和三元组的顺序并不重要。

**示例 2:**

- **输入:** `nums = [0,1,1]`
- **输出:** `[]`
- **解释:** 唯一可能的三元组和不为 0 。

**示例 3:**

- **输入:** `nums = [0,0,0]`
- **输出:** `[[0,0,0]]`
- **解释:** 唯一可能的三元组和为 0 。

**提示:**

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`

## 题解

这题麻烦的地方，不是找出三个数让它们加起来等于 `0`，而是要把所有答案找全，还不能出现重复。也正因为如此，它虽然表面上是求和题，最容易出错的却往往是搜索过程里的去重处理。

最直接的做法当然是三层循环，把所有三元组都枚举出来，再筛出和为 `0` 的组合。但这样是 `O(n^3)`，规模一上来肯定顶不住。所以这题更自然的思路，是先把它降一层：固定一个数，然后在剩下的区间里做“两数之和”。

这一步想成立，排序很关键。数组排好序以后，一方面相同的数会挨在一起，去重更顺手；另一方面，固定 `nums[i]` 以后，后面的区间就是一个标准的有序数组，完全可以用双指针去找 `nums[L] + nums[R] = -nums[i]`。也就是说，这题本质上是在做一件很经典的拆解：三数问题先固定一个数，再把剩下部分变成两数问题。

拿 `nums = [-1,0,1,2,-1,-4]` 排序以后会得到 `[-4,-1,-1,0,1,2]`。先固定 `-4` 时，后面怎么找都凑不出 `4`；接着固定第一个 `-1`，左右指针分别放在后面的 `-1` 和 `2` 上，三数和正好是 `0`，得到第一组答案 `[-1,-1,2]`。然后继续收缩指针，还会找到 `[-1,0,1]`。等外层走到第二个 `-1` 时，因为它和前一个固定值相同，直接跳过，否则只会把同样的答案再找一遍。

所以这题的去重其实分成三层。第一层是固定 `i` 时去重：如果 `nums[i] == nums[i - 1]`，这一轮直接跳过。第二层和第三层是在找到一组答案以后，`L` 和 `R` 都要继续跳过相同的值，否则下一次加入结果集的还是重复三元组。很多时候代码主逻辑都对，最后结果却不对，问题往往就出在这三层去重没有跟搜索过程绑在一起。

排序以后还有一个很自然的剪枝：如果当前固定的 `nums[i] > 0`，后面所有数都只会更大，三个正数不可能再加出 `0`，这时可以直接结束外层循环。这个优化不是必须的，但写上之后思路会更完整。

代码如下：

```java
public List<List<Integer>> threeSum(int[] nums) {
    List<List<Integer>> ans = new ArrayList<>();
    if (nums == null || nums.length < 3) {
        return ans;
    }

    Arrays.sort(nums);

    for (int i = 0; i < nums.length; i++) {
        if (nums[i] > 0) {
            break;
        }

        if (i > 0 && nums[i] == nums[i - 1]) {
            continue;
        }

        int L = i + 1;
        int R = nums.length - 1;

        while (L < R) {
            int sum = nums[i] + nums[L] + nums[R];

            if (sum == 0) {
                ans.add(Arrays.asList(nums[i], nums[L], nums[R]));
                L++;
                R--;

                while (L < R && nums[L] == nums[L - 1]) {
                    L++;
                }
                while (L < R && nums[R] == nums[R + 1]) {
                    R--;
                }
            } else if (sum < 0) {
                L++;
            } else {
                R--;
            }
        }
    }

    return ans;
}
```

时间复杂度是 `O(n^2)`，因为排序之后，外层固定一个数，内层双指针扫一遍；空间复杂度主要取决于排序实现，若不算结果集本身，额外空间不大。

这题更适合记住的，不只是“排序 + 双指针”这个结论，而是两件事：一是三数问题经常可以先固定一个数，降成两数问题；二是遇到去重要求时，最好在搜索过程中顺手处理，而不是最后再临时补救。

完整代码如下：

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Main {

    public static List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> ans = new ArrayList<>();
        if (nums == null || nums.length < 3) {
            return ans;
        }

        Arrays.sort(nums);

        for (int i = 0; i < nums.length; i++) {
            if (nums[i] > 0) {
                break;
            }

            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }

            int L = i + 1;
            int R = nums.length - 1;

            while (L < R) {
                int sum = nums[i] + nums[L] + nums[R];

                if (sum == 0) {
                    ans.add(Arrays.asList(nums[i], nums[L], nums[R]));
                    L++;
                    R--;

                    while (L < R && nums[L] == nums[L - 1]) {
                        L++;
                    }
                    while (L < R && nums[R] == nums[R + 1]) {
                        R--;
                    }
                } else if (sum < 0) {
                    L++;
                } else {
                    R--;
                }
            }
        }

        return ans;
    }

    public static void main(String[] args) {
        int[] nums1 = {-1, 0, 1, 2, -1, -4};
        System.out.println("示例 1 输入: " + Arrays.toString(nums1));
        List<List<Integer>> res1 = threeSum(nums1);
        System.out.println("示例 1 输出: " + res1);
        System.out.println("-----------------------------");

        int[] nums2 = {0, 1, 1};
        System.out.println("示例 2 输入: " + Arrays.toString(nums2));
        List<List<Integer>> res2 = threeSum(nums2);
        System.out.println("示例 2 输出: " + res2);
        System.out.println("-----------------------------");

        int[] nums3 = {0, 0, 0};
        System.out.println("示例 3 输入: " + Arrays.toString(nums3));
        List<List<Integer>> res3 = threeSum(nums3);
        System.out.println("示例 3 输出: " + res3);
    }
}
```
