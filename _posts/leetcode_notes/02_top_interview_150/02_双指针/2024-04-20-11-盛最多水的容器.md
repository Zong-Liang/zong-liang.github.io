## 题目

**描述：**

给定一个长度为 `n` 的整数数组 `height` 。有 `n` 条垂线，第 `i` 条线的两个端点是 `(i, 0)` 和 `(i, height[i])` 。

找出其中的两条线，使得它们与 `x` 轴共同构成的容器可以容纳最多的水。

返回容器可以储存的最大水量。

**说明：** 你不能倾斜容器。

**示例 1:**

- **输入:** `[1,8,6,2,5,4,8,3,7]`
- **输出:** 49
- **解释:** 图中垂直线代表输入数组 `[1,8,6,2,5,4,8,3,7]`。在此情况下，容器能够容纳水（表示为蓝色部分）的最大值为 49。

**示例 2:**

- **输入:** `height = [1,1]`
- **输出:** 1

**提示:**

- `n == height.length`
- `2 <= n <= 10^5`
- `0 <= height[i] <= 10^4`

## 题解

这题第一眼很容易去找“最高的两条线”，但它算的不是单纯的高度，而是面积：`(right - left) * min(height[left], height[right])`。也就是说，能装多少水，一边取决于两条线之间的距离，另一边取决于两端里较矮的那条线。后者才是这道题最关键的限制，因为水位不可能超过短板。

最直接的办法当然是把所有两条线的组合都枚举一遍，逐个算面积，再取最大值。但这样是 `O(n^2)`，在这题的数据范围下显然不够用。所以做这题时，更值得先问一句：能不能在枚举的过程中，一次性排掉一整类不可能更优的情况？

双指针就是在干这件事。先把 `left` 放在最左边，`right` 放在最右边，这样一开始宽度最大。每次先算当前面积，然后决定下一步缩哪一边。这里最重要的结论是：总是移动较短的那条边。如果左边更短，那么当前面积的瓶颈就在左边；此时去移动右边，只会让宽度变小，而短板未必变高，面积不可能因此变得更好。只有把左边这个短板换掉，才有机会遇到更高的线，把短板抬起来。

拿 `height = [1,8,6,2,5,4,8,3,7]` 来看，一开始左右两端是 `1` 和 `7`，宽度是 `8`，面积只有 `8 * 1 = 8`。因为左边更矮，所以左指针右移。下一步来到 `8` 和 `7`，虽然宽度缩成了 `7`，但短板一下抬到了 `7`，面积变成 `49`。这正说明：宽度每次内缩都会变小，没法避免；双指针在赌的，是把短板换成更高的值。

这个策略之所以成立，不是凭感觉，而是因为每次移动较短的一边，都能排掉一整批无效组合。比如当前左边更短，那以这个左边为边界、配上任何更靠内的右边，宽度只会更小，而短板上限仍然受左边限制，所以这些组合都不可能比当前更好。既然如此，当前这条短边就没有继续保留的价值，可以放心往里收。

代码如下：

```java
public int maxArea(int[] height) {
    int left = 0;
    int right = height.length - 1;
    int maxArea = 0;

    while (left < right) {
        int currentArea = (right - left) * Math.min(height[left], height[right]);
        maxArea = Math.max(maxArea, currentArea);

        if (height[left] < height[right]) {
            left++;
        } else {
            right--;
        }
    }

    return maxArea;
}
```

时间复杂度是 `O(n)`，因为两个指针都只会朝中间移动一次；空间复杂度是 `O(1)`，只用了几个额外变量。

这题很值得带走的一点，不只是“谁矮谁移动”这个口诀，而是它背后的判断方式：当一个结果同时受范围和瓶颈限制时，范围如果注定要缩，那就只能去动瓶颈，才有可能把答案做大。

完整代码如下：

```java
import java.util.Arrays;

public class Main {

    public static int maxArea(int[] height) {
        int left = 0;
        int right = height.length - 1;
        int maxArea = 0;

        while (left < right) {
            int h = Math.min(height[left], height[right]);
            int w = right - left;
            int area = h * w;

            maxArea = Math.max(maxArea, area);

            if (height[left] < height[right]) {
                left++;
            } else {
                right--;
            }
        }

        return maxArea;
    }

    public static void main(String[] args) {
        int[] height1 = new int[]{1, 8, 6, 2, 5, 4, 8, 3, 7};

        System.out.println("示例 1 输入:");
        System.out.println("height = " + Arrays.toString(height1));

        int result1 = maxArea(height1);

        System.out.println("示例 1 输出: " + result1);
        System.out.println("-----------------------------");

        int[] height2 = new int[]{1, 1};

        System.out.println("示例 2 输入:");
        System.out.println("height = " + Arrays.toString(height2));

        int result2 = maxArea(height2);

        System.out.println("示例 2 输出: " + result2);
    }
}
```
