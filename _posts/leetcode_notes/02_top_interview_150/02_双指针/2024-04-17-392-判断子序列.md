## 题目

**描述：**

给定字符串 **s** 和 **t** ，判断 **s** 是否为 **t** 的子序列。

字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，`"ace"`是`"abcde"`的一个子序列，而`"aec"`不是）。

**进阶:**
如果有大量输入的 S，称作 S1, S2, ..., Sk 其中 k >= 10 亿，你需要依次检查它们是否为 T 的子序列。在这种情况下，你会怎样改变代码？

**示例 1:**

- **输入:** `s = "abc"`, `t = "ahbgdc"`
- **输出:** `true`

**示例 2:**

- **输入:** `s = "axc"`, `t = "ahbgdc"`
- **输出:** `false`

**提示:**

- `0 <= s.length <= 100`
- `0 <= t.length <= 10^4`
- 两个字符串都只由小写字母组成。

## 题解

这题先别被“子序列”这个词吓到，它问的事情其实很朴素：`s` 里的字符，能不能在 `t` 里按顺序一个一个找到。注意这里强调的是“按顺序”，不是要求连续出现。所以像 `"abc"` 是 `"ahbgdc"` 的子序列，因为可以依次找到 `a`、`b`、`c`；但 `"axc"` 就不行，因为中间那个 `x` 根本找不到。

如果只看原题，不考虑后面的进阶，一对双指针已经很够用了。`i` 指向 `s`，`j` 指向 `t`，然后让 `j` 一路往后扫。只要 `s[i] == t[j]`，说明当前这个字符匹配上了，`i` 就往后走一步；不管匹没匹配上，`j` 都继续往后走。最后如果 `i` 已经走到 `s` 的末尾，就说明 `s` 的所有字符都按顺序在 `t` 里找到了。

这个过程里有个很重要的感觉：`t` 可以跳过字符，`s` 不行。因为我们是在 `t` 里“寻找”`s`，所以 `t` 当前这个字符如果用不上，略过就好了；但 `s` 只有当前字符真的匹配成功，才能继续往后找下一个。也正因为如此，这道题完全不需要回头，顺着扫过去就行。

这种写法其实还带一点很自然的贪心。每次在 `t` 里一旦看到了当前需要的字符，就立刻把它拿来匹配，肯定不会吃亏。因为子序列只关心相对顺序，不存在“先跳过这个相同字符，后面再选一个更优的位置”这种说法。越早匹配上，后面能留给剩余字符的空间反而越大。

拿 `s = "abc"`、`t = "ahbgdc"` 走一遍会很直观。开始时 `i` 指向 `a`，`j` 也从 `a` 开始，所以第一个字符立刻匹配上；接着 `j` 继续往后走，经过 `h` 时匹配不上，扫到 `b` 时又匹配成功；最后同样找到 `c`，于是 `s` 全部匹配完，返回 `true`。而 `"axc"` 的情况则会在寻找 `x` 的过程中卡住，直到 `t` 扫完都没找到，于是返回 `false`。

题目后面那个进阶值得单独记一下。要是只有一次查询，双指针已经是最顺的写法；但如果有大量不同的 `s`，而 `t` 是固定不变的，那每次都把 `t` 从头扫一遍就很亏了。这个时候更好的方向，是先把 `t` 里每个字符出现的位置都存下来，后面每次查询时用二分去找“当前下标之后的第一个可用位置”。也就是说，原题是一次线性扫描；一旦查询特别多，思路就该切到“先预处理，再重复利用”。

代码如下：

```java
public boolean isSubsequence(String s, String t) {
    int i = 0, j = 0;
    while (i < s.length() && j < t.length()) {
        if (s.charAt(i) == t.charAt(j)) {
            i++;
        }
        j++;
    }
    return i == s.length();
}
```

时间复杂度是 `O(n)`，这里的 `n` 可以理解为 `t` 的长度，因为本质上就是顺着扫一遍 `t`；空间复杂度是 `O(1)`，只用了两个指针。

这题可以顺手带走的，不只是双指针本身，而是一个判断习惯：单次查询时，直接顺着扫通常最划算；一旦变成“同一个大对象会被反复查询很多次”，就该考虑先把它预处理起来。

完整代码如下：

```java
public class Main {

    public static boolean isSubsequence(String s, String t) {
        if (s.length() > t.length()) {
            return false;
        }

        if (s.length() == 0) {
            return true;
        }

        int i = 0;
        int j = 0;

        while (i < s.length() && j < t.length()) {
            if (s.charAt(i) == t.charAt(j)) {
                i++;
            }
            j++;
        }

        return i == s.length();
    }

    public static void main(String[] args) {
        String s1 = "abc";
        String t1 = "ahbgdc";

        System.out.println("示例 1 输入: s = \"" + s1 + "\", t = \"" + t1 + "\"");
        System.out.println("示例 1 输出: " + isSubsequence(s1, t1));
        System.out.println("-----------------------------");

        String s2 = "axc";
        String t2 = "ahbgdc";

        System.out.println("示例 2 输入: s = \"" + s2 + "\", t = \"" + t2 + "\"");
        System.out.println("示例 2 输出: " + isSubsequence(s2, t2));
    }
}
```
