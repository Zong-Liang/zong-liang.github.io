## 题目

**描述：**

给你二叉树的根节点 `root`，返回其节点值的 **层序遍历**。（即逐层地，从左到右访问所有节点）。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[[3],[9,20],[15,7]]`

**示例 2:**

- **输入:** `root = [1]`
- **输出:** `[[1]]`

**示例 3:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中节点数目在范围 `[0, 2000]` 内
- `-1000 <= Node.val <= 1000`

## 题解

这题就是二叉树 BFS 的基础款：按层从左到右收集节点值。

核心写法是“外层 while 控层、内层 for 扫当前层”。每轮开始先读队列长度 `size`，它代表当前层节点数；内层处理完这 `size` 个节点后，一层结果就完整了。

这样可以天然得到 `[[...],[...],...]` 的结构，不需要额外标记层号。

时间复杂度 $O(n)$，空间复杂度最坏 $O(n)$。

这题虽然基础，但很关键：很多树题（右视图、锯齿层序、层平均值）都是在这个骨架上改一两行得到的。

**Java 实现：**

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) {
            return res;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            List<Integer> levelList = new ArrayList<>();
            int size = queue.size();

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                levelList.add(node.val);

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            res.add(levelList);
        }

        return res;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("示例 1 输出: " + levelOrder(root1));

        TreeNode root2 = new TreeNode(1);
        System.out.println("示例 2 输入: [1]");
        System.out.println("示例 2 输出: " + levelOrder(root2));

        TreeNode root3 = null;
        System.out.println("示例 3 输入: []");
        System.out.println("示例 3 输出: " + levelOrder(root3));
    }
}
```
