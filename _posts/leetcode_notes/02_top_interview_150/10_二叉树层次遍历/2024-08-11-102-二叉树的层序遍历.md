## 题目

**描述：**

给你二叉树的根节点 `root`，返回其节点值的 **层序遍历**。（即逐层地，从左到右访问所有节点）。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[[3],[9,20],[15,7]]`

**示例 2:**

- **输入:** `root = [1]`
- **输出:** `[[1]]`

**示例 3:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中节点数目在范围 `[0, 2000]` 内
- `-1000 <= Node.val <= 1000`

## 题解

### 这题其实就是在问：怎么一层一层地访问二叉树

题目要求返回二叉树节点值的层序遍历结果，也就是：

- 第一层放在第一个列表里
- 第二层放在第二个列表里
- 第三层放在第三个列表里

说白了，就是要我们按“层”来访问整棵树。

只要把这句话读明白，这题的方向其实已经很清楚了：  
**既然要按层处理，那最自然的做法就是层序遍历。**

### 为什么层序遍历最合适

层序遍历本身就是从上到下、从左到右，一层一层地扫描二叉树。

而队列恰好可以帮我们把这个过程实现出来：

- 先把根节点放进去
- 每次从队头取节点
- 再把它的左右孩子依次放到队尾

这样队列里的节点顺序，就会自然保持“按层、从左到右”的访问顺序。

所以这题并不是在考什么特别绕的技巧，本质上就是一个标准的 BFS 模板题。

### 这题的关键不只是遍历，而是怎么分层

如果只是单纯把节点按 BFS 顺序扫一遍，其实只能得到一条线性的结果，比如：

```text
[3,9,20,15,7]
```

但题目要的是：

```text
[[3],[9,20],[15,7]]
```

也就是说，不光要遍历，还得把同一层的节点收集到同一个列表里。

那怎么知道“这一层到哪里结束”呢？

办法很简单：  
在每轮开始时，先记录当前队列的大小 `size`。这个 `size` 就表示当前层一共有多少个节点。

接下来只处理这 `size` 个节点，就刚好能把这一层完整取出来，而且不会把下一层混进来。

这就是这题最核心的一步。

### 具体怎么做

整个过程可以拆成下面几步：

1. 如果根节点为空，直接返回空列表
2. 创建一个队列，把根节点放进去
3. 每次循环开始时，先记录当前队列长度 `size`
4. 创建一个新的列表 `levelList`，专门存这一层的节点值
5. 连续出队 `size` 次，把这一层所有节点都处理掉
6. 处理每个节点时：
   - 把节点值加入 `levelList`
   - 如果左孩子不为空，就入队
   - 如果右孩子不为空，就入队
7. 当前层处理完后，把 `levelList` 加入最终结果

这样一层一层扫完，答案就有了。

### 用示例看一遍

比如示例 1：

```text
root = [3,9,20,null,null,15,7]
```

这棵树大致长这样：

```text
    3
   / \
  9  20
    /  \
   15   7
```

层序遍历的过程是：

- 第 1 层：只有 `3`
- 第 2 层：有 `9` 和 `20`
- 第 3 层：有 `15` 和 `7`

所以结果就是：

```text
[[3],[9,20],[15,7]]
```

从这个例子也能看出来，这题本质上就是“按层分组”。

### 代码

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        List<Integer> levelList = new ArrayList<>();
        int size = queue.size();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            levelList.add(node.val);

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }

        res.add(levelList);
    }

    return res;
}
```

### 这段代码最关键的地方在哪

关键其实就一句：

```java
int size = queue.size();
```

为什么这句这么重要？

因为队列里此时放着的，正好是当前这一层的所有节点。  
把这个数量先记下来，我们就能明确知道这一轮该处理多少个节点。

如果没有这一步，你就只能一直出队，那下一层的节点也会被一起处理进去，结果就没法按层分组了。

所以这题真正的重点，不是“会不会用队列”，而是**会不会用队列大小来切分层级**。

### 再补一种写法：DFS

这题虽然最标准的写法是 BFS，但也可以用 DFS 来做。

DFS 的思路不是按层一层层扫，而是在递归时额外记录当前节点所在的层数 `level`。

每访问一个节点，就看它属于第几层：

- 如果结果列表里还没有这一层对应的列表，就先新建一个
- 然后把当前节点值加入 `res.get(level)`
- 再递归处理左右子树

这样虽然访问顺序不是层序，但最后仍然能把同一层的节点放到一起。

### DFS 为什么也能做

因为这题要的其实不是“必须按层顺序访问”，而是“最终结果要按层分组”。

BFS 是天然按层访问，所以写起来最顺。  
DFS 则是靠 `level` 参数，把分散访问到的节点重新归到对应层里。

也就是说：

- BFS 是遍历过程本身就带着层级结构
- DFS 是遍历过程中手动记录层级结构

两种方法都能做对，只是 BFS 更贴题一些。

### DFS 代码

```java
public List<List<Integer>> levelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    dfs(root, 0, res);
    return res;
}

private void dfs(TreeNode node, int level, List<List<Integer>> res) {
    if (node == null) return;

    if (res.size() == level) {
        res.add(new ArrayList<>());
    }

    res.get(level).add(node.val);

    dfs(node.left, level + 1, res);
    dfs(node.right, level + 1, res);
}
```

### 这题该怎么选写法

如果是第一次做这题，或者只是想写一个最稳的版本，直接上 BFS 就行。

原因很简单：  
题目问的就是层序遍历，而 BFS 本来就是层序遍历的标准写法，思路和题目完全一致，不需要额外转换。

DFS 也没问题，但更适合放在你已经熟悉树的递归写法之后，当成另一种实现方式来理解。

### 复杂度分析

无论是 BFS 还是 DFS，每个节点都只会访问一次，所以时间复杂度都是 $O(n)$，其中 $n$ 是二叉树节点总数。

空间复杂度要分开看：

- BFS 的空间复杂度是 $O(n)$，最坏情况下队列里可能存着某一整层的大量节点
- DFS 的空间复杂度是 $O(h)$，其中 $h$ 是树的高度，主要来自递归调用栈

### 这题真正值得记住的地方

这题是二叉树题里很典型的一类。

它最值得记住的不是代码本身，而是这个判断：

**只要题目是按层提问，就先想层序遍历。**

比如：

- 每层节点值
- 每层平均值
- 每层最大值
- 每层最右边节点

这类题的底层思路都差不多，核心都是先把“这一层”和“下一层”分开。

而这题，就是最基础的起点。

### 完整代码

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static List<List<Integer>> levelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) {
            return res;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            List<Integer> levelList = new ArrayList<>();
            int size = queue.size();

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                levelList.add(node.val);

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            res.add(levelList);
        }

        return res;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("示例 1 输出: " + levelOrder(root1));

        TreeNode root2 = new TreeNode(1);
        System.out.println("示例 2 输入: [1]");
        System.out.println("示例 2 输出: " + levelOrder(root2));

        TreeNode root3 = null;
        System.out.println("示例 3 输入: []");
        System.out.println("示例 3 输出: " + levelOrder(root3));
    }
}
```
