## 题目

**描述：**

给你二叉树的根节点 `root`，返回其节点值的 **锯齿形层序遍历**。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[[3],[20,9],[15,7]]`

**示例 2:**

- **输入:** `root = [1]`
- **输出:** `[[1]]`

**示例 3:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中节点数目在范围 `[0, 2000]` 内
- `-100 <= Node.val <= 100`

## 题解

### 先看这题比普通层序遍历多了什么

这题和「二叉树的层序遍历」几乎是同一类题，区别只在一点：

**每一层节点值的输出方向要交替变化。**

也就是说：

- 第 0 层从左往右
- 第 1 层从右往左
- 第 2 层再从左往右
- 后面继续这样来回切换

所以这题的核心不是“怎么遍历二叉树”，而是：

**在按层遍历的基础上，怎么让每一层的结果方向交替。**

### 主体还是层序遍历

既然题目仍然是按层来处理，那主体思路当然还是层序遍历，也就是 BFS。

这一点和普通层序遍历没有区别：

- 用队列保存当前要访问的节点
- 每轮先固定当前层的节点数
- 把这一层的节点依次取出来
- 再把下一层节点加入队列

真正有变化的地方，不在“节点怎么遍历”，而在“当前层的节点值怎么放进结果里”。

### 为什么不能只会普通层序遍历

如果我们完全照搬普通层序遍历，那么每一层收集出来的结果一定都是从左往右的。

例如示例 1：

```text
root = [3,9,20,null,null,15,7]
```

普通层序遍历会得到：

```text
[[3],[9,20],[15,7]]
```

但题目要的是锯齿形，也就是：

```text
[[3],[20,9],[15,7]]
```

问题就在第二层。  
节点访问顺序本身还是先 `9` 再 `20`，但结果要按从右往左输出，所以这一层收集结果时就不能像平时那样只会往后追加。

### 这题最顺手的处理方式

最自然的做法，是在收集每一层结果时用一个双端队列。

这样一来：

- 如果当前层应该从左往右，就把节点值放到尾部
- 如果当前层应该从右往左，就把节点值放到头部

节点本身的遍历顺序不需要变，还是正常的从左到右入队。  
真正变化的，只是“当前层结果的放入方式”。

这样写的好处是很直接：  
**遍历逻辑和方向控制是分开的。**

- 队列负责按层遍历整棵树
- 双端队列负责控制这一层最终长什么样

这个思路比“先收集完再反转”更顺，也更贴题。

### 具体怎么做

先把根节点放进队列。

然后每轮循环：

1. 先记录当前层节点个数 `size`
2. 新建一个双端队列 `levelList`
3. 把这一层的 `size` 个节点依次取出
4. 如果当前层方向是从左往右，就尾插
5. 如果当前层方向是从右往左，就头插
6. 当前层处理完后，把结果加入答案
7. 将方向取反，留给下一层使用

这样一层一层往下走，结果自然就是锯齿形的。

### 用示例走一遍

还是看示例 1：

```text
    3
   / \
  9  20
    /  \
   15   7
```

第 0 层要求从左往右，所以结果是：

```text
[3]
```

第 1 层要求从右往左。  
虽然遍历时还是先遇到 `9`，再遇到 `20``，但我们把 `9`先放进去后，再把`20` 插到头部，所以这一层最终变成：

```text
[20,9]
```

第 2 层又切回从左往右，所以结果是：

```text
[15,7]
```

最后答案就是：

```text
[[3],[20,9],[15,7]]
```

从这个过程可以看出来，关键不是改遍历顺序，而是改当前层的收集方式。

### 代码

```java
public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    if (root == null) return res;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);
    boolean leftToRight = true;

    while (!queue.isEmpty()) {
        LinkedList<Integer> levelList = new LinkedList<>();
        int size = queue.size();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();

            if (leftToRight) {
                levelList.addLast(node.val);
            } else {
                levelList.addFirst(node.val);
            }

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }

        res.add(levelList);
        leftToRight = !leftToRight;
    }

    return res;
}
```

### 这段代码最关键的地方

关键其实就两点。

第一点是：

```java
int size = queue.size();
```

这一步和普通层序遍历一样，作用是把当前层和下一层分开。没有它，就没法按层处理。

第二点是：

```java
if (leftToRight) {
    levelList.addLast(node.val);
} else {
    levelList.addFirst(node.val);
}
```

这才是锯齿形的核心。  
同样是访问到一个节点，放到头部还是尾部，决定了这一层最终是正着看还是反着看。

所以这题最值得抓住的，不是“双端队列”四个字，而是这个意识：

**遍历顺序可以不变，结果顺序可以单独控制。**

### 为什么子节点还是照常先左后右入队

这个地方很多人第一次写时会有点绕：  
既然当前层有时要从右往左，那下一层入队是不是也要反过来？

其实不用。

因为这里的“锯齿形”要求的是**每层结果的输出顺序交替**，不是让整棵树的 BFS 访问逻辑也来回切换。

队列里的节点，仍然保持标准层序遍历的顺序：

- 先左子节点
- 再右子节点

这样层级关系最清楚，也最不容易乱。  
至于某一层结果最终是正着还是反着，只交给 `levelList` 去处理就够了。

### 再补一种写法：DFS

这题也可以用 DFS，只不过思路和 BFS 有点不同。

DFS 不能天然按层一批一批地处理节点，所以需要在递归时手动记录当前层数 `level`。

做法是：

- 如果当前层还没有对应的列表，就先创建一个
- 如果当前层是偶数层，就把当前节点值加到末尾
- 如果当前层是奇数层，就把当前节点值插到开头
- 再递归左右子树

这样虽然访问过程不是层序的，但最后仍然能得到锯齿形层序遍历结果。

### DFS 为什么也成立

因为题目最终关心的是“每个节点属于哪一层，以及它在这一层的结果里该放到哪里”。

DFS 虽然不是一层层访问节点，但它知道当前节点所在的层数。  
只要层数知道了，我们就能判断这一层应该正着放还是反着放，然后把当前值插到合适的位置。

所以 BFS 和 DFS 的差别在于：

- BFS 是天然按层组织节点
- DFS 是靠 `level` 参数把节点归到正确的层里

这题里，BFS 会更自然；DFS 更适合作为另一种思路来理解。

### DFS 代码

```java
public List<List<Integer>> zigzagLevelOrder(TreeNode root) {
    List<List<Integer>> res = new ArrayList<>();
    dfs(root, 0, res);
    return res;
}

private void dfs(TreeNode node, int level, List<List<Integer>> res) {
    if (node == null) return;

    if (res.size() == level) {
        res.add(new LinkedList<>());
    }

    if (level % 2 == 0) {
        res.get(level).add(node.val);
    } else {
        res.get(level).add(0, node.val);
    }

    dfs(node.left, level + 1, res);
    dfs(node.right, level + 1, res);
}
```

### 这题该怎么选写法

如果是第一次做这题，直接写 BFS 就行。

因为它和普通层序遍历几乎一样，只是在收集当前层结果时多了一步方向判断，改动很小，也最容易写稳。

DFS 也能做，但它并不是这题最直观的入口。  
更适合作为你已经掌握层序遍历之后，再顺手补的一种写法。

### 复杂度分析

无论是 BFS 还是 DFS，每个节点都只会访问一次，所以时间复杂度都是 $O(n)$，其中 $n$ 是二叉树节点总数。

空间复杂度分开来看：

- BFS 的空间复杂度是 $O(n)$，最坏情况下队列里可能存某一整层节点
- DFS 的空间复杂度是 $O(h)$，其中 $h$ 是树的高度，主要来自递归栈

### 这题真正值得记住的地方

这题最值得记住的，其实不是“再背一个新模板”，而是下面这个变化方式：

**普通层序遍历的框架不变，只改每层结果的收集方式。**

很多二叉树题都是这样：

- 主体遍历框架不动
- 真正变化的是某一层统计什么、怎么统计、按什么顺序放

这题只是把“按层收集节点值”改成了“按层交替方向收集节点值”。

能看出这一点，后面很多层序遍历变种题都会顺很多。

### 完整代码

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) {
            return res;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        boolean leftToRight = true;

        while (!queue.isEmpty()) {
            LinkedList<Integer> levelList = new LinkedList<>();
            int size = queue.size();

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();

                if (leftToRight) {
                    levelList.addLast(node.val);
                } else {
                    levelList.addFirst(node.val);
                }

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            res.add(levelList);
            leftToRight = !leftToRight;
        }

        return res;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("示例 1 输出: " + zigzagLevelOrder(root1));

        TreeNode root2 = new TreeNode(1);
        System.out.println("示例 2 输入: [1]");
        System.out.println("示例 2 输出: " + zigzagLevelOrder(root2));

        TreeNode root3 = null;
        System.out.println("示例 3 输入: []");
        System.out.println("示例 3 输出: " + zigzagLevelOrder(root3));
    }
}
```
