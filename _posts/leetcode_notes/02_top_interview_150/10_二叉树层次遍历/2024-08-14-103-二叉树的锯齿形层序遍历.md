## 题目

**描述：**

给你二叉树的根节点 `root`，返回其节点值的 **锯齿形层序遍历**。（即先从左往右，再从右往左进行下一层遍历，以此类推，层与层之间交替进行）。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/02/19/tree1.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[[3],[20,9],[15,7]]`

**示例 2:**

- **输入:** `root = [1]`
- **输出:** `[[1]]`

**示例 3:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 树中节点数目在范围 `[0, 2000]` 内
- `-100 <= Node.val <= 100`

## 题解

和普通层序遍历相比，这题只多了一个要求：每层输出方向要交替。

主体仍然是 BFS。区别在于层内收集值时，当前层若是从左到右就尾插，若是从右到左就头插（或者最后统一反转也可以）。

代码里常见做法是用 `LinkedList` 存当前层结果，按方向选择 `addLast` / `addFirst`，这样不用额外反转数组。

时间复杂度 $O(n)$，空间复杂度最坏 $O(n)$。

这题的收获是“先复用主框架，再局部改输出策略”。别一上来重写遍历，改层内写入规则通常就够了。

**Java 实现：**

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static List<List<Integer>> zigzagLevelOrder(TreeNode root) {
        List<List<Integer>> res = new ArrayList<>();
        if (root == null) {
            return res;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);
        boolean leftToRight = true;

        while (!queue.isEmpty()) {
            LinkedList<Integer> levelList = new LinkedList<>();
            int size = queue.size();

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();

                if (leftToRight) {
                    levelList.addLast(node.val);
                } else {
                    levelList.addFirst(node.val);
                }

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            res.add(levelList);
            leftToRight = !leftToRight;
        }

        return res;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("示例 1 输出: " + zigzagLevelOrder(root1));

        TreeNode root2 = new TreeNode(1);
        System.out.println("示例 2 输入: [1]");
        System.out.println("示例 2 输出: " + zigzagLevelOrder(root2));

        TreeNode root3 = null;
        System.out.println("示例 3 输入: []");
        System.out.println("示例 3 输出: " + zigzagLevelOrder(root3));
    }
}
```
