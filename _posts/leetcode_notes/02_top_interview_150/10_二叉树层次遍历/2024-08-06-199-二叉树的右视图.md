## 题目

**描述：**

给定一个二叉树的 **根节点** `root`，想象自己站在它的右侧，按照从顶部到底部的顺序，返回从右侧所能看到的节点值。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2024/11/24/tmpd5jn43fs-1.png)

- **输入:** `root = [1,2,3,null,5,null,4]`
- **输出:** `[1,3,4]`

**示例 2:**

![示例2图片](https://assets.leetcode.com/uploads/2024/11/24/tmpkpe40xeh-1.png)

- **输入:** `root = [1,2,3,4,null,null,null,5]`
- **输出:** `[1,3,4,5]`

**示例 3:**

- **输入:** `root = [1,null,3]`
- **输出:** `[1,3]`

**示例 4:**

- **输入:** `root = []`
- **输出:** `[]`

**提示:**

- 二叉树的节点个数的范围是 `[0,100]`
- `-100 <= Node.val <= 100`

## 题解

### 先看清这题到底在求什么

所谓二叉树的右视图，其实可以直接理解成一句话：

**从右边看过去，每一层最后能看到的那个节点。**

注意这里的关键词是“每一层”。  
只要把这一点抓住，这题就不难了，因为它本质上不是在考什么复杂性质，而是在问你：

**怎样把二叉树一层一层地看过去，并且拿到每层最右边的那个节点。**

### 最顺手的做法：层序遍历

既然题目强调的是“每一层”，那最自然的思路就是层序遍历，也就是 BFS。

层序遍历的过程，本来就是按层从上到下扫描整棵树。  
而在扫描某一层时，最后访问到的那个节点，刚好就是这一层最右边的节点。

所以这题的思路其实很直接：

- 用队列做层序遍历
- 每次先记录当前层有多少个节点
- 把这一层的节点依次弹出
- 当前层遍历到最后一个节点时，把它的值加入答案

这就是整道题最核心的逻辑。

### 为什么“当前层最后一个节点”就是答案

这里要想清楚一点：

层序遍历时，队列里存的就是当前层从左到右的节点顺序。  
也就是说，当我们处理某一层时：

- 第一个弹出的，是这一层最左边的节点
- 最后一个弹出的，就是这一层最右边的节点

而右视图要的，正是每一层最右边那个节点，所以只要在每层遍历结束前，把最后一个节点记录下来就行。

这个思路和题目本身是完全贴合的，所以用起来会很顺。

### 具体怎么做

先把根节点放进队列。

只要队列不为空，就说明还有节点没处理完。每轮循环做三件事：

1. 先取出当前队列的长度 `size`，这个 `size` 就是当前层的节点数
2. 把这一层的 `size` 个节点依次弹出，并把它们的左右子节点加入队列
3. 如果当前弹出的是这一层的最后一个节点，也就是 `i == size - 1`，就把它加入结果列表

这样一层一层扫完之后，答案自然就出来了。

### 用示例走一遍

比如示例 1：

```text
root = [1,2,3,null,5,null,4]
```

这棵树大致是这样：

```text
    1
   / \
  2   3
   \   \
    5   4
```

按层来看：

- 第 1 层只有一个节点 `1`，最右边是 `1`
- 第 2 层是 `2, 3`，最右边是 `3`
- 第 3 层是 `5, 4`，最右边是 `4`

所以结果就是：

```text
[1,3,4]
```

其实你会发现，这题一旦用“每层最后一个节点”去理解，整个过程就非常直观了。

### 代码

```java
public List<Integer> rightSideView(TreeNode root) {
    List<Integer> res = new ArrayList<>();
    if (root == null) return res;

    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size();

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();

            if (i == size - 1) {
                res.add(node.val);
            }

            if (node.left != null) queue.offer(node.left);
            if (node.right != null) queue.offer(node.right);
        }
    }

    return res;
}
```

### 这段代码里最关键的地方是什么

其实就一句：

```java
if (i == size - 1) {
    res.add(node.val);
}
```

因为 `size` 表示当前层一共有多少个节点，而 `i == size - 1` 说明现在处理到这一层最后一个节点了。既然层序遍历本身就是从左到右扫这一层，那么最后一个自然就是右视图能看到的那个节点。

剩下的写法，基本就是标准的 BFS 模板。

### 再补一种写法：DFS

这题除了 BFS，也可以用 DFS 来做。

不过 DFS 的想法和 BFS 不太一样。  
它不是一层一层扫，而是利用“**先访问右子树**”这个顺序，来保证每一层第一次遇到的节点，就是这一层最右边的节点。

做法是这样的：

- 递归时记录当前深度 `depth`
- 如果当前深度刚好等于结果列表的大小，说明这是我们第一次到达这一层
- 因为我们是按“根 -> 右 -> 左”的顺序递归，所以第一次到达这一层的节点，一定就是这一层最右边的节点
- 把它加入结果即可

这个思路也很巧，而且代码会更短一些。

### DFS 为什么成立

正常前序遍历是“根 -> 左 -> 右”。

如果我们改成“根 -> 右 -> 左”，那每一层都会优先走到更靠右的位置。  
这样一来，某一层最先被访问到的节点，就是从右边看过去最先露出来的那个节点，也就是答案里应该记录的节点。

所以 DFS 的关键不是“递归”本身，而是：

**先右后左 + 每层只记录第一次遇到的节点。**

### DFS 代码

```java
public List<Integer> rightSideView(TreeNode root) {
    List<Integer> res = new ArrayList<>();
    dfs(root, 0, res);
    return res;
}

private void dfs(TreeNode node, int depth, List<Integer> res) {
    if (node == null) return;

    if (depth == res.size()) {
        res.add(node.val);
    }

    dfs(node.right, depth + 1, res);
    dfs(node.left, depth + 1, res);
}
```

### 两种写法怎么选

如果是做题时第一次看到这道题，我更建议先写 BFS。

原因很简单：  
这题本来就是按“每一层”来思考的，BFS 和题目的表述最一致，不容易绕。

DFS 当然也很好，而且写出来更简洁，但它对“先右后左”和“第一次到达这一层”的理解要求更高一点。  
如果不是已经比较熟悉二叉树 DFS 的写法，第一次做这题时，BFS 会更稳。

### 复杂度分析

无论是 BFS 还是 DFS，本质上都要把每个节点访问一遍，所以时间复杂度都是 $O(n)$，其中 $n$ 是二叉树的节点个数。

空间复杂度要分情况看：

- BFS 的空间复杂度是 $O(n)$)，最坏情况下队列里可能会存一整层节点
- DFS 的空间复杂度是 $O(h)$，其中 $h$ 是树的高度，主要来自递归栈

### 这题真正值得记住的点

这题其实很典型：  
**题目一旦按“层”来描述，优先想层序遍历；题目一旦强调“第一次看到”“最后一次看到”，就要想想遍历顺序能不能直接把答案卡出来。**

右视图这题里：

- 用 BFS，就是取每层最后一个节点
- 用 DFS，就是取每层第一次访问到的节点，但前提是先走右子树

这两种写法本质上都在做同一件事，只是切入角度不同。

### 完整代码

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static List<Integer> rightSideView(TreeNode root) {
        List<Integer> res = new ArrayList<>();
        if (root == null) {
            return res;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            int size = queue.size();

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();

                if (i == size - 1) {
                    res.add(node.val);
                }

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }
        }

        return res;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(1);
        root1.left = new TreeNode(2);
        root1.right = new TreeNode(3);
        root1.left.right = new TreeNode(5);
        root1.right.right = new TreeNode(4);

        System.out.println("示例 1 输入: [1,2,3,null,5,null,4]");
        System.out.println("示例 1 输出: " + rightSideView(root1));

        TreeNode root2 = new TreeNode(1);
        root2.left = new TreeNode(2);
        root2.right = new TreeNode(3);
        root2.left.left = new TreeNode(4);
        root2.left.left.left = new TreeNode(5);

        System.out.println("示例 2 输入: [1,2,3,4,null,null,null,5]");
        System.out.println("示例 2 输出: " + rightSideView(root2));

        TreeNode root3 = new TreeNode(1);
        root3.right = new TreeNode(3);

        System.out.println("示例 3 输入: [1,null,3]");
        System.out.println("示例 3 输出: " + rightSideView(root3));

        TreeNode root4 = null;
        System.out.println("示例 4 输入: []");
        System.out.println("示例 4 输出: " + rightSideView(root4));
    }
}
```
