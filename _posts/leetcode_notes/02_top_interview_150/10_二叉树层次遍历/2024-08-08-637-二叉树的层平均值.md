## 题目

**描述：**

给定一个非空二叉树的根节点 `root`，以数组的形式返回每一层节点的平均值。与实际答案相差 `10^-5` 以内的答案可以被接受。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/03/09/avg1-tree.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[3.00000,14.50000,11.00000]`
- **解释:**

  第 0 层的平均值为 3，第 1 层的平均值为 14.5，第 2 层的平均值为 11。

  因此返回 `[3, 14.5, 11]`。

**示例 2:**

![示例2图片](https://assets.leetcode.com/uploads/2021/03/09/avg2-tree.jpg)

- **输入:** `root = [3,9,20,15,7]`
- **输出:** `[3.00000,14.50000,11.00000]`

**提示:**

- 树中节点数量在 `[1, 10^4]` 范围内
- `-2^31 <= Node.val <= 2^31 - 1`

## 题解

### 先把题目翻成更直接的话

这题要我们做的事情，其实就是：

**把二叉树一层一层地看过去，算出每一层节点值的平均数。**

题目里最重要的词是“每一层”。  
只要抓住这一点，思路基本就出来了，因为一旦题目和“层”有关，最自然的做法通常就是层序遍历。

### 这题为什么最适合用层序遍历

层序遍历本身就是按层从上到下扫描整棵树的。

每一轮遍历时，队列里正好存着当前这一层的所有节点。  
那我们只需要在处理这一层的时候，顺手把节点值加起来，最后再除以这一层的节点个数，就能得到这一层的平均值。

所以这题其实没有什么额外绕弯的地方，核心就一句话：

**遍历一层，统计一层，计算一层。**

### 具体怎么做

先把根节点放进队列。

只要队列不为空，就说明还有层没有处理完。每轮循环做下面几件事：

1. 先取出当前队列的大小 `size`，这个值就表示当前层有多少个节点
2. 用一个变量 `sum` 记录当前层所有节点值之和
3. 把这一层的 `size` 个节点依次出队，并把它们的左右子节点加入队列
4. 当前层处理结束后，用 `sum / size` 算出平均值，加入结果列表

整个过程非常顺，因为题目本身就是按层来问的。

### 为什么这里要用 `double`

这题有个细节要注意：平均值不一定是整数，所以最后结果肯定要用浮点数表示。

另外，节点值的范围也比较大，如果直接用 `int` 去累计整层的和，理论上存在溢出的风险。  
所以这里用 `double` 来存 `sum` 会更稳，既能避免累加时出问题，也方便后面直接计算平均值。

这一点虽然不难，但写代码时最好提前想到，不然很容易在细节上丢分。

### 用示例看一遍

比如示例 1：

```text
root = [3,9,20,null,null,15,7]
```

这棵树可以看成：

```text
    3
   / \
  9  20
    /  \
   15   7
```

按层来看：

- 第 0 层只有一个节点 `3`，平均值是 `3`
- 第 1 层有两个节点 `9` 和 `20`，平均值是 `(9 + 20) / 2 = 14.5`
- 第 2 层有两个节点 `15` 和 `7`，平均值是 `(15 + 7) / 2 = 11`

所以结果就是：

```text
[3.00000, 14.50000, 11.00000]
```

这题本质上就是这样，没有更复杂的隐藏条件。

### 代码

```java
public List<Double> averageOfLevels(TreeNode root) {
    List<Double> res = new ArrayList<>();
    Queue<TreeNode> queue = new LinkedList<>();
    queue.offer(root);

    while (!queue.isEmpty()) {
        int size = queue.size();
        double sum = 0;

        for (int i = 0; i < size; i++) {
            TreeNode node = queue.poll();
            sum += node.val;

            if (node.left != null) {
                queue.offer(node.left);
            }
            if (node.right != null) {
                queue.offer(node.right);
            }
        }

        res.add(sum / size);
    }

    return res;
}
```

### 这段代码里最关键的地方

核心其实就两步：

第一步，先用 `size = queue.size()` 固定住当前层的节点数量。  
这样我们这一轮循环处理的，刚好就是同一层的所有节点，不会和下一层混在一起。

第二步，把这一层节点值的和算出来，再除以 `size`。  
因为我们要的是“每层平均值”，所以重点不是某个单独节点，而是要把一整层作为一个整体来处理。

剩下的写法，基本都是标准的 BFS 模板。

### 再补一种写法：DFS

虽然这题最适合用 BFS，但也可以用 DFS。

只不过 DFS 不能像 BFS 那样天然地“一层一层扫”，所以要自己额外维护信息。常见做法是准备两个列表：

- 一个记录每一层节点值之和
- 一个记录每一层节点数量

递归遍历时，带上当前深度 `level`。

每访问一个节点，就看它属于第几层：

- 如果这一层之前已经出现过，就把它的值加到对应层的总和里，同时计数加一
- 如果这是第一次到达这一层，就在列表里新增这一层的和与数量

等整棵树遍历完之后，再统一计算每一层的平均值。

### DFS 为什么也能做

因为 DFS 虽然不是按层访问节点，但它能在递归过程中知道当前节点位于第几层。

只要我们把“第几层”这个信息记录下来，那就算节点不是按层连续出现的，也照样能把同一层的数据累计到一起。

所以 BFS 和 DFS 的区别，不在结果能不能算出来，而在于：

- BFS 是天然按层组织数据
- DFS 是先遍历，再靠 `level` 把数据归到对应层里

这题里，BFS 会更顺手一些。

### DFS 代码

```java
public List<Double> averageOfLevels(TreeNode root) {
    List<Double> sums = new ArrayList<>();
    List<Integer> counts = new ArrayList<>();
    dfs(root, 0, sums, counts);

    List<Double> res = new ArrayList<>();
    for (int i = 0; i < sums.size(); i++) {
        res.add(sums.get(i) / counts.get(i));
    }
    return res;
}

private void dfs(TreeNode node, int level, List<Double> sums, List<Integer> counts) {
    if (node == null) return;

    if (level < sums.size()) {
        sums.set(level, sums.get(level) + node.val);
        counts.set(level, counts.get(level) + 1);
    } else {
        sums.add((double) node.val);
        counts.add(1);
    }

    dfs(node.left, level + 1, sums, counts);
    dfs(node.right, level + 1, sums, counts);
}
```

### 这题该怎么选写法

如果是第一次做这题，我更建议直接写 BFS。

原因很简单：  
题目问的是“每一层的平均值”，而 BFS 恰好就是按层处理节点的，思路和题目表述完全一致，写起来也最不容易乱。

DFS 当然也能做，而且不难，但它更像是“换一种方式把同层节点归类起来”。  
如果只是为了把题写对、写稳，BFS 会更合适。

### 复杂度分析

无论是 BFS 还是 DFS，每个节点都只会被访问一次，所以时间复杂度都是 $O(n)$，其中 $n$ 是二叉树的节点数。

空间复杂度分两种情况来看：

- BFS 的空间复杂度是 $O(m)$，其中 $m$ 是树的最大宽度，也就是某一层最多节点数
- DFS 的空间复杂度主要来自递归栈和层级统计，一般记作 $O(h)$，其中 $h$ 是树的高度

### 这题真正值得记住的地方

这题本身不难，但很典型。

它最值得记住的地方其实不是“怎么求平均值”，而是这个判断：

**只要题目明显是在按层提问，就优先想层序遍历。**

因为很多二叉树题，看起来写法不同，实际上关键就在于你有没有第一时间把题目和“层”联系起来。

像这题：

- 要求每层平均值，就按层统计和与数量
- 要求每层最大值、最小值、最右节点，思路也都类似

这一类题的底层逻辑其实是一样的。

### 完整代码

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

public class Main {

    public static class TreeNode {
        int val;
        TreeNode left;
        TreeNode right;

        TreeNode() {}

        TreeNode(int val) {
            this.val = val;
        }

        TreeNode(int val, TreeNode left, TreeNode right) {
            this.val = val;
            this.left = left;
            this.right = right;
        }
    }

    public static List<Double> averageOfLevels(TreeNode root) {
        List<Double> res = new ArrayList<>();
        if (root == null) {
            return res;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            int size = queue.size();
            double sum = 0;

            for (int i = 0; i < size; i++) {
                TreeNode node = queue.poll();
                sum += node.val;

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            res.add(sum / size);
        }

        return res;
    }

    public static void main(String[] args) {
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);

        System.out.println("示例 1 输入: [3,9,20,null,null,15,7]");
        System.out.println("示例 1 输出: " + averageOfLevels(root1));

        TreeNode root2 = new TreeNode(3);
        root2.left = new TreeNode(9);
        root2.right = new TreeNode(20);
        root2.left.left = new TreeNode(15);
        root2.left.right = new TreeNode(7);

        System.out.println("示例 2 输入: [3,9,20,15,7]");
        System.out.println("示例 2 输出: " + averageOfLevels(root2));

        TreeNode root3 = new TreeNode(2147483647);
        root3.right = new TreeNode(2147483647);

        System.out.println("示例 3 输入: [2147483647,null,2147483647]");
        System.out.println("示例 3 输出: " + averageOfLevels(root3));
    }
}
```
