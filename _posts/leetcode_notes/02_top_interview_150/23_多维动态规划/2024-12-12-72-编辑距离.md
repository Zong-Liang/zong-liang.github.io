## 题目

**描述：**

给你两个单词 `word1` 和 `word2`，请返回将 `word1` 转换成 `word2` 所使用的最少操作数。

你可以对一个单词进行如下三种操作：

- 插入一个字符
- 删除一个字符
- 替换一个字符

**示例 1:**

- **输入:** `word1 = "horse"`, `word2 = "ros"`
- **输出:** 3
- **解释:**

  horse -> rorse (将 'h' 替换为 'r')

  rorse -> rose (删除 'r')

  rose -> ros (删除 'e')

**示例 2:**

- **输入:** `word1 = "intention"`, `word2 = "execution"`
- **输出:** 5
- **解释:**

  intention -> inention (删除 't')

  inention -> enention (将 'i' 替换为 'e')

  enention -> exention (将 'n' 替换为 'x')

  exention -> exection (将 'n' 替换为 'c')

  exection -> execution (插入 'u')

**提示:**

- `0 <= word1.length, word2.length <= 500`
- `word1` 和 `word2` 由小写英文字母组成

## 题解

编辑距离这题看起来是字符串操作，核心其实是“最后一步做了什么”。如果直接盯着两个完整字符串，很容易在插入、删除、替换之间反复摇摆；换成前缀视角就清晰很多。

定义 `dp[i][j]` 为 `word1` 前 `i` 个字符变成 `word2` 前 `j` 个字符的最少操作数。这个定义的好处是，每个状态都能落到一个明确的“末尾处理”决策上。

边界先看两种最直接的情况。
- `dp[i][0] = i`，因为把长度为 `i` 的前缀变成空串，只能删除 `i` 次。
- `dp[0][j] = j`，因为空串变成长度为 `j` 的前缀，只能插入 `j` 次。

状态转移也分成两种情况。
- 若 `word1[i - 1] == word2[j - 1]`，末尾已经匹配，`dp[i][j] = dp[i - 1][j - 1]`。
- 若不相等，就在三种操作里选最小代价：替换（左上 + 1）、删除（上方 + 1）、插入（左方 + 1）。

```text
dp[i][j] = min(dp[i - 1][j - 1], dp[i - 1][j], dp[i][j - 1]) + 1
```

以 `horse -> ros` 为例，表格推进时你会看到所有决策都在重复同一件事：比较末尾是否相同；若不同，就从三种编辑动作里选代价最小的那个。

时间复杂度和空间复杂度可以分别来看。
- 时间复杂度 `O(mn)`，每个状态计算一次。
- 空间复杂度 `O(mn)`，使用二维表存状态。

这题可以沉淀成一个通用套路：遇到“把 A 变成 B，且操作只影响局部末尾”的问题，优先思考“最后一步来源”。转移关系通常就会自然出现。

完整代码如下：
```java
public class Main {

    public static int minDistance(String word1, String word2) {
        int m = word1.length();
        int n = word2.length();

        int[][] dp = new int[m + 1][n + 1];

        for (int i = 0; i <= m; i++) {
            dp[i][0] = i;
        }

        for (int j = 0; j <= n; j++) {
            dp[0][j] = j;
        }

        for (int i = 1; i <= m; i++) {
            for (int j = 1; j <= n; j++) {
                if (word1.charAt(i - 1) == word2.charAt(j - 1)) {
                    dp[i][j] = dp[i - 1][j - 1];
                } else {
                    dp[i][j] = Math.min(
                        dp[i - 1][j - 1],
                        Math.min(dp[i - 1][j], dp[i][j - 1])
                    ) + 1;
                }
            }
        }

        return dp[m][n];
    }

    public static void main(String[] args) {
        String w1_ex1 = "horse";
        String w2_ex1 = "ros";
        System.out.println("示例 1 输入: word1 = \"" + w1_ex1 + "\", word2 = \"" + w2_ex1 + "\"");
        System.out.println("示例 1 输出: " + minDistance(w1_ex1, w2_ex1));
        System.out.println("-----------------------------");

        String w1_ex2 = "intention";
        String w2_ex2 = "execution";
        System.out.println("示例 2 输入: word1 = \"" + w1_ex2 + "\", word2 = \"" + w2_ex2 + "\"");
        System.out.println("示例 2 输出: " + minDistance(w1_ex2, w2_ex2));
        System.out.println("-----------------------------");

        String w1_ex3 = "";
        String w2_ex3 = "a";
        System.out.println("示例 3 输入: word1 = \"" + w1_ex3 + "\", word2 = \"" + w2_ex3 + "\"");
        System.out.println("示例 3 输出: " + minDistance(w1_ex3, w2_ex3));
    }
}
```
