## 题目

**描述：**

给你一个字符串 `s`，找到 `s` 中最长的**回文子串**。

**示例 1:**

- **输入:** `s = "babad"`
- **输出:** `"bab"`
- **解释:** `"aba"` 同样是符合题意的答案。

**示例 2:**

- **输入:** `s = "cbbd"`
- **输出:** `"bb"`

**提示:**

- `1 <= s.length <= 1000`
- `s` 仅由数字和英文字母组成

## 题解

虽然这篇放在动态规划分组里，但当前代码用的是中心扩散法，这个选择很合理。因为回文串最本质的特征是“围绕中心对称”，从中心向两边扩比“先枚举子串再验证”更贴问题结构。

做法很直接：把每个位置都当成回文中心去扩散。需要注意中心有两种形态：一种是单字符中心（对应奇数长度回文），一种是双字符中心（对应偶数长度回文）。如果只处理前者，像 `"cbbd"` 里的 `"bb"` 就会漏掉。

扩散函数负责做同一件事：给定 `left` 和 `right`，只要不越界且两侧字符相等就继续外扩；循环停止时，上一次合法区间才是这个中心下的最长回文，所以长度返回 `right - left - 1`。

主循环每次取两种中心扩散长度的较大值，再根据中心位置把长度还原成区间边界。代码里的

```text
start = i - (len - 1) / 2
end = i + len / 2
```

就是在统一处理奇偶两类回文，不是技巧记忆点，理解成“中心 + 长度 -> 左右边界”就够了。

示例 `"babad"` 跑下来，`"bab"` 和 `"aba"` 都会被命中；示例 `"cbbd"` 则依赖双中心拿到 `"bb"`。这两组数据刚好覆盖了这题最容易漏的地方。

时间复杂度和空间复杂度可以分别来看。
- 时间复杂度 `O(n^2)`，最坏情况下每个中心都可能扩到两端。
- 空间复杂度 `O(1)`，只用了常数个变量。

如果题目对象本身有很强结构特征（这里是对称中心），优先围绕这个结构去枚举，通常比“先暴力枚举结果再验证”更自然，也更不容易写出碎逻辑。

完整代码如下：
```java
public class Main {

    public static String longestPalindrome(String s) {
        if (s == null || s.length() < 1) {
            return "";
        }

        int start = 0, end = 0;

        for (int i = 0; i < s.length(); i++) {
            int len1 = expandAroundCenter(s, i, i);
            int len2 = expandAroundCenter(s, i, i + 1);

            int len = Math.max(len1, len2);

            if (len > end - start) {
                start = i - (len - 1) / 2;
                end = i + len / 2;
            }
        }

        return s.substring(start, end + 1);
    }

    private static int expandAroundCenter(String s, int left, int right) {
        while (left >= 0 && right < s.length() && s.charAt(left) == s.charAt(right)) {
            left--;
            right++;
        }

        return right - left - 1;
    }

    public static void main(String[] args) {
        String s1 = "babad";
        System.out.println("示例 1 输入: s = \"" + s1 + "\"");
        System.out.println("示例 1 输出: \"" + longestPalindrome(s1) + "\"");
        System.out.println("-----------------------------");

        String s2 = "cbbd";
        System.out.println("示例 2 输入: s = \"" + s2 + "\"");
        System.out.println("示例 2 输出: \"" + longestPalindrome(s2) + "\"");
        System.out.println("-----------------------------");

        String s3 = "a";
        System.out.println("示例 3 输入: s = \"" + s3 + "\"");
        System.out.println("示例 3 输出: \"" + longestPalindrome(s3) + "\"");
        System.out.println("-----------------------------");

        String s4 = "aaaa";
        System.out.println("示例 4 输入: s = \"" + s4 + "\"");
        System.out.println("示例 4 输出: \"" + longestPalindrome(s4) + "\"");
    }
}
```
