## 题目

**描述：**

给定一个包含非负整数的 `m x n` 网格 `grid`，请找出一条从左上角到右下角的路径，使得路径上的数字总和为最小。

**说明：** 每次只能向下或者向右移动一步。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2020/11/05/minpath.jpg)

- **输入:** `grid = [[1,3,1],[1,5,1],[4,2,1]]`
- **输出:** 7
- **解释:** 因为路径 1→3→1→1→1 的总和最小。

**示例 2:**

- **输入:** `grid = [[1,2,3],[4,5,6]]`
- **输出:** 12

**提示:**

- `m == grid.length`
- `n == grid[i].length`
- `1 <= m, n <= 200`
- `0 <= grid[i][j] <= 200`

## 题解

这题看起来是“找一条路”，但真正该算的是“到每个格子的最小代价”。因为只能向右或向下移动，走到 `(i, j)` 的最后一步只可能来自上方 `(i - 1, j)` 或左侧 `(i, j - 1)`，所以它的最优值不会和其他方向发生关系。

把状态定成 `dp[i][j]`（从左上角走到 `(i, j)` 的最小路径和）之后，转移就很自然：

```text
dp[i][j] = min(dp[i - 1][j], dp[i][j - 1]) + grid[i][j]
```

边界处理也顺着这个定义来。第一行只能一路从左往右累加，第一列只能一路从上往下累加。把两条边界铺好以后，中间区域统一套转移公式即可。

示例 `[[1,3,1],[1,5,1],[4,2,1]]` 手推时会看到这个过程：第一行变成 `[1,4,5]`，第一列继续累加后左边界是 `[1,2,6]`，再按“上/左取小”填中间，最终右下角得到 `7`。

这题代码直接在 `grid` 上原地更新，不另开 `dp` 数组。这样做的原因是：某个格子的原始值一旦参与完转移，后续不再单独需要，写回最小路径和反而更紧凑。

时间复杂度和空间复杂度可以分别来看。
- 时间复杂度 `O(mn)`，每个格子只处理一次。
- 空间复杂度 `O(1)`（不计输入），因为直接复用了原网格。

这类网格题可以优先看一个信号：如果“当前位置最优值”只来自固定前驱（比如上和左），通常就是标准 DP 递推，不必枚举所有路径。

完整代码如下：
```java
import java.util.Arrays;

public class Main {

    public static int minPathSum(int[][] grid) {
        if (grid == null || grid.length == 0 || grid[0].length == 0) {
            return 0;
        }

        int m = grid.length;
        int n = grid[0].length;

        for (int j = 1; j < n; j++) {
            grid[0][j] += grid[0][j - 1];
        }

        for (int i = 1; i < m; i++) {
            grid[i][0] += grid[i - 1][0];
        }

        for (int i = 1; i < m; i++) {
            for (int j = 1; j < n; j++) {
                grid[i][j] += Math.min(grid[i - 1][j], grid[i][j - 1]);
            }
        }

        return grid[m - 1][n - 1];
    }

    public static void main(String[] args) {
        int[][] grid1 = {
            {1, 3, 1},
            {1, 5, 1},
            {4, 2, 1}
        };
        System.out.println("示例 1 输入: " + Arrays.deepToString(grid1));
        System.out.println("示例 1 输出: " + minPathSum(grid1));
        System.out.println("-----------------------------");

        int[][] grid2 = {
            {1, 2, 3},
            {4, 5, 6}
        };
        System.out.println("示例 2 输入: " + Arrays.deepToString(grid2));
        System.out.println("示例 2 输出: " + minPathSum(grid2));
    }
}
```
