## 题目

**描述：**

给你一个整数数组 `prices` 和一个整数 `k`，其中 `prices[i]` 是某支给定的股票在第 `i` 天的价格。

设计一个算法来计算你所能获取的最大利润。你最多可以完成 **k** 笔交易。也就是说，你最多可以买 `k` 次，卖 `k` 次。

**注意：** 你不能同时参与多笔交易（你必须在再次购买前出售掉之前的股票）。

**示例 1:**

- **输入:** `k = 2`, `prices = [2,4,1]`
- **输出:** 2
- **解释:** 在第 1 天（股票价格 = 2）的时候买入，在第 2 天（股票价格 = 4）的时候卖出，这笔交易所能获得利润 = 4-2 = 2。

**示例 2:**

- **输入:** `k = 2`, `prices = [3,2,6,5,0,3]`
- **输出:** 7
- **解释:**

  在第 2 天（股票价格 = 2）的时候买入，在第 3 天（股票价格 = 6）的时候卖出，这笔交易所能获得利润 = 6-2 = 4。

  随后，在第 5 天（股票价格 = 0）的时候买入，在第 6 天（股票价格 = 3）的时候卖出，这笔交易所能获得利润 = 3-0 = 3。

**提示:**

- `1 <= k <= 100`
- `1 <= prices.length <= 1000`
- `0 <= prices[i] <= 1000`

## 题解

这题可以看成 123 题的通用版：把“最多两笔交易”推广成“最多 `k` 笔交易”。思路本身没变，变的是状态数量从常数变成了和 `k` 相关。

对于第 `j` 笔交易，我们仍然维护两类状态。
- `buy[j]`：完成第 `j` 次买入后（持股）的最大利润。
- `sell[j]`：完成第 `j` 次卖出后（空仓）的最大利润。

每天价格 `p` 到来时，状态转移与两笔交易版本一一对应：

```text
buy[j]  = max(buy[j],  sell[j - 1] - p)
sell[j] = max(sell[j], buy[j] + p)
```

`buy[j]` 依赖 `sell[j - 1]`，本质上是在保证交易不重叠：第 `j` 次买入前，前 `j - 1` 笔必须已经卖完。

还有一个很实用的优化：当 `k >= n / 2` 时，交易次数上限已经不会成为约束，问题退化成“可无限次交易”。这时直接累加所有上升段差值即可，复杂度更低，也更直观。

为什么这套模型稳定？因为它没有猜具体买卖区间，而是让每一天都去更新“如果我正处于某个交易阶段，当前最好利润是多少”。最终 `sell[k]` 就是答案。

时间复杂度和空间复杂度可以分别来看。
- 一般情况：时间复杂度 `O(nk)`，空间复杂度 `O(k)`。
- 退化为无限次交易时：时间复杂度 `O(n)`，空间复杂度 `O(1)`。

这题最值得带走的是“固定状态模板可推广”这个能力：先把小 `k` 版本讲透，再把状态从单变量推广到数组，很多股票题都能用同一套语言描述清楚。

完整代码如下：
```java
import java.util.Arrays;

public class Main {

    public static int maxProfit(int k, int[] prices) {
        if (k <= 0 || prices == null || prices.length == 0) {
            return 0;
        }

        int n = prices.length;
        if (k >= n / 2) {
            return maxProfitUnlimited(prices);
        }

        int[] buy = new int[k + 1];
        int[] sell = new int[k + 1];

        Arrays.fill(buy, -prices[0]);
        Arrays.fill(sell, 0);

        for (int p : prices) {
            for (int j = 1; j <= k; j++) {
                buy[j] = Math.max(buy[j], sell[j - 1] - p);
                sell[j] = Math.max(sell[j], buy[j] + p);
            }
        }

        return sell[k];
    }

    private static int maxProfitUnlimited(int[] prices) {
        int maxProfit = 0;
        for (int i = 1; i < prices.length; i++) {
            if (prices[i] > prices[i - 1]) {
                maxProfit += prices[i] - prices[i - 1];
            }
        }
        return maxProfit;
    }

    public static void main(String[] args) {
        int k1 = 2;
        int[] prices1 = {2, 4, 1};
        System.out.println("示例 1 输入: k = " + k1 + ", prices = " + Arrays.toString(prices1));
        System.out.println("示例 1 输出: " + maxProfit(k1, prices1));
        System.out.println("-----------------------------");

        int k2 = 2;
        int[] prices2 = {3, 2, 6, 5, 0, 3};
        System.out.println("示例 2 输入: k = " + k2 + ", prices = " + Arrays.toString(prices2));
        System.out.println("示例 2 输出: " + maxProfit(k2, prices2));
        System.out.println("-----------------------------");

        int k3 = 100;
        int[] prices3 = {1, 2, 3, 4, 5};
        System.out.println("示例 3 输入: k = " + k3 + ", prices = " + Arrays.toString(prices3));
        System.out.println("示例 3 输出: " + maxProfit(k3, prices3));
    }
}
```
