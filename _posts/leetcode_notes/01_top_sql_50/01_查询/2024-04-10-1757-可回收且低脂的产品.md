## 题目：1757. 可回收且低脂的产品

**表:** `Products`

| Column Name | Type            |
| ----------- | --------------- |
| product_id  | int             |
| low_fats    | enum ('Y', 'N') |
| recyclable  | enum ('Y', 'N') |

- `product_id` 是该表的主键（具有唯一值的列）。
- `low_fats` 是枚举类型，取值为以下两种 ('Y', 'N')，其中 'Y' 表示该产品是低脂产品，'N' 表示不是低脂产品。
- `recyclable` 是枚举类型，取值为以下两种 ('Y', 'N')，其中 'Y' 表示该产品可回收，而 'N' 表示不可回收。

**问题描述:**

编写解决方案找出既是低脂又是可回收的产品编号。

返回结果 **无顺序要求**。

**示例 1:**

**输入:**

`Products` 表:

| product_id | low_fats | recyclable |
| ---------- | -------- | ---------- |
| 0          | Y        | N          |
| 1          | Y        | Y          |
| 2          | N        | Y          |
| 3          | Y        | Y          |
| 4          | N        | N          |

**输出:**

| product_id |
| ---------- |
| 1          |
| 3          |

**解释:**

只有产品 id 为 1 和 3 的产品, 既是低脂又是可回收的产品。

## 题解

这题和前面那种“满足任意一个条件”的题不一样，这里题目要求的是：**既是低脂产品，又是可回收产品**。

“既……又……”这种表述，放到 SQL 里通常就意味着：**两个条件要同时满足**，所以这里要用 `AND`。

题目里两个字段的含义也很清楚：

- `low_fats = 'Y'`：表示这个产品是低脂的；
- `recyclable = 'Y'`：表示这个产品是可回收的。

因此，我们只需要把这两个条件同时写在 `WHERE` 里即可。

```sql
SELECT product_id
FROM Products
WHERE low_fats = 'Y' AND recyclable = 'Y';
```

这条语句可以这样理解：

- `SELECT product_id`：题目只要求返回产品编号；
- `FROM Products`：数据来自 `Products` 表；
- `WHERE low_fats = 'Y' AND recyclable = 'Y'`：筛出既低脂又可回收的产品。

结合示例数据来看：

- 产品 `1` 的 `low_fats` 和 `recyclable` 都是 `'Y'`，所以要保留；
- 产品 `3` 的两个条件也都满足，所以也要保留；
- 像产品 `0`，虽然是低脂，但不可回收，就不能选出来；
- 像产品 `2`，虽然可回收，但不是低脂，也不能选出来。

所以最终结果就是：

- `1`
- `3`

这题本身不难，关键是读题时要分清楚条件之间的关系。  
题目说“满足任意一个”时，一般用 `OR`；题目说“同时满足”或者“既……又……”时，就要想到用 `AND`。
