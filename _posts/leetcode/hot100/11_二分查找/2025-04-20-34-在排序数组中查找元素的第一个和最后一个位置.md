## 问题描述:

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

**示例 1：**

```java
输入: nums = [5,7,7,8,8,10], target = 8
输出: [3,4]
```

**示例 2：**

```java
输入: nums = [5,7,7,8,8,10], target = 6
输出: [-1,-1]
```

**示例 3：**

```java
输入: nums = [], target = 0
输出: [-1,-1]
```

**提示：** `nums` 是一个非递减数组。

## 解题思路：

这是一道关于在排序数组中查找目标值的第一个和最后一个位置的问题。需要在 O(log n)时间复杂度内找到目标值的起始和结束位置。

### 方法一：两次二分查找（推荐）

核心思想是分别使用二分查找找到目标值的左边界和右边界。通过修改二分查找的条件来实现查找边界。

**算法步骤：**

1. 使用二分查找找到目标值的左边界（第一次出现的位置）
2. 使用二分查找找到目标值的右边界（最后一次出现的位置）
3. 如果找不到目标值，返回[-1, -1]
4. 返回[左边界, 右边界]

```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        if (nums == null || nums.length == 0) {
            return new int[]{-1, -1};
        }

        int leftBound = findLeftBound(nums, target);
        int rightBound = findRightBound(nums, target);

        return new int[]{leftBound, rightBound};
    }

    // 寻找左边界（第一次出现的位置）
    private int findLeftBound(int[] nums, int target) {
        int left = 0, right = nums.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                // 找到目标值，但继续向左搜索
                right = mid - 1;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        // 检查left是否越界，以及nums[left]是否等于target
        if (left >= nums.length || nums[left] != target) {
            return -1;
        }

        return left;
    }

    // 寻找右边界（最后一次出现的位置）
    private int findRightBound(int[] nums, int target) {
        int left = 0, right = nums.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                // 找到目标值，但继续向右搜索
                left = mid + 1;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        // 检查right是否越界，以及nums[right]是否等于target
        if (right < 0 || nums[right] != target) {
            return -1;
        }

        return right;
    }
}
```

**时间复杂度：** $O(\log n)$，进行了两次二分查找。

**空间复杂度：** $O(1)$，只使用常数额外空间。

### 方法二：一次二分查找 + 扩展搜索

先用标准二分查找找到目标值，然后向左右扩展找到边界。

```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        if (nums == null || nums.length == 0) {
            return new int[]{-1, -1};
        }

        // 标准二分查找
        int index = binarySearch(nums, target);

        if (index == -1) {
            return new int[]{-1, -1};
        }

        // 找到目标值后，向左右扩展找边界
        int left = index, right = index;

        // 向左扩展找左边界
        while (left > 0 && nums[left - 1] == target) {
            left--;
        }

        // 向右扩展找右边界
        while (right < nums.length - 1 && nums[right + 1] == target) {
            right++;
        }

        return new int[]{left, right};
    }

    private int binarySearch(int[] nums, int target) {
        int left = 0, right = nums.length - 1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                return mid;
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return -1;
    }
}
```

**时间复杂度：** 最坏情况下$O(n)$（当所有元素都是目标值时），平均情况$O(\log n)$。

**空间复杂度：** $O(1)$。

### 方法三：使用库函数优化的二分查找

利用更简洁的边界查找逻辑。

```java
class Solution {
    public int[] searchRange(int[] nums, int target) {
        if (nums == null || nums.length == 0) {
            return new int[]{-1, -1};
        }

        int leftBound = findBound(nums, target, true);

        if (leftBound == -1) {
            return new int[]{-1, -1};
        }

        int rightBound = findBound(nums, target, false);

        return new int[]{leftBound, rightBound};
    }

    // isLeft为true时找左边界，false时找右边界
    private int findBound(int[] nums, int target, boolean isLeft) {
        int left = 0, right = nums.length - 1;
        int result = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                result = mid; // 记录结果
                if (isLeft) {
                    right = mid - 1; // 继续向左搜索
                } else {
                    left = mid + 1;  // 继续向右搜索
                }
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }

        return result;
    }
}
```

**时间复杂度：** $O(\log n)$。

**空间复杂度：** $O(1)$。

### 关键点总结：

1. **边界查找思路**：找左边界时，遇到目标值继续向左搜索；找右边界时，遇到目标值继续向右搜索
2. **边界条件**：注意数组越界检查和目标值存在性验证
3. **优化策略**：可以先找左边界，如果不存在则直接返回[-1, -1]，避免不必要的右边界查找

**推荐使用方法一（两次二分查找）**，因为逻辑清晰，代码结构良好，且严格满足 O(log n)的时间复杂度要求。方法三的统一边界查找函数也是不错的选择，代码更简洁。
