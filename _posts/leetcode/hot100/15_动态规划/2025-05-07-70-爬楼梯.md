## 问题描述: 

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例 1:**

```java
输入: n = 2
输出: 2
解释: 有两种方法可以爬到楼顶 
(1) 1 阶 + 1 阶
(2) 2 阶
```

**示例 2:**

```java
输入: n = 3
输出: 3
解释: 有三种方法可以爬到楼顶
(1) 1 阶 + 1 阶 + 1 阶
(2) 1 阶 + 2 阶
(3) 2 阶 + 1 阶
```

## 解题思路: 

这是一个典型的斐波那契数列问题：

1. 基本情况：

   - `n=1` 时，只有 `1` 种方法：`1` 阶；
   - `n=2` 时，有 `2` 种方法：`1+1` 阶 或 `2` 阶；

2. 递推关系：

   - 要到达第 `n` 阶，可以从第 `(n-1)` 阶走 `1` 步，或从第 `(n-2)` 阶走 `2` 步；
   - 所以：`f(n) = f(n-1) + f(n-2)`。


3. 解法选择：

   - 递归（会超时）
   - 动态规划
   - 空间优化的动态规划

**推荐解法（空间优化的动态规划）：**

**核心思想：**

- 当前阶数的方法数 = 前一阶的方法数 + 前两阶的方法数
- 只需要保存前两个状态，不需要整个数组。

时间复杂度：$O(n)$ - 只需要遍历一次。

空间复杂度：$O(1)$ - 只用了三个变量。

**解题步骤：**

1. 边界处理：`n≤2` 时直接返回 `n`；
2. 初始化：设置前两阶的初始值；
3. 状态转移：从第三阶开始，依次计算每一阶的方法数；
4. 滚动更新：更新前两阶的值，准备下一次计算。

**数学本质：**

这个问题本质上是求斐波那契数列的第 `n+1` 项：

- `F(1) = 1`
- `F(2) = 2`
- `F(n) = F(n-1) + F(n-2)`

## Java代码: 
```java
public class Solution {
    
    /**
     * 方法1：动态规划 - 使用数组
     * 时间复杂度：O(n)，空间复杂度：O(n)
     */
    public int climbStairs(int n) {
        if (n <= 2) {
            return n;
        }
        
        int[] dp = new int[n + 1];
        dp[1] = 1;  // 1阶楼梯有1种方法
        dp[2] = 2;  // 2阶楼梯有2种方法
        
        // 从第3阶开始计算
        for (int i = 3; i <= n; i++) {
            dp[i] = dp[i - 1] + dp[i - 2];
        }
        
        return dp[n];
    }
    
    /**
     * 方法2：空间优化的动态规划
     * 时间复杂度：O(n)，空间复杂度：O(1)
     * 推荐使用这种方法
     */
    public int climbStairsOptimized(int n) {
        if (n <= 2) {
            return n;
        }
        
        int prev2 = 1;  // f(n-2)
        int prev1 = 2;  // f(n-1)
        int current = 0; // f(n)
        
        for (int i = 3; i <= n; i++) {
            current = prev1 + prev2;
            prev2 = prev1;
            prev1 = current;
        }
        
        return current;
    }
    
    /**
     * 方法3：递归（会超时，仅作演示）
     * 时间复杂度：O(2^n)，空间复杂度：O(n)
     */
    public int climbStairsRecursive(int n) {
        if (n <= 2) {
            return n;
        }
        return climbStairsRecursive(n - 1) + climbStairsRecursive(n - 2);
    }
    
    /**
     * 方法4：记忆化递归
     * 时间复杂度：O(n)，空间复杂度：O(n)
     */
    public int climbStairsMemo(int n) {
        int[] memo = new int[n + 1];
        return climbStairsHelper(n, memo);
    }
    
    private int climbStairsHelper(int n, int[] memo) {
        if (n <= 2) {
            return n;
        }
        
        if (memo[n] != 0) {
            return memo[n];
        }
        
        memo[n] = climbStairsHelper(n - 1, memo) + climbStairsHelper(n - 2, memo);
        return memo[n];
    }
    
    /**
     * 方法5：数学公式解法（斐波那契数列通项公式）
     * 时间复杂度：O(1)，空间复杂度：O(1)
     */
    public int climbStairsMath(int n) {
        if (n <= 2) {
            return n;
        }
        
        double sqrt5 = Math.sqrt(5);
        double phi = (1 + sqrt5) / 2;
        double psi = (1 - sqrt5) / 2;
        
        // 斐波那契数列的第n+1项
        return (int) Math.round((Math.pow(phi, n + 1) - Math.pow(psi, n + 1)) / sqrt5);
    }
    
    /**
     * 测试方法
     */
    public static void main(String[] args) {
        Solution solution = new Solution();
        
        // 测试用例
        int[] testCases = {1, 2, 3, 4, 5, 10};
        
        System.out.println("测试结果：");
        for (int n : testCases) {
            int result1 = solution.climbStairs(n);
            int result2 = solution.climbStairsOptimized(n);
            int result3 = solution.climbStairsMemo(n);
            
            System.out.printf("n=%d: DP数组=%d, 空间优化=%d, 记忆化=%d%n", 
                            n, result1, result2, result3);
        }
        
        // 详细解释过程
        System.out.println("\n解题过程演示 (n=5):");
        demonstrateProcess(5);
    }
    
    /**
     * 演示解题过程
     */
    private static void demonstrateProcess(int n) {
        System.out.println("f(1) = 1 (1种方法: 1阶)");
        System.out.println("f(2) = 2 (2种方法: 1+1阶 或 2阶)");
        
        int prev2 = 1, prev1 = 2;
        for (int i = 3; i <= n; i++) {
            int current = prev1 + prev2;
            System.out.printf("f(%d) = f(%d) + f(%d) = %d + %d = %d%n", 
                            i, i-1, i-2, prev1, prev2, current);
            prev2 = prev1;
            prev1 = current;
        }
    }
}

/**
 * 扩展：如果每次可以爬1、2或3阶怎么办？
 */
class ExtendedSolution {
    /**
     * 每次可以爬1、2或3阶楼梯
     */
    public int climbStairsThreeSteps(int n) {
        if (n == 0) return 1;
        if (n == 1) return 1;
        if (n == 2) return 2;
        if (n == 3) return 4;
        
        int[] dp = new int[n + 1];
        dp[0] = 1;
        dp[1] = 1;
        dp[2] = 2;
        dp[3] = 4;
        
        for (int i = 4; i <= n; i++) {
            dp[i] = dp[i-1] + dp[i-2] + dp[i-3];
        }
        
        return dp[n];
    }
    
    /**
     * 通用解法：给定可以爬的步数数组
     */
    public int climbStairsGeneral(int n, int[] steps) {
        int[] dp = new int[n + 1];
        dp[0] = 1;
        
        for (int i = 1; i <= n; i++) {
            for (int step : steps) {
                if (i >= step) {
                    dp[i] += dp[i - step];
                }
            }
        }
        
        return dp[n];
    }
}
```
