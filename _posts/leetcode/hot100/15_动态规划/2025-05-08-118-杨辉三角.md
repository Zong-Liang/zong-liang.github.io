## 问题描述: 

给定一个非负整数 *`numRows`，*生成「杨辉三角」的前 *`numRows`* 行。

在「杨辉三角」中，每个数是它左上方和右上方的数的和。

![img](../assets/img/posts/p118_0.gif)

**示例 1:**

```java
输入: numRows = 5
输出: [[1],[1,1],[1,2,1],[1,3,3,1],[1,4,6,4,1]]
```

**示例 2:**

```java
输入: numRows = 1
输出: [[1]]
```


## 解题思路: 

**杨辉三角的规律是：**

1. 第一行只有一个数字 `1`；
2. 每一行的第一个和最后一个数字都是 `1`；
3. 中间的数字等于上一行相邻两个数字的和。

**构建方法：**

1. 创建一个二维数组或列表来存储结果；
2. 逐行构建，每行的长度等于行号 `+1`；
3. 对于每一行，首尾设为 `1`，中间元素等于上一行对应位置的两个数之和。

## Java代码: 
```java
import java.util.*;

public class Solution {
    
    /**
     * 生成杨辉三角
     * @param numRows 行数
     * @return 杨辉三角的二维列表
     */
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> result = new ArrayList<>();
        
        // 边界条件检查
        if (numRows <= 0) {
            return result;
        }
        
        // 逐行生成杨辉三角
        for (int i = 0; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();
            
            // 每行的长度为 i + 1
            for (int j = 0; j <= i; j++) {
                // 每行的第一个和最后一个元素都是1
                if (j == 0 || j == i) {
                    row.add(1);
                } else {
                    // 中间的元素等于上一行相邻两个元素的和
                    int sum = result.get(i - 1).get(j - 1) + result.get(i - 1).get(j);
                    row.add(sum);
                }
            }
            
            result.add(row);
        }
        
        return result;
    }
    
    /**
     * 优化版本 - 空间复杂度更低的实现
     */
    public List<List<Integer>> generateOptimized(int numRows) {
        List<List<Integer>> result = new ArrayList<>();
        
        if (numRows <= 0) {
            return result;
        }
        
        for (int i = 0; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();
            
            // 先添加第一个元素1
            row.add(1);
            
            // 计算中间的元素
            for (int j = 1; j < i; j++) {
                row.add(result.get(i - 1).get(j - 1) + result.get(i - 1).get(j));
            }
            
            // 如果不是第一行，添加最后一个元素1
            if (i > 0) {
                row.add(1);
            }
            
            result.add(row);
        }
        
        return result;
    }
    
    /**
     * 测试方法
     */
    public static void main(String[] args) {
        Solution solution = new Solution();
        
        // 测试示例1
        System.out.println("示例1 - numRows = 5:");
        List<List<Integer>> result1 = solution.generate(5);
        printTriangle(result1);
        
        System.out.println("\n示例2 - numRows = 1:");
        List<List<Integer>> result2 = solution.generate(1);
        printTriangle(result2);
    }
    
    /**
     * 打印杨辉三角
     */
    private static void printTriangle(List<List<Integer>> triangle) {
        for (List<Integer> row : triangle) {
            System.out.println(row);
        }
    }
}

/**
 * 另一种实现方式 - 使用数组
 */
class SolutionArray {
    public List<List<Integer>> generate(int numRows) {
        List<List<Integer>> result = new ArrayList<>();
        
        if (numRows <= 0) {
            return result;
        }
        
        // 第一行
        result.add(Arrays.asList(1));
        
        for (int i = 1; i < numRows; i++) {
            List<Integer> row = new ArrayList<>();
            List<Integer> prevRow = result.get(i - 1);
            
            // 第一个元素
            row.add(1);
            
            // 中间元素
            for (int j = 1; j < i; j++) {
                row.add(prevRow.get(j - 1) + prevRow.get(j));
            }
            
            // 最后一个元素
            row.add(1);
            
            result.add(row);
        }
        
        return result;
    }
}
```
