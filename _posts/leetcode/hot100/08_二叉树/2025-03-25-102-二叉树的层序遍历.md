## 问题描述: 

给你二叉树的根节点 `root` ，返回其节点值的 **层序遍历** 。 （即逐层地，从左到右访问所有节点）。

**示例 1:**

![img](../assets/img/posts/p102_0.jpg)

```java
输入: root = [3,9,20,null,null,15,7]
输出: [[3],[9,20],[15,7]]
```

**示例 2:**

```java
输入: root = [1]
输出: [[1]]
```

**示例 3:**

```java
输入: root = []
输出: []
```
