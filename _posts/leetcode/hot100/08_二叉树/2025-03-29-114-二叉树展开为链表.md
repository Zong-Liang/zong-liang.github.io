## 问题描述:

给你二叉树的根结点 `root` ，请你将它展开为一个单链表:

- 展开后的单链表应该同样使用 `TreeNode` ，其中 `right` 子指针指向链表中下一个结点，而左子指针始终为 `null` 。
- 展开后的单链表应该与二叉树 [**先序遍历**](https://baike.baidu.com/item/先序遍历/6442839?fr=aladdin) 顺序相同。

**示例 1:**

![img](../assets/img/posts/leetcode/p114_0.jpg)

```java
输入: root = [1,2,5,3,4,null,6]
输出: [1,null,2,null,3,null,4,null,5,null,6]
```

**示例 2:**

```java
输入: root = []
输出: []
```

**示例 3:**

```java
输入: root = [0]
输出: [0]
```

**进阶:** 你可以使用原地算法（$O(1)$ 额外空间）展开这棵树吗？

## 解题思路：

这是一道关于将二叉树展开为链表的问题。需要将二叉树按照先序遍历的顺序重新构造成一个右偏的链表结构。

### 方法一：递归法（推荐）

核心思想是利用递归，对于每个节点：

- 先递归处理左右子树，使它们各自变成链表
- 将左子树插入到根节点和右子树之间
- 将左子树置空，右子树连接到原左子树的末尾

**算法步骤：**

1. 递归处理左右子树，使它们分别变成链表
2. 保存当前节点的右子树
3. 将左子树移动到右子树的位置
4. 将左子树置为空
5. 找到新右子树的末尾，连接原来的右子树

```java
// 定义二叉树节点
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode() {}
    TreeNode(int val) { this.val = val; }
    TreeNode(int val, TreeNode left, TreeNode right) {
        this.val = val;
        this.left = left;
        this.right = right;
    }
}

class Solution {
    public void flatten(TreeNode root) {
        if (root == null) {
            return;
        }

        // 递归处理左右子树
        flatten(root.left);
        flatten(root.right);

        // 保存右子树
        TreeNode rightSubtree = root.right;

        // 将左子树移动到右子树位置
        root.right = root.left;
        root.left = null;

        // 找到当前右子树的末尾
        TreeNode current = root;
        while (current.right != null) {
            current = current.right;
        }

        // 连接原来的右子树
        current.right = rightSubtree;
    }
}
```

**时间复杂度：** $O(n)$，其中 $n$ 是树中节点的数量，每个节点被访问一次。

**空间复杂度：** $O(h)$，其中 $h$ 是树的高度，递归调用栈的空间。

### 方法二：迭代法（栈实现）

使用栈来模拟先序遍历过程，同时构建链表。

**算法步骤：**

1. 使用栈进行先序遍历
2. 在遍历过程中，将当前节点的右子树和左子树依次入栈
3. 每次出栈一个节点，将其作为前一个节点的右子节点
4. 将所有节点的左子节点置为空

```java
import java.util.*;

class Solution {
    public void flatten(TreeNode root) {
        if (root == null) {
            return;
        }

        Stack<TreeNode> stack = new Stack<>();
        stack.push(root);
        TreeNode prev = null;

        while (!stack.isEmpty()) {
            TreeNode current = stack.pop();

            // 先将右子节点入栈，再将左子节点入栈（保证左子节点先被处理）
            if (current.right != null) {
                stack.push(current.right);
            }
            if (current.left != null) {
                stack.push(current.left);
            }

            // 构建链表结构
            if (prev != null) {
                prev.right = current;
                prev.left = null;
            }
            prev = current;
        }
    }
}
```

**时间复杂度：** $O(n)$，每个节点被访问一次。

**空间复杂度：** $O(h)$，栈的最大深度等于树的高度。

### 方法三：Morris 遍历法（O(1)空间）

使用 Morris 遍历的思想，通过修改树的结构来实现 O(1)空间复杂度。

**算法步骤：**

1. 对于当前节点，如果有左子树
2. 找到左子树的最右节点
3. 将当前节点的右子树连接到左子树最右节点的右子节点上
4. 将左子树移动到右子树位置，左子树置空
5. 继续处理下一个节点

```java
class Solution {
    public void flatten(TreeNode root) {
        TreeNode current = root;

        while (current != null) {
            if (current.left != null) {
                // 找到左子树的最右节点
                TreeNode rightmost = current.left;
                while (rightmost.right != null) {
                    rightmost = rightmost.right;
                }

                // 将当前节点的右子树连接到左子树的最右节点
                rightmost.right = current.right;

                // 将左子树移动到右子树位置
                current.right = current.left;
                current.left = null;
            }

            // 移动到下一个节点
            current = current.right;
        }
    }
}
```

**时间复杂度：** $O(n)$，每个节点被访问常数次。

**空间复杂度：** $O(1)$，只使用常数额外空间。

### 方法四：先序遍历+重建

先进行先序遍历收集所有节点，然后重新构建链表。

```java
import java.util.*;

class Solution {
    public void flatten(TreeNode root) {
        if (root == null) {
            return;
        }

        List<TreeNode> nodes = new ArrayList<>();
        preorderTraversal(root, nodes);

        // 重新构建链表
        for (int i = 0; i < nodes.size() - 1; i++) {
            TreeNode current = nodes.get(i);
            TreeNode next = nodes.get(i + 1);
            current.left = null;
            current.right = next;
        }
    }

    private void preorderTraversal(TreeNode node, List<TreeNode> nodes) {
        if (node == null) {
            return;
        }

        nodes.add(node);
        preorderTraversal(node.left, nodes);
        preorderTraversal(node.right, nodes);
    }
}
```

**时间复杂度：** $O(n)$，遍历两次树。

**空间复杂度：** $O(n)$，存储所有节点。

**推荐使用方法三（Morris 遍历法）**，因为它实现了 O(1)空间复杂度，是最优解；如果对空间复杂度要求不高，方法一（递归法）代码最简洁易懂。对于面试场景，建议先实现方法一，然后可以进一步优化到方法三。
