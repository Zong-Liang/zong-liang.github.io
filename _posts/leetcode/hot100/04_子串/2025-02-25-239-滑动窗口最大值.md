## 问题描述:

给你一个整数数组 `nums`，有一个大小为 `k` 的滑动窗口从数组的最左侧移动到数组的最右侧。你只可以看到在滑动窗口内的 `k` 个数字。滑动窗口每次只向右移动一位。

返回 _滑动窗口中的最大值_ 。

**示例 1：**

```java
输入: nums = [1,3,-1,-3,5,3,6,7], k = 3
输出: [3,3,5,5,6,7]
解释:
滑动窗口的位置                最大值
---------------               -----
[1  3  -1] -3  5  3  6  7       3
 1 [3  -1  -3] 5  3  6  7       3
 1  3 [-1  -3  5] 3  6  7       5
 1  3  -1 [-3  5  3] 6  7       5
 1  3  -1  -3 [5  3  6] 7       6
 1  3  -1  -3  5 [3  6  7]      7
```

**示例 2：**

```java
输入: nums = [1], k = 1
输出: [1]
```

## 解题思路:

这道题是"滑动窗口最大值"问题。给定一个整数数组和一个大小为 `k` 的滑动窗口，要求返回滑动窗口中每个位置的最大值。这是一个经典的滑动窗口问题。

### 方法一：单调双端队列（推荐）

使用双端队列（Deque）维护一个单调递减的队列，队列头部始终保存当前窗口的最大值索引。

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        if (nums == null || nums.length == 0 || k <= 0) return new int[0];

        int n = nums.length;
        int[] result = new int[n - k + 1];
        Deque<Integer> deque = new ArrayDeque<>(); // 存储数组索引

        for (int i = 0; i < n; i++) {
            // 移除窗口外的元素
            while (!deque.isEmpty() && deque.peekFirst() < i - k + 1) {
                deque.pollFirst();
            }

            // 维护单调递减队列，移除所有小于当前元素的索引
            while (!deque.isEmpty() && nums[deque.peekLast()] < nums[i]) {
                deque.pollLast();
            }

            // 添加当前元素索引
            deque.offerLast(i);

            // 当窗口形成时，记录最大值
            if (i >= k - 1) {
                result[i - k + 1] = nums[deque.peekFirst()];
            }
        }

        return result;
    }
}
```

**时间复杂度**：$O(n)$，其中 $n$ 是数组长度。每个元素最多被添加和删除一次。

**空间复杂度**：$O(k)$，双端队列最多存储 $k$ 个元素。

### 方法二：暴力解法

对每个窗口位置，遍历窗口内的所有元素找到最大值。

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        if (nums == null || nums.length == 0 || k <= 0) return new int[0];

        int n = nums.length;
        int[] result = new int[n - k + 1];

        for (int i = 0; i <= n - k; i++) {
            int max = Integer.MIN_VALUE;
            // 遍历当前窗口找最大值
            for (int j = i; j < i + k; j++) {
                max = Math.max(max, nums[j]);
            }
            result[i] = max;
        }

        return result;
    }
}
```

**时间复杂度**：$O(nk)$，其中 $n$ 是数组长度，$k$ 是窗口大小。

**空间复杂度**：$O(1)$，只使用了常数额外空间（不计算结果数组）。

### 方法三：优先队列（堆）

使用最大堆来维护窗口中的最大值，需要配合索引来判断元素是否在当前窗口内。

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        if (nums == null || nums.length == 0 || k <= 0) return new int[0];

        int n = nums.length;
        int[] result = new int[n - k + 1];

        // 使用最大堆，存储 [值, 索引] 对
        PriorityQueue<int[]> maxHeap = new PriorityQueue<>((a, b) -> b[0] - a[0]);

        // 初始化第一个窗口
        for (int i = 0; i < k; i++) {
            maxHeap.offer(new int[]{nums[i], i});
        }
        result[0] = maxHeap.peek()[0];

        // 滑动窗口
        for (int i = k; i < n; i++) {
            maxHeap.offer(new int[]{nums[i], i});

            // 移除窗口外的元素
            while (!maxHeap.isEmpty() && maxHeap.peek()[1] <= i - k) {
                maxHeap.poll();
            }

            result[i - k + 1] = maxHeap.peek()[0];
        }

        return result;
    }
}
```

**时间复杂度**：$O(n \log k)$，其中 $n$ 是数组长度。堆的插入和删除操作时间复杂度为 $O(\log k)$。

**空间复杂度**：$O(k)$，堆的空间开销。

### 方法四：分块处理

将数组分成若干块，预处理每块的前缀最大值和后缀最大值。

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        if (nums == null || nums.length == 0 || k <= 0) return new int[0];

        int n = nums.length;
        int[] result = new int[n - k + 1];

        // 前缀最大值：从每个块的开始到当前位置的最大值
        int[] prefixMax = new int[n];
        // 后缀最大值：从当前位置到每个块的结束的最大值
        int[] suffixMax = new int[n];

        for (int i = 0; i < n; i++) {
            if (i % k == 0) {
                prefixMax[i] = nums[i];
            } else {
                prefixMax[i] = Math.max(prefixMax[i - 1], nums[i]);
            }
        }

        for (int i = n - 1; i >= 0; i--) {
            if (i == n - 1 || (i + 1) % k == 0) {
                suffixMax[i] = nums[i];
            } else {
                suffixMax[i] = Math.max(suffixMax[i + 1], nums[i]);
            }
        }

        for (int i = 0; i <= n - k; i++) {
            // 窗口的最大值是右端点的后缀最大值和左端点的前缀最大值的较大者
            result[i] = Math.max(suffixMax[i], prefixMax[i + k - 1]);
        }

        return result;
    }
}
```

**时间复杂度**：$O(n)$，其中 $n$ 是数组长度。需要三次遍历数组。

**空间复杂度**：$O(n)$，需要额外的前缀和后缀数组。

### 方法五：动态规划优化

基于方法四的思想，但是实现更加简洁。

```java
class Solution {
    public int[] maxSlidingWindow(int[] nums, int k) {
        if (nums == null || nums.length == 0 || k <= 0) return new int[0];

        int n = nums.length;
        int[] result = new int[n - k + 1];

        // left[i] 表示以 i 为右端点，长度不超过 k 的子数组的最大值
        int[] left = new int[n];
        // right[i] 表示以 i 为左端点，长度不超过 k 的子数组的最大值
        int[] right = new int[n];

        left[0] = nums[0];
        right[n - 1] = nums[n - 1];

        for (int i = 1; i < n; i++) {
            // 每 k 个元素重新开始计算
            if (i % k == 0) {
                left[i] = nums[i];
            } else {
                left[i] = Math.max(left[i - 1], nums[i]);
            }

            // 从右往左计算
            int j = n - i - 1;
            if ((j + 1) % k == 0) {
                right[j] = nums[j];
            } else {
                right[j] = Math.max(right[j + 1], nums[j]);
            }
        }

        for (int i = 0; i <= n - k; i++) {
            result[i] = Math.max(right[i], left[i + k - 1]);
        }

        return result;
    }
}
```

**时间复杂度**：$O(n)$，其中 $n$ 是数组长度。

**空间复杂度**：$O(n)$，需要额外的 left 和 right 数组。

**推荐使用方法一（单调双端队列）**，它是最优解法，时间复杂度为 $O(n)$，空间复杂度为 $O(k)$，代码清晰易懂。

**核心思想**：

1. 使用双端队列维护窗口内元素的索引
2. 保持队列单调递减，队首始终是当前窗口的最大值
3. 滑动窗口时，及时移除过期元素和较小元素
4. 每次移动窗口时，队首元素就是当前窗口的最大值
