**示例 1:**

![img](../../../../assets/img/posts/p21_0.jpg)

```java
输入: l1 = [1,2,4], l2 = [1,3,4]
输出: [1,1,2,3,4,4]
```

**示例 2:**

```java
输入: l1 = [], l2 = []
输出: []
```

**示例 3:**

```java

输出: [0]
```
