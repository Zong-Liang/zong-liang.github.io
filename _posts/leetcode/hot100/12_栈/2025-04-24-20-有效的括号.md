## 问题描述: 

给定一个只包括 `'('`，`')'`，`'{'`，`'}'`，`'['`，`']'` 的字符串 `s` ，判断字符串是否有效。

有效字符串需满足: 

1. 左括号必须用相同类型的右括号闭合。
2. 左括号必须以正确的顺序闭合。
3. 每个右括号都有一个对应的相同类型的左括号。

**示例 1:**

```java
输入: s = "()"
输出: true
```

**示例 2:**

```java
输入: s = "()[]{}"
输出: true
```

**示例 3:**

```java
输入: s = "()[]{}"
输出: true
```

**示例 4:**

```java
输入: s = "([])"
输出: true
```

**提示:** `s` 仅由括号 `'()[]{}'` 组成
