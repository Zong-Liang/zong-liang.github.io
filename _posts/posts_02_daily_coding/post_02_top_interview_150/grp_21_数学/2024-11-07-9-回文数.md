### 题目：9. 回文数

给你一个整数 `x`，如果 `x` 是一个回文整数，返回 `true`；否则，返回 `false`。

**回文数** 是指正序（从左向右）和倒序（从右向左）读都是一样的整数。

- 例如，121 是回文，而 123 不是。

---

**示例 1:**

- **输入:** `x = 121`
- **输出:** `true`

**示例 2:**

- **输入:** `x = -121`
- **输出:** `false`
- **解释:** 从左向右读, 为 -121。 从右向左读, 为 121-。因此它不是一个回文数。

**示例 3:**

- **输入:** `x = 10`
- **输出:** `false`
- **解释:** 从右向左读, 为 01。因此它不是一个回文数。

---

**提示:**

- `-2^31 <= x <= 2^31 - 1`

**进阶:** 你能不将整数转为字符串来解决这个问题吗？

---

### Java 8 解答 (赛码网格式)

以下是根据“进阶”要求，不将整数转换为字符串的数学解法。该方法通过反转数字的一半来进行比较，效率更高且避免了整数溢出的问题。

```java
import java.util.Scanner;

/**
 * 赛码网要求的标准类名是 Main
 */
public class Main {

    /**
     * 判断一个整数是否是回文数，不使用字符串转换。
     *
     * @param x 待检查的整数。
     * @return 如果 x 是回文数则返回 true，否则返回 false。
     */
    public static boolean isPalindrome(int x) {
        // 根据题目解释，负数不是回文数。
        // 同时，如果数字的最后一位是 0，为了使该数字为回文，
        // 则其第一位数字也应该是 0，只有 0 满足这个条件。
        if (x < 0 || (x % 10 == 0 && x != 0)) {
            return false;
        }

        int revertedNumber = 0;
        // 当原始数字小于或等于反转后的数字时，说明已经处理了一半的数字
        while (x > revertedNumber) {
            revertedNumber = revertedNumber * 10 + x % 10;
            x /= 10;
        }

        // 对于数字长度为奇数的情况，我们可以通过 revertedNumber/10 去掉中间的数字。
        // 例如，当输入为 12321 时，在 while 循环结束后，我们会得到 x = 12, revertedNumber = 123。
        // 此时，123/10 = 12，与 x 相等。
        return x == revertedNumber || x == revertedNumber / 10;
    }


    /**
     * 主函数，包含题目给出的测试用例。
     * @param args 命令行参数（未使用）。
     */
    public static void main(String[] args) {
        System.out.println("--- 示例 1 ---");
        int x1 = 121;
        System.out.println("输入: x = " + x1);
        boolean result1 = isPalindrome(x1);
        System.out.println("输出: " + result1); // 预期输出: true
        System.out.println();

        System.out.println("--- 示例 2 ---");
        int x2 = -121;
        System.out.println("输入: x = " + x2);
        boolean result2 = isPalindrome(x2);
        System.out.println("输出: " + result2); // 预期输出: false
        System.out.println();

        System.out.println("--- 示例 3 ---");
        int x3 = 10;
        System.out.println("输入: x = " + x3);
        boolean result3 = isPalindrome(x3);
        System.out.println("输出: " + result3); // 预期输出: false
    }
}
```
