### 题目：66. 加一

给定一个表示 **大整数** 的整数数组 `digits`，其中 `digits[i]` 是整数的第 `i` 位数字。这些数字按从左到右，从最高位到最低位排列。这个大整数不包含任何前导 0。

将大整数加 1，并返回结果的数字数组。

---

**示例 1:**

- **输入:** `digits = [1,2,3]`
- **输出:** `[1,2,4]`
- **解释:** 输入数组表示数字 123。加 1 后得到 123 + 1 = 124。因此，结果应该是 `[1,2,4]`。

**示例 2:**

- **输入:** `digits = [4,3,2,1]`
- **输出:** `[4,3,2,2]`
- **解释:** 输入数组表示数字 4321。加 1 后得到 4321 + 1 = 4322。因此，结果应该是 `[4,3,2,2]`。

**示例 3:**

- **输入:** `digits = [9]`
- **输出:** `[1,0]`
- **解释:** 输入数组表示数字 9。加 1 后得到 9 + 1 = 10。因此，结果应该是 `[1,0]`。

---

**提示:**

- `1 <= digits.length <= 100`
- `0 <= digits[i] <= 9`
- `digits` 不包含任何前导 0。

---

### Java 8 解答 (赛码网格式)

以下是使用 Java 8 实现的解决方案。该算法模拟了手动加法的过程，从数组的末尾开始处理进位。

```java
import java.util.Arrays;

/**
 * 赛码网要求的标准类名是 Main
 */
public class Main {

    /**
     * 将由整数数组表示的大整数加一。
     *
     * @param digits 代表大整数的数组。
     * @return 加一后的结果数组。
     */
    public static int[] plusOne(int[] digits) {
        // 从数组的最后一位开始遍历
        for (int i = digits.length - 1; i >= 0; i--) {
            // 当前位加 1
            digits[i]++;
            // 对 10 取模，处理进位
            digits[i] = digits[i] % 10;
            // 如果当前位不为 0，说明没有产生进位，直接返回结果
            if (digits[i] != 0) {
                return digits;
            }
        }

        // 如果循环结束，说明所有位都是 9（例如 [9,9,9]）
        // 这种情况下，需要一个新的、长度加一的数组
        digits = new int[digits.length + 1];
        // 将新数组的第一位置为 1，其余位默认为 0
        digits[0] = 1;
        return digits;
    }

    /**
     * 主函数，包含题目给出的测试用例。
     * @param args 命令行参数（未使用）。
     */
    public static void main(String[] args) {
        System.out.println("--- 示例 1 ---");
        int[] digits1 = {1, 2, 3};
        System.out.println("输入: digits = " + Arrays.toString(digits1));
        int[] result1 = plusOne(digits1);
        System.out.println("输出: " + Arrays.toString(result1)); // 预期输出: [1, 2, 4]
        System.out.println();

        System.out.println("--- 示例 2 ---");
        int[] digits2 = {4, 3, 2, 1};
        System.out.println("输入: digits = " + Arrays.toString(digits2));
        int[] result2 = plusOne(digits2);
        System.out.println("输出: " + Arrays.toString(result2)); // 预期输出: [4, 3, 2, 2]
        System.out.println();

        System.out.println("--- 示例 3 ---");
        int[] digits3 = {9};
        System.out.println("输入: digits = " + Arrays.toString(digits3));
        int[] result3 = plusOne(digits3);
        System.out.println("输出: " + Arrays.toString(result3)); // 预期输出: [1, 0]
        System.out.println();

        System.out.println("--- 边界测试用例 ---");
        int[] digits4 = {9, 9, 9};
        System.out.println("输入: digits = " + Arrays.toString(digits4));
        int[] result4 = plusOne(digits4);
        System.out.println("输出: " + Arrays.toString(result4)); // 预期输出: [1, 0, 0, 0]
    }
}
```
