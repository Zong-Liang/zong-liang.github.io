## 题目：210. 课程表 II

现在你总共有 `numCourses` 门课需要选，记为 `0` 到 `numCourses - 1`。给你一个数组 `prerequisites`，其中 `prerequisites[i] = [ai, bi]`，表示在选修课程 `ai` 前 **必须** 先选修 `bi`。

- 例如，想要学习课程 `0`，你需要先完成课程 `1`，我们用一个匹配来表示：`[0,1]`。

返回你为了学完所有课程所安排的学习顺序。可能会有多个正确的顺序，你只要返回 **任意一种** 就可以了。如果不可能完成所有课程，返回一个 **空数组**。

**示例 1:**

- **输入:** `numCourses = 2`, `prerequisites = [[1,0]]`
- **输出:** `[0,1]`
- **解释:** 总共有 2 门课程。要学习课程 1，你需要先完成课程 0。因此，正确的课程顺序为 `[0,1]`。

**示例 2:**

- **输入:** `numCourses = 4`, `prerequisites = [[1,0],[2,0],[3,1],[3,2]]`
- **输出:** `[0,2,1,3]`
- **解释:** 总共有 4 门课程。要学习课程 3，你应该先完成课程 1 和课程 2。并且课程 1 和课程 2 都应该排在课程 0 之后。
  因此，一个正确的课程顺序是 `[0,1,2,3]`。另一个正确的排序是 `[0,2,1,3]`。

**示例 3:**

- **输入:** `numCourses = 1`, `prerequisites = []`
- **输出:** `[0]`

**提示:**

- `1 <= numCourses <= 2000`
- `0 <= prerequisites.length <= numCourses * (numCourses - 1)`
- `prerequisites[i].length == 2`
- `0 <= ai, bi < numCourses`
- `ai != bi`
- 所有 `[ai, bi]` **互不相同**

## Java 代码

**算法思路：(拓扑排序 / 广度优先搜索 BFS)**

这个问题本质上是要求一个有向无环图（DAG）的 **拓扑排序**。课程是图中的节点，先修关系是图中的有向边。如果图中存在环，则说明课程之间存在循环依赖，无法完成所有课程。

我们可以使用基于 **广度优先搜索 (BFS)** 的 Kahn 算法来解决：

1.  **构建图和入度数组:**
    - 创建一个邻接表 `adj` 来表示课程之间的依赖关系（`bi -> ai`）。
    - 创建一个 `inDegree` 数组，`inDegree[i]` 表示课程 `i` 有多少门先修课程。
2.  **初始化队列:**
    - 找到所有入度为 0 的课程（即没有先修课程的课），将它们加入一个队列 `queue`。这些是我们可以最先开始学习的课程。
3.  **BFS 过程:**
    - 当队列不为空时，循环执行：
      a. 从队列中取出一门课程 `u`，并将其加入结果序列。
      b. 遍历所有依赖于 `u` 的后续课程 `v`（即 `u` 指向的邻居）。
      c. 将 `v` 的入度减 1，表示 `u` 这门先修课已经完成。
      d. 如果 `v` 的入度变为 0，说明它的所有先修课都已完成，可以将 `v` 加入队列。
4.  **检查结果:**
    - 如果最终结果序列中的课程数量等于 `numCourses`，说明所有课程都已成功排序，返回该序列。
    - 否则，说明图中存在环，无法完成所有课程，返回一个空数组。

```java
import java.util.*;


public class Main {

    public static int[] findOrder(int numCourses, int[][] prerequisites) {
        // 1. 构建邻接表和入度数组
        List<List<Integer>> adj = new ArrayList<>(numCourses);
        for (int i = 0; i < numCourses; i++) {
            adj.add(new ArrayList<>());
        }
        int[] inDegree = new int[numCourses];

        for (int[] p : prerequisites) {
            int prev = p[1];
            int curr = p[0];
            adj.get(prev).add(curr);
            inDegree[curr]++;
        }

        // 2. 初始化队列，加入所有入度为 0 的节点
        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < numCourses; i++) {
            if (inDegree[i] == 0) {
                queue.offer(i);
            }
        }

        // 3. BFS 过程
        int[] result = new int[numCourses];
        int count = 0; // 记录已排序的课程数量

        while (!queue.isEmpty()) {
            int course = queue.poll();
            result[count++] = course;

            // 遍历当前课程的所有后续课程
            for (int neighbor : adj.get(course)) {
                // 将后续课程的入度减 1
                inDegree[neighbor]--;
                // 如果入度变为 0，则加入队列
                if (inDegree[neighbor] == 0) {
                    queue.offer(neighbor);
                }
            }
        }

        // 4. 检查结果
        if (count == numCourses) {
            return result;
        } else {
            // 如果 count < numCourses，说明存在环，无法完成所有课程
            return new int[0];
        }
    }


    public static void main(String[] args) {
        System.out.println("--- 示例 1 ---");
        int numCourses1 = 2;
        int[][] prerequisites1 = {{1, 0}};
        System.out.println("输入: numCourses = " + numCourses1 + ", prerequisites = " + Arrays.deepToString(prerequisites1));
        int[] result1 = findOrder(numCourses1, prerequisites1);
        System.out.println("输出: " + Arrays.toString(result1)); // 预期输出: [0, 1]
        System.out.println();

        System.out.println("--- 示例 2 ---");
        int numCourses2 = 4;
        int[][] prerequisites2 = {{1, 0}, {2, 0}, {3, 1}, {3, 2}};
        System.out.println("输入: numCourses = " + numCourses2 + ", prerequisites = " + Arrays.deepToString(prerequisites2));
        int[] result2 = findOrder(numCourses2, prerequisites2);
        System.out.println("输出: " + Arrays.toString(result2)); // 预期输出: [0, 1, 2, 3] 或 [0, 2, 1, 3]
        System.out.println();

        System.out.println("--- 示例 3 ---");
        int numCourses3 = 1;
        int[][] prerequisites3 = {};
        System.out.println("输入: numCourses = " + numCourses3 + ", prerequisites = " + Arrays.deepToString(prerequisites3));
        int[] result3 = findOrder(numCourses3, prerequisites3);
        System.out.println("输出: " + Arrays.toString(result3)); // 预期输出: [0]
    }
}
```
