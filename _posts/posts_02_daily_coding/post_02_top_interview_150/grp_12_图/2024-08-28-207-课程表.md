### 题目：207. 课程表

你这个学期必须选修 `numCourses` 门课程，记为 `0` 到 `numCourses - 1`。

在选修某些课程之前需要一些先修课程。 先修课程按数组 `prerequisites` 给出，其中 `prerequisites[i] = [ai, bi]`，表示如果要学习课程 `ai` 则 **必须** 先学习课程 `bi`。

- 例如，先修课程对 `[0, 1]` 表示：想要学习课程 `0`，你需要先完成课程 `1`。

请你判断是否可能完成所有课程的学习？如果可以，返回 `true`；否则，返回 `false`。

---

**示例 1:**

- **输入:** `numCourses = 2`, `prerequisites = [[1,0]]`
- **输出:** `true`
- **解释:** 总共有 2 门课程。学习课程 1 之前，你需要完成课程 0。这是可能的。

**示例 2:**

- **输入:** `numCourses = 2`, `prerequisites = [[1,0],[0,1]]`
- **输出:** `false`
- **解释:** 总共有 2 门课程。学习课程 1 之前，你需要先完成课程 0；并且学习课程 0 之前，你还应先完成课程 1。这是不可能的。

---

**提示:**

- `1 <= numCourses <= 2000`
- `0 <= prerequisites.length <= 5000`
- `prerequisites[i].length == 2`
- `0 <= ai, bi < numCourses`
- `prerequisites[i]` 中的所有课程对 **互不相同**

---

### Java 8 解答 (赛码网格式)

**算法思路：(拓扑排序 / 判断环)**
这个问题本质上是在一个有向图中判断是否存在 **环**。课程是图中的节点，先修关系是有向边。如果图中存在环（例如 A -> B -> C -> A），则意味着课程之间存在循环依赖，无法完成学习。

我们可以使用基于 **广度优先搜索 (BFS)** 的拓扑排序算法 (Kahn's algorithm) 来检测环：

1.  **构建图和入度数组:**
    - 创建一个邻接表 `adj` 来表示课程间的依赖关系 (`bi -> ai`)。
    - 创建一个 `inDegree` 数组，`inDegree[i]` 记录课程 `i` 的先修课程数量。
2.  **初始化队列:**
    - 找到所有入度为 0 的课程（没有先修课的课程），将它们加入队列 `queue`。
3.  **BFS 过程:**
    - 当队列不为空时，从队列中取出一门课程 `u`。这代表我们“完成”了这门课。
    - 每完成一门课，就将一个计数器 `count` 加一。
    - 遍历所有依赖于 `u` 的后续课程 `v`（即 `u` 的邻居）。
    - 将 `v` 的入度减 1。如果 `v` 的入度变为 0，说明它的所有先修课都已完成，可以将 `v` 加入队列。
4.  **判断结果:**
    - 在 BFS 结束后，如果 `count` 等于总课程数 `numCourses`，说明所有课程都能够被“完成”，图中没有环，返回 `true`。
    - 如果 `count` 小于 `numCourses`，说明有些课程因为循环依赖而无法入队，图中存在环，返回 `false`。

```java
import java.util.*;

/**
 * 赛码网要求的标准类名是 Main
 */
public class Main {

    public static boolean canFinish(int numCourses, int[][] prerequisites) {
        // 1. 构建邻接表和入度数组
        List<List<Integer>> adj = new ArrayList<>(numCourses);
        for (int i = 0; i < numCourses; i++) {
            adj.add(new ArrayList<>());
        }
        int[] inDegree = new int[numCourses];

        for (int[] p : prerequisites) {
            int prev = p[1];
            int curr = p[0];
            adj.get(prev).add(curr);
            inDegree[curr]++;
        }

        // 2. 初始化队列，加入所有入度为 0 的节点
        Queue<Integer> queue = new LinkedList<>();
        for (int i = 0; i < numCourses; i++) {
            if (inDegree[i] == 0) {
                queue.offer(i);
            }
        }

        // 3. BFS 过程
        int count = 0; // 记录已完成的课程数量

        while (!queue.isEmpty()) {
            int course = queue.poll();
            count++;

            // 遍历当前课程的所有后续课程
            for (int neighbor : adj.get(course)) {
                inDegree[neighbor]--;
                if (inDegree[neighbor] == 0) {
                    queue.offer(neighbor);
                }
            }
        }

        // 4. 判断结果
        return count == numCourses;
    }

    /**
     * 主函数，包含题目给出的测试用例。
     * @param args 命令行参数（未使用）。
     */
    public static void main(String[] args) {
        System.out.println("--- 示例 1 ---");
        int numCourses1 = 2;
        int[][] prerequisites1 = {{1, 0}};
        System.out.println("输入: numCourses = " + numCourses1 + ", prerequisites = " + Arrays.deepToString(prerequisites1));
        boolean result1 = canFinish(numCourses1, prerequisites1);
        System.out.println("输出: " + result1); // 预期输出: true
        System.out.println();

        System.out.println("--- 示例 2 ---");
        int numCourses2 = 2;
        int[][] prerequisites2 = {{1, 0}, {0, 1}};
        System.out.println("输入: numCourses = " + numCourses2 + ", prerequisites = " + Arrays.deepToString(prerequisites2));
        boolean result2 = canFinish(numCourses2, prerequisites2);
        System.out.println("输出: " + result2); // 预期输出: false
        System.out.println();
    }
}
```
