## 题目

**描述：**

假设你正在爬楼梯。需要 `n` 阶你才能到达楼顶。

每次你可以爬 `1` 或 `2` 个台阶。你有多少种不同的方法可以爬到楼顶呢？

**示例 1:**

- **输入:** `n = 2`
- **输出:** 2
- **解释:** 有两种方法可以爬到楼顶。

  1.  1 阶 + 1 阶
  2.  2 阶

**示例 2:**

- **输入:** `n = 3`
- **输出:** 3
- **解释:** 有三种方法可以爬到楼顶。

  1.  1 阶 + 1 阶 + 1 阶
  2.  1 阶 + 2 阶
  3.  2 阶 + 1 阶

**提示:**

- `1 <= n <= 45`

## 题解

### 解法：动态规划 (Dynamic Programming)

**思路：**

这是一个经典的动态规划问题。我们可以采用**倒推**的思维方式：

想要爬到第 $n$ 阶楼顶，最后一步只有两种可能：

1.  从第 $n-1$ 阶爬 1 个台阶到达。
2.  从第 $n-2$ 阶爬 2 个台阶到达。

因此，爬到第 $n$ 阶的方法总数就等于爬到第 $n-1$ 阶的方法数加上爬到第 $n-2$ 阶的方法数。

即状态转移方程为：$f(n) = f(n-1) + f(n-2)$。

这实际上就是一个**斐波那契数列**。

- **边界条件**：

  - $f(1) = 1$ （只有 1 种方法：1 阶）
  - $f(2) = 2$ （有 2 种方法：1+1，2）

为了节省空间，我们不需要开辟一个长度为 $n$ 的数组来存储所有状态，只需要维护两个变量（分别记录 $f(i-2)$ 和 $f(i-1)$）即可，这种方法称为**滚动数组优化**。

**复杂度分析：**

- **时间复杂度：** $O(n)$，需要循环遍历一次从 3 到 $n$。
- **空间复杂度：** $O(1)$，只利用了常数个变量。

**代码片段：**

```java
public int climbStairs(int n) {
    if (n <= 2) {
        return n;
    }
    int p = 1, q = 2;
    for (int i = 3; i <= n; i++) {
        int sum = p + q;
        p = q;
        q = sum;
    }
    return q;
}
```

### 完整代码

```java
public class Main {

    /**
     * 70. 爬楼梯 - 动态规划 (滚动数组优化)
     */
    public static int climbStairs(int n) {
        // 边界情况处理
        if (n <= 1) return 1; // 题目说 n >= 1，若 n=1 只有1种
        if (n == 2) return 2; // n=2 有2种

        // 初始化 f(1) 和 f(2)
        int first = 1;  // 代表 f(i-2)
        int second = 2; // 代表 f(i-1)

        // 从第 3 阶开始计算
        for (int i = 3; i <= n; i++) {
            int third = first + second; // f(i) = f(i-1) + f(i-2)

            // 滚动更新变量
            first = second;
            second = third;
        }

        // 循环结束后 second 存储的就是 f(n)
        return second;
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        int n1 = 2;
        System.out.println("示例 1 输入: n = " + n1);
        System.out.println("示例 1 输出: " + climbStairs(n1));
        // 解释: 1+1, 2
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        int n2 = 3;
        System.out.println("示例 2 输入: n = " + n2);
        System.out.println("示例 2 输出: " + climbStairs(n2));
        // 解释: 1+1+1, 1+2, 2+1
        System.out.println("-----------------------------");

        // --- 额外示例 ---
        int n3 = 5;
        System.out.println("额外示例 输入: n = " + n3);
        System.out.println("额外示例 输出: " + climbStairs(n3));
        // 斐波那契数列: 1, 2, 3, 5, 8... 预期输出 8
    }
}
```
