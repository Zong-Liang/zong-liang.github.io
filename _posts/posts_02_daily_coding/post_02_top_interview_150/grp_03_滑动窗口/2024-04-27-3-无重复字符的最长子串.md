### 题目：3. 无重复字符的最长子串

**描述**
给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长子串** 的长度。

**示例 1:**

- **输入:** `s = "abcabcbb"`
- **输出:** 3
- **解释:** 因为无重复字符的最长子串是 `"abc"`，所以其长度为 3。注意 `"bca"` 和 `"cab"` 也是正确答案。

**示例 2:**

- **输入:** `s = "bbbbb"`
- **输出:** 1
- **解释:** 因为无重复字符的最长子串是 `"b"`，所以其长度为 1。

**示例 3:**

- **输入:** `s = "pwwkew"`
- **输出:** 3
- **解释:** 因为无重复字符的最长子串是 `"wke"`，所以其长度为 3。请注意，你的答案必须是 **子串** 的长度，`"pwke"` 是一个子序列，不是子串。

**提示:**

- `0 <= s.length <= 5 * 10^4`
- `s` 由英文字母、数字、符号和空格组成

---

### 赛码网格式 Java 8 代码

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    /**
     * 找出不含有重复字符的最长子串的长度。
     * 采用滑动窗口算法。
     *
     * @param s 输入字符串
     * @return 最长子串的长度
     */
    public static int lengthOfLongestSubstring(String s) {
        if (s == null || s.length() == 0) {
            return 0;
        }

        // maxLength 用于记录最长子串的长度
        int maxLength = 0;
        // left 是滑动窗口的左边界
        int left = 0;

        // map 用于存储字符及其在字符串中最后一次出现的位置
        // Key: 字符, Value: 索引
        Map<Character, Integer> map = new HashMap<>();

        // right 是滑动窗口的右边界
        for (int right = 0; right < s.length(); right++) {
            char currentChar = s.charAt(right);

            // 检查当前字符是否在 map 中存在
            if (map.containsKey(currentChar)) {
                // 如果存在，我们需要移动左边界 left
                // new left 应该是重复字符上次出现位置的下一个位置
                // 同时要确保 left 指针不会向左回退
                left = Math.max(left, map.get(currentChar) + 1);
            }

            // 更新当前字符的最新位置
            map.put(currentChar, right);

            // 计算当前窗口的长度，并更新最大长度
            maxLength = Math.max(maxLength, right - left + 1);
        }

        return maxLength;
    }

    public static void main(String[] args) {
        // 示例 1
        String s1 = "abcabcbb";
        System.out.println("输入: \"" + s1 + "\"");
        System.out.println("输出: " + lengthOfLongestSubstring(s1)); // 预期: 3
        System.out.println("---");

        // 示例 2
        String s2 = "bbbbb";
        System.out.println("输入: \"" + s2 + "\"");
        System.out.println("输出: " + lengthOfLongestSubstring(s2)); // 预期: 1
        System.out.println("---");

        // 示例 3
        String s3 = "pwwkew";
        System.out.println("输入: \"" + s3 + "\"");
        System.out.println("输出: " + lengthOfLongestSubstring(s3)); // 预期: 3
        System.out.println("---");

        // 额外测试用例
        String s4 = "tmmzuxt";
        System.out.println("输入: \"" + s4 + "\"");
        System.out.println("输出: " + lengthOfLongestSubstring(s4)); // 预期: 5
        System.out.println("---");
    }
}
```
