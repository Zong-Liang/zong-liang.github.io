## 题目：637. 二叉树的层平均值

给定一个非空二叉树的根节点 `root`，以数组的形式返回每一层节点的平均值。与实际答案相差 `10^-5` 以内的答案可以被接受。

**示例 1:**

![示例1图片](https://assets.leetcode.com/uploads/2021/03/09/avg1-tree.jpg)

- **输入:** `root = [3,9,20,null,null,15,7]`
- **输出:** `[3.00000,14.50000,11.00000]`
- **解释:**
  第 0 层的平均值为 3，第 1 层的平均值为 14.5，第 2 层的平均值为 11。
  因此返回 `[3, 14.5, 11]`。

**示例 2:**

![示例2图片](https://assets.leetcode.com/uploads/2021/03/09/avg2-tree.jpg)

- **输入:** `root = [3,9,20,15,7]`
- **输出:** `[3.00000,14.50000,11.00000]`

**提示:**

- 树中节点数量在 `[1, 10^4]` 范围内
- `-2^31 <= Node.val <= 2^31 - 1`

## Java 代码

**算法思路：(广度优先搜索 BFS)**

这个问题是典型的层序遍历应用。我们可以使用 **广度优先搜索 (BFS)** 和一个队列来逐层处理树的节点。

1.  **初始化:**
    - 创建一个队列 `queue` 并将根节点 `root` 加入。
    - 创建一个列表 `result` 用于存储每一层的平均值。
2.  **BFS 循环:**
    - 当队列不为空时，进行逐层遍历。
    - 在每一层的开始，记录下当前队列的大小 `levelSize`，这代表了当前层的节点数。
    - 初始化当前层的总和 `levelSum = 0.0` (使用 `double` 类型以避免整数溢出和保证精度)。
    - 循环 `levelSize` 次，处理当前层的所有节点：
      a. 从队列中取出一个节点 `node`。
      b. 将 `node.val` 累加到 `levelSum`。
      c. 将该节点的非空子节点（左、右）加入队列，为下一层做准备。
    - 当前层所有节点处理完毕后，计算平均值 `levelSum / levelSize`，并将其加入 `result` 列表。
3.  循环结束后，`result` 列表就包含了所有层的平均值。

```java
import java.util.ArrayList;
import java.util.LinkedList;
import java.util.List;
import java.util.Queue;

/**
 * 二叉树节点的定义
 */
class TreeNode {
    int val;
    TreeNode left;
    TreeNode right;
    TreeNode(int val) { this.val = val; }
}


public class Main {

    /**
     * 计算二叉树每一层节点的平均值。
     */
    public static List<Double> averageOfLevels(TreeNode root) {
        List<Double> result = new ArrayList<>();
        if (root == null) {
            return result;
        }

        Queue<TreeNode> queue = new LinkedList<>();
        queue.offer(root);

        while (!queue.isEmpty()) {
            int levelSize = queue.size();
            double levelSum = 0.0;

            for (int i = 0; i < levelSize; i++) {
                TreeNode node = queue.poll();
                levelSum += node.val;

                if (node.left != null) {
                    queue.offer(node.left);
                }
                if (node.right != null) {
                    queue.offer(node.right);
                }
            }

            result.add(levelSum / levelSize);
        }

        return result;
    }

    /**
     * 主函数，包含题目给出的测试用例。
     * @param args 命令行参数（未使用）。
     */
    public static void main(String[] args) {
        System.out.println("--- 示例 1 ---");
        TreeNode root1 = new TreeNode(3);
        root1.left = new TreeNode(9);
        root1.right = new TreeNode(20);
        root1.right.left = new TreeNode(15);
        root1.right.right = new TreeNode(7);
        System.out.println("输入: root = [3,9,20,null,null,15,7]");
        List<Double> result1 = averageOfLevels(root1);
        System.out.println("输出: " + result1); // 预期输出: [3.0, 14.5, 11.0]
        System.out.println();

        System.out.println("--- 示例 2 ---");
        TreeNode root2 = new TreeNode(3);
        root2.left = new TreeNode(9);
        root2.right = new TreeNode(20);
        root2.left.left = new TreeNode(15);
        root2.left.right = new TreeNode(7);
        System.out.println("输入: root = [3,9,20,15,7]");
        List<Double> result2 = averageOfLevels(root2);
        System.out.println("输出: " + result2); // 预期输出: [3.0, 14.5, 11.0]
    }
}
```
