## 题目：13. 罗马数字转整数

**描述**

罗马数字包含以下七种字符: `I`, `V`, `X`, `L`, `C`, `D` 和 `M`。

| 字符 | 数值 |
| :--- | :--- |
| I    | 1    |
| V    | 5    |
| X    | 10   |
| L    | 50   |
| C    | 100  |
| D    | 500  |
| M    | 1000 |

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 `IIII`，而是 `IV`。这个特殊的规则只适用于以下六种情况：

- `I` 可以放在 `V` (5) 和 `X` (10) 的左边，来表示 4 和 9。
- `X` 可以放在 `L` (50) 和 `C` (100) 的左边，来表示 40 和 90。
- `C` 可以放在 `D` (500) 和 `M` (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。

**示例 1:**

- **输入:** `s = "III"`
- **输出:** 3

**示例 2:**

- **输入:** `s = "IV"`
- **输出:** 4

**示例 3:**

- **输入:** `s = "IX"`
- **输出:** 9

**示例 4:**

- **输入:** `s = "LVIII"`
- **输出:** 58

**示例 5:**

- **输入:** `s = "MCMXCIV"`
- **输出:** 1994

**提示:**

- `1 <= s.length <= 15`
- `s` 仅含字符 `('I', 'V', 'X', 'L', 'C', 'D', 'M')`
- 题目数据保证 `s` 是一个有效的罗马数字，且表示整数在范围 `[1, 3999]` 内
- 题目所给测试用例皆符合罗马数字书写规则，不会出现跨位等情况。

## 题解

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    /**
     * 将罗马数字字符串转换为整数。
     *
     * @param s 罗马数字字符串
     * @return 对应的整数
     */
    public static int romanToInt(String s) {
        // 创建一个 Map 存储罗马字符与数值的对应关系
        Map<Character, Integer> map = new HashMap<>();
        map.put('I', 1);
        map.put('V', 5);
        map.put('X', 10);
        map.put('L', 50);
        map.put('C', 100);
        map.put('D', 500);
        map.put('M', 1000);

        int result = 0;

        // 遍历字符串，检查每个字符与它后面字符的关系
        for (int i = 0; i < s.length(); i++) {
            int currentValue = map.get(s.charAt(i));

            // 如果不是最后一个字符，并且当前字符代表的值小于下一个字符代表的值
            // (例如 "IV" 中的 'I' 和 'V')，则减去当前值
            if (i < s.length() - 1 && currentValue < map.get(s.charAt(i + 1))) {
                result -= currentValue;
            } else {
                // 否则，加上当前值
                result += currentValue;
            }
        }

        return result;
    }

    public static void main(String[] args) {
        // 示例 1
        String s1 = "III";
        System.out.println("输入: \"" + s1 + "\"");
        System.out.println("输出: " + romanToInt(s1)); // 预期: 3
        System.out.println("---");

        // 示例 2
        String s2 = "IV";
        System.out.println("输入: \"" + s2 + "\"");
        System.out.println("输出: " + romanToInt(s2)); // 预期: 4
        System.out.println("---");

        // 示例 3
        String s3 = "IX";
        System.out.println("输入: \"" + s3 + "\"");
        System.out.println("输出: " + romanToInt(s3)); // 预期: 9
        System.out.println("---");

        // 示例 4
        String s4 = "LVIII";
        System.out.println("输入: \"" + s4 + "\"");
        System.out.println("输出: " + romanToInt(s4)); // 预期: 58
        System.out.println("---");

        // 示例 5
        String s5 = "MCMXCIV";
        System.out.println("输入: \"" + s5 + "\"");
        System.out.println("输出: " + romanToInt(s5)); // 预期: 1994
        System.out.println("---");
    }
}
```
