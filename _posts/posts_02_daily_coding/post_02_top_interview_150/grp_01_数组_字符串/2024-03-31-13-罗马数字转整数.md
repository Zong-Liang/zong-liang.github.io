## 题目

**描述：**

罗马数字包含以下七种字符: `I`, `V`, `X`, `L`, `C`, `D` 和 `M`。

| 字符 | 数值 |
| :--- | :--- |
| I    | 1    |
| V    | 5    |
| X    | 10   |
| L    | 50   |
| C    | 100  |
| D    | 500  |
| M    | 1000 |

通常情况下，罗马数字中小的数字在大的数字的右边。但也存在特例，例如 4 不写做 `IIII`，而是 `IV`。这个特殊的规则只适用于以下六种情况：

- `I` 可以放在 `V` (5) 和 `X` (10) 的左边，来表示 4 和 9。
- `X` 可以放在 `L` (50) 和 `C` (100) 的左边，来表示 40 和 90。
- `C` 可以放在 `D` (500) 和 `M` (1000) 的左边，来表示 400 和 900。

给定一个罗马数字，将其转换成整数。

**示例 1:**

- **输入:** `s = "III"`
- **输出:** 3

**示例 2:**

- **输入:** `s = "IV"`
- **输出:** 4

**示例 3:**

- **输入:** `s = "IX"`
- **输出:** 9

**示例 4:**

- **输入:** `s = "LVIII"`
- **输出:** 58

**示例 5:**

- **输入:** `s = "MCMXCIV"`
- **输出:** 1994

**提示:**

- `1 <= s.length <= 15`
- `s` 仅含字符 `('I', 'V', 'X', 'L', 'C', 'D', 'M')`
- 题目数据保证 `s` 是一个有效的罗马数字，且表示整数在范围 `[1, 3999]` 内
- 题目所给测试用例皆符合罗马数字书写规则，不会出现跨位等情况。

## 题解

### 解法：遍历模拟法

**思路：**

罗马数字的转换规则主要看 **当前字符** 和 **下一个字符** 的大小关系：

1.  **一般情况**：罗马数字通常是从大到小排列的。如果当前字符代表的数值 $\ge$ 下一个字符代表的数值，则将当前数值 **加** 到总和中（例如 `VI`，`5 >= 1`，结果是 `5 + 1`）。
2.  **特殊情况**（减法规则）：如果当前字符代表的数值 $<$ 下一个字符代表的数值，说明这是类似 `IV`（4）或 `IX`（9）的情况。此时，当前数值应该被 **减去**（例如 `IV`，`1 < 5`，结果是 `-1 + 5`）。

**具体步骤：**

1.  建立一个辅助函数（或 switch 语句），用于获取字符对应的数值。
2.  初始化结果变量 `sum = 0`。
3.  遍历字符串 `s`：

    - 获取当前字符的值 `value`。
    - 判断是否存在下一个字符，且 `value` 是否小于下一个字符的值。
    - 如果是，`sum -= value`。
    - 否则，`sum += value`。

4.  返回 `sum`。

**复杂度分析：**

- **时间复杂度：** $O(n)$，其中 $n$ 是字符串的长度。我们需要遍历一次字符串。
- **空间复杂度：** $O(1)$，只需要常数级别的额外空间（尽管使用了 switch 或哈希表，但字符集固定为 7 个）。

**代码片段：**

```java
public int romanToInt(String s) {
    int sum = 0;
    int n = s.length();

    for (int i = 0; i < n; i++) {
        int value = getValue(s.charAt(i));
        // 如果当前值小于下一个值，执行减法；否则执行加法
        if (i < n - 1 && value < getValue(s.charAt(i + 1))) {
            sum -= value;
        } else {
            sum += value;
        }
    }
    return sum;
}

private int getValue(char ch) {
    switch(ch) {
        case 'I': return 1;
        case 'V': return 5;
        case 'X': return 10;
        case 'L': return 50;
        case 'C': return 100;
        case 'D': return 500;
        case 'M': return 1000;
        default: return 0;
    }
}
```

### 完整代码

```java
public class Main {

    // 辅助函数：将罗马字符映射为整数
    // 使用 switch 语句比 HashMap 性能更好
    private static int getValue(char ch) {
        switch(ch) {
            case 'I': return 1;
            case 'V': return 5;
            case 'X': return 10;
            case 'L': return 50;
            case 'C': return 100;
            case 'D': return 500;
            case 'M': return 1000;
            default: return 0;
        }
    }

    // 核心算法实现
    public static int romanToInt(String s) {
        int sum = 0;
        int n = s.length();

        for (int i = 0; i < n; i++) {
            int value = getValue(s.charAt(i));

            // 核心逻辑：
            // 如果当前位置的数值 < 下一个位置的数值，说明是减法情况（如 IV, IX）
            // 否则是加法情况
            if (i < n - 1 && value < getValue(s.charAt(i + 1))) {
                sum -= value;
            } else {
                sum += value;
            }
        }
        return sum;
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        String s1 = "III";
        System.out.println("示例 1 输入: s = \"" + s1 + "\"");
        System.out.println("示例 1 输出: " + romanToInt(s1));
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        String s2 = "IV";
        System.out.println("示例 2 输入: s = \"" + s2 + "\"");
        System.out.println("示例 2 输出: " + romanToInt(s2));
        System.out.println("-----------------------------");

        // --- 示例 3 ---
        String s3 = "IX";
        System.out.println("示例 3 输入: s = \"" + s3 + "\"");
        System.out.println("示例 3 输出: " + romanToInt(s3));
        System.out.println("-----------------------------");

        // --- 示例 4 ---
        String s4 = "LVIII";
        System.out.println("示例 4 输入: s = \"" + s4 + "\"");
        System.out.println("示例 4 输出: " + romanToInt(s4));
        System.out.println("-----------------------------");

        // --- 示例 5 ---
        String s5 = "MCMXCIV";
        System.out.println("示例 5 输入: s = \"" + s5 + "\"");
        System.out.println("示例 5 输出: " + romanToInt(s5));
        // 解释: M=1000, CM=900, XC=90, IV=4 -> 1994
    }
}
```
