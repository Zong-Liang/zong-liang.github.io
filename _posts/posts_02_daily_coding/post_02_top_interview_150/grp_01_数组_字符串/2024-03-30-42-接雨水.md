### 题目：42. 接雨水

**描述**
给定 `n` 个非负整数表示每个宽度为 1 的柱子的高度图，计算按此排列的柱子，下雨之后能接多少雨水。

**示例 1:**

- **输入:** `height = [0,1,0,2,1,0,1,3,2,1,2,1]`
- **输出:** 6
- **解释:** 上面是由数组 `[0,1,0,2,1,0,1,3,2,1,2,1]` 表示的高度图，在这种情况下，可以接 6 个单位的雨水（蓝色部分表示雨水）。

**示例 2:**

- **输入:** `height = [4,2,0,3,2,5]`
- **输出:** 9

**提示:**

- `n == height.length`
- `1 <= n <= 2 * 10^4`
- `0 <= height[i] <= 10^5`

---

### 赛码网格式 Java 8 代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 计算柱子之间能接多少雨水。
     * 采用双指针法，时间复杂度 O(n)，空间复杂度 O(1)。
     *
     * @param height 代表柱子高度的数组
     * @return 可以接的雨水总量
     */
    public static int trap(int[] height) {
        if (height == null || height.length < 3) {
            return 0;
        }

        int totalWater = 0;
        int left = 0;
        int right = height.length - 1;

        // leftMax 记录从左到 right 指针遇到的最高柱子
        int leftMax = 0;
        // rightMax 记录从右到 left 指针遇到的最高柱子
        int rightMax = 0;

        while (left < right) {
            // 水位的高度由左右两边较短的柱子决定
            if (height[left] < height[right]) {
                // 如果左边的柱子比 leftMax 高，它不能存水，更新 leftMax
                if (height[left] >= leftMax) {
                    leftMax = height[left];
                } else {
                    // 否则，它可以存水，水量为 leftMax - 当前柱子高度
                    totalWater += leftMax - height[left];
                }
                // 移动左指针
                left++;
            } else {
                // 如果右边的柱子比 rightMax 高，它不能存水，更新 rightMax
                if (height[right] >= rightMax) {
                    rightMax = height[right];
                } else {
                    // 否则，它可以存水，水量为 rightMax - 当前柱子高度
                    totalWater += rightMax - height[right];
                }
                // 移动右指针
                right--;
            }
        }

        return totalWater;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] height1 = {0, 1, 0, 2, 1, 0, 1, 3, 2, 1, 2, 1};
        System.out.println("输入: " + Arrays.toString(height1));
        System.out.println("输出: " + trap(height1)); // 预期: 6
        System.out.println("---");

        // 示例 2
        int[] height2 = {4, 2, 0, 3, 2, 5};
        System.out.println("输入: " + Arrays.toString(height2));
        System.out.println("输出: " + trap(height2)); // 预期: 9
        System.out.println("---");
    }
}
```
