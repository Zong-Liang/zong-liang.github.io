## 题目

**描述：**

给你一个非负整数数组 `nums` ，你最初位于数组的 **第一个下标** 。数组中的每个元素代表你在该位置可以跳跃的最大长度。

判断你是否能够到达最后一个下标，如果可以，返回 `true` ；否则，返回 `false` 。

**示例 1:**

- **输入:** `nums = [2,3,1,1,4]`
- **输出:** `true`
- **解释:** 可以先跳 1 步，从下标 0 到达下标 1, 然后再从下标 1 跳 3 步到达最后一个下标。

**示例 2:**

- **输入:** `nums = [3,2,1,0,4]`
- **输出:** `false`
- **解释:** 无论怎样，总会到达下标为 3 的位置。但该下标的最大跳跃长度是 0 ，所以永远不可能到达最后一个下标。

**提示:**

- `1 <= nums.length <= 10^4`
- `0 <= nums[i] <= 10^5`

## 题解

### 解法：贪心算法

**思路：**

这是一个典型的贪心算法问题。我们不需要具体计算每一步跳到哪里，只需要关注**在当前位置能跳到的最远位置**（`rightmost`）。

1.  维护一个变量 `rightmost`，表示目前能够到达的最远下标，初始为 0。
2.  遍历数组，对于每一个位置 `i`：

    - **首先判断当前位置是否可达**：如果 `i > rightmost`，说明无论如何都无法跳到位置 `i`，既然连 `i` 都到不了，更不可能到达终点，直接返回 `false`。
    - **更新最远距离**：如果当前位置可达，尝试从这里起跳，计算能到达的最远位置 `i + nums[i]`，并更新 `rightmost`（取最大值）。
    - **判断终点**：如果在遍历过程中，`rightmost` 已经大于或等于最后一个下标（`nums.length - 1`），说明终点可达，直接返回 `true`。

**复杂度分析：**

- **时间复杂度：** $O(n)$，其中 $n$ 是数组的长度。我们只需要遍历数组一次。
- **空间复杂度：** $O(1)$，只需要常数级别的额外空间维护变量 `rightmost`。

**代码片段：**

```java
public boolean canJump(int[] nums) {
    int rightmost = 0;
    for (int i = 0; i < nums.length; i++) {
        // 如果当前位置已经超过了能到达的最远距离，说明无法到达这里，失败
        if (i > rightmost) {
            return false;
        }
        // 更新目前能到达的最远位置
        rightmost = Math.max(rightmost, i + nums[i]);

        // 如果最远位置已经覆盖了终点，直接返回 true
        if (rightmost >= nums.length - 1) {
            return true;
        }
    }
    return true;
}
```

### 完整代码

```java
import java.util.Arrays;

public class Main {

    // 核心算法实现
    public static boolean canJump(int[] nums) {
        // 记录能到达的最远下标
        int rightmost = 0;

        for (int i = 0; i < nums.length; i++) {
            // 如果当前索引 i 已经超过了最大可达范围，说明断档了，跳不过去
            if (i > rightmost) {
                return false;
            }

            // 贪心策略：维护能跳到的最远位置
            rightmost = Math.max(rightmost, i + nums[i]);

            // 剪枝：如果已经可以到达最后一个位置（或超过），提前结束
            if (rightmost >= nums.length - 1) {
                return true;
            }
        }

        return true;
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        int[] nums1 = new int[]{2, 3, 1, 1, 4};

        System.out.println("示例 1 输入:");
        System.out.println("nums = " + Arrays.toString(nums1));

        boolean result1 = canJump(nums1);

        System.out.println("示例 1 输出: " + result1);
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        int[] nums2 = new int[]{3, 2, 1, 0, 4};

        System.out.println("示例 2 输入:");
        System.out.println("nums = " + Arrays.toString(nums2));

        boolean result2 = canJump(nums2);

        System.out.println("示例 2 输出: " + result2);
    }
}
```
