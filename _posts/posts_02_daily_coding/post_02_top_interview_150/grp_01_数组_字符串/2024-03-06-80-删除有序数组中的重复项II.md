## 题目

**描述：**

给你一个有序数组 `nums` ，请你 **原地** 删除重复出现的元素，使得出现次数超过两次的元素**只出现两次** ，返回删除后数组的新长度。

不要使用额外的数组空间，你必须在 **原地 修改输入数组** 并在使用 O(1) 额外空间的条件下完成。

**说明：**

为什么返回数值是整数，但输出的答案是数组呢？

请注意，输入数组是以 **「引用」** 方式传递的，这意味着在函数里修改输入数组对于调用者是可见的。

你可以想象内部操作如下:

```java
// nums 是以“引用”方式传递的。也就是说，不对实参做任何拷贝
int len = removeDuplicates(nums);

// 在函数里修改输入数组对于调用者是可见的。
// 根据你的函数返回的长度, 它会打印出数组中 该长度范围内 的所有元素。
for (int i = 0; i < len; i++) {
    print(nums[i]);
}
```

**示例 1:**

- **输入:** `nums = [1,1,1,2,2,3]`
- **输出:** `5`, `nums = [1,1,2,2,3,_]`
- **解释:** 函数应返回新长度 `length = 5`, 并且原数组的前五个元素被修改为 `1, 1, 2, 2, 3` 。 不需要考虑数组中超出新长度后面的元素。

**示例 2:**

- **输入:** `nums = [0,0,1,1,1,1,2,3,3]`
- **输出:** `7`, `nums = [0,0,1,1,2,3,3,_,_]`
- **解释:** 函数应返回新长度 length = 7, 并且原数组的前七个元素被修改为 0, 0, 1, 1, 2, 3, 3。不需要考虑数组中超出新长度后面的元素。

**提示:**

- `1 <= nums.length <= 3 * 10^4`
- `-10^4 <= nums[i] <= 10^4`
- `nums` 已按升序排列

## 题解

### 解法：双指针法

**思路：**

这是一道经典的双指针题目，是“删除有序数组中的重复项”的进阶版。题目要求每个元素最多出现 **两次**。

1.  定义两个指针：

    - `slow`（慢指针）：指向当前即将放置有效元素的位置，同时也代表当前有效数组的长度。
    - `fast`（快指针）：用于遍历数组。

2.  **核心逻辑**：

    - 由于数组是有序的，我们需要检查当前遍历到的元素 `nums[fast]` 是否会导致结果数组中出现三个相同的元素。
    - 因为允许保留两个副本，所以我们需要比较的是 **当前待检查元素** `nums[fast]` 和 **结果数组倒数第二个元素** `nums[slow - 2]`。

3.  **判断规则**：

    - 如果 `slow < 2`：前两个元素不需要检查，直接保留。
    - 如果 `nums[fast] != nums[slow - 2]`：说明 `nums[fast]` 可以加入结果数组。因为数组有序，如果它等于 `nums[slow - 2]`，那么意味着 `nums[slow - 2]`, `nums[slow - 1]`, `nums[fast]` 三者都相等（即出现了三次），这正是我们要避免的。
    - 如果条件满足，将 `nums[fast]` 赋值给 `nums[slow]`，并将 `slow` 向后移动。

此逻辑可以通用推广：如果要保留最多 `k` 个重复项，只需比较 `nums[fast]` 和 `nums[slow - k]`。

**复杂度分析：**

- **时间复杂度：** $O(n)$，其中 $n$ 是数组的长度。我们只需要遍历数组一次。
- **空间复杂度：** $O(1)$，原地修改，只需要常数级额外空间。

**代码片段：**

```java
public int removeDuplicates(int[] nums) {
    int slow = 0;
    for (int fast = 0; fast < nums.length; fast++) {
        // 1. 如果新数组长度小于 2，直接加入
        // 2. 如果当前元素不等于新数组倒数第 2 个元素，说明没有连续 3 个重复，可以加入
        if (slow < 2 || nums[fast] != nums[slow - 2]) {
            nums[slow] = nums[fast];
            slow++;
        }
    }
    return slow;
}
```

### 完整代码

```java
import java.util.Arrays;

public class Main {

    // 核心算法实现
    public static int removeDuplicates(int[] nums) {
        int slow = 0;
        for (int fast = 0; fast < nums.length; fast++) {
            // 只要 slow 指向的位置小于 2，或者当前遍历的数值不等于 slow 前两位的数值
            // 就说明这个数值是有效的（不会造成超过 2 个重复）
            if (slow < 2 || nums[fast] != nums[slow - 2]) {
                nums[slow] = nums[fast];
                slow++;
            }
        }
        return slow;
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        int[] nums1 = new int[]{1, 1, 1, 2, 2, 3};

        System.out.println("示例 1 输入:");
        System.out.println("nums = " + Arrays.toString(nums1));

        int k1 = removeDuplicates(nums1);

        // 截取有效部分打印
        int[] result1 = Arrays.copyOfRange(nums1, 0, k1);
        System.out.println("示例 1 输出: " + k1 + ", nums = " + Arrays.toString(result1));
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        int[] nums2 = new int[]{0, 0, 1, 1, 1, 1, 2, 3, 3};

        System.out.println("示例 2 输入:");
        System.out.println("nums = " + Arrays.toString(nums2));

        int k2 = removeDuplicates(nums2);

        int[] result2 = Arrays.copyOfRange(nums2, 0, k2);
        System.out.println("示例 2 输出: " + k2 + ", nums = " + Arrays.toString(result2));
    }
}
```
