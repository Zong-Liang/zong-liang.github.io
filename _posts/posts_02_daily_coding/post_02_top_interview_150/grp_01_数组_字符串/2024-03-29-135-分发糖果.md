### 题目：135. 分发糖果

**描述**
`n` 个孩子站成一排。给你一个整数数组 `ratings` 表示每个孩子的评分。

你需要按照以下要求，给这些孩子分发糖果：

- 每个孩子至少分配到 1 个糖果。
- 相邻两个孩子中，评分更高的那个会获得更多的糖果。

请你给每个孩子分发糖果，计算并返回需要准备的 **最少糖果数目** 。

**示例 1:**

- **输入:** `ratings = [1,0,2]`
- **输出:** 5
- **解释:** 你可以分别给第一个、第二个、第三个孩子分发 2、1、2 颗糖果。

**示例 2:**

- **输入:** `ratings = [1,2,2]`
- **输出:** 4
- **解释:** 你可以分别给第一个、第二个、第三个孩子分发 1、2、1 颗糖果。第三个孩子只得到 1 颗糖果，这满足题面中的两个条件。

**提示:**

- `n == ratings.length`
- `1 <= n <= 2 * 10^4`
- `0 <= ratings[i] <= 2 * 10^4`

---

### 赛码网格式 Java 8 代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 计算分发糖果需要准备的最少糖果数目。
     * 采用两次遍历的贪心算法。
     *
     * @param ratings 每个孩子的评分数组
     * @return 最少需要的糖果总数
     */
    public static int candy(int[] ratings) {
        int n = ratings.length;
        if (n == 0) {
            return 0;
        }

        // 创建一个数组来存储每个孩子得到的糖果数
        int[] candies = new int[n];
        // 规则1：每个孩子至少1个糖果
        Arrays.fill(candies, 1);

        // 规则2 (部分)：从左到右遍历，确保评分更高的右边孩子比左边孩子糖果多
        for (int i = 1; i < n; i++) {
            if (ratings[i] > ratings[i - 1]) {
                candies[i] = candies[i - 1] + 1;
            }
        }

        // 规则2 (另一部分)：从右到左遍历，确保评分更高的左边孩子比右边孩子糖果多
        // 同时要取当前值和新计算值的最大值，以保证不破坏第一次遍历的结果
        for (int i = n - 2; i >= 0; i--) {
            if (ratings[i] > ratings[i + 1]) {
                candies[i] = Math.max(candies[i], candies[i + 1] + 1);
            }
        }

        // 计算总糖果数
        int totalCandies = 0;
        for (int c : candies) {
            totalCandies += c;
        }

        return totalCandies;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] ratings1 = {1, 0, 2};
        System.out.println("输入: " + Arrays.toString(ratings1));
        System.out.println("输出: " + candy(ratings1)); // 预期: 5
        System.out.println("---");

        // 示例 2
        int[] ratings2 = {1, 2, 2};
        System.out.println("输入: " + Arrays.toString(ratings2));
        System.out.println("输出: " + candy(ratings2)); // 预期: 4
        System.out.println("---");
    }
}
```
