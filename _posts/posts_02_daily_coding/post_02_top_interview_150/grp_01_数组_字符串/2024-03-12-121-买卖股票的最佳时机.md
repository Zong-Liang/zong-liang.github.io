## 题目：121. 买卖股票的最佳时机

**描述**

给定一个数组 `prices` ，它的第 `i` 个元素 `prices[i]` 表示一支给定股票第 `i` 天的价格。

你只能选择 **某一天** 买入这只股票，并选择在 **未来的某一个不同的日子** 卖出该股票。设计一个算法来计算你所能获取的最大利润。

返回你可以从这笔交易中获取的最大利润。如果你不能获取任何利润，返回 `0` 。

**示例 1:**

- **输入:** `[7,1,5,3,6,4]`
- **输出:** 5
- **解释:** 在第 2 天（股票价格 = 1）的时候买入，在第 5 天（股票价格 = 6）的时候卖出，最大利润 = 6-1 = 5 。注意利润不能是 7-1 = 6, 因为卖出价格需要大于买入价格；同时，你不能在买入前卖出股票。

**示例 2:**

- **输入:** `prices = [7,6,4,3,1]`
- **输出:** 0
- **解释:** 在这种情况下, 没有交易完成, 所以最大利润为 0。

**提示:**

- `1 <= prices.length <= 10^5`
- `0 <= prices[i] <= 10^4`

## Java 代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 计算一次买卖可以获得的最大利润。
     *
     * @param prices 股票每日价格的数组
     * @return 最大利润
     */
    public static int maxProfit(int[] prices) {
        if (prices == null || prices.length < 2) {
            return 0;
        }

        // minPrice 记录到目前为止遇到的最低股价
        int minPrice = Integer.MAX_VALUE;
        // maxProfit 记录到目前为止可以获得的最大利润
        int maxProfit = 0;

        // 遍历整个价格数组
        for (int price : prices) {
            // 如果今天的价格更低，就更新最低买入价
            if (price < minPrice) {
                minPrice = price;
            }
            // 否则，尝试以今天的价格卖出，计算利润，并更新最大利润
            else if (price - minPrice > maxProfit) {
                maxProfit = price - minPrice;
            }
        }

        return maxProfit;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] prices1 = {7, 1, 5, 3, 6, 4};
        System.out.println("输入: " + Arrays.toString(prices1));
        System.out.println("输出: " + maxProfit(prices1)); // 预期: 5
        System.out.println("---");

        // 示例 2
        int[] prices2 = {7, 6, 4, 3, 1};
        System.out.println("输入: " + Arrays.toString(prices2));
        System.out.println("输出: " + maxProfit(prices2)); // 预期: 0
        System.out.println("---");
    }
}
```
