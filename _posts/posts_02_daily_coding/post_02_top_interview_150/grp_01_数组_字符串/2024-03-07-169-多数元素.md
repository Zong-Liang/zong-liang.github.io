## 题目：169. 多数元素

**描述**

给定一个大小为 `n` 的数组 `nums` ，返回其中的 **多数元素** 。多数元素是指在数组中出现次数 **大于** `⌊ n/2 ⌋` 的元素。

你可以假设数组是非空的，并且给定的数组总是存在多数元素。

**示例 1:**

- **输入:** `nums = [3,2,3]`
- **输出:** 3

**示例 2:**

- **输入:** `nums = [2,2,1,1,1,2,2]`
- **输出:** 2

**提示:**

- `n == nums.length`
- `1 <= n <= 5 * 10^4`
- `-10^9 <= nums[i] <= 10^9`

**进阶：** 尝试设计时间复杂度为 O(n)、空间复杂度为 O(1) 的算法解决此问题。

## 题解

```java
import java.util.Arrays;

public class Main {

    /**
     * 找出数组中的多数元素。
     * 采用 Boyer-Moore 投票算法，时间复杂度 O(n)，空间复杂度 O(1)。
     *
     * @param nums 整数数组
     * @return 多数元素
     */
    public static int majorityElement(int[] nums) {
        // candidate 用于存储当前的候选多数元素
        Integer candidate = null;
        // count 用于计数
        int count = 0;

        // 遍历数组
        for (int num : nums) {
            // 如果 count 为 0，说明之前的元素都相互抵消了，
            // 将当前元素设为新的候选者
            if (count == 0) {
                candidate = num;
            }

            // 如果当前元素等于候选者，count 加 1；否则减 1
            count += (num == candidate) ? 1 : -1;
        }

        // 因为题目保证多数元素总是存在，所以最后留下的 candidate 就是答案
        return candidate;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1 = {3, 2, 3};
        System.out.println("输入: " + Arrays.toString(nums1));
        System.out.println("输出: " + majorityElement(nums1)); // 预期: 3
        System.out.println("---");

        // 示例 2
        int[] nums2 = {2, 2, 1, 1, 1, 2, 2};
        System.out.println("输入: " + Arrays.toString(nums2));
        System.out.println("输出: " + majorityElement(nums2)); // 预期: 2
        System.out.println("---");
    }
}
```
