### 题目：88. 合并两个有序数组

**描述**
给你两个按 **非递减顺序** 排列的整数数组 `nums1` 和 `nums2`，另有两个整数 `m` 和 `n` ，分别表示 `nums1` 和 `nums2` 中的元素数目。

请你 **合并** `nums2` 到 `nums1` 中，使合并后的数组同样按 **非递减顺序** 排列。

**注意：** 最终，合并后数组不应由函数返回，而是存储在数组 `nums1` 中。为了应对这种情况，`nums1` 的初始长度为 `m + n`，其中前 `m` 个元素表示应合并的元素，后 `n` 个元素为 `0` ，应忽略。`nums2` 的长度为 `n` 。

**示例 1:**

- **输入:** `nums1 = [1,2,3,0,0,0]`, `m = 3`, `nums2 = [2,5,6]`, `n = 3`
- **输出:** `[1,2,2,3,5,6]`
- **解释:** 需要合并 `[1,2,3]` 和 `[2,5,6]` 。合并结果是 `[1,2,2,3,5,6]` ，其中斜体加粗标注的为 `nums1` 中的元素。

**示例 2:**

- **输入:** `nums1 = [1]`, `m = 1`, `nums2 = []`, `n = 0`
- **输出:** `[1]`

**示例 3:**

- **输入:** `nums1 = [0]`, `m = 0`, `nums2 = [1]`, `n = 1`
- **输出:** `[1]`

**提示:**

- `nums1.length == m + n`
- `nums2.length == n`
- `0 <= m, n <= 200`
- `1 <= m + n <= 200`
- `-10^9 <= nums1[i], nums2[j] <= 10^9`

**进阶：** 你可以设计实现一个时间复杂度为 `O(m + n)` 的算法解决此问题吗？

---

### 赛码网格式 Java 8 代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 原地合并两个有序数组。
     * 采用从后往前的双指针法，时间复杂度 O(m + n)。
     *
     * @param nums1 目标数组，有足够空间
     * @param m     nums1 中的有效元素数量
     * @param nums2 源数组
     * @param n     nums2 中的有效元素数量
     */
    public static void merge(int[] nums1, int m, int[] nums2, int n) {
        // p1 指向 nums1 有效元素的末尾
        int p1 = m - 1;
        // p2 指向 nums2 的末尾
        int p2 = n - 1;
        // p 指向 nums1 数组的最终末尾，用于放置元素
        int p = m + n - 1;

        // 从后往前遍历，将较大的元素放到 nums1 的末尾
        while (p1 >= 0 && p2 >= 0) {
            if (nums1[p1] > nums2[p2]) {
                nums1[p] = nums1[p1];
                p1--;
            } else {
                nums1[p] = nums2[p2];
                p2--;
            }
            p--;
        }

        // 如果 nums2 中还有剩余元素（说明它们都比 nums1 的所有元素小），
        // 将它们复制到 nums1 的前面
        // System.arraycopy(源数组, 源数组起始位置, 目标数组, 目标数组起始位置, 拷贝长度)
        System.arraycopy(nums2, 0, nums1, 0, p2 + 1);
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1_ex1 = {1, 2, 3, 0, 0, 0};
        int m1 = 3;
        int[] nums2_ex1 = {2, 5, 6};
        int n1 = 3;
        System.out.println("输入: nums1 = " + Arrays.toString(nums1_ex1) + ", m = " + m1 + ", nums2 = " + Arrays.toString(nums2_ex1) + ", n = " + n1);
        merge(nums1_ex1, m1, nums2_ex1, n1);
        System.out.println("输出: " + Arrays.toString(nums1_ex1)); // 预期: [1, 2, 2, 3, 5, 6]
        System.out.println("---");

        // 示例 2
        int[] nums1_ex2 = {1};
        int m2 = 1;
        int[] nums2_ex2 = {};
        int n2 = 0;
        System.out.println("输入: nums1 = " + Arrays.toString(nums1_ex2) + ", m = " + m2 + ", nums2 = " + Arrays.toString(nums2_ex2) + ", n = " + n2);
        merge(nums1_ex2, m2, nums2_ex2, n2);
        System.out.println("输出: " + Arrays.toString(nums1_ex2)); // 预期: [1]
        System.out.println("---");

        // 示例 3
        int[] nums1_ex3 = {0};
        int m3 = 0;
        int[] nums2_ex3 = {1};
        int n3 = 1;
        System.out.println("输入: nums1 = " + Arrays.toString(nums1_ex3) + ", m = " + m3 + ", nums2 = " + Arrays.toString(nums2_ex3) + ", n = " + n3);
        merge(nums1_ex3, m3, nums2_ex3, n3);
        System.out.println("输出: " + Arrays.toString(nums1_ex3)); // 预期: [1]
        System.out.println("---");
    }
}
```
