## 题目：14. 最长公共前缀

**描述**

编写一个函数来查找字符串数组中的最长公共前缀。

如果不存在公共前缀，返回空字符串 `""`。

**示例 1:**

- **输入:** `strs = ["flower","flow","flight"]`
- **输出:** `"fl"`

**示例 2:**

- **输入:** `strs = ["dog","racecar","car"]`
- **输出:** `""`
- **解释:** 输入不存在公共前缀。

**提示:**

- `1 <= strs.length <= 200`
- `0 <= strs[i].length <= 200`
- `strs[i]` 如果非空，则仅由小写英文字母组成

## Java 代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 查找字符串数组中的最长公共前缀。
     *
     * @param strs 字符串数组
     * @return 最长公共前缀字符串
     */
    public static String longestCommonPrefix(String[] strs) {
        // 如果数组为空或为 null，则没有公共前缀
        if (strs == null || strs.length == 0) {
            return "";
        }

        // 将数组中的第一个字符串作为初始的公共前缀
        String prefix = strs[0];

        // 遍历数组中从第二个开始的每一个字符串
        for (int i = 1; i < strs.length; i++) {
            // 不断缩短前缀，直到它成为当前字符串的前缀
            // String.indexOf(prefix) != 0 表示当前字符串不是以 prefix 开头的
            while (strs[i].indexOf(prefix) != 0) {
                // 从后向前缩短 prefix
                prefix = prefix.substring(0, prefix.length() - 1);

                // 如果 prefix 缩短为空，说明没有公共前缀，直接返回
                if (prefix.isEmpty()) {
                    return "";
                }
            }
        }

        return prefix;
    }

    public static void main(String[] args) {
        // 示例 1
        String[] strs1 = {"flower", "flow", "flight"};
        System.out.println("输入: " + Arrays.toString(strs1));
        System.out.println("输出: \"" + longestCommonPrefix(strs1) + "\""); // 预期: "fl"
        System.out.println("---");

        // 示例 2
        String[] strs2 = {"dog", "racecar", "car"};
        System.out.println("输入: " + Arrays.toString(strs2));
        System.out.println("输出: \"" + longestCommonPrefix(strs2) + "\""); // 预期: ""
        System.out.println("---");
    }
}
```
