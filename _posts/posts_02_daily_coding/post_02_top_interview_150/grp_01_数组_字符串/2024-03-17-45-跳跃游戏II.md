### 题目：45. 跳跃游戏 II

**描述**
给定一个长度为 `n` 的 **0 索引** 整数数组 `nums`。初始位置在下标 0。

每个元素 `nums[i]` 表示从索引 `i` 向后跳转的最大长度。换句话说，如果你在索引 `i` 处，你可以跳转到任意 `(i + j)` 处:

- `0 <= j <= nums[i]` 且
- `i + j < n`

返回到达 `n - 1` 的最小跳跃次数。测试用例保证可以到达 `n - 1`。

**示例 1:**

- **输入:** `nums = [2,3,1,1,4]`
- **输出:** 2
- **解释:** 跳到最后一个位置的最小跳跃数是 2。从下标 0 跳到下标 1 的位置，跳 1 步，然后跳 3 步到达数组的最后一个位置。

**示例 2:**

- **输入:** `nums = [2,3,0,1,4]`
- **输出:** 2

**提示:**

- `1 <= nums.length <= 10^4`
- `0 <= nums[i] <= 1000`
- 题目保证可以到达 `n - 1`

---

### 赛码网格式 Java 8 代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 计算到达数组最后一个位置的最小跳跃次数。
     * 采用贪心算法。
     *
     * @param nums 整数数组，表示每一步可以跳跃的最大长度
     * @return 最小跳跃次数
     */
    public static int jump(int[] nums) {
        int n = nums.length;
        if (n <= 1) {
            return 0;
        }

        // jumps: 当前的跳跃次数
        int jumps = 0;
        // currentEnd: 当前跳跃能够到达的最远边界
        int currentEnd = 0;
        // farthest: 在当前边界内，所有位置能够跳到的最远位置
        int farthest = 0;

        // 遍历数组，但不包括最后一个元素
        // 因为当 i 到达最后一个元素时，我们已经到达目的地了
        for (int i = 0; i < n - 1; i++) {
            // 在 [i, currentEnd] 的范围内，不断更新能跳到的最远距离
            farthest = Math.max(farthest, i + nums[i]);

            // 如果到达了当前跳跃的边界
            if (i == currentEnd) {
                // 就必须进行下一次跳跃
                jumps++;
                // 更新下一次跳跃的边界为刚刚计算出的最远距离
                currentEnd = farthest;

                // 优化：如果新的边界已经可以覆盖或超过终点，可以提前结束
                if (currentEnd >= n - 1) {
                    break;
                }
            }
        }

        return jumps;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1 = {2, 3, 1, 1, 4};
        System.out.println("输入: " + Arrays.toString(nums1));
        System.out.println("输出: " + jump(nums1)); // 预期: 2
        System.out.println("---");

        // 示例 2
        int[] nums2 = {2, 3, 0, 1, 4};
        System.out.println("输入: " + Arrays.toString(nums2));
        System.out.println("输出: " + jump(nums2)); // 预期: 2
        System.out.println("---");
    }
}
```
