## 题目：189. 轮转数组

**描述**

给定一个整数数组 `nums`，将数组中的元素向右轮转 `k` 个位置，其中 `k` 是非负数。

**示例 1:**

- **输入:** `nums = [1,2,3,4,5,6,7], k = 3`
- **输出:** `[5,6,7,1,2,3,4]`
- **解释:**

  向右轮转 1 步: `[7,1,2,3,4,5,6]`

  向右轮转 2 步: `[6,7,1,2,3,4,5]`

  向右轮转 3 步: `[5,6,7,1,2,3,4]`

**示例 2:**

- **输入:** `nums = [-1,-100,3,99], k = 2`
- **输出:** `[3,99,-1,-100]`
- **解释:**

  向右轮转 1 步: `[99,-1,-100,3]`

  向右轮转 2 步: `[3,99,-1,-100]`

**提示:**

- `1 <= nums.length <= 10^5`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `0 <= k <= 10^5`

**进阶:**

- 尽可能想出更多的解决方案，至少有 **三种** 不同的方法可以解决这个问题。
- 你可以使用空间复杂度为 `O(1)` 的 **原地** 算法解决这个问题吗？

## 题解

```java
import java.util.Arrays;

public class Main {

    /**
     * 将数组中的元素向右轮转 k 个位置 (原地算法)。
     * 采用“三次反转”的方法，空间复杂度为 O(1)。
     *
     * @param nums 整数数组
     * @param k    轮转的步数
     */
    public static void rotate(int[] nums, int k) {
        int n = nums.length;
        // 如果 k 大于数组长度，取模以避免不必要的重复轮转
        k %= n;

        // 1. 反转整个数组
        // 例如：[1,2,3,4,5,6,7] -> [7,6,5,4,3,2,1]
        reverse(nums, 0, n - 1);

        // 2. 反转前 k 个元素
        // 例如：[7,6,5,4,3,2,1] -> [5,6,7,4,3,2,1] (k=3)
        reverse(nums, 0, k - 1);

        // 3. 反转剩下的 n-k 个元素
        // 例如：[5,6,7,4,3,2,1] -> [5,6,7,1,2,3,4]
        reverse(nums, k, n - 1);
    }

    /**
     * 辅助方法：反转数组中指定范围的元素。
     * @param nums  数组
     * @param start 起始索引
     * @param end   结束索引
     */
    private static void reverse(int[] nums, int start, int end) {
        while (start < end) {
            int temp = nums[start];
            nums[start] = nums[end];
            nums[end] = temp;
            start++;
            end--;
        }
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1 = {1, 2, 3, 4, 5, 6, 7};
        int k1 = 3;
        System.out.println("输入: nums = " + Arrays.toString(nums1) + ", k = " + k1);
        rotate(nums1, k1);
        System.out.println("输出: " + Arrays.toString(nums1)); // 预期: [5, 6, 7, 1, 2, 3, 4]
        System.out.println("---");

        // 示例 2
        int[] nums2 = {-1, -100, 3, 99};
        int k2 = 2;
        System.out.println("输入: nums = " + Arrays.toString(nums2) + ", k = " + k2);
        rotate(nums2, k2);
        System.out.println("输出: " + Arrays.toString(nums2)); // 预期: [3, 99, -1, -100]
        System.out.println("---");
    }
}
```
