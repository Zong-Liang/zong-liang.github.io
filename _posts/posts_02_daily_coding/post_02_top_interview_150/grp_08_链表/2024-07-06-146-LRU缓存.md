## 题目

**描述：**

请你设计并实现一个满足 **LRU (最近最少使用)** 缓存约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1`。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value`；如果不存在，则向缓存中插入该组 `key-value`。如果插入操作导致关键字数量超过 `capacity`，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 都必须以 `O(1)` 的平均时间复杂度运行。

**示例：**

- **输入**
  `["LRUCache", "put", "put", "get", "put", "get", "put", "get", "get", "get"]`

  `[[2], [1, 1], [2, 2], [1], [3, 3], [2], [4, 4], [1], [3], [4]]`

- **输出**

  `[null, null, null, 1, null, -1, null, -1, 3, 4]`

- **解释**

  ```
  LRUCache lRUCache = new LRUCache(2);
  lRUCache.put(1, 1); // 缓存是 {1=1}
  lRUCache.put(2, 2); // 缓存是 {1=1, 2=2}
  lRUCache.get(1);    // 返回 1，{2=2, 1=1}
  lRUCache.put(3, 3); // 该操作会使得关键字 2 作废，缓存是 {1=1, 3=3}
  lRUCache.get(2);    // 返回 -1 (未找到)
  lRUCache.put(4, 4); // 该操作会使得关键字 1 作废，缓存是 {3=3, 4=4}
  lRUCache.get(1);    // 返回 -1 (未找到)
  lRUCache.get(3);    // 返回 3
  lRUCache.get(4);    // 返回 4
  ```

**提示:**

- `1 <= capacity <= 3000`
- `0 <= key <= 10000`
- `0 <= value <= 10^5`
- 最多调用 `2 * 10^5` 次 `get` 和 `put`

## 题解

### 解法一：哈希表 + 双向链表 (Hash Map + Double Linked List)

**思路：**

LRU 缓存的核心在于两点：

1.  **快速查找**：需要 $O(1)$ 时间内判断 Key 是否存在并获取 Value。这显然需要 **哈希表 (HashMap)**。
2.  **快速排序与移动**：需要维护元素的访问顺序（最近使用的在最前，最久未使用的在最后），且需要在 $O(1)$ 时间内将某个元素移动到头部，或者删除尾部元素。数组无法满足插入删除 $O(1)$，单向链表删除需要遍历前驱节点，只有 **双向链表 (Doubly Linked List)** 能满足在已知节点引用的情况下 $O(1)$ 删除和插入。

**设计方案：**

我们结合哈希表和双向链表：

- **哈希表** (`Map<Integer, Node>`)：存储 `key` 到 `链表节点 (Node)` 的映射。这样我们就能通过 key 快速找到链表中的节点。
- **双向链表**：存储具体的 `key` 和 `value`。

  - 为了方便操作，我们定义两个**哨兵节点**：`head` (虚拟头) 和 `tail` (虚拟尾)。
  - **规则**：靠近 `head` 的是最近使用的 (Most Recently Used)，靠近 `tail` 的是最久未使用的 (Least Recently Used)。

**算法流程：**

1.  **`get(key)`**:

    - 查询哈希表。如果不存在，返回 -1。
    - 如果存在，找到对应的 `Node`。
    - **核心操作**：将该 `Node` 从链表当前位置删除，并插入到 `head` 后面（标记为最近使用）。
    - 返回 `Node.value`。

2.  **`put(key, value)`**:

    - 查询哈希表。
    - **如果 key 存在**：更新 `Node.value`，并将该 `Node` 移动到 `head` 后面。
    - **如果 key 不存在**：

      - 创建新 `Node`。
      - 添加到哈希表，并插入到链表 `head` 后面。
      - **检查容量**：如果 `map.size() > capacity`，则需要**淘汰**。
      - **淘汰操作**：移除链表倒数第一个节点（即 `tail` 的前驱），同时从哈希表中移除对应的 key。

**复杂度分析：**

- **时间复杂度：** `get` 和 `put` 均为 $O(1)$。哈希表查找是 $O(1)$，双向链表的节点移动也是 $O(1)$。
- **空间复杂度：** $O(capacity)$。需要存储最多 `capacity` 个节点。

**代码片段：**

```java
class LRUCache {
    // 定义双向链表节点
    class Node {
        int key, value;
        Node prev, next;
        Node(int k, int v) { key = k; value = v; }
    }

    private int capacity;
    private Map<Integer, Node> map;
    private Node head, tail; // 哨兵节点

    public LRUCache(int capacity) {
        this.capacity = capacity;
        this.map = new HashMap<>();
        // 初始化哨兵节点，避免边界判断
        head = new Node(-1, -1);
        tail = new Node(-1, -1);
        head.next = tail;
        tail.prev = head;
    }

    public int get(int key) {
        if (!map.containsKey(key)) {
            return -1;
        }
        Node node = map.get(key);
        // 访问后，将其移动到头部
        moveToHead(node);
        return node.value;
    }

    public void put(int key, int value) {
        if (map.containsKey(key)) {
            Node node = map.get(key);
            node.value = value;
            moveToHead(node);
        } else {
            Node newNode = new Node(key, value);
            map.put(key, newNode);
            addToHead(newNode);

            if (map.size() > capacity) {
                // 超出容量，删除双向链表尾部的节点（最久未使用）
                Node res = removeTail();
                map.remove(res.key);
            }
        }
    }

    // --- 辅助函数：操作双向链表 ---

    // 将节点移动到头部
    private void moveToHead(Node node) {
        removeNode(node);
        addToHead(node);
    }

    // 从链表中移除节点
    private void removeNode(Node node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    // 将节点添加到头部（head 之后）
    private void addToHead(Node node) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }

    // 删除尾部节点（tail 之前），并返回该节点（以便在 map 中删除）
    private Node removeTail() {
        Node res = tail.prev;
        removeNode(res);
        return res;
    }
}
```

### 解法二：利用 Java LinkedHashMap (偷懒法/面试备选)

**思路：**

Java 的 `java.util.LinkedHashMap` 本质上就是哈希表 + 双向链表的实现。它提供了一个构造函数，可以开启 `accessOrder` 模式（访问顺序模式，而非默认的插入顺序模式）。此外，它提供了一个 `removeEldestEntry` 方法，当返回 `true` 时会自动删除最老的元素。

虽然在面试中直接用这个可能被判定为“作弊”或“无法展示数据结构能力”，但了解它非常重要，在工程实践中通常直接使用它。

**代码片段：**

```java
class LRUCache extends LinkedHashMap<Integer, Integer> {
    private int capacity;

    public LRUCache(int capacity) {
        // accessOrder = true 表示按照访问顺序排序
        super(capacity, 0.75F, true);
        this.capacity = capacity;
    }

    public int get(int key) {
        // LinkedHashMap 若 key 不存在返回 null，需手动处理
        return super.getOrDefault(key, -1);
    }

    public void put(int key, int value) {
        super.put(key, value);
    }

    @Override
    protected boolean removeEldestEntry(Map.Entry<Integer, Integer> eldest) {
        // 当 size 超过容量时，自动移除最老的元素
        return size() > capacity;
    }
}
```

### 完整代码

这里提供 **解法一（手写双向链表 + 哈希表）** 的完整可运行代码，这是面试中最标准、最能体现能力的写法。

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    static class LRUCache {
        // 内部类：双向链表节点
        class Node {
            int key;
            int value;
            Node prev;
            Node next;

            public Node() {}
            public Node(int key, int value) {
                this.key = key;
                this.value = value;
            }
        }

        private Map<Integer, Node> cache = new HashMap<>();
        private int capacity;
        private int size;
        private Node head, tail;

        public LRUCache(int capacity) {
            this.capacity = capacity;
            this.size = 0;
            // 初始化伪头部和伪尾部节点
            head = new Node();
            tail = new Node();
            head.next = tail;
            tail.prev = head;
        }

        public int get(int key) {
            Node node = cache.get(key);
            if (node == null) {
                return -1;
            }
            // 如果 key 存在，先通过哈希表定位，再移到头部
            moveToHead(node);
            return node.value;
        }

        public void put(int key, int value) {
            Node node = cache.get(key);
            if (node == null) {
                // 如果 key 不存在，创建一个新的节点
                Node newNode = new Node(key, value);
                // 添加进哈希表
                cache.put(key, newNode);
                // 添加至双向链表的头部
                addToHead(newNode);
                size++;

                if (size > capacity) {
                    // 如果超出容量，删除双向链表的尾部节点
                    Node tail = removeTail();
                    // 删除哈希表中对应的项
                    cache.remove(tail.key);
                    size--;
                }
            } else {
                // 如果 key 存在，修改 value，并移到头部
                node.value = value;
                moveToHead(node);
            }
        }

        // --- 双向链表操作 ---

        private void addToHead(Node node) {
            node.prev = head;
            node.next = head.next;
            head.next.prev = node;
            head.next = node;
        }

        private void removeNode(Node node) {
            node.prev.next = node.next;
            node.next.prev = node.prev;
        }

        private void moveToHead(Node node) {
            removeNode(node);
            addToHead(node);
        }

        private Node removeTail() {
            Node res = tail.prev;
            removeNode(res);
            return res;
        }
    }

    public static void main(String[] args) {
        // --- 测试示例 ---
        System.out.println("开始测试 LRUCache:");

        LRUCache lRUCache = new LRUCache(2);

        System.out.println("put(1, 1)");
        lRUCache.put(1, 1); // 缓存是 {1=1}

        System.out.println("put(2, 2)");
        lRUCache.put(2, 2); // 缓存是 {1=1, 2=2}

        System.out.println("get(1) -> " + lRUCache.get(1));    // 返回 1，缓存变为 {2=2, 1=1}

        System.out.println("put(3, 3)");
        lRUCache.put(3, 3); // 导致 key 2 作废，缓存是 {1=1, 3=3}

        System.out.println("get(2) -> " + lRUCache.get(2));    // 返回 -1 (未找到)

        System.out.println("put(4, 4)");
        lRUCache.put(4, 4); // 导致 key 1 作废，缓存是 {3=3, 4=4}

        System.out.println("get(1) -> " + lRUCache.get(1));    // 返回 -1 (未找到)
        System.out.println("get(3) -> " + lRUCache.get(3));    // 返回 3
        System.out.println("get(4) -> " + lRUCache.get(4));    // 返回 4

        System.out.println("-----------------------------");
        System.out.println("测试完成");
    }
}
```
