## 题目：146. LRU 缓存

请你设计并实现一个满足 **LRU (最近最少使用)** 缓存约束的数据结构。

实现 `LRUCache` 类：

- `LRUCache(int capacity)` 以 **正整数** 作为容量 `capacity` 初始化 LRU 缓存
- `int get(int key)` 如果关键字 `key` 存在于缓存中，则返回关键字的值，否则返回 `-1`。
- `void put(int key, int value)` 如果关键字 `key` 已经存在，则变更其数据值 `value`；如果不存在，则向缓存中插入该组 `key-value`。如果插入操作导致关键字数量超过 `capacity`，则应该 **逐出** 最久未使用的关键字。

函数 `get` 和 `put` 都必须以 `O(1)` 的平均时间复杂度运行。

**示例：**

- **输入**
  `["LRUCache", "put", "put", "get", "put", "get", "put", "get", "get", "get"]`
  `[[2], [1, 1], [2, 2], [1], [3, 3], [2], [4, 4], [1], [3], [4]]`
- **输出**
  `[null, null, null, 1, null, -1, null, -1, 3, 4]`
- **解释**
  ```
  LRUCache lRUCache = new LRUCache(2);
  lRUCache.put(1, 1); // 缓存是 {1=1}
  lRUCache.put(2, 2); // 缓存是 {1=1, 2=2}
  lRUCache.get(1);    // 返回 1，{2=2, 1=1}
  lRUCache.put(3, 3); // 该操作会使得关键字 2 作废，缓存是 {1=1, 3=3}
  lRUCache.get(2);    // 返回 -1 (未找到)
  lRUCache.put(4, 4); // 该操作会使得关键字 1 作废，缓存是 {3=3, 4=4}
  lRUCache.get(1);    // 返回 -1 (未找到)
  lRUCache.get(3);    // 返回 3
  lRUCache.get(4);    // 返回 4
  ```

**提示:**

- `1 <= capacity <= 3000`
- `0 <= key <= 10000`
- `0 <= value <= 10^5`
- 最多调用 `2 * 10^5` 次 `get` 和 `put`

## 题解

**算法思路：(哈希表 + 双向链表)**

为了实现 `get` 和 `put` 操作的 `O(1)` 时间复杂度，我们需要一个能够快速查找的数据结构（**哈希表**），以及一个能够快速插入和删除节点的数据结构（**双向链表**）。

1.  **哈希表 `cacheMap`:**
    - `key` 是缓存的键。
    - `value` 是一个指向双向链表中对应节点的引用。
    - 这保证了我们可以在 `O(1)` 时间内找到任何一个缓存项。
2.  **双向链表 `DLinkedList`:**
    - 用于维护缓存项的“新鲜度”。链表头部是最新使用的项，链表尾部是最久未使用的项。
    - 当我们访问（`get`）或更新（`put`）一个缓存项时，我们将其对应的节点移动到链表头部。
    - 当缓存已满并需要插入新项时，我们删除链表尾部的节点。

**`get(key)` 操作:**

1.  在 `cacheMap` 中查找 `key`。
2.  如果不存在，返回 `-1`。
3.  如果存在，获取对应的节点，将其从当前位置移动到链表头部，并返回节点的值。

**`put(key, value)` 操作:**

1.  在 `cacheMap` 中查找 `key`。
2.  **如果存在:** 更新节点的值，并将其移动到链表头部。
3.  **如果不存在:**
    a. 创建一个新节点。
    b. 将新节点插入到链表头部。
    c. 在 `cacheMap` 中添加 `(key, newNode)` 的映射。
    d. 检查缓存大小是否超过 `capacity`。如果是，则删除链表尾部的节点，并从 `cacheMap` 中移除对应的键。

Java 的 `LinkedHashMap` 实际上可以更简洁地实现 LRU 缓存，但为了展示底层原理，这里使用手动实现的哈希表和双向链表。

```java
import java.util.HashMap;
import java.util.Map;

/**
 * 双向链表节点
 */
class DLinkedNode {
    int key;
    int value;
    DLinkedNode prev;
    DLinkedNode next;

    public DLinkedNode() {}
    public DLinkedNode(int key, int value) {
        this.key = key;
        this.value = value;
    }
}

/**
 * LRUCache 类
 */
class LRUCache {
    private final Map<Integer, DLinkedNode> cacheMap;
    private final int capacity;
    private int size;
    private final DLinkedNode head;
    private final DLinkedNode tail;

    public LRUCache(int capacity) {
        this.capacity = capacity;
        this.size = 0;
        this.cacheMap = new HashMap<>();
        // 使用伪头部和伪尾部节点
        this.head = new DLinkedNode();
        this.tail = new DLinkedNode();
        head.next = tail;
        tail.prev = head;
    }

    public int get(int key) {
        DLinkedNode node = cacheMap.get(key);
        if (node == null) {
            return -1;
        }
        // 移动到头部，表示最近使用
        moveToHead(node);
        return node.value;
    }

    public void put(int key, int value) {
        DLinkedNode node = cacheMap.get(key);
        if (node != null) {
            // 更新值并移动到头部
            node.value = value;
            moveToHead(node);
        } else {
            DLinkedNode newNode = new DLinkedNode(key, value);
            cacheMap.put(key, newNode);
            addToHead(newNode);
            size++;
            if (size > capacity) {
                // 移除最久未使用的节点
                DLinkedNode tailNode = removeTail();
                cacheMap.remove(tailNode.key);
                size--;
            }
        }
    }

    // 链表操作辅助函数
    private void addToHead(DLinkedNode node) {
        node.prev = head;
        node.next = head.next;
        head.next.prev = node;
        head.next = node;
    }

    private void removeNode(DLinkedNode node) {
        node.prev.next = node.next;
        node.next.prev = node.prev;
    }

    private void moveToHead(DLinkedNode node) {
        removeNode(node);
        addToHead(node);
    }

    private DLinkedNode removeTail() {
        DLinkedNode res = tail.prev;
        removeNode(res);
        return res;
    }
}



public class Main {
    public static void main(String[] args) {
        System.out.println("--- 示例 ---");

        LRUCache lRUCache = new LRUCache(2);
        System.out.print("[null");

        lRUCache.put(1, 1);
        System.out.print(", null");

        lRUCache.put(2, 2);
        System.out.print(", null");

        int res1 = lRUCache.get(1);
        System.out.print(", " + res1); // 1

        lRUCache.put(3, 3);
        System.out.print(", null");

        int res2 = lRUCache.get(2);
        System.out.print(", " + res2); // -1

        lRUCache.put(4, 4);
        System.out.print(", null");

        int res3 = lRUCache.get(1);
        System.out.print(", " + res3); // -1

        int res4 = lRUCache.get(3);
        System.out.print(", " + res4); // 3

        int res5 = lRUCache.get(4);
        System.out.print(", " + res5); // 4

        System.out.println("]");
    }
}
```
