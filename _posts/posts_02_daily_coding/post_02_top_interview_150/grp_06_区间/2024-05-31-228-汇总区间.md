## 题目：228. 汇总区间

**描述**

给定一个 **无重复元素** 的 **有序** 整数数组 `nums` 。

区间 `[a,b]` 是从 `a` 到 `b` (包含) 的所有整数的集合。

返回 **恰好覆盖** 数组中所有数字的 **最小有序** 区间范围列表。也就是说，`nums` 的每个元素都恰好被某个区间范围所覆盖，并且不存在属于某个区间但不属于 `nums` 的数字 `x` 。

列表中的每个区间范围 `[a,b]` 应该按如下格式输出：

- `"a->b"` ，如果 `a != b`
- `"a"` ，如果 `a == b`

**示例 1:**

- **输入:** `nums = [0,1,2,4,5,7]`
- **输出:** `["0->2","4->5","7"]`
- **解释:** 区间范围是:
  - `[0,2] --> "0->2"`
  - `[4,5] --> "4->5"`
  - `[7,7] --> "7"`

**示例 2:**

- **输入:** `nums = [0,2,3,4,6,8,9]`
- **输出:** `["0","2->4","6","8->9"]`
- **解释:** 区间范围是:
  - `[0,0] --> "0"`
  - `[2,4] --> "2->4"`
  - `[6,6] --> "6"`
  - `[8,9] --> "8->9"`

**提示:**

- `0 <= nums.length <= 20`
- `-2^31 <= nums[i] <= 2^31 - 1`
- `nums` 中的所有值都 **互不相同**
- `nums` 按 **升序** 排列


## Java 代码

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Main {

    /**
     * 找到覆盖有序数组中所有数字的最小区间范围列表。
     *
     * @param nums 无重复元素的有序整数数组
     * @return 表示区间范围的字符串列表
     */
    public static List<String> summaryRanges(int[] nums) {
        List<String> result = new ArrayList<>();
        if (nums == null || nums.length == 0) {
            return result;
        }

        // 使用双指针思想，i 是每个区间的起始索引
        int i = 0;
        while (i < nums.length) {
            // 记录区间的起始值
            int start = nums[i];

            // 向后查找，直到数字不再连续
            // nums[i+1] == nums[i] + 1
            while (i + 1 < nums.length && nums[i + 1] - nums[i] == 1) {
                i++;
            }

            // 记录区间的结束值
            int end = nums[i];

            // 根据起始值和结束值是否相同，格式化并添加结果
            if (start == end) {
                result.add(String.valueOf(start));
            } else {
                result.add(start + "->" + end);
            }

            // 移动到下一个可能的区间起始点
            i++;
        }

        return result;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1 = {0, 1, 2, 4, 5, 7};
        System.out.println("输入: " + Arrays.toString(nums1));
        System.out.println("输出: " + summaryRanges(nums1)); // 预期: ["0->2", "4->5", "7"]
        System.out.println("---");

        // 示例 2
        int[] nums2 = {0, 2, 3, 4, 6, 8, 9};
        System.out.println("输入: " + Arrays.toString(nums2));
        System.out.println("输出: " + summaryRanges(nums2)); // 预期: ["0", "2->4", "6", "8->9"]
        System.out.println("---");

        // 额外测试用例：空数组
        int[] nums3 = {};
        System.out.println("输入: " + Arrays.toString(nums3));
        System.out.println("输出: " + summaryRanges(nums3)); // 预期: []
        System.out.println("---");

        // 额外测试用例：单个元素
        int[] nums4 = {5};
        System.out.println("输入: " + Arrays.toString(nums4));
        System.out.println("输出: " + summaryRanges(nums4)); // 预期: ["5"]
        System.out.println("---");
    }
}
```
