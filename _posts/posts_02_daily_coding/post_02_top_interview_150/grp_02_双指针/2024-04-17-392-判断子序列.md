## 题目：392. 判断子序列

**描述**

给定字符串 **s** 和 **t** ，判断 **s** 是否为 **t** 的子序列。

字符串的一个子序列是原始字符串删除一些（也可以不删除）字符而不改变剩余字符相对位置形成的新字符串。（例如，`"ace"`是`"abcde"`的一个子序列，而`"aec"`不是）。

**进阶:**
如果有大量输入的 S，称作 S1, S2, ..., Sk 其中 k >= 10 亿，你需要依次检查它们是否为 T 的子序列。在这种情况下，你会怎样改变代码？

**示例 1:**

- **输入:** `s = "abc"`, `t = "ahbgdc"`
- **输出:** `true`

**示例 2:**

- **输入:** `s = "axc"`, `t = "ahbgdc"`
- **输出:** `false`

**提示:**

- `0 <= s.length <= 100`
- `0 <= t.length <= 10^4`
- 两个字符串都只由小写字母组成。



## Java 代码

```java
public class Main {

    /**
     * 判断 s 是否是 t 的子序列。
     * 采用双指针法。
     *
     * @param s 子序列字符串
     * @param t 源字符串
     * @return 如果 s 是 t 的子序列，返回 true；否则返回 false。
     */
    public static boolean isSubsequence(String s, String t) {
        // 如果 s 为空，它总是 t 的子序列
        if (s.length() == 0) {
            return true;
        }

        // i 指向 s，j 指向 t
        int i = 0;
        int j = 0;

        // 遍历 t 字符串
        while (j < t.length()) {
            // 如果 t 中的当前字符与 s 中的当前字符匹配
            if (s.charAt(i) == t.charAt(j)) {
                // 移动 s 的指针，去匹配下一个字符
                i++;
            }
            // 无论是否匹配，都移动 t 的指针
            j++;

            // 如果 s 的所有字符都已匹配完，则提前结束并返回 true
            if (i == s.length()) {
                return true;
            }
        }

        // 如果遍历完 t，s 的指针 i 还没有走完，说明 s 不是 t 的子序列
        return false;
    }

    public static void main(String[] args) {
        // 示例 1
        String s1 = "abc";
        String t1 = "ahbgdc";
        System.out.println("输入: s = \"" + s1 + "\", t = \"" + t1 + "\"");
        System.out.println("输出: " + isSubsequence(s1, t1)); // 预期: true
        System.out.println("---");

        // 示例 2
        String s2 = "axc";
        String t2 = "ahbgdc";
        System.out.println("输入: s = \"" + s2 + "\", t = \"" + t2 + "\"");
        System.out.println("输出: " + isSubsequence(s2, t2)); // 预期: false
        System.out.println("---");

        // 额外测试用例
        String s3 = "";
        String t3 = "ahbgdc";
        System.out.println("输入: s = \"" + s3 + "\", t = \"" + t3 + "\"");
        System.out.println("输出: " + isSubsequence(s3, t3)); // 预期: true
        System.out.println("---");

        String s4 = "b";
        String t4 = "c";
        System.out.println("输入: s = \"" + s4 + "\", t = \"" + t4 + "\"");
        System.out.println("输出: " + isSubsequence(s4, t4)); // 预期: false
        System.out.println("---");
    }
}
```
