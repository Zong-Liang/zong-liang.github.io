## 题目：15. 三数之和

**描述**

给你一个整数数组 `nums` ，判断是否存在三元组 `[nums[i], nums[j], nums[k]]` 满足 `i != j`、`i != k` 且 `j != k` ，同时还满足 `nums[i] + nums[j] + nums[k] == 0` 。请你返回所有和为 `0` 且不重复的三元组。

**注意：** 答案中不可以包含重复的三元组。

**示例 1:**

- **输入:** `nums = [-1,0,1,2,-1,-4]`
- **输出:** `[[-1,-1,2],[-1,0,1]]`
- **解释:**
  - `nums[0] + nums[1] + nums[2] = (-1) + 0 + 1 = 0` 。
  - `nums[1] + nums[2] + nums[4] = 0 + 1 + (-1) = 0` 。
  - `nums[0] + nums[3] + nums[4] = (-1) + 2 + (-1) = 0` 。
  - 不同的三元组是 `[-1,0,1]` 和 `[-1,-1,2]` 。
  - 注意，输出的顺序和三元组的顺序并不重要。

**示例 2:**

- **输入:** `nums = [0,1,1]`
- **输出:** `[]`
- **解释:** 唯一可能的三元组和不为 0 。

**示例 3:**

- **输入:** `nums = [0,0,0]`
- **输出:** `[[0,0,0]]`
- **解释:** 唯一可能的三元组和为 0 。

**提示:**

- `3 <= nums.length <= 3000`
- `-10^5 <= nums[i] <= 10^5`



## Java 代码

```java
import java.util.ArrayList;
import java.util.Arrays;
import java.util.List;

public class Main {

    /**
     * 找出数组中所有和为 0 且不重复的三元组。
     * 采用“排序 + 双指针”的解法。
     *
     * @param nums 整数数组
     * @return 所有满足条件的三元组列表
     */
    public static List<List<Integer>> threeSum(int[] nums) {
        List<List<Integer>> result = new ArrayList<>();
        if (nums == null || nums.length < 3) {
            return result;
        }

        // 1. 对数组进行排序
        Arrays.sort(nums);

        // 2. 遍历数组，固定第一个数 a
        for (int i = 0; i < nums.length - 2; i++) {
            // 如果第一个数就大于 0，由于数组已排序，后面的数都大于0，和不可能为0
            if (nums[i] > 0) {
                break;
            }

            // 去重：如果当前数字与前一个数字相同，则跳过，避免重复解
            if (i > 0 && nums[i] == nums[i - 1]) {
                continue;
            }

            // 3. 使用双指针 L 和 R 在 i 后面的区间寻找 b 和 c
            int left = i + 1;
            int right = nums.length - 1;

            while (left < right) {
                int sum = nums[i] + nums[left] + nums[right];

                if (sum == 0) {
                    // 找到了一个解
                    result.add(Arrays.asList(nums[i], nums[left], nums[right]));

                    // 去重：跳过所有与当前 left 和 right 相同的数字
                    while (left < right && nums[left] == nums[left + 1]) {
                        left++;
                    }
                    while (left < right && nums[right] == nums[right - 1]) {
                        right--;
                    }

                    // 移动指针寻找下一个可能的解
                    left++;
                    right--;
                } else if (sum < 0) {
                    // 和小于0，需要增大和，左指针右移
                    left++;
                } else { // sum > 0
                    // 和大于0，需要减小和，右指针左移
                    right--;
                }
            }
        }
        return result;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1 = {-1, 0, 1, 2, -1, -4};
        System.out.println("输入: " + Arrays.toString(nums1));
        System.out.println("输出: " + threeSum(nums1));
        // 预期: [[-1, -1, 2], [-1, 0, 1]] (顺序可能不同)
        System.out.println("---");

        // 示例 2
        int[] nums2 = {0, 1, 1};
        System.out.println("输入: " + Arrays.toString(nums2));
        System.out.println("输出: " + threeSum(nums2)); // 预期: []
        System.out.println("---");

        // 示例 3
        int[] nums3 = {0, 0, 0};
        System.out.println("输入: " + Arrays.toString(nums3));
        System.out.println("输出: " + threeSum(nums3)); // 预期: [[0, 0, 0]]
        System.out.println("---");
    }
}
```
