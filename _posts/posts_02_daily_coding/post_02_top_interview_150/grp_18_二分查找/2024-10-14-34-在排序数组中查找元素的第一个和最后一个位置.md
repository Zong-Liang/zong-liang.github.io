### 题目：34. 在排序数组中查找元素的第一个和最后一个位置

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

---

**示例 1:**

- **输入:** `nums = [5,7,7,8,8,10]`, `target = 8`
- **输出:** `[3,4]`

**示例 2:**

- **输入:** `nums = [5,7,7,8,8,10]`, `target = 6`
- **输出:** `[-1,-1]`

**示例 3:**

- **输入:** `nums = []`, `target = 0`
- **输出:** `[-1,-1]`

---

**提示:**

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `nums` 是一个非递减数组
- `-10^9 <= target <= 10^9`

---

### Java 8 解答 (赛码网格式)

**算法思路：(二分查找)**
为了满足 `O(log n)` 的时间复杂度，我们必须使用二分查找。标准的二分查找只能找到目标值的一个实例，但我们需要找到它的范围。因此，我们可以执行两次修改版的二分查找：

1.  **第一次二分查找**：寻找目标值的 **第一个** 出现位置（左边界）。
    - 当我们找到一个 `nums[mid] == target` 时，我们不立即停止，而是记录下这个位置，并尝试在它的 **左边** 继续寻找更早出现的位置（即 `right = mid - 1`）。
2.  **第二次二分查找**：寻找目标值的 **最后一个** 出现位置（右边界）。
    - 当我们找到一个 `nums[mid] == target` 时，我们同样记录下位置，但这次是尝试在它的 **右边** 继续寻找更晚出现的位置（即 `left = mid + 1`）。

如果第一次查找没有找到任何目标值，那么数组中就不存在该值，可以直接返回 `[-1, -1]`。

```java
import java.util.Arrays;

/**
 * 赛码网要求的标准类名是 Main
 */
public class Main {

    /**
     * 找出给定目标值在数组中的开始位置和结束位置。
     *
     * @param nums   非递减顺序排列的整数数组
     * @param target 目标值
     * @return 包含开始和结束位置的数组，或 [-1, -1]
     */
    public static int[] searchRange(int[] nums, int target) {
        int first = findBound(nums, target, true); // 查找左边界

        // 如果没有找到左边界，说明元素不存在
        if (first == -1) {
            return new int[]{-1, -1};
        }

        int last = findBound(nums, target, false); // 查找右边界

        return new int[]{first, last};
    }

    /**
     * 修改版的二分查找，用于寻找目标值的左边界或右边界。
     *
     * @param nums      数组
     * @param target    目标值
     * @param isFirst   是否查找第一个出现的位置（左边界）
     * @return 边界的索引，如果不存在则返回 -1
     */
    private static int findBound(int[] nums, int target, boolean isFirst) {
        int left = 0;
        int right = nums.length - 1;
        int ans = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                ans = mid; // 找到了一个目标值
                if (isFirst) {
                    // 如果是找左边界，继续向左搜索
                    right = mid - 1;
                } else {
                    // 如果是找右边界，继续向右搜索
                    left = mid + 1;
                }
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return ans;
    }

    /**
     * 主函数，包含题目给出的测试用例。
     * @param args 命令行参数（未使用）。
     */
    public static void main(String[] args) {
        System.out.println("--- 示例 1 ---");
        int[] nums1 = {5, 7, 7, 8, 8, 10};
        int target1 = 8;
        System.out.println("输入: nums = " + Arrays.toString(nums1) + ", target = " + target1);
        int[] result1 = searchRange(nums1, target1);
        System.out.println("输出: " + Arrays.toString(result1)); // 预期输出: [3, 4]
        System.out.println();

        System.out.println("--- 示例 2 ---");
        int[] nums2 = {5, 7, 7, 8, 8, 10};
        int target2 = 6;
        System.out.println("输入: nums = " + Arrays.toString(nums2) + ", target = " + target2);
        int[] result2 = searchRange(nums2, target2);
        System.out.println("输出: " + Arrays.toString(result2)); // 预期输出: [-1, -1]
        System.out.println();

        System.out.println("--- 示例 3 ---");
        int[] nums3 = {};
        int target3 = 0;
        System.out.println("输入: nums = " + Arrays.toString(nums3) + ", target = " + target3);
        int[] result3 = searchRange(nums3, target3);
        System.out.println("输出: " + Arrays.toString(result3)); // 预期输出: [-1, -1]
        System.out.println();
    }
}
```
