## 题目

**描述：**

给你一个按照非递减顺序排列的整数数组 `nums`，和一个目标值 `target`。请你找出给定目标值在数组中的开始位置和结束位置。

如果数组中不存在目标值 `target`，返回 `[-1, -1]`。

你必须设计并实现时间复杂度为 `O(log n)` 的算法解决此问题。

**示例 1:**

- **输入:** `nums = [5,7,7,8,8,10]`, `target = 8`
- **输出:** `[3,4]`

**示例 2:**

- **输入:** `nums = [5,7,7,8,8,10]`, `target = 6`
- **输出:** `[-1,-1]`

**示例 3:**

- **输入:** `nums = []`, `target = 0`
- **输出:** `[-1,-1]`

**提示:**

- `0 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `nums` 是一个非递减数组
- `-10^9 <= target <= 10^9`

## 题解

### 解法：两次二分查找 (Two-pass Binary Search)

**思路：**

题目要求时间复杂度为 $O(\log n)$，且数组是有序的，必须使用**二分查找**。

常规的二分查找只能找到“一个”目标值，无法确定它是第一个还是最后一个。为了找到范围，我们需要进行两次二分查找：

1.  **查找左边界（First Position）**：

    - 在二分查找遇到 `nums[mid] == target` 时，不立即停止，而是记录当前位置，并尝试向**左**收缩区间（`right = mid - 1`），看左边是否还有 `target`。

2.  **查找右边界（Last Position）**：

    - 在二分查找遇到 `nums[mid] == target` 时，记录当前位置，并尝试向**右**收缩区间（`left = mid + 1`），看右边是否还有 `target`。

我们可以编写一个辅助函数，通过传入一个布尔标志（例如 `isFirst`）来复用二分查找的逻辑。

**复杂度分析：**

- **时间复杂度：** $O(\log n)$，我们需要进行两次二分查找，每次的时间复杂度是 $\log n$，总复杂度仍为 $O(\log n)$。
- **空间复杂度：** $O(1)$，仅使用常数个变量。

**代码片段：**

```java
public int[] searchRange(int[] nums, int target) {
    int leftIdx = binarySearch(nums, target, true);
    int rightIdx = binarySearch(nums, target, false);
    return new int[]{leftIdx, rightIdx};
}

// 辅助函数：findFirst 为 true 找左边界，false 找右边界
private int binarySearch(int[] nums, int target, boolean findFirst) {
    int left = 0;
    int right = nums.length - 1;
    int ans = -1; // 初始化为 -1，表示没找到

    while (left <= right) {
        int mid = left + (right - left) / 2;
        if (nums[mid] < target) {
            left = mid + 1;
        } else if (nums[mid] > target) {
            right = mid - 1;
        } else {
            // nums[mid] == target
            ans = mid; // 记录当前找到的位置
            if (findFirst) {
                right = mid - 1; // 继续向左找
            } else {
                left = mid + 1;  // 继续向右找
            }
        }
    }
    return ans;
}
```

### 完整代码

```java
import java.util.Arrays;

public class Main {

    /**
     * 在排序数组中查找元素的第一个和最后一个位置
     */
    public static int[] searchRange(int[] nums, int target) {
        // 分别寻找左边界和右边界
        int first = findBound(nums, target, true);
        int last = findBound(nums, target, false);

        return new int[]{first, last};
    }

    /**
     * 二分查找核心逻辑
     * @param nums 数组
     * @param target 目标值
     * @param isFirst true 表示查找第一个位置(左边界)，false 表示查找最后一个位置(右边界)
     * @return 找到的索引，如果不存在返回 -1
     */
    private static int findBound(int[] nums, int target, boolean isFirst) {
        int left = 0;
        int right = nums.length - 1;
        int res = -1;

        while (left <= right) {
            int mid = left + (right - left) / 2;

            if (nums[mid] == target) {
                res = mid; // 这是一个潜在的答案，先记录下来
                if (isFirst) {
                    // 如果是要找第一个，说明左边可能还有，缩小右边界
                    right = mid - 1;
                } else {
                    // 如果是要找最后一个，说明右边可能还有，缩小左边界
                    left = mid + 1;
                }
            } else if (nums[mid] < target) {
                left = mid + 1;
            } else {
                right = mid - 1;
            }
        }
        return res;
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        int[] nums1 = {5, 7, 7, 8, 8, 10};
        int target1 = 8;
        System.out.println("示例 1 输入: nums = " + Arrays.toString(nums1) + ", target = " + target1);
        int[] res1 = searchRange(nums1, target1);
        System.out.println("示例 1 输出: " + Arrays.toString(res1));
        // 解释: 8 在索引 3 和 4 出现
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        int[] nums2 = {5, 7, 7, 8, 8, 10};
        int target2 = 6;
        System.out.println("示例 2 输入: nums = " + Arrays.toString(nums2) + ", target = " + target2);
        int[] res2 = searchRange(nums2, target2);
        System.out.println("示例 2 输出: " + Arrays.toString(res2));
        // 解释: 6 不存在
        System.out.println("-----------------------------");

        // --- 示例 3 ---
        int[] nums3 = {};
        int target3 = 0;
        System.out.println("示例 3 输入: nums = " + Arrays.toString(nums3) + ", target = " + target3);
        int[] res3 = searchRange(nums3, target3);
        System.out.println("示例 3 输出: " + Arrays.toString(res3));
        // 解释: 空数组返回 [-1, -1]
        System.out.println("-----------------------------");

        // --- 额外示例 ---
        int[] nums4 = {1};
        int target4 = 1;
        System.out.println("额外示例 输入: nums = " + Arrays.toString(nums4) + ", target = " + target4);
        int[] res4 = searchRange(nums4, target4);
        System.out.println("额外示例 输出: " + Arrays.toString(res4));
        // 解释: 只有一个元素且匹配，返回 [0, 0]
    }
}
```
