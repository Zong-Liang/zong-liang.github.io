## 题目：219. 存在重复元素 II

**描述：**

给你一个整数数组 `nums` 和一个整数 `k` ，判断数组中是否存在两个 **不同** 的索引 `i` 和 `j` ，满足 `nums[i] == nums[j]` 且 `abs(i - j) <= k` 。如果存在，返回 `true` ；否则，返回 `false` 。

**示例 1:**

- **输入:** `nums = [1,2,3,1], k = 3`
- **输出:** `true`

**示例 2:**

- **输入:** `nums = [1,0,1,1], k = 1`
- **输出:** `true`

**示例 3:**

- **输入:** `nums = [1,2,3,1,2,3], k = 2`
- **输出:** `false`

**提示:**

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `0 <= k <= 10^5`

## 题解

### 解法一：哈希表（记录索引）

**思路：**

题目要求找到两个相同的元素，且它们的索引之差不超过 `k`。这是一个典型的利用哈希表（Map）来记录信息的题目。

我们可以遍历数组，使用哈希表记录**每个元素最后一次出现的索引**。

1.  定义一个哈希表 `map`，键（Key）存储数组元素的值，值（Value）存储该元素当前的索引。
2.  遍历数组 `nums`，对于当前元素 `nums[i]`：

    - 检查 `map` 中是否已经存在该元素。
    - 如果存在，说明之前出现过。取出之前存储的索引 `prevIndex`，判断当前索引 `i` 与 `prevIndex` 的差值是否满足 `i - prevIndex <= k`。

      - 如果满足，直接返回 `true`。

    - 无论是否满足，都更新（或添加）该元素在 `map` 中的索引为当前索引 `i`。这样做是为了保证下次比较时，计算的是与当前位置最近的距离（贪心思想）。

3.  如果遍历完数组都没有找到符合条件的索引对，返回 `false`。

**复杂度分析：**

- **时间复杂度：** $O(n)$，其中 $n$ 是数组的长度。我们需要遍历数组一次，哈希表的查找和插入操作平均时间复杂度为 $O(1)$。
- **空间复杂度：** $O(n)$，最坏情况下数组中所有元素都不相同，哈希表需要存储 $n$ 个键值对。

**代码片段：**

```java
public boolean containsNearbyDuplicate(int[] nums, int k) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int num = nums[i];
        // 如果已经包含了该元素，并且索引差值小于等于 k
        if (map.containsKey(num) && i - map.get(num) <= k) {
            return true;
        }
        // 更新当前元素的索引
        map.put(num, i);
    }
    return false;
}
```

### 解法二：滑动窗口（哈希集合）

**思路：**

我们可以维护一个大小不超过 `k` 的滑动窗口。我们只关心**当前元素之前的 `k` 个元素**中是否存在与当前元素相同的值。

1.  使用一个哈希集合 `set` 来存储滑动窗口内的元素。
2.  遍历数组 `nums`，对于索引 `i`：

    - **维护窗口大小**：如果 `i > k`，说明窗口的大小已经超过了 `k`，需要移除窗口最左侧的元素，即移除 `nums[i - k - 1]`。
    - **查重**：检查 `set` 中是否包含当前元素 `nums[i]`。

      - 如果包含，说明在最近的 `k` 个范围内出现了重复元素，返回 `true`。

    - **加入窗口**：将 `nums[i]` 加入 `set`。

3.  遍历结束未发现重复，返回 `false`。

此方法相对于解法一，在 `k` 远小于 `n` 时可以节省空间。

**复杂度分析：**

- **时间复杂度：** $O(n)$，每个元素最多被添加和移除一次。
- **空间复杂度：** $O(k)$，哈希集合中最多存储 $k + 1$ 个元素。

**代码片段：**

```java
public boolean containsNearbyDuplicate(int[] nums, int k) {
    Set<Integer> set = new HashSet<>();
    for (int i = 0; i < nums.length; i++) {
        // 如果窗口大小超过了 k，移除最早进入的元素
        if (i > k) {
            set.remove(nums[i - k - 1]);
        }
        // 尝试添加当前元素，如果 set 中已存在，add 方法返回 false
        // 说明在 k 距离内找到了重复元素
        if (!set.add(nums[i])) {
            return true;
        }
    }
    return false;
}
```

### 完整代码

这里采用 **解法一（哈希表）** 作为主要实现，因其逻辑最为直观且适用性广。

```java
import java.util.Arrays;
import java.util.HashMap;
import java.util.Map;

public class Main {

    // 核心算法实现：使用哈希表
    public static boolean containsNearbyDuplicate(int[] nums, int k) {
        // Key: 数组元素值, Value: 对应的索引
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            int num = nums[i];
            // 检查是否存在重复元素，并判断距离
            if (map.containsKey(num)) {
                int prevIndex = map.get(num);
                if (i - prevIndex <= k) {
                    return true;
                }
            }
            // 无论是否找到，都更新当前元素的索引为最新位置
            // 这样能保证下一次比较时，距离是最小的
            map.put(num, i);
        }

        return false;
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        int[] nums1 = {1, 2, 3, 1};
        int k1 = 3;
        System.out.println("示例 1 输入: nums = " + Arrays.toString(nums1) + ", k = " + k1);
        System.out.println("示例 1 输出: " + containsNearbyDuplicate(nums1, k1));
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        int[] nums2 = {1, 0, 1, 1};
        int k2 = 1;
        System.out.println("示例 2 输入: nums = " + Arrays.toString(nums2) + ", k = " + k2);
        System.out.println("示例 2 输出: " + containsNearbyDuplicate(nums2, k2));
        System.out.println("-----------------------------");

        // --- 示例 3 ---
        int[] nums3 = {1, 2, 3, 1, 2, 3};
        int k3 = 2;
        System.out.println("示例 3 输入: nums = " + Arrays.toString(nums3) + ", k = " + k3);
        System.out.println("示例 3 输出: " + containsNearbyDuplicate(nums3, k3));
    }
}
```
