## 题目：219. 存在重复元素 II

**描述**

给你一个整数数组 `nums` 和一个整数 `k` ，判断数组中是否存在两个 **不同** 的索引 `i` 和 `j` ，满足 `nums[i] == nums[j]` 且 `abs(i - j) <= k` 。如果存在，返回 `true` ；否则，返回 `false` 。

**示例 1:**

- **输入:** `nums = [1,2,3,1], k = 3`
- **输出:** `true`

**示例 2:**

- **输入:** `nums = [1,0,1,1], k = 1`
- **输出:** `true`

**示例 3:**

- **输入:** `nums = [1,2,3,1,2,3], k = 2`
- **输出:** `false`

**提示:**

- `1 <= nums.length <= 10^5`
- `-10^9 <= nums[i] <= 10^9`
- `0 <= k <= 10^5`

## Java 代码

```java
import java.util.HashMap;
import java.util.Map;

public class Main {

    /**
     * 判断数组中是否存在满足条件的重复元素。
     * 条件为：nums[i] == nums[j] 且 abs(i - j) <= k
     *
     * @param nums 整数数组
     * @param k    整数，表示索引差的绝对值的最大值
     * @return 如果存在则返回 true，否则返回 false
     */
    public static boolean containsNearbyDuplicate(int[] nums, int k) {
        // 使用 HashMap 存储数字及其最近出现的索引
        // Key: 数组中的数字, Value: 该数字的最新索引
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            int currentNum = nums[i];

            // 检查 Map 中是否已存在当前数字
            if (map.containsKey(currentNum)) {
                // 如果存在，获取上一次出现的索引
                int prevIndex = map.get(currentNum);

                // 计算当前索引与上一次索引的差值
                // 如果差值小于等于 k，则找到了满足条件的两个数
                if (i - prevIndex <= k) {
                    return true;
                }
            }

            // 无论是否找到，都更新（或插入）当前数字的最新索引
            map.put(currentNum, i);
        }

        // 遍历完整个数组都没有找到，则返回 false
        return false;
    }

    public static void main(String[] args) {
        // 示例 1
        int[] nums1 = {1, 2, 3, 1};
        int k1 = 3;
        System.out.println("输入: nums = [1,2,3,1], k = 3");
        System.out.println("输出: " + containsNearbyDuplicate(nums1, k1)); // 预期: true
        System.out.println("---");

        // 示例 2
        int[] nums2 = {1, 0, 1, 1};
        int k2 = 1;
        System.out.println("输入: nums = [1,0,1,1], k = 1");
        System.out.println("输出: " + containsNearbyDuplicate(nums2, k2)); // 预期: true
        System.out.println("---");

        // 示例 3
        int[] nums3 = {1, 2, 3, 1, 2, 3};
        int k3 = 2;
        System.out.println("输入: nums = [1,2,3,1,2,3], k = 2");
        System.out.println("输出: " + containsNearbyDuplicate(nums3, k3)); // 预期: false
        System.out.println("---");
    }
}
```
