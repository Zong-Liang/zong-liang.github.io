## 题目：383. 赎金信

**描述**

给你两个字符串：`ransomNote` 和 `magazine` ，判断 `ransomNote` 能不能由 `magazine` 里面的字符构成。

如果可以，返回 `true` ；否则返回 `false` 。

`magazine` 中的每个字符只能在 `ransomNote` 中使用一次。

**示例 1:**

- **输入:** `ransomNote = "a"`, `magazine = "b"`
- **输出:** `false`

**示例 2:**

- **输入:** `ransomNote = "aa"`, `magazine = "ab"`
- **输出:** `false`

**示例 3:**

- **输入:** `ransomNote = "aa"`, `magazine = "aab"`
- **输出:** `true`

**提示:**

- `1 <= ransomNote.length, magazine.length <= 10^5`
- `ransomNote` 和 `magazine` 由小写英文字母组成

## Java 代码

```java
public class Main {

    /**
     * 判断 ransomNote 能否由 magazine 里面的字符构成。
     *
     * @param ransomNote 赎金信字符串
     * @param magazine   杂志字符串
     * @return 如果可以构成，返回 true；否则返回 false。
     */
    public static boolean canConstruct(String ransomNote, String magazine) {
        // 优化：如果 ransomNote 比 magazine 长，则不可能构成
        if (ransomNote.length() > magazine.length()) {
            return false;
        }

        // 1. 创建一个数组来统计 magazine 中每个字符的出现次数
        // 因为题目说明只有小写英文字母，所以数组大小为 26 即可
        int[] charCounts = new int[26];
        for (char c : magazine.toCharArray()) {
            charCounts[c - 'a']++;
        }

        // 2. 遍历 ransomNote，消耗 magazine 中的字符
        for (char c : ransomNote.toCharArray()) {
            // 将对应字符的计数减 1
            charCounts[c - 'a']--;

            // 如果某个字符的计数变为负数，说明 magazine 中的字符不够用
            if (charCounts[c - 'a'] < 0) {
                return false;
            }
        }

        // 3. 如果成功遍历完 ransomNote，说明 magazine 中的字符足够
        return true;
    }

    public static void main(String[] args) {
        // 示例 1
        String ransomNote1 = "a";
        String magazine1 = "b";
        System.out.println("输入: ransomNote = \"" + ransomNote1 + "\", magazine = \"" + magazine1 + "\"");
        System.out.println("输出: " + canConstruct(ransomNote1, magazine1)); // 预期: false
        System.out.println("---");

        // 示例 2
        String ransomNote2 = "aa";
        String magazine2 = "ab";
        System.out.println("输入: ransomNote = \"" + ransomNote2 + "\", magazine = \"" + magazine2 + "\"");
        System.out.println("输出: " + canConstruct(ransomNote2, magazine2)); // 预期: false
        System.out.println("---");

        // 示例 3
        String ransomNote3 = "aa";
        String magazine3 = "aab";
        System.out.println("输入: ransomNote = \"" + ransomNote3 + "\", magazine = \"" + magazine3 + "\"");
        System.out.println("输出: " + canConstruct(ransomNote3, magazine3)); // 预期: true
        System.out.println("---");
    }
}
```
