## 题目

**描述：**

给定一个整数数组 `nums` 和一个整数目标值 `target`，请你在该数组中找出 **和为目标值 `target`** 的那 **两个** 整数，并返回它们的数组下标。

你可以假设每种输入只会对应一个答案。但是，数组中同一个元素在答案里不能重复出现。

你可以按任意顺序返回答案。

**示例 1:**

- **输入:** `nums = [2,7,11,15]`, `target = 9`
- **输出:** `[0,1]`
- **解释:** 因为 `nums[0] + nums[1] == 9` ，返回 `[0, 1]` 。

**示例 2:**

- **输入:** `nums = [3,2,4]`, `target = 6`
- **输出:** `[1,2]`

**示例 3:**

- **输入:** `nums = [3,3]`, `target = 6`
- **输出:** `[0,1]`

**提示:**

- `2 <= nums.length <= 10^4`
- `-10^9 <= nums[i] <= 10^9`
- `-10^9 <= target <= 10^9`
- **只会存在一个有效答案**

**进阶：** 你可以想出一个时间复杂度小于 `O(n^2)` 的算法吗？

## 题解

### 标准解法：暴力枚举法

**思路：**

最容易想到的方法是遍历数组中的每一个数 `x`，然后寻找数组中是否存在另一个数 `y`，使得 `x + y = target`。

为了避免重复和自身相加，我们在寻找 `y` 时，只需要在 `x` 后面的元素中进行搜索。

1.  第一层循环遍历下标 `i` 从 `0` 到 `n-1`。
2.  第二层循环遍历下标 `j` 从 `i+1` 到 `n-1`。
3.  判断 `nums[i] + nums[j]` 是否等于 `target`。如果是，则返回 `[i, j]`。

**复杂度分析：**

- **时间复杂度：** $O(n^2)$，其中 $n$ 是数组的长度。最坏情况下，数组中任意两个数都要被匹配一次。
- **空间复杂度：** $O(1)$，只需要常数级别的额外空间。

**代码片段：**

```java
public int[] twoSum(int[] nums, int target) {
    for (int i = 0; i < nums.length; i++) {
        // j 从 i + 1 开始，避免重复和自身相加
        for (int j = i + 1; j < nums.length; j++) {
            if (nums[i] + nums[j] == target) {
                return new int[]{i, j};
            }
        }
    }
    return new int[0]; // 理论上不会执行到这里，因为题目保证有解
}
```

### 进阶解法：哈希表法（空间换时间）

**思路：**

为了将时间复杂度从 $O(n^2)$ 降低到 $O(n)$，我们需要一种更快速的方法来检查数组中是否存在目标元素。哈希表（HashMap）是保持数组中的每个元素与其索引相互对应的最好方法。

我们可以通过一次遍历完成：

1.  创建一个哈希表，用于存储 `数值 -> 下标` 的映射。
2.  遍历数组，对于每一个元素 `nums[i]`：

    - 计算它需要的补数：`complement = target - nums[i]`。
    - 查询哈希表中是否存在这个 `complement`。
    - **如果存在**：说明之前已经遍历过了这个补数，直接返回补数的下标和当前下标 `i`。
    - **如果不存在**：将当前元素 `nums[i]` 和它的下标 `i` 存入哈希表，供后面的元素查询。

**复杂度分析：**

- **时间复杂度：** $O(n)$，我们只遍历了包含 $n$ 个元素的列表一次。在哈希表中进行添加和查找操作的平均时间复杂度为 $O(1)$。
- **空间复杂度：** $O(n)$，主要取决于哈希表中存储的元素数量，最坏情况下需要存储 $n$ 个元素。

**代码片段：**

```java
public int[] twoSum(int[] nums, int target) {
    Map<Integer, Integer> map = new HashMap<>();
    for (int i = 0; i < nums.length; i++) {
        int complement = target - nums[i];
        // 检查 map 中是否已经包含我们要找的那个补数
        if (map.containsKey(complement)) {
            // 如果有，直接返回那个数的下标和当前的下标
            return new int[] { map.get(complement), i };
        }
        // 如果没有，把当前的数和下标存进去
        map.put(nums[i], i);
    }
    return new int[0];
}
```

### 完整代码

```java
import java.util.Arrays;
import java.util.HashMap;
import java.util.Map;

public class Main {

    // 核心算法实现（进阶解法：哈希表法）
    public static int[] twoSum(int[] nums, int target) {
        // Key: 数值, Value: 下标
        Map<Integer, Integer> map = new HashMap<>();

        for (int i = 0; i < nums.length; i++) {
            int complement = target - nums[i];

            // 查找哈希表中是否存在补数
            if (map.containsKey(complement)) {
                return new int[] { map.get(complement), i };
            }

            // 将当前数存入哈希表
            map.put(nums[i], i);
        }
        // 题目保证有解，所以这里只需返回空数组或抛异常
        return new int[0];
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        int[] nums1 = new int[]{2, 7, 11, 15};
        int target1 = 9;

        System.out.println("示例 1 输入: nums = " + Arrays.toString(nums1) + ", target = " + target1);
        int[] res1 = twoSum(nums1, target1);
        System.out.println("示例 1 输出: " + Arrays.toString(res1));
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        int[] nums2 = new int[]{3, 2, 4};
        int target2 = 6;

        System.out.println("示例 2 输入: nums = " + Arrays.toString(nums2) + ", target = " + target2);
        int[] res2 = twoSum(nums2, target2);
        System.out.println("示例 2 输出: " + Arrays.toString(res2));
        System.out.println("-----------------------------");

        // --- 示例 3 ---
        int[] nums3 = new int[]{3, 3};
        int target3 = 6;

        System.out.println("示例 3 输入: nums = " + Arrays.toString(nums3) + ", target = " + target3);
        int[] res3 = twoSum(nums3, target3);
        System.out.println("示例 3 输出: " + Arrays.toString(res3));
    }
}
```
