## 题目：49. 字母异位词分组

**描述：**

给你一个字符串数组，请你将 **字母异位词** 组合在一起。可以按任意顺序返回结果列表。

**字母异位词** 是由相同字母按不同顺序排列形成的字符串。

**示例 1:**

- **输入:** `strs = ["eat","tea","tan","ate","nat","bat"]`
- **输出:** `[["bat"],["nat","tan"],["ate","eat","tea"]]`

**示例 2:**

- **输入:** `strs = [""]`
- **输出:** `[[""]]`

**示例 3:**

- **输入:** `strs = ["a"]`
- **输出:** `[["a"]]`

**提示:**

- `1 <= strs.length <= 10^4`
- `0 <= strs[i].length <= 100`
- `strs[i]` 仅包含小写字母

## 题解

### 解法：哈希表 + 排序

**思路：**

两个字符串互为**字母异位词**，意味着它们包含的字符种类和数量完全相同。

因此，如果我们对这两个字符串中的字符进行**排序**，它们得到的字符串一定是相同的。

利用这个性质，我们可以使用一个哈希表（`Map`）来进行分组：

1.  **键（Key）**：将当前字符串转换为字符数组，进行排序，然后转回字符串。这个排序后的字符串就是这一组异位词的唯一标识。
2.  **值（Value）**：一个列表，存储所有排序后等于 Key 的原始字符串。

**算法流程：**

1.  初始化一个 `Map<String, List<String>>`。
2.  遍历输入数组 `strs` 中的每一个字符串 `s`。
3.  将 `s` 转换为字符数组并排序，得到 `key`。
4.  将 `s` 添加到 `map` 中对应的 `key` 的列表中。
5.  最后，将 `map` 中所有的值（Values）作为结果返回。

**复杂度分析：**

- **时间复杂度：** $O(N \cdot K \log K)$，其中 $N$ 是字符串数组的长度，$K$ 是字符串的最大长度。我们需要遍历 $N$ 个字符串，对每个字符串排序需要 $O(K \log K)$ 的时间。
- **空间复杂度：** $O(N \cdot K)$，需要用哈希表存储全部字符串。

**代码片段：**

```java
public List<List<String>> groupAnagrams(String[] strs) {
    if (strs == null || strs.length == 0) {
        return new ArrayList<>();
    }

    // Key: 排序后的字符串, Value: 原始字符串列表
    Map<String, List<String>> map = new HashMap<>();

    for (String s : strs) {
        // 1. 将字符串转换为字符数组并排序
        char[] ca = s.toCharArray();
        Arrays.sort(ca);
        // 2. 将排序后的字符数组转回字符串作为 Key
        String key = new String(ca);

        // 3. 放入 Map 中
        if (!map.containsKey(key)) {
            map.put(key, new ArrayList<>());
        }
        map.get(key).add(s);
    }

    // 返回 Map 中所有的 Values
    return new ArrayList<>(map.values());
}
```

### 完整代码

```java
import java.util.*;

public class Main {

    // 核心算法实现：哈希表 + 排序
    public static List<List<String>> groupAnagrams(String[] strs) {
        if (strs == null || strs.length == 0) {
            return new ArrayList<>();
        }

        Map<String, List<String>> map = new HashMap<>();

        for (String s : strs) {
            // 将字符串转为字符数组
            char[] ca = s.toCharArray();
            // 对字符数组进行排序，"eat" -> "aet", "tea" -> "aet"
            Arrays.sort(ca);
            // 将排序后的数组转回字符串，作为哈希表的 Key
            String key = new String(ca);

            // 如果 Key 不存在，初始化一个新的列表
            if (!map.containsKey(key)) {
                map.put(key, new ArrayList<>());
            }
            // 将原始字符串加入对应的列表
            map.get(key).add(s);
        }

        // map.values() 返回的是 Collection，需要转换为 List
        return new ArrayList<>(map.values());
    }

    public static void main(String[] args) {
        // --- 示例 1 ---
        String[] strs1 = {"eat", "tea", "tan", "ate", "nat", "bat"};

        System.out.println("示例 1 输入: " + Arrays.toString(strs1));

        List<List<String>> result1 = groupAnagrams(strs1);

        System.out.println("示例 1 输出: " + result1);
        // 解释: 分组结果类似于 [[bat], [nat, tan], [ate, eat, tea]]
        System.out.println("-----------------------------");

        // --- 示例 2 ---
        String[] strs2 = {""};

        System.out.println("示例 2 输入: " + Arrays.toString(strs2));

        List<List<String>> result2 = groupAnagrams(strs2);

        System.out.println("示例 2 输出: " + result2);
        System.out.println("-----------------------------");

        // --- 示例 3 ---
        String[] strs3 = {"a"};

        System.out.println("示例 3 输入: " + Arrays.toString(strs3));

        List<List<String>> result3 = groupAnagrams(strs3);

        System.out.println("示例 3 输出: " + result3);
    }
}
```
