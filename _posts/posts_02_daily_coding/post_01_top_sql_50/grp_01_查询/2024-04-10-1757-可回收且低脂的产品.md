## 题目：1757. 可回收且低脂的产品

**表:** `Products`

| Column Name | Type            |
| ----------- | --------------- |
| product_id  | int             |
| low_fats    | enum ('Y', 'N') |
| recyclable  | enum ('Y', 'N') |

- `product_id` 是该表的主键（具有唯一值的列）。
- `low_fats` 是枚举类型，取值为以下两种 ('Y', 'N')，其中 'Y' 表示该产品是低脂产品，'N' 表示不是低脂产品。
- `recyclable` 是枚举类型，取值为以下两种 ('Y', 'N')，其中 'Y' 表示该产品可回收，而 'N' 表示不可回收。

**问题描述:**

编写解决方案找出既是低脂又是可回收的产品编号。

返回结果 **无顺序要求**。

**示例 1:**

**输入:**
`Products` 表:

| product_id | low_fats | recyclable |
| ---------- | -------- | ---------- |
| 0          | Y        | N          |
| 1          | Y        | Y          |
| 2          | N        | Y          |
| 3          | Y        | Y          |
| 4          | N        | N          |

**输出:**

| product_id |
| ---------- |
| 1          |
| 3          |

**解释:**
只有产品 id 为 1 和 3 的产品, 既是低脂又是可回收的产品。

## 正确答案 (MySQL)

你可以使用 `WHERE` 子句和 `AND` 操作符来同时筛选满足两个条件的产品。

```sql
SELECT product_id
FROM Products
WHERE low_fats = 'Y'
  AND recyclable = 'Y';
```

**查询逻辑分解:**

1. **`SELECT product_id`**:

   - 我们的目标是找出符合条件的产品 ID。

2. **`FROM Products`**:

   - 指定从 `Products` 表中查询数据。

3. **`WHERE low_fats = 'Y' AND recyclable = 'Y'`**:
   - 这是筛选条件的核心。`WHERE` 子句用于过滤行。
   - `low_fats = 'Y'`: 筛选出 `low_fats` 值为 'Y' 的产品。
   - `recyclable = 'Y'`: 筛选出 `recyclable` 值为 'Y' 的产品。
   - `AND`: `AND` 操作符确保只有 **同时** 满足上述两个条件的行才会被选中。
